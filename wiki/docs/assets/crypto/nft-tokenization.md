# NFT & 실물자산 토큰화

NFT(Non-Fungible Token, 대체불가토큰)는 블록체인 위에서 고유한 디지털 자산의 소유권을 증명하는 기술입니다. 디지털 아트에서 시작된 NFT는 게임, 음악, 부동산, 채권까지 그 응용 범위를 넓히고 있습니다. 특히 실물자산 토큰화(Real World Asset Tokenization, RWA)는 전통 금융과 블록체인을 연결하는 핵심 기술로, BlackRock, JPMorgan 같은 대형 기관들이 적극적으로 참여하고 있습니다.

---

## NFT 기술 표준

### ERC-721

이더리움에서 NFT의 기본 표준입니다. 각 토큰이 고유한 tokenId를 가지며, 하나의 컨트랙트에서 다양한 고유 자산을 표현할 수 있습니다.

**핵심 기능**:
- `ownerOf(tokenId)` — 특정 NFT의 소유자 조회
- `transferFrom(from, to, tokenId)` — NFT 전송
- `approve(to, tokenId)` — 특정 NFT에 대한 전송 권한 부여
- `tokenURI(tokenId)` — NFT의 메타데이터 URI 반환

**메타데이터 구조** (JSON):
```json
{
  "name": "작품명",
  "description": "작품 설명",
  "image": "ipfs://...",
  "attributes": [
    {"trait_type": "희귀도", "value": "레전더리"},
    {"trait_type": "색상", "value": "금색"}
  ]
}
```

### ERC-1155

하나의 컨트랙트에서 대체가능(Fungible)과 대체불가(Non-Fungible) 토큰을 모두 관리할 수 있는 멀티토큰 표준입니다.

**ERC-721 대비 장점**:
- 배치 전송 지원 — 여러 토큰을 한 번의 트랜잭션으로 전송 (가스비 절약)
- 대체가능/불가능 토큰 혼합 — 게임 아이템(고유 무기 + 소모품)에 적합
- 효율적인 스토리지 — 비슷한 토큰을 묶어 관리
- 세미 펀저블(Semi-Fungible) — 동일 종류의 NFT를 수량으로 표현 가능

### 기타 표준

| 표준 | 체인 | 용도 |
|-----|------|------|
| ERC-6551 | Ethereum | 토큰 바운드 계정 (NFT가 자체 지갑을 가짐) |
| ERC-4907 | Ethereum | 임대 가능한 NFT (소유권과 사용권 분리) |
| SPL Token | Solana | 솔라나의 NFT 표준 (cNFT: 압축 NFT) |
| FA (Fungible Asset) | Aptos/Sui | Move 기반 NFT 표준 |

## NFT 마켓플레이스 생태계

### 주요 마켓플레이스 비교

| 플랫폼 | 체인 | 수수료 | 특징 |
|-------|------|-------|------|
| OpenSea | Ethereum, Polygon, Base 등 | 2.5% | 최대 종합 마켓플레이스 |
| Blur | Ethereum | 0% (마켓플레이스) | 프로 트레이더 특화, 입찰 중심 |
| Magic Eden | Solana, Bitcoin, Ethereum | 2% | 멀티체인, Ordinals 지원 |
| X2Y2 | Ethereum | 0.5% | 로열티 선택적 |
| Tensor | Solana | 1.5% | 솔라나 트레이딩 특화 |

### NFT 로열티 논쟁

NFT 창작자 로열티(Creator Royalty)는 2차 판매 시 원작자에게 지급되는 수수료(보통 2.5~10%)입니다. 그러나 2023년 이후 로열티 우회가 일반화되면서 논쟁이 심화되었습니다.

**로열티 강제 메커니즘**:
- **오퍼레이터 필터 레지스트리** — OpenSea가 도입, 로열티를 우회하는 마켓플레이스 차단
- **온체인 로열티** — 스마트 컨트랙트 수준에서 전송 시 자동 로열티 징수
- **토큰 바운드 로열티** — ERC-2981 표준으로 로열티 정보를 온체인에 기록

:::caution NFT 로열티의 현실
현재 대부분의 마켓플레이스에서 로열티는 선택적이거나 우회 가능합니다. NFT 프로젝트의 수익 모델을 평가할 때 로열티 수입에 과도하게 의존하는 프로젝트는 리스크가 높다는 점을 인식하세요. 지속 가능한 NFT 프로젝트는 로열티 외의 다양한 수익원을 확보해야 합니다.
:::

## NFT 밸류에이션

### 가치 평가 접근법

NFT의 가치 평가는 전통 자산과 다른 독특한 도전을 제시합니다.

**비교 가치 접근법 (Comparable Sales)**:
- 동일 컬렉션 내 유사 특성 NFT의 최근 거래 가격 비교
- 희귀도 점수(Rarity Score)에 기반한 가격 프리미엄 분석
- 바닥가(Floor Price) 대비 프리미엄 산정

**수익 접근법**:
- 로열티 수익 기반 DCF (창작자 관점)
- IP 라이선싱 수익 (BAYC → ApeCoin, 브랜드 확장)
- 유틸리티 가치 (멤버십, 접근 권한)

**희귀도 기반 접근법**:
```
예상 가격 = 바닥가 × 희귀도 배수

희귀도 배수는 다음 요소로 결정:
- 특성(Trait)별 발생 빈도
- 특성 조합의 희귀성
- 시각적 매력도 (주관적)
- 역사적 의미 (초기 민팅 번호 등)
```

## 실물자산 토큰화 (RWA Tokenization)

### 개념과 시장 규모

실물자산 토큰화란 부동산, 채권, 주식, 미술품 등 실물 자산의 소유권이나 수익권을 블록체인 토큰으로 표현하는 것입니다. Boston Consulting Group은 2030년까지 토큰화 자산 시장 규모가 **16조 달러**에 달할 것으로 예측합니다.

**토큰화의 장점**:
1. **분할 소유** — 고가 자산을 소액 단위로 분할하여 접근성 확대
2. **유동성 향상** — 24/7 거래 가능, P2P 전송
3. **투명성** — 소유권과 거래 이력이 블록체인에 기록
4. **효율성** — 중개기관 축소, 정산 시간 단축
5. **프로그래밍 가능성** — 자동 배당, 규정 준수 로직 등을 코드로 구현

### 주요 RWA 카테고리

**국채/채권 토큰화**:

| 프로젝트 | 발행자 | 기초자산 | TVL |
|---------|-------|---------|-----|
| BUIDL | BlackRock | 미국 단기 국채 | $500M+ |
| USDY | Ondo Finance | 미국 국채 + 은행예금 | $300M+ |
| sDAI | MakerDAO/Sky | 미국 국채 | $1B+ |
| OUSG | Ondo Finance | 미국 단기 국채 | $200M+ |

**부동산 토큰화**:
- RealT — 미국 임대 부동산을 토큰화, 일일 임대 수익 분배
- Centrifuge — 부동산 대출 등 실물자산 담보 대출 토큰화
- Tangible — 부동산, 금, 와인 등 다양한 실물자산 토큰화

**미술품/수집품 토큰화**:
- Masterworks — 고가 미술품 분할 소유
- Courtyard — 트레이딩 카드, 수집품 토큰화
- 4K — 한국 미술품 분할 소유 플랫폼

:::tip RWA 토큰화의 핵심 과제
RWA 토큰화의 가장 큰 과제는 **온체인과 오프체인의 연결**입니다. 블록체인 토큰의 소유권이 실제 법적 소유권과 일치하도록 보장하는 법적 구조(SPV, 신탁 등)가 필수적입니다. 또한 오라클을 통한 자산 가치 업데이트, 감사, 보험 등의 인프라가 갖춰져야 합니다.
:::

## STO (Security Token Offering)

### 개념

STO는 증권형 토큰을 발행하여 자금을 조달하는 방식입니다. ICO와 달리 증권법 규제를 준수하며, 토큰이 주식, 채권, 수익권 등의 증권적 성격을 가집니다.

**STO vs ICO vs IPO 비교**:

| 항목 | STO | ICO | IPO |
|-----|-----|-----|-----|
| 규제 | 증권법 적용 | 비규제/불명확 | 증권법 적용 |
| 투자자 보호 | 높음 | 낮음 | 높음 |
| 기초자산 | 실물자산/수익권 | 유틸리티 토큰 | 회사 지분 |
| 진입 장벽 | 중간 | 낮음 | 높음 |
| 유동성 | 중간 (ATS/DEX) | 높음 (DEX/CEX) | 높음 (증권거래소) |
| 비용 | 중간 | 낮음 | 높음 |

### 한국의 STO 규제

한국은 2023년부터 STO(토큰 증권) 제도화를 추진하고 있습니다.

**주요 내용**:
- **토큰 증권의 정의** — 분산원장 기술을 활용하여 발행된 증권
- **발행 요건** — 증권신고서 제출, 금융위원회 심사
- **유통 플랫폼** — 한국거래소(KRX) 디지털자산 거래 플랫폼, 장외거래(ATS)
- **투자자 보호** — 일반 증권과 동일한 수준의 투자자 보호 적용
- **허용 자산** — 부동산, 미술품, 음악 저작권, 매출채권 등

**국내 STO 추진 사례**:

| 주체 | 유형 | 상태 |
|-----|------|------|
| 카카오페이증권 | 부동산 토큰증권 | 시범 운영 |
| 신한투자증권 | 미술품 조각투자 | 운영 중 |
| KB증권 | 부동산 수익증권 | 기획 중 |
| 두나무(업비트) | 디지털자산 증권 인프라 | 개발 중 |

:::caution 한국 STO의 현재 상황
한국의 STO 제도는 아직 초기 단계로, 구체적인 시행령과 감독 규정이 계속 발전하고 있습니다. 투자 전 최신 규제 동향을 확인하고, 허가받은 플랫폼을 통해서만 거래하세요. 해외 STO 플랫폼 참여 시 한국 증권법과의 충돌 가능성도 검토해야 합니다.
:::

## 분할 소유 (Fractional Ownership)

### 법적 구조

분할 소유를 실현하기 위한 주요 법적 구조입니다.

**SPV (Special Purpose Vehicle)**:
- 특수목적법인이 실물자산을 보유
- SPV의 지분을 토큰화하여 투자자에게 판매
- 자산과 발행사의 법적 분리 (파산 격리)

**신탁 구조**:
- 신탁회사가 자산을 보관하고 수익을 배분
- 수익증권을 토큰으로 발행
- 한국에서 부동산 토큰화에 주로 사용

**조합 구조**:
- 투자자들이 조합을 구성하여 공동 투자
- 조합 지분을 토큰으로 표현
- 소규모 프로젝트에 적합

### 기술적 구현

**ERC-20 기반 분할화**:
- 고가 NFT(ERC-721)를 금고(Vault) 컨트랙트에 잠금
- 금고가 ERC-20 토큰을 발행하여 소유권 분할
- Fractional.art(현 Tessera)가 대표적 사례

**분할 소유의 과제**:
1. **거버넌스** — 다수 소유자 간 의사결정 (매각, 보험, 관리 등)
2. **유동성** — 분할 토큰의 거래 시장이 충분히 활성화되어야 함
3. **가격 발견** — 분할된 자산의 적정 가격 산정 어려움
4. **규제** — 대부분의 국가에서 증권으로 분류될 가능성
5. **재구성** — 분할된 소유권을 다시 합치는 메커니즘 필요

## NFT 세금 (한국)

### 현행 과세 체계

한국에서 NFT에 대한 명확한 과세 기준은 아직 완전히 확립되지 않았으나, 다음과 같은 방향으로 해석됩니다.

**NFT 거래 수익**:
- 가상자산으로 분류되는 NFT의 양도차익 — 기타소득 또는 양도소득으로 과세 논의 중
- 수집품적 성격의 NFT — 기타소득으로 과세될 가능성
- 유틸리티 NFT — 재화/서비스 거래로 부가가치세 적용 가능성

**NFT 창작자 수익**:
- 1차 판매 수익 — 사업소득 또는 기타소득
- 로열티 수익 — 사업소득 (계속적, 반복적인 경우)

**RWA 토큰 수익**:
- 배당/이자 수익 — 금융소득 (이자소득 or 배당소득)
- 매매차익 — 자산 유형에 따라 다름 (증권형이면 양도소득)

:::tip NFT 세금 기록 관리
NFT 거래가 활발한 투자자는 반드시 모든 거래 기록(민팅, 구매, 판매, 스왑)을 보관하세요. 거래 시점의 가격을 원화 기준으로 기록하고, 가스비 등 부대 비용도 함께 기록하면 향후 과세 시 유리합니다. Koinly, CoinTracker 같은 세금 추적 도구의 활용을 권장합니다.
:::

## NFT & RWA 투자 체크리스트

### NFT 투자 시 확인 사항

1. **팀/아티스트** — 실명, 이력, 커뮤니티 참여도
2. **컨트랙트 검증** — Etherscan에서 소스코드 공개 여부, 소유권 포기(renounce) 여부
3. **메타데이터 저장** — IPFS/Arweave(분산) vs 중앙 서버 (서버 다운 시 이미지 소실)
4. **유동성** — 바닥가, 일일 거래량, 리스팅 수
5. **유틸리티** — 보유 혜택의 실질적 가치
6. **커뮤니티** — Discord/Twitter 활성도, 홀더 구조

### RWA 토큰 투자 시 확인 사항

1. **법적 구조** — SPV/신탁의 설계, 투자자 권리 보장
2. **기초자산 검증** — 실물자산의 존재 및 가치 평가
3. **감사/검증** — 독립 감사 보고서, 자산 평가 보고서
4. **규제 준수** — 발행 허가, 투자자 자격 요건
5. **수익 분배** — 배당 빈도, 세금 처리, 수수료 구조
