# 채권 가치평가 개요

## 채권의 기본 개념

### 채권이란?

**채권 (Bond)** 은 발행자가 투자자에게 원금과 이자를 지급하기로 약속한 채무증서입니다.

```
발행자 (차입자)          투자자 (대출자)
      │                      │
      │◄──── 투자금 ─────────│
      │                      │
      │──── 이자 (쿠폰) ────►│
      │                      │
      │──── 원금 상환 ──────►│
      └──────────────────────┘
           만기 시
```

### 채권의 구성 요소

| 요소 | 설명 | 예시 |
|------|------|------|
| 액면가 (Face Value) | 만기 시 상환 금액 | 10,000원 |
| 쿠폰 금리 (Coupon Rate) | 연간 이자율 | 3% |
| 쿠폰 (Coupon) | 연간 이자 금액 | 300원 |
| 만기 (Maturity) | 원금 상환 시점 | 5년 |
| 현재가 (Price) | 시장 거래 가격 | 9,800원 |

### 채권 종류

```
발행 주체별:
├── 국채 (Government Bond)
├── 지방채 (Municipal Bond)
├── 특수채 (Agency Bond)
├── 회사채 (Corporate Bond)
└── 금융채 (Financial Bond)

이자 지급별:
├── 이표채 (Coupon Bond)
├── 할인채 (Discount Bond)
└── 복리채 (Compound Bond)

담보 여부:
├── 담보부채권 (Secured)
└── 무담보채권 (Unsecured/Debenture)
```

## 채권 가격 결정

### 현재가치 계산

```
채권가격 = Σ(쿠폰의 현재가치) + 액면가의 현재가치

        C         C              C + F
P = ─────── + ─────── + ... + ─────────
    (1+r)¹   (1+r)²          (1+r)ⁿ

C = 쿠폰 (이자)
F = 액면가
r = 할인율 (시장금리)
n = 만기
```

**예시: 5년 만기, 액면가 10,000원, 쿠폰 3%, 시장금리 4%**

```
P = 300/(1.04)¹ + 300/(1.04)² + 300/(1.04)³
  + 300/(1.04)⁴ + 10,300/(1.04)⁵

P = 288.46 + 277.37 + 266.70 + 256.44 + 8,465.47
P = 9,554.44원
```

### 금리와 채권 가격의 관계

```
금리 ↑ → 채권 가격 ↓
금리 ↓ → 채권 가격 ↑

      가격
        │
        │╲
        │ ╲
        │  ╲
        │   ╲
        │    ╲
        └──────────── 금리

역의 관계 (Inverse Relationship)
```

## 수익률 지표

### 1. 경상수익률 (Current Yield)

```
경상수익률 = 연간 쿠폰 / 현재 가격

예: 쿠폰 300원, 가격 9,500원
경상수익률 = 300 / 9,500 = 3.16%

특징: 자본손익 미반영
```

### 2. 만기수익률 (YTM: Yield to Maturity)

```
채권을 만기까지 보유 시 연환산 수익률

        C         C              C + F
P = ─────── + ─────── + ... + ─────────
    (1+YTM)¹  (1+YTM)²        (1+YTM)ⁿ

YTM을 푸는 것은 IRR 계산과 동일
```

**YTM 해석:**
```
YTM > 쿠폰금리 → 할인채 (Discount)
YTM = 쿠폰금리 → 액면채 (Par)
YTM < 쿠폰금리 → 할증채 (Premium)
```

### 3. 콜수익률 (YTC: Yield to Call)

```
콜옵션 부여 채권의 경우

YTC는 콜 시점까지의 수익률

콜 가능성 높으면:
- YTM보다 YTC 중시
- 콜 가격 고려 필요
```

## 듀레이션 (Duration)

### 맥컬리 듀레이션

```
가중평균 현금흐름 회수 기간

        Σ t × PV(CFt)
D = ─────────────────
           P

t = 현금흐름 시점
PV(CFt) = t시점 현금흐름의 현재가치
P = 채권 가격
```

### 수정 듀레이션

```
Modified Duration = Macaulay Duration / (1 + YTM)

가격 변화율 ≈ -Modified Duration × Δ금리

예: 수정 듀레이션 4년, 금리 1% 상승
가격 변화 ≈ -4 × 1% = -4%
```

### 듀레이션 특성

```
듀레이션 결정 요인:

만기 ↑ → 듀레이션 ↑
쿠폰 ↑ → 듀레이션 ↓
YTM ↑ → 듀레이션 ↓

제로쿠폰채: 듀레이션 = 만기
```

## 신용 분석

### 신용등급 체계

```
투자등급 (Investment Grade):
AAA  │ 최고 등급
AA   │ 매우 우수
A    │ 우수
BBB  │ 양호
─────┼─────────────────
투기등급 (Speculative Grade):
BB   │ 투기적
B    │ 매우 투기적
CCC  │ 상당한 위험
CC   │ 높은 위험
C    │ 채무불이행 임박
D    │ 채무불이행
```

### 신용 스프레드

```
신용 스프레드 = 회사채 금리 - 국채 금리

스프레드 결정 요인:
├── 신용등급
├── 만기
├── 유동성
├── 담보 여부
└── 경기 상황

스프레드 확대 → 신용위험 증가
스프레드 축소 → 신용위험 감소
```

### 신용 분석 체크리스트

```
정량 분석:
□ 이자보상배율 (Interest Coverage)
□ 부채비율 (Debt/Equity)
□ 순부채/EBITDA
□ 유동비율
□ 현금흐름 안정성

정성 분석:
□ 산업 전망
□ 경쟁 지위
□ 경영진 역량
□ 지배구조
□ ESG 요소
```

## 채권 투자 전략

### 패시브 전략

```
1. 매입보유 (Buy and Hold)
   - 만기까지 보유
   - 안정적 현금흐름
   - 재투자 위험 존재

2. 래더링 (Laddering)
   - 다양한 만기 분산
   - 금리 위험 평균화
   - 유동성 확보

   예: 1년, 3년, 5년, 7년, 10년 채권 균등 투자
```

### 액티브 전략

```
1. 금리 예측 전략
   - 금리 하락 예상 → 듀레이션 확대
   - 금리 상승 예상 → 듀레이션 축소

2. 수익률 곡선 전략
   - 불링 (Bullet): 특정 만기 집중
   - 바벨 (Barbell): 단기/장기 집중
   - 래더 (Ladder): 균등 분산

3. 크레딧 전략
   - 등급 변화 예측
   - 스프레드 변화 예측
```
