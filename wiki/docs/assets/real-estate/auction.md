# 경매 & 공매

부동산 경매는 시세 대비 20~40% 저렴하게 부동산을 취득할 수 있는 투자 방법입니다. 다만 권리분석 실패 시 큰 손실이 발생할 수 있으므로, 체계적인 학습과 실전 경험이 필수입니다.

---

## 경매 vs 공매

| 항목 | 경매 | 공매 |
|:-----|:-----|:-----|
| **집행 기관** | 법원 | 한국자산관리공사(캠코), 세무서 등 |
| **근거 법률** | 민사집행법 | 국세징수법, 자산관리공사법 |
| **발생 원인** | 채권자 신청 (대출 미상환) | 세금 체납, 공공 자산 매각 |
| **입찰 방식** | 법원 기일입찰 (오프라인/온라인) | 온비드 전자입찰 |
| **권리분석** | 복잡 (말소기준권리 분석 필요) | 비교적 단순 |
| **명도 책임** | 낙찰자 | 낙찰자 |
| **매각대금** | 잔금 납부 후 소유권이전 | 잔금 납부 후 소유권이전 |

## 경매 절차

### 전체 타임라인

| 단계 | 시기 | 내용 |
|:-----|:-----|:-----|
| **1. 물건 검색** | 상시 | 대법원 경매 사이트, 경매 앱으로 물건 탐색 |
| **2. 권리분석** | 입찰 전 | 등기부등본, 배당표, 현황조사서 분석 |
| **3. 현장 답사** | 입찰 전 | 물건 상태, 점유자, 주변 환경 확인 |
| **4. 입찰가 산정** | 입찰 전 | 시세 조사 → 목표 낙찰가 결정 |
| **5. 입찰** | 매각기일 | 입찰서 제출 + 보증금 납부 (최저가의 10%) |
| **6. 개찰·낙찰** | 매각기일 | 최고가 입찰자 낙찰 |
| **7. 매각허가결정** | 낙찰 후 7일 | 법원의 매각 허가 (이의신청 가능) |
| **8. 잔금 납부** | 허가 후 1개월 | 낙찰 대금 납부 |
| **9. 소유권이전등기** | 잔금 후 | 법원 촉탁 등기 |
| **10. 명도** | 등기 후 | 점유자 퇴거 (합의 또는 강제집행) |

### 경매 비용 구조

| 비용 | 내용 | 금액 |
|:-----|:-----|:-----|
| **입찰 보증금** | 최저매각가의 10% (낙찰 시 대금에 충당) | 환급 가능 |
| **취득세** | 일반 매매와 동일 | 1~12% |
| **등록면허세** | 소유권이전등기 | 약 0.2~0.4% |
| **법무사 비용** | 등기 대행 | 50~100만원 |
| **명도 비용** | 이사비 합의금 또는 강제집행 비용 | 100~500만원 |

## 권리분석 핵심

### 말소기준권리

경매에서 가장 중요한 개념입니다. 말소기준권리보다 **후순위** 권리는 경매로 소멸되고, **선순위** 권리는 인수해야 합니다.

| 말소기준권리 | 설명 |
|:-----------|:-----|
| **(근)저당권** | 가장 흔한 말소기준권리 |
| **가압류** | 채권자의 재산 보전 처분 |
| **담보가등기** | 담보 목적의 가등기 |
| **경매개시결정등기** | 경매 신청에 의한 등기 |

:::danger 인수되는 권리 (낙찰자가 떠안는 것)
말소기준권리보다 **선순위**인 다음 권리는 낙찰자가 인수해야 합니다:
- 선순위 전세권
- 선순위 임차인의 보증금 (대항력 있는 임차인)
- 선순위 지상권
- 법정지상권
- 유치권

이 권리들의 금액을 낙찰가에 더해야 **실제 취득 비용**입니다.
:::

### 배당 분석

| 순위 | 권리 | 설명 |
|:-----|:-----|:-----|
| 1순위 | 경매 비용 | 법원 집행 비용 |
| 2순위 | 임금·퇴직금 | 근로자 우선변제 |
| 3순위 | 소액임차인 | 최우선변제금 (지역별 상이) |
| 4순위 | 당해세 | 재산세, 종부세 등 |
| 5순위 | 선순위 담보권 | 근저당, 전세권 등 (설정일 순) |
| 6순위 | 후순위 담보권 | 나중에 설정된 권리 |
| 7순위 | 일반 채권 | 무담보 채권 |

### 소액임차인 최우선변제금

| 지역 | 보증금 기준 | 최우선변제금 |
|:-----|:----------|:----------|
| **서울** | 1억 6,500만원 이하 | 5,500만원 |
| **수도권 과밀억제권역** | 1억 4,500만원 이하 | 4,800만원 |
| **광역시** | 8,500만원 이하 | 2,800만원 |
| **기타** | 7,500만원 이하 | 2,500만원 |

## 입찰 전략

### 낙찰가 산정

$$
\text{적정 입찰가} = \text{시세} - \text{인수 권리} - \text{명도 비용} - \text{수리 비용} - \text{기대 수익}
$$

| 물건 유형 | 일반 낙찰가율 | 특징 |
|:---------|:-----------|:-----|
| **아파트** | 시세의 80~95% | 경쟁 치열, 시세 투명 |
| **빌라·다세대** | 시세의 60~80% | 시세 파악 어려움, 수익률 높음 |
| **상가** | 시세의 50~70% | 권리분석 복잡, 공실 리스크 |
| **토지** | 감정가의 40~70% | 유찰 많음, 활용 계획 필수 |

### 유찰과 최저가 하락

| 회차 | 최저매각가 | 비고 |
|:-----|:---------|:-----|
| 1회 | 감정가의 100% | 첫 입찰 |
| 2회 (1차 유찰) | 감정가의 80% | 20% 하락 |
| 3회 (2차 유찰) | 감정가의 64% | 추가 20% 하락 |
| 4회 (3차 유찰) | 감정가의 51% | 추가 20% 하락 |
| 5회 이상 | 특별매각 또는 취소 | - |

## 명도 (점유자 퇴거)

| 방법 | 설명 | 비용 | 기간 |
|:-----|:-----|:-----|:-----|
| **합의 명도** | 이사비 지급 후 원만 퇴거 | 100~300만원 | 1~3개월 |
| **인도명령** | 법원에 인도명령 신청 | 신청비 + 집행비 | 2~4개월 |
| **명도소송** | 대항력 있는 점유자 대상 | 소송비 + 집행비 | 6개월~1년 |
| **강제집행** | 법원 집행관 동행 퇴거 | 200~500만원 | 인도명령 후 |

:::tip 명도 실전 팁
- 대부분의 점유자는 **합의 명도**를 선호합니다. 이사비 100~300만원 제시가 일반적입니다
- 인도명령은 낙찰 대금 납부 후 **6개월 이내** 신청해야 합니다
- 유치권을 주장하는 점유자는 법적 분쟁이 길어질 수 있으므로 입찰 전 반드시 확인하세요
:::

## 공매 (온비드)

### 공매 절차

| 단계 | 내용 |
|:-----|:-----|
| **1. 물건 검색** | 온비드(onbid.co.kr)에서 매각 물건 검색 |
| **2. 입찰** | 온라인 전자입찰 (공인인증서 필요) |
| **3. 낙찰** | 최고가 입찰자 낙찰 |
| **4. 대금 납부** | 납부 기한 내 전액 납부 |
| **5. 소유권 이전** | 캠코가 소유권이전 서류 발급 |

### 공매의 장점

| 장점 | 설명 |
|:-----|:-----|
| **온라인 입찰** | 법원 방문 불필요 |
| **경쟁 적음** | 경매 대비 참여자 적음 |
| **절차 간편** | 경매보다 단순한 권리관계 |
| **국유재산** | 정부 보유 부동산도 매각 |
