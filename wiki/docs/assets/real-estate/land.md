# 토지·임야·농지 투자

토지 투자는 부동산 투자 중 가장 높은 수익률을 기대할 수 있는 반면, 가장 긴 투자 기간과 가장 복잡한 규제를 수반합니다. 개발 호재에 의한 가치 상승이 핵심이며, 토지의 법적 성격(용도지역, 지목, 규제)을 정확히 파악하는 것이 성패를 좌우합니다.

---

## 토지 투자의 기본

### 토지 가격 결정 요인

| 요인 | 설명 | 영향도 |
|:-----|:-----|:------|
| **용도지역** | 주거, 상업, 공업, 녹지 등 | 매우 높음 (건축 가능 규모 결정) |
| **접도 조건** | 도로 접면 여부 및 폭 | 매우 높음 (맹지는 건축 불가) |
| **개발 계획** | GTX, 신도시, 산업단지 등 | 높음 (투기 요인) |
| **지형·형상** | 정방형, 경사도, 고저 차이 | 중간 (공사비 영향) |
| **기반시설** | 상하수도, 전기, 가스 | 중간 (추가 비용 발생) |
| **인근 시세** | 주변 거래 사례 | 기본 (비교 기준) |

### 용도지역별 특성

| 용도지역 | 건폐율 | 용적률 | 투자 특성 |
|:---------|:------|:------|:---------|
| **제1종 전용주거** | 50% | 50~100% | 단독주택, 저밀도 |
| **제1종 일반주거** | 60% | 100~200% | 저층 주거 |
| **제2종 일반주거** | 60% | 150~250% | 중층 아파트 가능 |
| **제3종 일반주거** | 50% | 200~300% | 고층 아파트 가능 |
| **준주거지역** | 70% | 200~500% | 주거+상업 복합 |
| **일반상업지역** | 80% | 200~1,300% | 오피스·상가 |
| **준공업지역** | 70% | 200~400% | 주거 전환 핵심 (용도변경 기대) |
| **계획관리지역** | 40% | 100% | 비도시지역 중 개발 가능 |
| **보전관리지역** | 20% | 80% | 개발 극도로 제한 |
| **농림지역** | 20% | 80% | 농업 용도만 가능 |
| **자연환경보전** | 20% | 80% | 개발 불가 |

:::caution 토지 투자 핵심: 용도지역 확인 필수
같은 "대"지목 100평이라도:
- **일반상업지역**: 용적률 1,300% → 1,300평 건물 가능 → 평당 수천만원
- **자연환경보전지역**: 용적률 80% → 80평 건물도 어려움 → 평당 수만원

토지 가격은 지목이 아니라 **용도지역**이 결정합니다. 반드시 토지이용계획확인서를 통해 확인하세요.
:::

## 농지

### 농지의 법적 정의

농지법상 농지는 **전(밭), 답(논), 과수원** 등 실제 농업에 사용되는 토지입니다.

### 농지취득자격증명

농지를 취득하려면 반드시 **농지취득자격증명**을 발급받아야 합니다.

| 항목 | 내용 |
|:-----|:-----|
| **발급 기관** | 농지 소재지 읍·면·동사무소 |
| **발급 요건** | 자경 농민, 농업법인, 또는 자경 계획 제출 |
| **자경 의무** | 취득 후 실제 영농 종사 (위반 시 처분 명령) |
| **비농업인** | 주말농장 목적 시 1,000㎡(약 300평) 이하만 가능 |
| **처리 기간** | 4~7일 |

### 농업진흥지역 vs 비진흥지역

| 구분 | 농업진흥지역 | 비진흥지역 |
|:-----|:-----------|:---------|
| **규제 강도** | 매우 강함 (절대농지) | 상대적으로 완화 |
| **개발 가능성** | 거의 불가 (농업 용도만) | 제한적 개발 가능 |
| **전용 허가** | 원칙적 불허 (예외적 허용) | 농지전용 신청 가능 |
| **투자 가치** | 낮음 (농업 수익만) | 중간~높음 (개발 기대) |
| **농지전용부담금** | 높음 | 상대적으로 낮음 |

### 농지전용

농지를 다른 용도로 사용하려면 농지전용허가를 받아야 합니다.

| 항목 | 내용 |
|:-----|:-----|
| **전용 허가** | 시장·군수·구청장 허가 |
| **전용 부담금** | 공시지가의 30% (농업진흥지역 외) |
| **전용 불가 사유** | 농업진흥지역, 경지정리 완료 농지, 보전 필요 농지 |
| **소규모 전용** | 660㎡ 이하 농업인 주택 등은 신고로 가능 |

## 임야 (산림)

### 임야의 특성

| 항목 | 내용 |
|:-----|:-----|
| **면적** | 한국 국토의 약 63% (매우 넓음) |
| **가격** | 평당 수만~수십만원 (저렴) |
| **개발** | 산지전용허가 필요, 매우 까다로움 |
| **투자 매력** | 장기 개발 기대, 소액 투자 가능 |
| **리스크** | 개발 불허, 환경 규제, 접근성 |

### 산지 구분

| 구분 | 내용 | 개발 가능성 |
|:-----|:-----|:----------|
| **보전산지** | 공익용·임업용 산지 | 거의 불가 |
| **준보전산지** | 보전산지 외 산지 | 제한적 개발 가능 |

### 산지전용허가

| 항목 | 내용 |
|:-----|:-----|
| **허가 기관** | 산림청, 시·도지사, 시장·군수 |
| **전용 부담금** | 공시지가의 산정 비율 (산지 유형별 차등) |
| **환경영향평가** | 10,000㎡ 이상 시 필요 |
| **복구비 예치** | 산지 복구를 위한 보증금 예치 필요 |

:::danger 임야 투자 주의사항
- **맹지 여부**: 도로에 접하지 않은 임야는 개발이 사실상 불가능합니다
- **경사도**: 경사도 25도 이상은 산지전용 불허 대상입니다
- **보전산지**: 보전산지로 지정된 임야는 개발 기대가 거의 없습니다
- **환경 규제**: 생태자연도 1등급, 상수원보호구역 등은 추가 규제
- **사기 주의**: "곧 개발된다"는 브로커 말을 믿고 투자하면 안 됩니다
:::

## 토지 실사 (Due Diligence) 체크리스트

### 서류 확인

| 서류 | 확인 내용 | 발급처 |
|:-----|:---------|:------|
| **등기부등본** | 소유자, 가압류, 근저당 | 인터넷등기소 |
| **토지이용계획확인서** | 용도지역, 도시계획, 규제 | 토지이음 |
| **토지대장** | 지목, 면적, 소유 이력 | 정부24 |
| **지적도** | 토지 경계, 형상, 도로 접면 | 정부24 |
| **개별공시지가** | 세금·보상 기준 가격 | 부동산공시가격 알리미 |

### 현장 확인

| 항목 | 확인 내용 |
|:-----|:---------|
| **접도** | 실제 도로 접면 여부, 도로 폭 (건축법상 4m 이상) |
| **기반시설** | 상하수도, 전기, 가스 인입 가능 여부 |
| **경사도** | 급경사지 여부 (개발 비용 증가) |
| **주변 환경** | 혐오시설, 군사시설, 상수원 보호구역 |
| **실제 사용** | 지목과 실제 사용 용도 일치 여부 |

## 토지 투자 전략

| 전략 | 방법 | 리스크 | 기대 수익 |
|:-----|:-----|:------|:---------|
| **신도시 예정지** | 보상 대상 토지 매입 | 보상가 불확실 | 보상가 프리미엄 |
| **용도변경 기대** | 준공업→주거 전환 지역 | 변경 불확실 | 매우 높음 |
| **개발행위허가** | 농지·임야 전용 후 건축 | 허가 불확실 | 중간~높음 |
| **수용 보상** | 공공사업 편입 토지 | 보상 시기 불확실 | 보상금 |
| **지목 변경** | 전답→대 전환 | 전용 비용, 허가 | 높음 |
