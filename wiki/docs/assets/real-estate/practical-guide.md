# 부동산 실전 가이드

이 문서는 실제로 방을 구하고, 계약하고, 입주하는 과정에서 필요한 모든 실전 정보를 담았습니다. 사회초년생이 원룸을 구하든, 신혼부부가 아파트를 매수하든 이 가이드를 따라가면 됩니다.

---

## 1단계: 방 구하기 준비

### 예산 설정

| 항목 | 월세 기준 | 전세 기준 | 매매 기준 |
|:-----|:---------|:---------|:---------|
| **보증금** | 월소득 3~6배 | 저축 + 대출 | 자기자본 30~40% |
| **월 지출** | 월소득의 30% 이하 | 관리비만 | 대출 이자 + 관리비 |
| **추가 비용** | 중개비 + 이사비 | 중개비 + 이사비 | 취득세 + 중개비 + 법무사비 |

### 지역 선정 기준

| 기준 | 확인 방법 |
|:-----|:---------|
| 직장 통근 거리 | 네이버 지도 대중교통 검색 (30~40분 이내 권장) |
| 역세권 여부 | 지하철역 도보 10분 이내 |
| 주변 편의시설 | 편의점, 마트, 병원, 세탁소 |
| 치안 | 안심귀갓길, CCTV, 가로등 (저녁에 현장 방문 추천) |
| 수해·침수 이력 | 정부 홍수 지도(floodmap.go.kr) |

## 2단계: 매물 검색 (앱 활용)

### 앱별 활용법

**직방/다방 사용 순서:**
1. 앱 설치 → 원하는 지역 설정
2. 거래 유형 선택 (월세/전세/매매)
3. 필터 설정: 보증금 범위, 월세 범위, 면적, 층수
4. 관심 매물 3~5개 저장
5. 각 매물의 중개사에게 방문 예약 전화

**호갱노노 사용 순서 (아파트):**
1. 지도에서 관심 지역 확인
2. 단지별 시세·실거래가 비교
3. 전세가율 확인 (전세 안전도 판단)
4. 최근 거래 이력·가격 추이 그래프 확인

**피터팬의 좋은방 (직거래):**
1. 매물 검색 → 집주인 직접 게시 매물 확인
2. 중개수수료 절약 가능 (직거래)
3. 단, 권리관계 확인은 본인이 직접 해야 함
4. 가짜 매물·사기 주의 (반드시 등기부등본 확인)

:::caution 허위 매물 주의
- 시세 대비 지나치게 저렴한 매물은 **미끼 매물**일 확률이 높습니다
- "방금 나갔는데 비슷한 거 있다"며 다른 비싼 매물을 소개하면 미끼
- 반드시 **내가 앱에서 본 그 매물**을 보여달라고 요청하세요
- 사진과 실물이 다르면 바로 나오세요
:::

## 3단계: 공인중개사 방문

### 방문 전 준비

| 준비 사항 | 이유 |
|:---------|:-----|
| 관심 매물 3~5개 리스트 | 한 번 방문에 여러 매물 볼 수 있음 |
| 예산·조건 명확히 정리 | 중개사가 맞는 매물 추천 가능 |
| 체크리스트 인쇄/메모 | 현장에서 빠짐없이 확인 |
| 줄자 (또는 줄자 앱) | 가구 배치 확인 |

### 공인중개사 방문 시 대화법

| 상황 | 말하면 좋은 것 | 주의할 것 |
|:-----|:------------|:---------|
| **처음 만날 때** | "이 지역에서 OO조건으로 찾고 있습니다" | 예산 최대치를 바로 말하지 마세요 |
| **매물 볼 때** | "이 집 언제부터 나왔나요?" "직전 세입자가 왜 나갔나요?" | 급하다는 인상을 주지 마세요 |
| **흥정할 때** | "다른 곳도 보고 있어서 조금 깎아주시면 바로 결정하겠습니다" | 무리한 흥정은 역효과 |
| **계약 단계** | "등기부등본 먼저 확인하겠습니다" | 등기부등본 확인 전 계약금 주지 마세요 |

### 중개사 방문 시 확인 사항

| 확인 항목 | 질문 |
|:---------|:-----|
| 중개사 자격 | 사무실에 공인중개사 자격증·사업자등록증 게시 여부 |
| 해당 매물 전문성 | "이 근처에서 얼마나 오래 하셨어요?" |
| 중개보수 | "중개수수료는 얼마인가요?" (법정 상한 확인) |
| 하자 이력 | "이 건물 누수·결로·소음 문제 있었나요?" |

## 4단계: 현장 방문 체크리스트

### 월세방·전세방 체크리스트

#### 건물 외부

| 항목 | 확인 내용 | O/X |
|:-----|:---------|:-----|
| 건물 외관 상태 | 균열, 외벽 파손, 타일 탈락 | |
| 주차장 | 주차 가능 여부, 1세대 1주차 | |
| CCTV | 주출입구, 주차장, 엘리베이터 | |
| 우편함·택배함 | 위치와 보안 | |
| 쓰레기 분리수거장 | 위치, 관리 상태 | |

#### 실내 — 필수 확인

| 항목 | 확인 내용 | O/X |
|:-----|:---------|:-----|
| **채광** | 남향/동향 확인, 실제 햇빛 들어오는지 | |
| **통풍** | 창문 2개 이상 (맞통풍 가능 여부) | |
| **습기·곰팡이** | 벽면·천장·장판 하부 확인 | |
| **수압** | 화장실·주방 수도꼭지 실제 틀어보기 | |
| **온수** | 온수 나오는 데 걸리는 시간 | |
| **배수** | 화장실·싱크대 물 내리고 배수 속도 | |
| **보일러** | 작동 확인, 제조연도 (10년 이상이면 교체 필요) | |
| **전기** | 콘센트 위치·개수, 에어컨 전용 콘센트 유무 | |
| **방음** | 벽 두드려보기, 층간소음 (위층에서 걸어보면 좋음) | |
| **인터넷** | 통신사별 설치 가능 여부, 기존 배선 | |
| **벌레** | 배관 틈새, 방충망 상태, 1층은 특히 확인 | |
| **창문** | 이중창 여부, 틈새 바람, 방범창 | |

#### 실내 — 추가 확인

| 항목 | 확인 내용 |
|:-----|:---------|
| 세탁기 자리 | 실내/베란다/공용, 배수 연결 |
| 에어컨 | 기존 설치 여부, 설치 가능 위치 |
| 신발장·수납 | 수납 공간 충분한지 |
| 도어록 | 디지털/일반, 비밀번호 변경 가능 |
| 화재 감지기 | 설치 여부 |

#### 주변 환경

| 항목 | 확인 내용 |
|:-----|:---------|
| 소음원 | 대로변, 공사장, 유흥가, 철로 |
| 편의시설 | 편의점, 마트, 세탁소, 병원 거리 |
| 대중교통 | 지하철·버스 정류장 도보 거리 |
| 야간 치안 | 저녁 시간에 다시 방문 추천 |
| 주변 시설 | 학교(소음), 장례식장, 공장 등 혐오시설 |

:::tip 현장 방문 꿀팁
- **오후 2~4시**에 방문하면 채광을 정확히 확인할 수 있습니다
- **비 오는 날** 방문하면 누수·습기를 바로 발견할 수 있습니다
- **저녁 시간**에 한번 더 방문하면 야간 소음·치안을 확인할 수 있습니다
- 사진과 영상을 많이 찍어두세요 (나중에 비교할 때 유용)
- 화장실 물을 직접 틀어보고, 보일러를 직접 켜보세요
:::

## 5단계: 계약

### 계약 전 필수 확인

| 순서 | 확인 사항 | 방법 |
|:-----|:---------|:-----|
| 1 | **등기부등본** 열람 | 인터넷등기소 (계약 당일 발급) |
| 2 | **건축물대장** 확인 | 정부24 (위반건축물 여부) |
| 3 | 소유자 = 계약 상대방 | 등기부 소유자와 신분증 대조 |
| 4 | 근저당·가압류 확인 | 등기부등본 갑구·을구 |
| 5 | 임대인 체납 세금 확인 | 임대인에게 납세증명서 요청 (미납 세금은 선순위) |

### 전세 계약 시 추가 확인

| 확인 사항 | 안전 기준 |
|:---------|:---------|
| 전세가율 | 매매가의 70% 이하 |
| 근저당 + 전세금 합계 | 매매가의 80% 이하 |
| 전세보증보험(HUG/SGI) 가입 가능 여부 | 가입 가능해야 안전 |
| 임대인 다주택 여부 | 갭투자자 주의 |

### 계약서 작성 핵심

| 항목 | 확인·기재 사항 |
|:-----|:-------------|
| **임대인 정보** | 성명, 주민등록번호, 연락처 (등기부 소유자와 일치 확인) |
| **보증금/월세** | 금액, 지급일, 지급 방법 (계좌이체 권장) |
| **계약 기간** | 시작일·종료일 명확히 (최소 2년 보장) |
| **관리비** | 포함 항목(수도·전기·인터넷·TV 등) 명확히 |
| **수선 의무** | 임대인 부담 범위 (보일러, 수도, 구조적 하자) |
| **특약사항** | 옵션 인수, 반려동물, 입주 전 수리 사항 등 |
| **중도 해지** | 위약금 규정 |

### 계약금·잔금 납부

| 단계 | 금액 | 주의사항 |
|:-----|:-----|:--------|
| **계약금** | 보증금의 5~10% | 등기부 확인 후 지급, 임대인 계좌로 이체 |
| **잔금** | 나머지 전액 | 잔금일 당일 등기부 재확인 후 지급 |

:::danger 계약 사기 예방 수칙
1. **계약금은 반드시 등기부 확인 후** 지급하세요
2. 입금은 **임대인(소유자) 본인 명의 계좌**로만 하세요
3. 대리인과 계약 시 **위임장 + 인감증명서** 확인 필수
4. 전세는 반드시 **전세보증보험 가입 가능 여부**를 사전에 확인하세요
5. "급하게 계약하라"는 압박에 절대 넘어가지 마세요
:::

## 6단계: 입주 & 전입신고

### 입주일 당일 할 일

| 순서 | 할 일 | 방법 |
|:-----|:------|:-----|
| 1 | 잔금 지급 | 임대인 계좌 이체 (이체 확인증 보관) |
| 2 | 열쇠 수령 | 도어록 비밀번호 변경 |
| 3 | 실제 입주 | 짐 반입, 거주 시작 |
| 4 | **전입신고** | 주민센터 방문 또는 정부24 온라인 |
| 5 | **확정일자** | 주민센터 또는 정부24 (전입신고와 동시 처리 가능) |

:::caution 전입신고는 입주 당일 반드시
- **대항력** 발생: 전입신고 + 입주 → **다음날 0시**부터 효력
- **우선변제권**: 대항력 + 확정일자 → 경매 시 보증금 우선 회수
- 하루라도 늦으면 그 사이에 근저당이 설정될 수 있습니다
:::

### 전세보증보험 가입

| 보험 | 가입 시기 | 비용 | 신청 방법 |
|:-----|:---------|:-----|:---------|
| **HUG 전세보증금반환보증** | 전세 계약 후 | 연 보증금의 0.115~0.154% | HUG 홈페이지 또는 은행 |
| **SGI 전세금보장신용보험** | 전세 계약 후 | 연 보증금의 0.183~0.347% | SGI 홈페이지 또는 대리점 |

## 7단계: 퇴거

### 퇴거 절차

| 순서 | 할 일 | 시기 |
|:-----|:------|:-----|
| 1 | 임대인에게 계약 종료 통보 | 계약 만료 6~2개월 전 |
| 2 | 보증금 반환 일정 협의 | 퇴거 1개월 전 |
| 3 | 원상복구 | 퇴거 전 |
| 4 | 전기·수도·가스 정산 | 퇴거 당일 |
| 5 | 보증금 수령 | 퇴거 당일 (동시이행) |
| 6 | 전출신고 | 이사 후 14일 이내 |

### 보증금 미반환 시 대처법

| 단계 | 방법 | 소요 기간 |
|:-----|:-----|:---------|
| 1단계 | 내용증명 발송 | 즉시 |
| 2단계 | 임차권등기명령 신청 (법원) | 1~2주 |
| 3단계 | 보증금반환소송 | 3~6개월 |
| 4단계 | 강제집행 (경매 신청) | 소송 후 |
| **대안** | 전세보증보험으로 대위변제 | 2~4주 |

| 항목 | 시기 | 내용 |
