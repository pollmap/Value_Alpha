# 등기 & 권리분석

부동산 투자에서 가장 기본이면서도 가장 중요한 단계가 **권리분석**입니다. 등기부등본 한 장을 정확히 읽을 수 있으면 사기 피해의 90% 이상을 예방할 수 있습니다. 이 문서에서는 등기부등본의 구조, 각 항목의 의미, 위험 신호 식별법을 체계적으로 다룹니다.

---

## 등기부등본의 구조

등기부등본(정식 명칭: 부동산등기사항전부증명서)은 **표제부**, **갑구**, **을구** 세 부분으로 구성됩니다.

| 구분 | 내용 | 핵심 확인 사항 |
|:-----|:-----|:-------------|
| **표제부** | 부동산의 물리적 현황 | 소재지, 면적, 구조, 용도 |
| **갑구** | 소유권 관련 사항 | 소유자, 가압류, 가처분, 경매 개시 |
| **을구** | 소유권 이외의 권리 | 근저당, 전세권, 지상권, 임차권 |

---

## 표제부 분석

표제부는 해당 부동산의 **물리적 정보**를 담고 있습니다.

### 토지 표제부 확인 항목

| 항목 | 확인 내용 | 주의사항 |
|:-----|:---------|:--------|
| **소재지번** | 정확한 위치 | 도로명주소와 지번주소 불일치 시 확인 |
| **지목** | 대, 전, 답, 임야 등 | 실제 사용 용도와 지목 불일치 여부 |
| **면적** | 등기 면적 (㎡) | 실측 면적과 차이 가능성 |

### 건물 표제부 확인 항목

| 항목 | 확인 내용 | 주의사항 |
|:-----|:---------|:--------|
| **건물 내역** | 구조, 용도, 면적 | 위반 건축물 여부 |
| **전유부분** | 전용면적, 층수, 호수 | 분양면적 ≠ 전용면적 |
| **대지권 비율** | 토지 지분 비율 | 대지권 미등기 시 문제 |

:::info 전용면적 vs 공급면적 vs 분양면적
- **전용면적**: 벽 안쪽 실사용 면적 (방, 거실, 주방, 화장실)
- **주거공용면적**: 계단, 복도, 엘리베이터 등
- **공급면적** = 전용면적 + 주거공용면적
- **분양면적(계약면적)** = 공급면적 + 기타공용면적(주차장, 관리실 등)

실제 생활 공간은 **전용면적**만 해당되므로, 아파트 비교 시 반드시 전용면적 기준으로 비교해야 합니다. 전용 84㎡(약 34평)가 한국에서 가장 보편적인 아파트 면적입니다.
:::

## 갑구 분석 (소유권 관련)

갑구는 **누가 소유자인지**, 소유권에 영향을 미치는 **권리 제한이 있는지**를 보여줍니다.

### 정상적인 갑구 항목

| 등기 유형 | 의미 | 위험도 |
|:---------|:-----|:------|
| **소유권이전** | 매매·상속·증여로 소유자 변경 | 없음 |
| **소유권보존** | 최초 소유권 등기 (신축 시) | 없음 |

### 주의가 필요한 갑구 항목

| 등기 유형 | 의미 | 위험도 | 조치 |
|:---------|:-----|:------|:-----|
| **가압류** | 채권자가 재산 처분 금지 신청 | 높음 | 매매 전 반드시 해제 확인 |
| **가처분** | 소유권 분쟁 중 처분 금지 | 매우 높음 | 절대 매수 금지 |
| **경매개시결정** | 강제 경매 진행 중 | 매우 높음 | 경매로만 취득 가능 |
| **가등기** | 소유권 이전 예약 | 높음 | 본등기 시 후순위 권리 소멸 |
| **환매등기** | 매도인의 환매권 보유 | 중간 | 환매 기간·조건 확인 |
| **신탁등기** | 신탁회사에 소유권 이전 | 중간 | 수익자·신탁 조건 확인 |

:::danger 갑구에서 절대 피해야 할 3가지
1. **가처분 등기**: 소유권 자체가 분쟁 중이므로 매수하면 안 됩니다
2. **경매개시결정 후 매매**: 경매 절차가 진행 중이므로 정상 매매가 불가능합니다
3. **가등기 + 가압류 동시 존재**: 복수의 채권자가 관여된 부동산은 권리관계가 매우 복잡합니다
:::

## 을구 분석 (소유권 이외의 권리)

을구는 **담보권과 용익권** 등 소유권 이외의 권리를 보여줍니다. 대출과 직결되는 핵심 항목입니다.

### 주요 을구 항목

| 등기 유형 | 의미 | 투자자 확인사항 |
|:---------|:-----|:-------------|
| **근저당권** | 금융기관 담보 설정 | 채권최고액, 채무자 확인 |
| **저당권** | 특정 금액 담보 설정 | 근저당보다 드묾 |
| **전세권** | 전세금에 대한 물권적 보장 | 전세 보증금 규모 확인 |
| **지상권** | 타인 토지 위 건물 소유·사용 권리 | 토지·건물 소유자 분리 시 |
| **지역권** | 통행 등 타인 토지 사용 권리 | 접도 의무 관련 |
| **임차권** | 임차인의 등기된 권리 | 대항력 확인 |

### 근저당권 분석법

근저당권은 부동산 등기에서 가장 흔하게 나타나는 을구 항목입니다.

| 용어 | 의미 | 예시 |
|:-----|:-----|:-----|
| **채권최고액** | 담보로 보장하는 최대 금액 | 실제 대출금의 120~130% |
| **채무자** | 대출을 받은 사람 | 소유자와 다를 수 있음 |
| **근저당권자** | 대출해준 금융기관 | 은행, 저축은행, 캐피탈 등 |

:::tip 근저당 채권최고액에서 실제 대출금 역산하기
채권최고액 = 실제 대출금 × 120~130% (은행마다 다름)

- 채권최고액 3억 9,000만원 → 실제 대출금 약 3억원 (130% 기준)
- 채권최고액 3억 6,000만원 → 실제 대출금 약 3억원 (120% 기준)

전세 계약 시 **시세 - 근저당 채권최고액 = 안전한 전세금 상한선**으로 계산합니다.
:::

## 권리분석 실전 체크리스트

부동산 거래 전 아래 체크리스트를 순서대로 확인합니다.

### 1단계: 기본 확인

| 체크 항목 | 확인 방법 |
|:---------|:---------|
| 등기부등본 발급일이 최근인가? | 계약 당일 발급 권장 |
| 소유자와 매도인이 동일한가? | 신분증 대조 |
| 면적이 계약서와 일치하는가? | 표제부 전용면적 확인 |

### 2단계: 갑구 확인

| 체크 항목 | 위험 여부 |
|:---------|:---------|
| 가압류·가처분 유무 | 있으면 거래 중단 |
| 소유권 이전 이력이 정상적인가? | 단기간 잦은 이전은 주의 |
| 신탁 등기 유무 | 있으면 신탁회사 동의 필요 |

### 3단계: 을구 확인

| 체크 항목 | 판단 기준 |
|:---------|:---------|
| 근저당 채권최고액 합계 | 시세의 70% 이하면 안전 |
| 전세권 설정 유무 | 기존 전세금 규모 파악 |
| 선순위 권리 총액 | 매매가 대비 적정 비율 확인 |

## 등기부등본 외 추가 확인 서류

| 서류 | 발급처 | 확인 내용 |
|:-----|:------|:---------|
| **건축물대장** | 정부24 | 위반건축물, 용도, 면적, 주차장 |
| **토지이용계획확인서** | 토지이음 | 용도지역, 규제, 도로 |
| **토지대장** | 정부24 | 지목, 면적, 소유자 |
| **지적도·임야도** | 정부24 | 경계, 형상, 도로 접면 |
| **건물 에너지효율등급** | 에너지관리공단 | 단열·냉난방 효율 |
| **주택가격공시** | 부동산공시가격 알리미 | 공시지가, 재산세 기준 |

:::
