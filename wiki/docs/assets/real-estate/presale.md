# 분양 & 분양권

분양은 건설사가 새 아파트를 짓기 전(또는 건설 중에) 미리 구매자를 모집하는 제도입니다. 청약 당첨 후 계약을 체결하면 **분양권**이 생기며, 이 분양권 자체가 거래 대상이 됩니다. 분양가와 시세의 차이가 곧 수익이 되므로, 분양 구조를 정확히 이해하는 것이 핵심입니다.

---

## 분양의 종류

| 유형 | 공급 주체 | 분양가 | 대상 | 특징 |
|:-----|:--------|:------|:-----|:-----|
| **공공분양** | LH, SH 등 공공기관 | 시세의 50~80% | 무주택 서민 | 소득·자산 기준 엄격, 전매 제한 강함 |
| **민간분양** | 민간 건설사 | 시세 반영 | 청약통장 보유자 | 분양가상한제 적용 여부에 따라 차이 |
| **사전청약** | 공공 (LH) | 추정 분양가 | 무주택자 | 입주까지 5~7년, 본청약 전 사전 당첨 |
| **후분양** | 민간/공공 | 완공 후 확정가 | 일반 | 실물 확인 후 계약, 분양가 높을 수 있음 |
| **임대 후 분양** | 공공 | 감정평가가 | 임대 거주자 | 5~10년 임대 후 분양 전환 |

## 분양가 구성

### 분양가의 구조

| 항목 | 내용 | 비중 |
|:-----|:-----|:-----|
| **택지비** | 토지 매입 비용 | 30~50% |
| **건축비** | 직접 공사비 + 간접 공사비 | 40~60% |
| **가산비** | 설계비, 감리비, 부대비용 | 5~10% |

### 분양가상한제

| 항목 | 내용 |
|:-----|:-----|
| **적용 대상** | 공공택지 + 민간택지 중 분양가상한제 적용 지역 |
| **산정 방식** | 표준건축비(국토부 고시) + 택지비(감정평가) |
| **효과** | 주변 시세 대비 저렴한 분양가 → 시세 차익 기대 |
| **부작용** | 건설사 수익성 저하 → 마감재 품질 하락, 공급 축소 |

:::info 분양가상한제 아파트의 시세 차익 구조
분양가상한제 적용 아파트는 주변 시세 대비 수천만~수억 원 저렴하게 공급됩니다.

**예시**: 주변 시세 15억 원 지역에서 분양가 10억 원으로 공급
- 시세 차익 잠재력: 약 5억 원
- 단, 전매 제한(최대 10년)과 실거주 의무(2~5년)가 적용됩니다.
:::

## 분양 대금 납부 구조

### 일반적인 납부 스케줄

| 단계 | 시기 | 금액 | 자금 조달 |
|:-----|:-----|:-----|:---------|
| **계약금 1차** | 계약 시 | 분양가의 10% | 자기자본 |
| **계약금 2차** | 계약 후 1~2개월 | 분양가의 10% | 자기자본 |
| **중도금 (1~6차)** | 착공~입주 전 (6회 분할) | 분양가의 60% | 중도금 대출 |
| **잔금** | 입주 시 | 분양가의 20% | 주택담보대출 전환 |

### 중도금 대출

| 항목 | 내용 |
|:-----|:-----|
| **대출 비율** | 분양가의 최대 60% |
| **이자 납부** | 건설사 이자 후불제 or 무이자 (분양 조건에 따라) |
| **전환** | 입주 시 주택담보대출로 전환 |
| **DSR 적용** | 중도금 대출도 DSR 산정에 포함 |

## 분양권

### 분양권의 정의

분양권은 **아직 완공되지 않은 아파트를 인도받을 수 있는 권리**입니다. 소유권이 아닌 채권적 권리이며, 분양 계약 체결 시점부터 소유권 이전등기 완료 시점까지 존재합니다.

### 분양권 전매

| 항목 | 내용 |
|:-----|:-----|
| **전매 의미** | 분양권을 제3자에게 양도 (프리미엄 거래) |
| **프리미엄** | 분양가 대비 시세 상승분 (전매 가격 - 기납부금) |
| **전매 제한** | 지역·분양가에 따라 전매 제한 기간 상이 |

### 분양권 전매 제한 기간

| 지역 | 제한 기간 |
|:-----|:---------|
| 투기과열지구 (분양가상한제) | 소유권이전등기 시까지 (전매 불가) |
| 투기과열지구 (비상한제) | 3년 |
| 조정대상지역 | 1년~3년 |
| 비규제지역 (공공택지) | 6개월~1년 |
| 비규제지역 (민간택지) | 6개월 |

### 분양권 양도세

2021년 이후 분양권도 주택 수에 포함되어 양도소득세가 부과됩니다.

| 보유 기간 | 세율 |
|:---------|:-----|
| 1년 미만 | 70% |
| 1년 이상 2년 미만 | 60% |
| 2년 이상 | 기본세율 (6~45%) + 다주택 중과 가능 |

## 분양 옵션 선택

입주 전 선택하는 유상옵션(유상 추가 사양)은 주거 편의성에 큰 영향을 미칩니다.

| 옵션 | 비용 (대략) | 추천 여부 | 이유 |
|:-----|:----------|:---------|:-----|
| **시스템 에어컨** | 300~800만원 | 강력 추천 | 입주 후 설치 시 더 비쌈 |
| **발코니 확장** | 500~1,500만원 | 강력 추천 | 실사용 면적 20~30% 증가 |
| **빌트인 가전** | 200~500만원 | 선택적 | 브랜드·사양 확인 |
| **고급 마감재** | 100~300만원 | 선택적 | 실거주 시 추천 |
| **천장형 환기 시스템** | 100~200만원 | 추천 | 공기질 개선 |
| **바닥재 업그레이드** | 100~200만원 | 선택적 | 장판 → 강화마루 전환 |

:::tip 옵션 비용 vs 나중 시공 비용
시스템 에어컨과 발코니 확장은 분양 시 옵션으로 선택하는 것이 입주 후 별도 시공보다 **30~50% 저렴**합니다. 특히 발코니 확장은 입주 후에는 사실상 불가능하므로, 실거주 계획이 있다면 반드시 선택하세요.
:::

## 입주 절차

| 단계 | 시기 | 내용 |
|:-----|:-----|:-----|
| **사전점검** | 입주 2~3개월 전 | 하자 확인 (균열, 누수, 마감 불량) |
| **하자 보수** | 사전점검 후 | 건설사가 보수 진행 |
| **잔금 납부** | 입주지정일 | 잔금 + 등기비용 + 취득세 준비 |
| **입주** | 잔금 후 | 열쇠 수령, 이사 |
| **소유권이전등기** | 입주 후 60일 이내 | 법무사 통해 등기 |

### 사전점검 체크리스트

| 점검 영역 | 확인 사항 |
|:---------|:---------|
| **벽면·천장** | 균열, 도배 들뜸, 곰팡이 |
| **바닥** | 수평, 마루 들뜸, 소리 |
| **창호** | 개폐 상태, 기밀성, 결로 |
| **배수** | 화장실·싱크대 배수 속도, 누수 |
| **전기** | 콘센트 작동, 조명, 분전반 |
| **설비** | 보일러, 환기, 인터폰, 디지털 도어록 |
