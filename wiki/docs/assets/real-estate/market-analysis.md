# 부동산 시장 분석 & 앱·도구 가이드

부동산 투자의 성패는 **시장을 얼마나 정확히 읽느냐**에 달려 있습니다. 이 문서에서는 부동산 시장을 분석하는 핵심 지표와 정책 프레임워크, 그리고 실전에서 반드시 활용해야 하는 앱과 도구를 다룹니다.

---

## 부동산 시장 핵심 지표

### 가격 지표

| 지표 | 발표 기관 | 주기 | 활용 |
|:-----|:--------|:-----|:-----|
| **KB부동산 시세** | KB국민은행 | 주간 | 매매·전세 시세 기준 (대출 감정 활용) |
| **한국부동산원 지수** | 한국부동산원 | 주간/월간 | 공식 가격 동향 (정책 기준) |
| **실거래가** | 국토교통부 | 실시간 | 실제 거래 가격 (가장 정확) |
| **공시지가** | 국토교통부 | 연 1회 | 세금·보상 기준 |
| **감정평가액** | 감정평가사 | 필요시 | 대출·경매·분양 기준 |

### 수급 지표

| 지표 | 의미 | 해석 |
|:-----|:-----|:-----|
| **미분양** | 분양 후 미계약 물량 | 증가 → 공급 과잉 신호 |
| **입주 물량** | 향후 입주 예정 세대수 | 대량 입주 → 전세가 하락 압력 |
| **인허가 물량** | 건축 허가 받은 물량 | 2~3년 후 공급 선행지표 |
| **착공 물량** | 실제 공사 시작 물량 | 인허가보다 현실적 지표 |
| **매매·전세 거래량** | 실제 거래 건수 | 증가 → 시장 활성화 |

### 심리 지표

| 지표 | 의미 | 기준 |
|:-----|:-----|:-----|
| **매매수급지수** | 매수세 vs 매도세 | 100 초과: 매수세 우위, 100 미만: 매도세 우위 |
| **전세수급지수** | 전세 수요 vs 공급 | 100 초과: 수요 우위 (전세가 상승) |
| **PIR** (가격/소득비) | 집값 / 연소득 | 서울 평균 15~20배 (높을수록 부담) |
| **HAI** (주택구매력지수) | 소득 대비 구매 능력 | 100 이상이면 구매 가능 |
| **전세가율** | 전세가 / 매매가 | 80% 이상이면 위험 신호 |

## 부동산 정책 프레임워크

| 정책 도구 | 긴축 (가격 안정) | 완화 (시장 활성화) |
|:---------|:-------------|:---------------|
| **LTV** | 40~50%로 축소 | 70~80%로 확대 |
| **DSR** | 40%로 강화 | 완화 또는 예외 적용 |
| **취득세** | 다주택 중과 (8~12%) | 중과 완화·감면 |
| **양도세** | 다주택 중과 (+20~30%p) | 중과 배제·한시 감면 |
| **종부세** | 세율 인상, 공제 축소 | 세율 인하, 공제 확대 |
| **분양가상한제** | 적용 지역 확대 | 적용 지역 축소 |
| **전매 제한** | 기간 연장 | 기간 단축 |
| **청약 규제** | 1순위 요건 강화 | 요건 완화 |
| **대출 규제** | 총량 제한, 금리 인상 | 우대금리, 한도 확대 |

## 필수 부동산 앱 & 도구

부동산 거래에서 앱을 적극 활용하면 정보 비대칭을 줄이고, 더 나은 의사결정을 할 수 있습니다.

### 시세·실거래가 조회

| 앱/서비스 | 핵심 기능 | 사용법 | 추천 상황 |
|:---------|:---------|:------|:---------|
| **아파트 실거래가** (앱) | 국토부 실거래가 기반, 단지별 매매·전세·월세 이력 | 단지 검색 → 거래 이력 확인 → 시세 추이 그래프 | 매매·전세 시세 확인할 때 가장 먼저 |
| **호갱노노** (앱/웹) | 아파트 시세, 분양가 비교, 청약 분석, 학군 정보 | 지도에서 단지 선택 → 시세·실거래가·전세가율 확인 | 여러 단지 한눈에 비교할 때 |
| **KB부동산** (앱/웹) | KB시세 (은행 대출 기준), 시장 동향 리포트 | KB시세 조회 → 대출 가능 금액 추정에 활용 | 대출 한도 확인, 시세 공신력 필요할 때 |
| **부동산 리치고** (앱) | AI 시세 예측, 투자 분석, 수익률 계산 | 단지 검색 → AI 가격 전망·수익률 시뮬레이션 | 투자 의사결정 보조 도구로 |

### 매물 검색·중개

| 앱/서비스 | 핵심 기능 | 사용법 | 추천 상황 |
|:---------|:---------|:------|:---------|
| **직방** (앱) | 원룸·투룸·오피스텔·아파트 매물 검색, 3D VR 투어 | 지역 설정 → 조건 필터(가격·면적·층) → 매물 상세 확인 → 중개사 연결 | 원룸·오피스텔 구할 때 |
| **다방** (앱) | 원룸~아파트 매물, 실매물 인증, 시세 정보 | 지도에서 지역 선택 → 매물 필터링 → 관심 매물 저장 → 방문 예약 | 월세·전세방 구할 때 |
| **피터팬의 좋은방** (앱) | 직거래 플랫폼 (중개수수료 절약), 커뮤니티 | 직거래 매물 검색 → 집주인과 직접 연락 → 계약 | 중개수수료 아끼고 싶을 때 |
| **네이버 부동산** (웹/앱) | 아파트·오피스텔·상가 매물, 단지 정보, 학군 | 네이버 앱 → 부동산 탭 → 단지·매물 검색 | 가장 많은 매물 물량 |

### 청약·분양

| 앱/서비스 | 핵심 기능 | 사용법 |
|:---------|:---------|:------|
| **LH 청약플러스** (앱) | LH 공공분양·임대 청약 전용 | LH 분양 공고 확인 → 자격 조회 → 앱에서 바로 청약 신청 |
| **청약홈** (웹/앱) | 모든 청약 통합 (공공+민간) | 청약 일정 확인 → 자격 점수 계산 → 온라인 청약 신청 |

### 경매·투자 분석

| 앱/서비스 | 핵심 기능 | 사용법 |
|:---------|:---------|:------|
| **경매마당** (앱) | 법원 경매 물건 조회, 권리분석 보조, 입찰 이력 | 지역·물건유형 필터 → 물건 상세(감정가·최저가·권리분석) → 입찰 준비 |
| **온비드** (웹) | 공매 물건 조회·전자입찰 | 물건 검색 → 공매 조건 확인 → 온라인 입찰 |
| **핀볼** (앱) | 부동산 투자 수익률 분석, 세금 계산, 대출 시뮬레이션 | 물건 정보 입력 → 수익률·세금·대출 시뮬레이션 결과 확인 |

:::tip 앱 활용 실전 순서
부동산을 구할 때 이 순서로 앱을 활용하면 효율적입니다:

1. **시세 파악**: 아파트 실거래가 / 호갱노노 → 해당 지역 시세 확인
2. **매물 검색**: 직방 / 다방 / 네이버 부동산 → 조건에 맞는 매물 목록 작성
3. **권리 확인**: 인터넷등기소 → 등기부등본 열람
4. **대출 확인**: KB부동산 → KB시세 확인 (대출 가능 금액 추정)
5. **투자 분석**: 리치고 / 핀볼 → 수익률·세금 시뮬레이션
6. **현장 방문**: 중개사 예약 → 실물 확인 → 계약
:::

## 시장 사이클 분석

부동산 시장은 **확장-과열-수축-침체** 사이클을 반복합니다.

| 단계 | 특징 | 지표 | 투자 전략 |
|:-----|:-----|:-----|:---------|
| **확장기** | 거래량 증가, 가격 완만 상승 | 거래량↑, 미분양↓ | 매수 적기 |
| **과열기** | 가격 급등, 투기 심화 | 매수수급지수 120+, PIR 급등 | 보유·차익 실현 |
| **수축기** | 거래량 감소, 가격 정체·하락 | 거래량↓, 미분양↑ | 관망 |
| **침체기** | 가격 하락, 매수세 실종 | 매수수급지수 70-, 급매 증가 | 저가 매수 기회 |
