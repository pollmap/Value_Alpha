# 임대사업 가이드

임대사업은 부동산을 보유하고 임대료를 받아 수익을 창출하는 사업입니다. 안정적인 현금흐름을 만들 수 있지만, 공실 리스크, 세입자 관리, 세금 부담 등을 종합적으로 고려해야 합니다.

---

## 임대 수익 구조

### 수익률 계산

$$
\text{총 수익률} = \frac{\text{연간 임대 수입}}{\text{총 투자금}} \times 100
$$

$$
\text{순 수익률} = \frac{\text{연간 임대 수입} - \text{연간 비용}}{\text{총 투자금}} \times 100
$$

### 수익률 계산 예시

| 항목 | 금액 |
|:-----|:-----|
| 매입가 | 5억원 |
| 대출 (LTV 60%) | 3억원 |
| 자기자본 | 2억원 |
| 보증금 | 5,000만원 |
| 월세 | 150만원 |
| **연간 임대 수입** | 1,800만원 |
| 대출 이자 (연 4%) | -1,200만원 |
| 재산세·종부세 | -100만원 |
| 관리·수선비 | -200만원 |
| **연간 순수입** | **300만원** |
| **자기자본 수익률** | 300만 / 2억 = **1.5%** |

:::info 임대 수익률의 현실
서울 아파트 임대 수익률은 2~3% 수준으로, 대출 이자를 차감하면 순수익률이 1~2%에 불과합니다. 임대 수익만으로는 수익성이 낮고, **시세 차익과 결합**해야 의미 있는 투자가 됩니다. 빌라, 오피스텔, 상가는 임대 수익률이 4~8%로 높지만 시세 차익 기대가 낮습니다.
:::

## 임대사업자 등록

### 등록 유형

| 유형 | 임대 기간 | 대상 | 세제 혜택 |
|:-----|:---------|:-----|:---------|
| **민간임대 (단기)** | 4년 | 모든 주택 | 제한적 |
| **민간임대 (장기)** | 10년 | 모든 주택 | 양도세·종부세 감면 |
| **공공지원 민간임대** | 10년 | 요건 충족 주택 | 최대 혜택 |

### 등록 절차

| 단계 | 내용 | 기관 |
|:-----|:-----|:-----|
| 1. 임대사업자 등록 | 지자체에 등록 신청 | 시·군·구청 |
| 2. 사업자등록 | 세무서에 사업자등록 | 관할 세무서 |
| 3. 임대차계약 신고 | 임대 조건 신고 | 렌트홈(renthome.go.kr) |
| 4. 표준임대차계약서 사용 | 법정 서식 사용 의무 | - |

### 의무사항

| 의무 | 내용 | 위반 시 |
|:-----|:-----|:--------|
| **임대 의무기간** | 4년 또는 10년 | 과태료·세제 혜택 환수 |
| **임대료 증액 제한** | 연 5% 이내 | 과태료 |
| **임대차계약 신고** | 계약 후 3개월 이내 | 과태료 |
| **임대조건 신고** | 변경 시 신고 | 과태료 |

## 임대 관리 실무

### 세입자 선정

| 확인 항목 | 방법 |
|:---------|:-----|
| 신분 확인 | 신분증 사본 보관 |
| 소득 수준 | 재직증명서, 소득증빙 (선택) |
| 신용 상태 | 신용등급 확인 (동의 필요) |
| 전적 확인 | 이전 임대인 연락 (선택) |

### 운영 비용

| 비용 항목 | 연간 추정 | 비고 |
|:---------|:---------|:-----|
| 재산세 | 공시가 기준 산정 | 매년 7·9월 |
| 종부세 | 합산 공시가 초과분 | 매년 12월 |
| 종합소득세 | 임대소득 기준 | 매년 5월 |
| 수선·유지비 | 매입가의 0.5~1% | 노후도에 따라 |
| 공실 손실 | 연 1~2개월분 | 지역·물건에 따라 |
| 관리비 (공용) | 물건에 따라 | 오피스텔·상가 해당 |

## 임대소득세 신고

| 구분 | 과세 방식 | 세율 |
|:-----|:---------|:-----|
| 연 2,000만원 이하 | 분리과세 14% 또는 종합과세 선택 | 14% or 6~45% |
| 연 2,000만원 초과 | 종합과세 | 6~45% |

### 분리과세 계산

| 항목 | 등록 임대사업자 | 미등록 |
|:-----|:-------------|:------|
| 필요경비율 | 60% | 50% |
| 기본공제 | 400만원 | 200만원 |

**예시**: 연 임대수입 1,800만원, 등록 임대사업자
- 소득금액: 1,800 - (1,800 × 60%) - 400만 = 320만원
- 세금: 320만 × 14% = 44.8만원

## 임대 유형별 전략

| 유형 | 타겟 | 장점 | 단점 |
|:-----|:-----|:-----|:-----|
| **아파트 전세** | 가족 세대 | 안정적, 관리 편함 | 수익률 낮음 |
| **아파트 월세** | 다양 | 현금흐름 안정 | 공실·관리 부담 |
| **원룸·오피스텔** | 1인 가구 | 수익률 높음 | 세입자 교체 빈번 |
| **상가** | 자영업자 | 높은 수익률 | 공실 리스크 큼 |
| **다가구 통임대** | 다수 세대 | 규모의 경제 | 초기 투자금 높음 |
| **쉐어하우스** | 청년·외국인 | 최고 수익률 | 관리 노동 집약적 |
