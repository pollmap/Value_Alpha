# 매매·전세·월세 거래 가이드

한국 부동산 거래는 크게 **매매**(소유권 이전), **전세**(보증금 기반 임대), **월세**(월 임대료 기반 임대)로 구분됩니다. 각 거래 유형마다 법적 절차, 세금, 리스크가 다르므로 거래 전 정확한 이해가 필수입니다.

---

## 매매 거래

### 매매 절차 (전체 타임라인)

| 단계 | 소요 기간 | 주요 행위 | 비용 |
|:-----|:---------|:---------|:-----|
| **1. 매물 탐색** | 1~4주 | 네이버부동산·직방·호갱노노 검색, 현장 방문 | 없음 |
| **2. 시세 조사** | 1~2주 | 실거래가 조회(국토부), KB시세, 주변 매물 비교 | 없음 |
| **3. 계약 체결** | 1일 | 매매계약서 작성, 계약금 지급 (보통 매매가 10%) | 계약금 |
| **4. 중도금** | 계약~잔금 사이 | 중도금 지급 (선택, 대형 거래 시) | 중도금 |
| **5. 대출 실행** | 잔금일 2~4주 전 | 금융기관 대출 심사·승인 | 대출 수수료 |
| **6. 잔금 지급** | 계약 후 1~3개월 | 잔금 지급 + 등기 이전 + 열쇠 인수 | 잔금 + 제비용 |
| **7. 소유권 이전등기** | 잔금일~60일 이내 | 법무사 통해 등기 신청 | 등기 비용 |
| **8. 취득세 납부** | 취득일로부터 60일 이내 | 관할 시·군·구청에 납부 | 취득세 |

### 매매 시 발생 비용

| 비용 항목 | 산정 기준 | 대략적 금액 (10억 아파트 기준) |
|:---------|:---------|:---------------------------|
| **취득세** | 1~3% (가액·주택수별 차등) | 1,000~3,000만원 |
| **중개수수료** | 0.4~0.7% (거래가액별 차등) | 400~700만원 |
| **법무사 비용** | 등기 대행 수수료 | 50~100만원 |
| **국민주택채권** | 매입 후 즉시 할인 매도 | 30~50만원 |
| **인지세** | 매매가 1억 초과 시 15만원 | 15만원 |

:::caution 취득세율 (2024년 기준)
| 주택 수 | 조정대상지역 | 비조정대상지역 |
|:--------|:-----------|:------------|
| 1주택 | 1~3% | 1~3% |
| 2주택 | 8% (중과) | 1~3% |
| 3주택 이상 | 12% (중과) | 8% |
| 법인 | 12% | 12% |

다주택자 취득세 중과는 정책에 따라 한시적으로 완화·강화될 수 있으므로 거래 시점의 최신 규정을 반드시 확인하세요.
:::

### 매매 계약서 핵심 확인 항목

| 항목 | 확인 내용 |
|:-----|:---------|
| **매도인 정보** | 등기부등본 소유자와 일치 여부, 신분증 확인 |
| **매매 대금** | 계약금·중도금·잔금 금액과 지급일 |
| **특약 사항** | 하자 보수, 이사 일정, 기존 임차인 승계 여부 |
| **기존 담보** | 잔금일까지 근저당 말소 조건 명시 |
| **위약금** | 계약 해제 시 위약금 규정 (보통 계약금 몰수/배액배상) |

## 전세 거래

전세는 한국 고유의 주거 제도로, 목돈(전세 보증금)을 맡기고 월 임대료 없이 거주하는 방식입니다.

### 전세의 구조

| 항목 | 내용 |
|:-----|:-----|
| **보증금** | 시세의 60~80% 수준, 계약 기간 동안 임대인이 보관 |
| **계약 기간** | 보통 2년 (주택임대차보호법 최소 보장) |
| **월 임대료** | 없음 (전세의 핵심 특징) |
| **갱신 청구권** | 2020년 도입, 1회 갱신 가능 (증액 5% 상한) |
| **수익 구조 (임대인)** | 전세금 운용 수익 or 갭투자 레버리지 |
| **수익 구조 (임차인)** | 월세 절약, 보증금 회수 시 원금 보전 |

### 전세 보증금 보호 3가지 요건

임차인이 전세 보증금을 법적으로 보호받으려면 **3가지 요건**을 모두 갖춰야 합니다.

| 요건 | 방법 | 효력 발생 |
|:-----|:-----|:---------|
| **1. 주택 인도 (입주)** | 실제로 해당 주택에 거주 시작 | 입주일 |
| **2. 전입신고** | 주민센터에 전입신고 | 전입신고 다음날 0시 |
| **3. 확정일자** | 주민센터 또는 온라인(정부24)에서 받음 | 확정일자 부여일 |

:::danger 전세 사기 예방 핵심
- **대항력** = 입주 + 전입신고 → 제3자에 대한 보증금 주장 가능
- **우선변제권** = 대항력 + 확정일자 → 경매 시 보증금 우선 회수
- 반드시 **잔금 지급일 당일**에 전입신고와 확정일자를 동시에 처리하세요
- 등기부등본은 **잔금일 아침**에 재발급하여 갑구·을구 변동 확인
:::

### 전세 안전 진단

| 점검 항목 | 안전 기준 | 위험 기준 |
|:---------|:---------|:---------|
| **전세가율** (전세가/매매가) | 60% 이하 | 80% 이상 (깡통전세 위험) |
| **근저당 + 전세금** | 매매가의 70% 이하 | 매매가 초과 (역전세 위험) |
| **임대인 보유 주택 수** | 소수 | 다수 (갭투자자 위험) |
| **전세보증보험 가입** | 가능 | 가입 불가 (HUG, SGI 거부) |
| **집주인 세금 체납** | 없음 | 있음 (경매 위험) |

### 전세보증금 보호 보험

| 보험 | 운영기관 | 보장 한도 | 보험료 |
|:-----|:--------|:---------|:------|
| **전세보증금반환보증** | HUG (주택도시보증공사) | 수도권 7억, 기타 5억 | 연 0.115~0.154% |
| **전세금보장신용보험** | SGI서울보증 | 보증금의 80% | 연 0.183~0.347% |
| **전세자금보증** | HF (한국주택금융공사) | 수도권 7억, 기타 5억 | 연 0.05~0.15% |

## 월세 거래

월세는 보증금 + 월 임대료를 지불하는 가장 일반적인 임대차 형태입니다.

### 월세 유형

| 유형 | 보증금 | 월세 | 특징 |
|:-----|:------|:-----|:-----|
| **순수 월세** | 100~500만원 | 높음 | 보증금 부담 최소화 |
| **반전세** | 높음 (전세의 50~70%) | 낮음 | 전세 전환 추세에서 증가 |
| **사글세** | 없음 | 선불 | 거의 사라진 형태 |

### 전월세 전환율

전세와 월세 간 전환 시 사용하는 비율입니다.

$$
\text{월세} = \frac{(\text{전세금} - \text{보증금}) \times \text{전환율}}{12}
$$

| 항목 | 기준 |
|:-----|:-----|
| **법정 전환율 상한** | 기준금리 + 2% (2024년 기준 약 5.5%) |
| **시장 전환율** | 지역·물건에 따라 4~8% |
| **계산 예시** | 전세 3억 → 보증금 1억 + 월세 환산: (3억-1억) × 5%/12 = 약 83만원 |

### 월세 세액공제

| 요건 | 내용 |
|:-----|:-----|
| **대상** | 총급여 7,000만원 이하 근로자 (종합소득 6,000만원 이하) |
| **공제율** | 총급여 5,500만원 이하: 17%, 초과: 15% |
| **한도** | 연 1,000만원 |
| **절세 효과** | 월세 70만원 × 12개월 = 840만원 → 최대 142.8만원 세액공제 |

## 임대차 3법 핵심 정리

2020년 시행된 임대차 3법은 임차인 보호를 강화한 법률 개정입니다.

| 법률 | 핵심 내용 | 영향 |
|:-----|:---------|:-----|
| **계약갱신청구권** | 임차인이 1회 갱신 요구 가능 (총 4년 거주 보장) | 임대인 갱신 거절 사유 제한 |
| **전월세상한제** | 갱신 시 보증금·월세 인상률 5% 상한 | 기존 전세 가격 고정 효과 |
| **전월세신고제** | 보증금 6,000만원 또는 월세 30만원 초과 시 신고 의무 | 시장 투명성 제고 |

## 거래 유형별 비교

| 항목 | 매매 | 전세 | 월세 |
|:-----|:-----|:-----|:-----|
| **초기 자금** | 매우 높음 | 높음 | 낮음 |
| **월 지출** | 대출 이자 + 관리비 | 관리비만 | 월세 + 관리비 |
| **자산 형성** | O (시세차익 가능) | △ (원금 보전) | X |
| **리스크** | 가격 하락, 금리 상승 | 보증금 미반환 | 없음 (소액 손실만) |
| **세금 혜택** | 1주택 비과세 가능 | 없음 | 월세 세액공제 |
| **계약 기간** | 영구 | 2년 (갱신 시 4년) | 1~2년 |
| **적합 대상** | 장기 거주 + 투자 | 목돈 있는 무주택자 | 단기 거주, 사회초년생 |
