# 부동산 가치평가 개요

## 부동산 투자의 특징

### 다른 자산과의 비교

| 특성 | 부동산 | 주식 | 채권 |
|------|--------|------|------|
| 유동성 | 낮음 | 높음 | 중간 |
| 레버리지 | 70-80% 가능 | 제한적 | 제한적 |
| 소득 안정성 | 높음 (임대) | 변동적 | 높음 (이자) |
| 인플레이션 헷지 | 우수 | 중간 | 취약 |
| 관리 필요성 | 높음 | 낮음 | 없음 |
| 진입 장벽 | 높음 | 낮음 | 중간 |

### 부동산 수익 구조

```
총 수익 = 소득 수익 + 자본 수익

소득 수익 (Income Return):
├── 임대료 수입
├── 주차장 수입
└── 기타 부대 수입

자본 수익 (Capital Return):
└── 매각차익 (시세 상승)
```

## 핵심 가치평가 방법

### 3가지 접근법

```
1. 원가법 (Cost Approach)
   └── 토지가 + 건물 재조달원가 - 감가상각

2. 비교법 (Sales Comparison)
   └── 유사 거래 사례 비교

3. 수익법 (Income Approach)
   └── 미래 소득의 현재가치
```

## 수익 환원법 (Income Approach)

### 직접 환원법

```
부동산 가치 = NOI / Cap Rate

NOI (Net Operating Income) = 순영업소득
Cap Rate = 자본환원율
```

**예시:**
```
연간 NOI: 1억원
Cap Rate: 5%

부동산 가치 = 1억 / 0.05 = 20억원
```

### NOI 계산

```
총 임대료 수입 (GPI)
    │
    ├── (-) 공실손실
    │
    └── = 유효총수입 (EGI)
            │
            ├── (-) 운영비용
            │   ├── 관리비
            │   ├── 수선유지비
            │   ├── 보험료
            │   └── 재산세
            │
            └── = 순영업소득 (NOI)
```

### Cap Rate (자본환원율)

```
Cap Rate = NOI / 매매가격

낮은 Cap Rate (3-4%):
- 안정적 지역
- 프라임 자산
- 낮은 위험

높은 Cap Rate (7-10%):
- 위험한 지역
- 노후 자산
- 높은 위험
```

## 현금흐름 분석

### BTCF (Before-Tax Cash Flow)

```
NOI
    │
    ├── (-) 부채원리금 상환 (Debt Service)
    │
    └── = BTCF (세전현금흐름)
```

### 주요 수익률 지표

| 지표 | 공식 | 의미 |
|------|------|------|
| **Cap Rate** | NOI / 가격 | 전체 수익률 |
| **Cash-on-Cash** | BTCF / 자기자본 | 현금 수익률 |
| **ROE** | 순이익 / 자기자본 | 자기자본이익률 |
| **IRR** | 투자기간 내부수익률 | 종합 수익률 |

### 레버리지 효과

```
무레버리지:
가격: 10억, 전액 자기자본
NOI: 5천만, Cap Rate: 5%
수익률 = 5%

레버리지 (LTV 70%):
가격: 10억, 자기자본 3억, 대출 7억 (금리 3%)
NOI: 5천만, 이자: 2,100만
BTCF: 2,900만
Cash-on-Cash = 2,900만/3억 = 9.7%

→ 레버리지로 수익률 향상
```

## 부동산 유형별 분석

### 주거용

```
분석 포인트:
□ 입지 (교통, 학군, 편의시설)
□ 공급/수요 동향
□ 전세가율
□ 재건축/재개발 가능성
□ 인구 추이
```

### 상업용 (오피스)

```
분석 포인트:
□ 공실률
□ 임대료 추이
□ 주요 임차인 신용도
□ 리스 만기 분포
□ 주변 개발 계획
```

### 물류/산업용

```
분석 포인트:
□ 물류 접근성
□ 천장고/하중
□ 주요 임차인 (3PL, 이커머스)
□ 물류 수요 추세
□ 스펙 (냉장/냉동, 자동화)
```

### 리테일

```
분석 포인트:
□ 유동인구
□ 앵커테넌트 유무
□ 온라인 쇼핑 영향
□ 임대차 조건 (% rent)
□ 상권 분석
```

## 투자 체크리스트

### 실사 (Due Diligence)

```
법률 검토:
□ 등기부등본 확인
□ 토지이용계획 확인
□ 건축물대장 확인
□ 임대차계약서 검토
□ 하자/분쟁 이력

재무 검토:
□ 임대료 수준 적정성
□ 공실률 분석
□ 운영비 검증
□ 자본적 지출 예상
□ 임차인 신용 분석

물리적 검토:
□ 건물 상태 점검
□ 기계/전기 설비
□ 환경 오염 여부
□ 수선 필요 사항
```

### 투자 의사결정

```
투자 기준 예시:
□ Cap Rate > 목표 수익률 - 부채비용
□ LTV < 70%
□ DSCR > 1.2x
□ Cash-on-Cash > 8%
□ IRR > 15% (5년 보유 기준)
```

## REITs 분석

### 리츠 가치평가

```
주요 지표:
1. FFO (Funds From Operations)
   = 순이익 + 감가상각 - 매각차익

2. AFFO (Adjusted FFO)
   = FFO - 유지보수 자본지출

3. P/FFO
   = 주가 / 주당 FFO

4. NAV
   = 자산가치 - 부채
```

### 리츠 투자 체크리스트

```
□ 배당수익률 vs 국채 금리
□ FFO 성장률
□ 부채 비율
□ 자산 포트폴리오 품질
□ 경영진 Track Record
□ NAV 대비 할인/할증
```
