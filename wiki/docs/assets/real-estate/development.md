# 재건축·재개발·리모델링

재건축과 재개발은 노후 부동산의 가치를 극대화하는 정비사업입니다. 완료 시 신축 아파트를 시세 대비 저렴하게 취득하는 효과가 있어 투자 수요가 많지만, 사업 기간이 10~20년에 달하고 분담금 부담, 사업 지연 리스크가 큽니다.

---

## 재건축 vs 재개발 vs 리모델링

| 항목 | 재건축 | 재개발 | 리모델링 |
|:-----|:------|:------|:--------|
| **대상** | 아파트·연립주택 | 단독·다세대 밀집 지역 | 기존 건물 (15년 이상) |
| **근거법** | 도시정비법 | 도시정비법 | 주택법 |
| **정비 방식** | 기존 건물 철거 → 신축 | 도시기반시설 + 건물 신축 | 기존 골조 유지 + 증축 |
| **사업 주체** | 조합 + 시공사 | 조합 + 시공사 | 조합 + 시공사 |
| **초과이익환수** | 적용 | 미적용 | 미적용 |
| **안전진단** | 필요 (D·E등급) | 불필요 | 안전성 검토 |
| **용적률 증가** | 대폭 증가 가능 | 대폭 증가 가능 | 제한적 (기존 + 최대 30%) |
| **소요 기간** | 10~20년 | 10~15년 | 5~10년 |

## 재건축 진행 단계

| 단계 | 내용 | 소요 기간 |
|:-----|:-----|:---------|
| **1. 안전진단** | 구조 안전성 평가 (D·E등급 시 추진 가능) | 6~12개월 |
| **2. 정비구역 지정** | 지자체 도시계획위 심의 | 1~2년 |
| **3. 추진위원회 설립** | 토지 소유자 과반수 동의 | 6~12개월 |
| **4. 조합 설립** | 토지·건물 소유자 75% 이상 동의 | 1~2년 |
| **5. 사업시행계획 인가** | 건축 설계, 분양 계획 확정 | 1~2년 |
| **6. 관리처분계획 인가** | 조합원 분담금 확정, 이주 | 1~2년 |
| **7. 이주·철거** | 세입자 이주, 기존 건물 철거 | 6~12개월 |
| **8. 착공** | 신축 공사 시작 | - |
| **9. 준공·입주** | 완공 후 조합원 입주 | 공사 2~4년 |
| **10. 조합 해산** | 정산 후 조합 해산 | 입주 후 1~3년 |

## 재건축 투자의 수익 구조

### 핵심 개념

| 용어 | 의미 |
|:-----|:-----|
| **권리가액** | 조합원이 기존에 보유한 부동산의 평가 금액 |
| **분담금** | 신축 아파트 분양가 - 권리가액 (조합원이 추가 납부하는 금액) |
| **비례율** | 총 사업 수입 / 총 사업 비용 × 100 (100% 이상이면 사업성 양호) |
| **일반분양** | 조합원 외 일반에게 분양 (사업 수입의 핵심) |
| **프리미엄** | 조합원 입주권의 시세 - 실제 투자금 |

### 분담금 계산 예시

| 항목 | 금액 |
|:-----|:-----|
| 기존 아파트 매입가 | 12억원 |
| 권리가액 (감정평가) | 10억원 |
| 신축 아파트 조합원 분양가 | 15억원 |
| **분담금** | 15억 - 10억 = **5억원** |
| **총 투자금** | 12억 + 5억 = **17억원** |
| 신축 아파트 시세 (완공 후) | 22억원 |
| **예상 수익** | 22억 - 17억 = **5억원** |

:::caution 재건축 투자 리스크
- **분담금 증가**: 공사비 상승, 일반분양 미달 시 분담금이 수억 원 추가될 수 있음
- **사업 지연**: 안전진단 불합격, 조합 내분, 인허가 지연으로 10년 이상 지연 가능
- **초과이익환수제**: 재건축 이익의 최대 50%를 환수 (3,000만원 초과분)
- **매몰 비용**: 사업 무산 시 이미 납부한 조합 비용 회수 불가
:::

## 재개발 진행 단계

재개발은 재건축과 유사하나, 도시기반시설(도로, 공원 등) 정비가 포함됩니다.

| 재개발 고유 특성 | 내용 |
|:-------------|:-----|
| **도시기반시설** | 도로 확장, 공원 조성 등 포함 |
| **안전진단 불요** | 재건축과 달리 안전진단 불필요 |
| **토지 보상** | 도로·공원 부지 수용 보상 |
| **세입자 보상** | 주거이전비, 이사비 등 |
| **임대주택** | 일정 비율 임대주택 의무 건설 |

## 리모델링

리모델링은 기존 골조를 유지하면서 증축·수직증축하는 방식입니다.

| 항목 | 내용 |
|:-----|:-----|
| **대상** | 준공 후 15년 이상 경과 공동주택 |
| **증축 범위** | 기존 세대수의 최대 15%, 수직증축 최대 3개 층 |
| **동의 요건** | 구분소유자 2/3 이상 + 토지면적 2/3 이상 |
| **장점** | 재건축보다 빠름, 초과이익환수 미적용 |
| **단점** | 평면 변경 제한, 증축 규모 한계 |

## 정비사업 투자 비교

| 항목 | 재건축 | 재개발 | 리모델링 |
|:-----|:------|:------|:--------|
| **기대 수익** | 높음 | 높음 | 중간 |
| **소요 기간** | 10~20년 | 10~15년 | 5~10년 |
| **분담금** | 높음 | 중간~높음 | 중간 |
| **리스크** | 높음 | 높음 | 중간 |
| **진입 장벽** | 높음 (매입가 높음) | 중간 | 중간 |
| **적합 투자자** | 장기 투자, 고액 자산가 | 장기 투자 | 중기 투자 |
