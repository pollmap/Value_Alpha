# 금융권 직무 종류 및 특성

금융권은 섹터별로, 그리고 같은 섹터 내에서도 직무에 따라 업무 성격이 크게 다릅니다. 이 페이지에서는 각 섹터의 주요 직무를 상세히 분석합니다. 직무 선택은 취업 준비의 방향을 결정짓는 가장 중요한 요소이므로, 충분한 시간을 들여 자신에게 맞는 직무를 탐색하시기 바랍니다.

:::tip 직무 선택 시 고려사항
직무를 선택할 때는 (1) 본인의 성향과 강점, (2) 진입 가능성, (3) 장기적 커리어 전망, (4) 워라밸 선호도를 종합적으로 고려해야 합니다. 화려해 보이는 직무가 반드시 본인에게 맞는 것은 아닙니다.
:::

## 1. 은행 직무

### 1-1. 기업금융 (Corporate Banking)

| 항목 | 내용 |
|------|------|
| **업무내용** | 기업 고객 대상 대출, 여신 심사, 프로젝트파이낸싱(PF), 신디케이트론 등의 기업금융 업무를 수행합니다. 기업의 재무상태를 분석하고, 적절한 금융 솔루션을 제안하며, 여신 한도를 설정합니다. 대기업/중소기업 담당에 따라 업무 규모와 성격이 달라집니다. |
| **필요역량** | 재무제표 분석 능력, 산업/기업 분석 능력, 여신 심사 관련 지식, 커뮤니케이션 능력, 엑셀 활용 능력 |
| **적합한 성향** | 숫자에 강하면서도 대인관계가 원활한 사람, 기업의 사업 모델과 산업 동향에 관심이 있는 사람, 꼼꼼하고 리스크를 냉정하게 판단할 수 있는 사람 |
| **커리어패스** | 주니어 여신심사역 → 시니어 심사역 → 팀장 → 본부장 / 또는 증권사 IB, PE 등으로 이직 가능 |
| **현실적 진입장벽** | 은행 공채로 입사 후 배치받는 경우가 일반적. 본인이 원한다고 바로 기업금융 부서에 배치되는 것은 아닙니다. 영업점에서 일정 기간 근무 후 본부로 이동하는 패턴이 흔합니다. |

### 1-2. 여신심사

| 항목 | 내용 |
|------|------|
| **업무내용** | 개인 및 기업 대출 신청에 대한 심사를 수행합니다. 차주(대출 신청인)의 상환 능력, 담보 가치, 신용도를 종합적으로 판단하여 대출 승인 여부와 조건을 결정합니다. |
| **필요역량** | 재무분석, 신용분석, 담보평가 지식, 관련 법규 이해, 판단력 |
| **적합한 성향** | 신중하고 분석적인 성격, 규정과 기준을 정확히 따르는 성향, 리스크에 민감한 사람 |
| **커리어패스** | 심사역 → 수석심사역 → 심사부서 관리자 / 여신심사역 자격증 취득 시 전문성 인정 |
| **현실적 진입장벽** | 은행 입사 후 내부 배치를 통해 접근. 여신심사역 자격증은 실무 경험 이후 취득하는 경우가 많음 |

### 1-3. 외환딜링

| 항목 | 내용 |
|------|------|
| **업무내용** | 외환시장에서 통화를 매매하여 수익을 창출하거나, 고객의 환 헤지 수요를 충족시킵니다. 환율 동향 분석, 포지션 관리, 리스크 모니터링 등을 수행합니다. |
| **필요역량** | 거시경제/국제금융 이해, 환율 분석, 빠른 의사결정 능력, 영어 커뮤니케이션, 스트레스 관리 |
| **적합한 성향** | 시장 변동에 빠르게 반응할 수 있는 사람, 수치에 강하고 집중력이 높은 사람, 심리적 압박감을 잘 관리하는 사람 |
| **커리어패스** | 주니어 딜러 → 시니어 딜러 → 딜링룸 매니저 → 트레져리(Treasury) 부서장 |
| **현실적 진입장벽** | 채용 인원이 매우 적음. 외환전문역 자격증 우대. 은행 내부에서도 경쟁이 있는 보직. 외환 관련 인턴 경험이 도움됨 |

### 1-4. PB (Private Banking)

| 항목 | 내용 |
|------|------|
| **업무내용** | 고액자산가(HNWI) 고객을 대상으로 종합 자산관리 서비스를 제공합니다. 투자, 세무, 부동산, 상속/증여 등 다양한 영역의 자문을 수행하며, 고객 포트폴리오를 설계하고 관리합니다. |
| **필요역량** | 투자/금융상품 지식, 세무/부동산/상속 기초 지식, 대인관계/영업 능력, 신뢰 구축 능력, 시장 동향 파악 |
| **적합한 성향** | 사람을 만나고 관계를 형성하는 것을 즐기는 사람, 서비스 마인드가 강한 사람, 지속적으로 학습하고 다양한 분야의 지식을 넓힐 수 있는 사람 |
| **커리어패스** | 영업점 경험 → PB센터 배치 → 시니어 PB → PB센터장 / 또는 증권사 WM으로 이직 |
| **현실적 진입장벽** | 대부분 은행 내부 이동을 통해 PB 부서로 배치됨. 신입으로 바로 PB를 하기보다는 영업점에서 경험을 쌓은 후 이동하는 경우가 일반적. AFPK, CFP 등 자격증 보유 시 유리 |

### 1-5. 리스크관리

| 항목 | 내용 |
|------|------|
| **업무내용** | 은행의 신용리스크, 시장리스크, 운영리스크, 유동성리스크 등을 측정, 모니터링, 관리합니다. 리스크 모델 개발 및 검증, 스트레스 테스트, 바젤 규제 대응 등의 업무를 수행합니다. |
| **필요역량** | 통계/수학, 리스크 모델링, 금융공학 기초, 바젤 규제 이해, 프로그래밍(Python, SAS 등), 데이터 분석 |
| **적합한 성향** | 정량적 분석에 강한 사람, 꼼꼼하고 체계적인 사람, 규제와 기준을 정확히 이해하고 적용할 수 있는 사람 |
| **커리어패스** | 리스크관리 주니어 → 시니어 리스크 매니저 → 리스크관리부장 → CRO(Chief Risk Officer) |
| **현실적 진입장벽** | 은행 공채 후 내부 배치 또는 리스크 관련 경력직 채용. 통계/수학/금융공학 전공자 우대. FRM 자격증 보유 시 유리 |

### 1-6. IT/디지털

| 항목 | 내용 |
|------|------|
| **업무내용** | 은행의 디지털 트랜스포메이션, 모바일뱅킹 앱 개발, 데이터 분석 플랫폼 구축, AI/ML 모델 개발, 정보보안, 인프라 관리 등을 수행합니다. |
| **필요역량** | 소프트웨어 개발(Java, Python 등), 데이터베이스, 클라우드, 정보보안, 데이터 분석 등 (세부 직무에 따라 다름) |
| **적합한 성향** | 기술적 문제 해결을 즐기는 사람, 금융과 IT의 교차 영역에 관심이 있는 사람 |
| **커리어패스** | IT 개발자/데이터 분석가 → 팀장 → IT 부문 관리자 → CDO/CIO |
| **현실적 진입장벽** | 별도 IT 직군 채용이 증가 추세. 컴퓨터공학/데이터사이언스 전공 우대. 금융 도메인 지식이 있으면 경쟁력 있음 |

### 1-7. 준법감시 (Compliance)

| 항목 | 내용 |
|------|------|
| **업무내용** | 은행의 모든 업무가 법규와 내부 규정을 준수하는지 감시합니다. 자금세탁방지(AML), 내부통제, 금융소비자보호, 규제 변화 모니터링 등을 수행합니다. |
| **필요역량** | 금융관련 법규 이해(은행법, 자본시장법 등), 분석력, 커뮤니케이션, 영어(국제규제 관련) |
| **적합한 성향** | 법과 규정에 대한 이해력이 뛰어난 사람, 원칙적이고 독립적 판단이 가능한 사람 |
| **커리어패스** | 준법감시 담당 → 시니어 → 준법감시인 → CCO(Chief Compliance Officer) |
| **현실적 진입장벽** | 법학 전공자 또는 법률 관련 자격 보유자 우대. 은행 내부에서 이동하거나 별도 채용. 최근 AML/컴플라이언스 인력 수요 증가 추세 |

## 2. 증권사 직무

### 2-1. IB (투자은행) - 기업금융/M&A/IPO/유상증자

| 항목 | 내용 |
|------|------|
| **업무내용** | 기업의 자금조달(IPO, 유상증자, 회사채 발행), M&A 자문, 구조화금융 등을 수행합니다. 딜(거래) 발굴부터 실행까지 전 과정에 참여하며, 기업 밸류에이션, 법률/회계 실사 조율, 투자설명서 작성, 규제기관 대응 등 다양한 업무를 합니다. |
| **필요역량** | 회계/재무 분석(필수), 밸류에이션(DCF, Comparable 등), 엑셀 모델링(필수), 파워포인트 프레젠테이션, 자본시장법 등 관련 법규 이해, 영어(해외 딜 시), 체력(장시간 근무) |
| **적합한 성향** | 지적 호기심이 강하고 학습 속도가 빠른 사람, 높은 업무 강도를 견딜 수 있는 사람, 디테일에 강하면서도 큰 그림을 볼 수 있는 사람, 동시다발적 업무 처리가 가능한 사람 |
| **커리어패스** | 애널리스트(1~3년) → 어소시에이트(3~5년) → VP(5~8년) → 시니어VP/디렉터 → MD(Managing Director). PE, 사업회사 전략기획, 벤처캐피탈 등으로 이직하는 경우도 흔함 |
| **현실적 진입장벽** | **금융권 최고 수준의 진입장벽.** SKY 및 상위권 대학 출신이 대다수를 차지합니다. 학벌 필터가 현실적으로 존재하며, CFA/CPA/변호사 자격 보유자도 많습니다. 관련 인턴 경험이 거의 필수적이며, 회계법인/로펌 출신 경력직 이직도 일반적 경로입니다. 지원자 대비 채용 인원이 매우 적어 경쟁이 극도로 치열합니다. |

### 2-2. 리서치 (애널리스트)

| 항목 | 내용 |
|------|------|
| **업무내용** | 특정 산업 및 기업을 분석하여 투자 리포트를 작성합니다. 기업 탐방, 실적 분석, 밸류에이션, 투자의견 제시 등의 업무를 수행합니다. 섹터별로 전담 애널리스트가 배정되며(반도체, 자동차, 제약/바이오, 금융 등), 매크로/퀀트/채권 등 비섹터 애널리스트도 있습니다. |
| **필요역량** | 해당 산업에 대한 깊은 이해, 재무분석/밸류에이션, 글쓰기(리포트 작성), 프레젠테이션, 영어(해외 기업/리서치 자료 참조), 엑셀 모델링 |
| **적합한 성향** | 특정 산업이나 기업에 대해 깊이 파고드는 것을 즐기는 사람, 논리적 글쓰기에 강한 사람, 자신의 의견을 명확히 표현할 수 있는 사람, 새벽 출근과 실적 시즌의 강도 높은 업무를 감당할 수 있는 사람 |
| **커리어패스** | 리서치 어시스턴트(RA) → 주니어 애널리스트 → 시니어 애널리스트 → 헤드 애널리스트. 자산운용사 펀드매니저, 기업 IR, 언론 등으로 이직 가능 |
| **현실적 진입장벽** | IB에 준하는 높은 진입장벽. RA(리서치 어시스턴트)로 시작하는 것이 일반적이며, RA 채용도 경쟁이 치열합니다. 상위권 대학 출신이 많고, 금융투자분석사(CIFA), CFA 등 자격증 보유가 유리합니다. 해당 산업에 대한 전문 지식이 있으면 경쟁력이 높아집니다. |

### 2-3. S&T (세일즈 & 트레이딩)

| 항목 | 내용 |
|------|------|
| **업무내용** | **세일즈**: 기관투자자(연기금, 운용사, 보험사 등)를 대상으로 투자 아이디어를 제공하고 주문을 수탁합니다. **트레이딩**: 자기매매(Prop Trading) 또는 고객 주문 체결을 위한 시장 거래를 수행합니다. 주식, 채권, 파생상품 등 상품군에 따라 세분화됩니다. |
| **필요역량** | 시장 분석(매크로/미시), 금융상품 이해(주식/채권/파생), 빠른 판단력과 실행력, 대인관계(세일즈), 수학적/통계적 사고(트레이딩), 프로그래밍(퀀트 트레이딩) |
| **적합한 성향** | 시장 움직임에 민감하고 빠르게 반응하는 사람, 결과에 대한 책임을 질 수 있는 사람, 세일즈의 경우 외향적이고 관계 구축에 능한 사람 |
| **커리어패스** | 주니어 세일즈/트레이더 → 시니어 → 데스크 헤드 → S&T 총괄. 운용사, 헤지펀드 등으로 이직 가능 |

| **적합한 성향** | 증권사 리서치 애널리스트와 유사하나, 실제 투자 의사결정에 더 직접적으로 관여하므로 투자 성과에 대한 책임감이 중요 |
| **커리어패스** | 리서치 애널리스트 → 시니어 애널리스트 → 펀드매니저 |
| **현실적 진입장벽** | 증권사 리서치 출신이 이직하는 경우가 많음. 신입 채용 시에는 상위권 대학 + CFA/투자자산운용사 자격증 보유자 선호 |

### 4-3. 트레이더

| 항목 | 내용 |
|------|------|
| **업무내용** | 펀드매니저의 투자 의사결정에 따라 실제 시장에서 매매를 집행합니다. 최적의 가격에 체결하기 위한 실행 전략을 수립하고, 시장 충격을 최소화하면서 거래를 완료합니다. |
| **필요역량** | 시장 미시구조 이해, 주문 집행 최적화, 빠른 판단력, 관련 시스템 활용 능력 |
| **적합한 성향** | 집중력이 높고 빠른 의사결정이 가능한 사람, 시장 상황을 실시간으로 모니터링하는 것에 적합한 사람 |
| **커리어패스** | 주니어 트레이더 → 시니어 트레이더 → 트레이딩 데스크 헤드 |
| **현실적 진입장벽** | 채용 인원이 매우 적음. 금융공학/수학/컴퓨터과학 전공자 선호. 프로그래밍 능력이 점점 더 중시됨 |

### 4-4~4-6. 리스크관리 / 컴플라이언스 / 마케팅

| 직무 | 업무 개요 | 진입장벽 |
|------|----------|----------|
| **리스크관리** | 펀드의 투자 한도, 포트폴리오 리스크 모니터링, 규제 자본 관리 | 수학/통계 배경 + FRM 자격 우대 |
| **컴플라이언스** | 자본시장법, 펀드 운용 규정 준수 감시, 내부통제, 불공정거래 방지 | 법학 배경 또는 금융법규 이해도 필요 |
| **마케팅** | 펀드 판매 채널(은행, 증권사) 관리, 판매사 교육, 펀드 프로모션 | 커뮤니케이션/영업 역량 중심, 상대적으로 진입장벽 낮음 |

## 5. VC/PE 직무

### 5-1. 딜소싱 (Deal Sourcing)

| 항목 | 내용 |
|------|------|
| **업무내용** | 투자 대상 기업을 발굴합니다. 산업 리서치, 네트워킹, 딜 파이프라인 관리, 초기 스크리닝 등을 수행합니다. VC의 경우 스타트업 생태계와의 접점이 중요하며, PE의 경우 IB/회계법인/변호사 등의 네트워크를 통해 딜이 유입되는 경우가 많습니다. |
| **필요역량** | 산업 분석, 네트워킹, 사업 모델 평가, 재무 분석 기초 |
| **적합한 성향** | 다양한 산업과 기업에 호기심이 강한 사람, 네트워킹에 능하고 인맥을 관리할 수 있는 사람 |
| **커리어패스** | 어소시에이트 → 시니어 어소시에이트 → VP → 디렉터 → 파트너 |
| **현실적 진입장벽** | VC: 다양한 배경 가능하나 스타트업 생태계 이해 필수. PE: 대부분 IB/회계법인/컨설팅 2~3년 경력자를 채용하며, 신입 채용은 매우 드묾 |

### 5-2. 실사 (Due Diligence)

| 항목 | 내용 |
|------|------|
| **업무내용** | 투자 대상 기업에 대한 깊이 있는 조사를 수행합니다. 재무 실사(FDD), 법률 실사(LDD), 사업 실사(BDD), 세무 실사(TDD) 등을 직접 수행하거나 외부 전문가(회계법인, 로펌 등)를 관리합니다. |
| **필요역량** | 재무분석(회계/재무제표 분석 필수), 법률 기초 지식, 산업 분석, 프로젝트 관리, 엑셀 모델링 |
| **적합한 성향** | 디테일에 강하고 빈틈없이 조사할 수 있는 사람, 방대한 양의 자료를 체계적으로 정리할 수 있는 사람, 시간 압박 하에서도 정확한 분석을 수행할 수 있는 사람 |
| **커리어패스** | VC/PE 내 딜 팀에서 전반적으로 수행 (별도 직무라기보다 핵심 업무의 일부) |
| **현실적 진입장벽** | CPA/CFA 자격증 보유가 매우 유리. 회계법인 FAS/FDD 부서 출신이 PE로 이직하는 경로가 흔함 |

### 5-3. 포트폴리오 관리

| 항목 | 내용 |
|------|------|
| **업무내용** | 투자 이후 피투자 기업의 가치를 제고하기 위한 활동을 수행합니다. 이사회 참여, 경영 지원(재무, 인사, 전략), 후속 투자 유치 지원, 성과 모니터링 등을 합니다. |
| **필요역량** | 경영 전략, 재무관리, 리더십, 커뮤니케이션, 해당 산업에 대한 깊은 이해 |
| **적합한 성향** | 기업가적 마인드셋을 가진 사람, 다양한 이해관계자와 소통할 수 있는 사람 |
| **커리어패스** | VC/PE 시니어 직급에서 주로 담당. 피투자 기업의 경영진으로 이동하기도 함 |
| **현실적 진입장벽** | 주니어보다 시니어 레벨에서 주로 수행하는 업무. 컨설팅/사업회사 경영 경험이 도움됨 |

### 5-4. 밸류에이션

| 항목 | 내용 |
|------|------|
| **업무내용** | 투자 대상 기업의 가치를 산정합니다. DCF, Comparable Analysis, Precedent Transaction, LBO Model(PE의 경우) 등의 방법론을 활용합니다. |
| **필요역량** | 재무모델링(필수), 밸류에이션 방법론, 엑셀(고급), 산업 이해 |
| **적합한 성향** | 숫자와 모델링에 강한 사람, 가정(assumption)의 변화에 따른 민감도를 직관적으로 이해할 수 있는 사람 |
| **커리어패스** | 딜 팀 내에서 핵심 역량으로 기능 |
| **현실적 진입장벽** | CFA, CPA 자격증이 도움됨. IB 또는 회계법인 밸류에이션 부서 경험이 매우 유리 |

### 5-5. EXIT 전략

| 항목 | 내용 |
|------|------|
| **업무내용** | 투자를 회수(EXIT)하기 위한 전략을 수립합니다. IPO, 세컨더리 매각, 전략적 매각(Trade Sale), 경영진 매수(MBO) 등의 방법을 검토하고 실행합니다. |
| **필요역량** | M&A 프로세스 이해, 자본시장 이해, 협상력, 밸류에이션, 법률 기초 지식 |
| **적합한 성향** | 전략적 사고와 협상에 능한 사람, 시장 타이밍에 대한 감각이 있는 사람 |
| **커리어패스** | 시니어 레벨에서 주도하는 업무. 성공적인 EXIT 트랙레코드가 커리어에 결정적 |
| **현실적 진입장벽** | 시니어 직급의 업무이므로, 주니어 시절에는 보조적으로 참여하며 경험을 축적 |

| **현실적 진입장벽** | 법학 전공 또는 금융기관 컴플라이언스 경험 보유자 선호. 핀테크 산업이 성장하면서 수요 증가 추세 |

## 8. 정책금융/공공기관 직무

### 8-1. 정책기획

| 항목 | 내용 |
|------|------|
| **업무내용** | 금융 정책 수립 및 시행을 위한 조사/분석/기획 업무를 수행합니다. 산업 정책 금융, 수출지원금융, 중소기업 지원 정책 등을 기획합니다. |
| **필요역량** | 거시경제 분석, 정책 분석, 보고서 작성, 통계 분석, 영어(국제 비교 등) |
| **적합한 성향** | 거시적 관점에서 사고할 수 있는 사람, 공익적 가치를 중시하는 사람, 체계적으로 보고서를 작성하는 데 강한 사람 |
| **커리어패스** | 주니어 → 과장 → 부장 → 실장 → 임원. 공공기관 특유의 직급 체계 |
| **현실적 진입장벽** | NCS 기반 필기시험 + 면접. 경제학/경영학 전공자가 많으며, 석사 이상 학력자가 적지 않음. 한국은행, 금융감독원 등 상위 기관은 경쟁이 매우 치열 |

### 8-2. 심사/평가

| 항목 | 내용 |
|------|------|
| **업무내용** | 정책금융기관에서 기업/프로젝트에 대한 금융 지원 심사를 수행합니다. 기업 신용분석, 프로젝트 타당성 평가, 기술 평가(기술보증기금 등) 등을 합니다. |
| **필요역량** | 재무분석, 산업분석, 기술 이해(기술금융의 경우), 보고서 작성 |
| **적합한 성향** | 분석적이고 객관적 판단이 가능한 사람, 다양한 산업에 대한 학습 의지가 있는 사람 |
| **커리어패스** | 심사역 → 수석심사역 → 심사부서 관리자 |
| **현실적 진입장벽** | NCS 채용 통과 후 배치. 관련 자격증(신용분석사, CFA 등) 보유 시 유리 |

### 8-3. 리스크관리 (공공기관)

공공기관의 리스크관리는 민간 금융사와 유사하되, 정책적 목적에 따른 리스크 수용 범위가 다를 수 있습니다. 정책금융의 특성상 민간이 감당하기 어려운 리스크를 수용하는 경우가 있으므로, 정책적 판단과 리스크 분석을 균형 있게 수행해야 합니다.

### 8-4. 국제협력

| 항목 | 내용 |
|------|------|
| **업무내용** | 국제금융기구(IMF, World Bank, ADB 등)와의 협력, 해외 금융기관과의 교류, 국제 금융규제 동향 분석, 해외사무소 관리 등을 수행합니다. |
| **필요역량** | 영어(비즈니스 수준 이상 필수), 국제금융 이해, 외교적 커뮤니케이션, 글로벌 경제 분석 |
| **적합한 성향** | 국제적 환경에서 일하고 싶은 사람, 외국어 커뮤니케이션에 자신 있는 사람, 해외 근무에 대한 의지가 있는 사람 |
| **커리어패스** | 국제협력 담당 → 해외사무소 파견 → 시니어 → 국제협력부서장 |
| **현실적 진입장벽** | 영어 능통 필수. 해외 유학/근무 경험 우대. 해당 부서 채용이 별도로 진행되거나 내부 이동을 통해 배치 |

## 직무 선택을 위한 자기 점검 질문

직무를 선택하기 전에 다음 질문에 대해 솔직하게 답해 보시기 바랍니다.

1. **분석 vs 관계**: 혼자 데이터를 분석하는 것이 좋은가, 사람을 만나고 관계를 맺는 것이 좋은가?
2. **안정 vs 성과**: 안정적인 급여를 선호하는가, 성과에 따라 높은 보상을 받을 수 있는 구조를 선호하는가?
3. **워라밸 우선순위**: 업무 외 시간이 얼마나 중요한가? 커리어 초기에 시간을 투자할 의향이 있는가?
4. **전문성 vs 다양성**: 한 분야를 깊이 파고드는 것이 좋은가, 다양한 업무를 경험하는 것이 좋은가?
5. **현실적 진입 가능성**: 내 현재 스펙과 역량으로 해당 직무에 진입할 수 있는가? 부족한 부분을 채울 시간과 자원이 있는가?
