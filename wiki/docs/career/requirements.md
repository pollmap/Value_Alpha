## 1. 학력

### 섹터별 학력 영향도

| 섹터 | 영향도 | 설명 |
|:-----|:-----:|:-----|
| 증권사 IB | ★★★★★ | SKY 대다수. 서성한도 가능하나 비율 낮음. 서류 단계 학벌 필터 사실상 존재 |
| 증권사 리서치 | ★★★★★ | IB와 유사. 상위권 대학원 학력자 다수 |
| 증권사 S&T | ★★★★☆ | IB/리서치보다 다소 유연, 대형사는 학력 비중 높음 |
| 시중은행 | ★★★☆☆ | 인서울 4년제 주류. 비서울권 합격자 존재. NCS/면접 등 다양한 평가 요소 |
| 보험사 사무직 | ★★★☆☆ | 은행과 유사 수준 |
| 보험사 계리직 | ★★☆☆☆ | 수학/통계 전공 및 계리사 시험이 학벌보다 중요 |
| 자산운용사 (대형) | ★★★★☆ | 학벌 + 투자 역량 모두 요구 |
| 자산운용사 (중소) | ★★★☆☆ | 실력 중심 평가 경향, 기업별 편차 큼 |
| VC | ★★★☆☆ | 다양한 배경 수용, 상위 VC는 학력 높은 편 |
| PE | ★★★★★ | IB와 유사. 대부분 경력직, 전직장+학력 모두 중요 |
| 카드사/캐피탈 | ★★★☆☆ | 은행과 유사하거나 다소 유연 |
| 핀테크 | ★★☆☆☆ | 학벌보다 실무 역량 중심 |
| 정책금융/공공기관 | ★★☆☆☆\~★★★☆☆ | NCS 기반. 한은/금감원은 합격자 학력 수준 여전히 높음 |

### 현실적 조언

- **학벌 필터가 존재하는 곳은 존재합니다.** 자신의 학력으로 진입이 현실적인 섹터/직무를 먼저 파악하세요.
- **학벌이 절대적이지는 않습니다.** 자격증, 인턴, 포트폴리오로 보완 가능합니다.
- **비서울권 전략**: 공공기관(NCS), 보험 계리직(전문 자격), 핀테크(실무 역량), 중소형 금융사를 우선 고려하세요.

## 2. 학점

| 섹터 | 권장 학점 (4.5 만점) | 비고 |
|:-----|:-------------------|:-----|
| 대형 증권사 IB/리서치 | 3.7+ | 3.5 미만 서류 통과 어려움 |
| 대형 증권사 기타 | 3.5+ | — |
| 시중은행 | 3.5+ | 3.3 이상이면 서류 통과 가능성 |
| 보험사 | 3.3+ | 계리직은 수학 과목 학점 더 중요 |
| 자산운용사 | 3.5+ | 투자 역량이 학점보다 중요할 수 있음 |
| 카드사/캐피탈 | 3.3+ | — |
| 핀테크 | 큰 영향 없음 | 포트폴리오·실무 역량 중심 |
| 공공기관 | NCS 중심 | 학점 비중 낮거나 미반영 |

> 학점은 **최소 기준선**으로 기능합니다. 높다고 합격이 보장되지 않지만, 기준 이하이면 서류에서 걸릴 수 있습니다. 이미 졸업했다면 다른 요소를 강화하세요.

## 4. 어학

| 섹터 | TOEIC 기준 | Speaking/OPIc | 비고 |
|:-----|:----------|:-------------|:-----|
| 대형 증권사 | 850+ | Lv.6+ / IH+ | IB는 실제 영어 업무 능력 중요 |
| 시중은행 | 800+ | Lv.5+ / IH | 합격자 평균은 더 높을 수 있음 |
| 보험사 | 800+ | Lv.5+ / IH | 기업별 편차 |
| 자산운용사 | 850+ | 실무 수준 | CFA 공부 시 자연 향상 |
| 카드사/캐피탈 | 750+ | Lv.5+ / IM+ | — |
| 핀테크 | 기업별 상이 | 실무 중심 | 영어 면접 가능 |
| 공공기관 | 700+ | 기관별 상이 | 한은/금감원은 높은 수준 요구 |
| 외국계 | 토익보다 실무 영어 | 비즈니스 영어 필수 | 영어 면접 필수 |

> TOEIC은 **지원 자격(최소 기준)**으로 기능합니다. 900점과 950점의 차이가 합격에 결정적이지 않습니다. 성적 유효기간 2년에 유의하세요.

## 5. 인턴 경험

### 인턴 유형별 비교

| 유형 | 특징 | 채용 연결 | 가치 |
|:-----|:-----|:---------|:-----|
| 채용연계형 | 인턴 후 정규직 전환 평가 | 높음 (전환율 50\~80%) | **가장 효과적인 취업 경로** |
| 체험형 (2\~4주) | 실무 체험·기업 문화 경험 | 낮음\~중간 | 경험·네트워크 형성 도움 |
| 장기 인턴 (6개월+) | 실무 깊이 참여, 스타트업/핀테크 | 기업별 상이 | 포트폴리오·자소서 활용 가치 높음 |

### 섹터별 인턴 중요도

| 섹터 | 중요도 | 설명 |
|:-----|:-----:|:-----|
| 증권사 IB | ★★★★★ | IB 인턴 → 정규직이 주요 채용 경로 |
| 증권사 리서치 | ★★★★★ | RA 경험이 핵심. 미경험 시 입사 매우 어려움 |
| 시중은행 | ★★★☆☆ | 도움 되지만 필수 아님 |
| 보험사 | ★★★☆☆ | 계리직은 계리 인턴 유리 |
| 자산운용사 | ★★★★☆ | 운용사/리서치 인턴 매우 유리 |
| VC/PE | ★★★★★ | 관련 경력 거의 필수. PE는 대부분 경력직 |
| 핀테크 | ★★★★☆ | 스타트업 인턴/프로젝트 중요 |
| 공공기관 | ★★☆☆☆ | NCS 기반, 자소서 소재로 활용 가능 |

> **전략**: 채용연계형 최우선 지원 · 인턴 시 구체적 업무 내용과 본인 기여 정리 · 대학 3학년 여름/겨울이 일반적 시기
