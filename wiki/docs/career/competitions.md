# 대회·공모전·포트폴리오 가이드

금융권 취업을 준비하면서 "어떤 대외활동을 해야 하는가"는 가장 많이 받는 질문 중 하나입니다. 이 페이지에서는 실제로 금융권 채용에서 유의미한 경쟁력이 되는 활동과, 과대/과소평가되는 활동을 구분하여 정리합니다.

:::tip 핵심 원칙
활동의 **양보다 질**이 중요합니다. 10개의 활동에 이름만 올리는 것보다, 2~3개의 활동에서 깊이 있는 성과를 내는 것이 채용에서 훨씬 유리합니다. 면접에서 구체적으로 설명할 수 있는 경험이 핵심입니다.
:::

**주식투자 대회에 대한 솔직한 평가:**
- 수상 경력은 이력서에 한 줄 추가할 수 있는 수준이지만, 결정적인 합격 요인이 되기는 어렵습니다.
- 단기 수익률 경쟁은 투자 역량보다 운(luck)이나 과도한 리스크 테이킹의 결과일 수 있으며, 금융업 종사자들은 이를 잘 알고 있습니다.
- 다만, 참가 과정에서 투자 논리를 정리하고 리포트를 작성하는 경험 자체는 가치가 있습니다.

## 2. 포트폴리오 구성 가이드

포트폴리오는 "나의 역량을 구체적으로 증명할 수 있는 자료"입니다. 직무에 따라 효과적인 포트폴리오의 형태가 다릅니다.

### 2-1. 투자 분석 리포트 작성법

리서치, 자산운용, IB 직무를 목표로 한다면, **자체 작성 투자 분석 리포트**가 가장 강력한 포트폴리오입니다.

#### 리포트 기본 구조

```
1. 투자 의견 요약 (Executive Summary)
   - 종목명, 투자의견(매수/매도/중립), 목표주가, 현재가 대비 상승여력
   - 핵심 투자 포인트 3~5가지

2. 기업 개요 (Company Overview)
   - 사업 구조, 매출 비중, 핵심 제품/서비스
   - 경영진 분석, 주주구성

3. 산업 분석 (Industry Analysis)
   - 시장 규모 및 성장성
   - 경쟁 환경 (Porter's 5 Forces 등)
   - 규제 환경, 기술 트렌드

4. 재무 분석 (Financial Analysis)
   - 최근 3~5년 손익계산서/재무상태표/현금흐름표 분석
   - 주요 재무비율(수익성, 안정성, 성장성, 효율성)
   - 동종업계 대비 비교

5. 밸류에이션 (Valuation)
   - DCF (Discounted Cash Flow) 모델
   - Comparable Valuation (PER, PBR, EV/EBITDA 등)
   - 민감도 분석 (Sensitivity Analysis)

6. 투자 리스크 (Risk Factors)
   - 기업 특이 리스크
   - 산업 리스크
   - 매크로 리스크

7. 결론 및 목표주가 도출
```

#### 리포트 작성 시 주의사항

- **단순 요약이 아닌 분석이어야 합니다.** 기업의 사업보고서를 그대로 옮겨 적는 것은 의미가 없습니다. 본인만의 분석적 시각과 판단이 드러나야 합니다.
- **근거 없는 낙관/비관은 피하세요.** "이 기업은 앞으로 성장할 것이다"가 아니라, "A 근거에 의해 B 시나리오에서 C 수준의 성장이 예상된다"는 식으로 논리 구조를 갖추세요.
- **밸류에이션은 반드시 포함하세요.** 투자 리포트에서 밸류에이션 없이 "좋은 기업이다"라고만 하는 것은 불완전합니다.
- **분량**: 본문 10~20페이지가 적정. 너무 짧으면 분석이 부실하고, 너무 길면 핵심이 묻힙니다.
- **양보다 질**: 리포트 10편을 대충 쓰는 것보다, 3~5편을 제대로 쓰는 것이 낫습니다.

### 2-2. 개인 블로그 / 브런치 활용

| 플랫폼 | 장점 | 단점 | 추천 대상 |
|--------|------|------|----------|
| **네이버 블로그** | 국내 검색 노출, 접근성 높음 | 전문적 이미지 형성이 다소 어려움 | 일반 금융 콘텐츠 |
| **브런치** | 깔끔한 디자인, 전문적 이미지 | 입점 심사 필요, 검색 노출 제한적 | 심층 분석 글 |
| **티스토리** | 자유로운 커스터마이징, 검색 노출 | 초기 설정 필요 | 기술적/분석적 콘텐츠 |
| **Substack/Medium** | 글로벌 독자, 전문적 이미지 | 한국 시장 인지도 낮음, 영어가 주류 | 영어 리서치/분석 |

#### 블로그 운영 팁

- **정기적으로 글을 올리세요.** 한두 편 올리고 방치하면 효과가 없습니다. 주 1회 또는 격주 1회 정도가 적정합니다.
- **주제를 좁히세요.** "경제 전반"보다 "반도체 산업 분석" 또는 "한국 은행업 분석" 등 특정 영역에 집중하는 것이 전문성을 보여줍니다.
- **면접 시 활용**: 블로그를 면접 전에 제출하거나 면접에서 언급하면, 면접관이 실제로 확인하는 경우가 있습니다. 부실한 내용이면 오히려 마이너스가 될 수 있으므로 품질 관리가 중요합니다.

### 2-3. GitHub 포트폴리오 (퀀트/데이터분석)

퀀트, 데이터 분석, 핀테크 직무를 목표로 한다면 GitHub 포트폴리오가 매우 효과적입니다.

#### 추천 프로젝트 아이디어

| 프로젝트 | 난이도 | 관련 직무 | 설명 |
|---------|--------|----------|------|
| **주가 예측 모델** | ★★★☆☆ | 퀀트, 데이터분석 | 머신러닝/딥러닝을 활용한 주가 예측. 단, 결과의 한계를 인지하고 서술하는 것이 중요 |
| **포트폴리오 최적화** | ★★★☆☆ | 자산운용, 퀀트 | MVO(Mean-Variance Optimization), Risk Parity 등 실제 운용 전략 구현 |
| **팩터 투자 모델** | ★★★★☆ | 퀀트, 자산운용 | Fama-French 등 팩터 모델 구현 및 백테스트 |
| **신용평가 모델** | ★★★☆☆ | 리스크관리, 카드사 | 머신러닝 기반 신용평가(Credit Scoring) 모델 구축 |
| **부정거래 탐지 (FDS)** | ★★★★☆ | 핀테크, 카드사 | 이상 탐지(Anomaly Detection) 알고리즘 구현 |
| **경제 데이터 대시보드** | ★★☆☆☆ | 리서치, 데이터분석 | 경제지표 시각화 대시보드 (Python + Streamlit/Dash 등) |
| **자연어 처리(NLP) 뉴스 분석** | ★★★★☆ | 퀀트, 데이터분석 | 뉴스/공시 텍스트 분석을 통한 감성 분석(Sentiment Analysis) |
| **재무제표 자동 분석 도구** | ★★★☆☆ | 리서치, IB | DART API를 활용한 재무제표 자동 수집 및 분석 |

#### GitHub 포트폴리오 구성 시 주의사항

- **README를 잘 작성하세요.** 프로젝트의 목적, 방법론, 결과, 한계를 명확히 서술해야 합니다. 코드만 있고 설명이 없으면 가치가 반감됩니다.
- **코드 품질을 관리하세요.** 주석, 모듈화, 가독성 등 기본적인 코드 품질이 중요합니다.
- **결과의 한계를 인정하세요.** "백테스트에서 연 50% 수익률을 달성했다"고만 쓰면 신뢰를 잃습니다. 과적합(overfitting) 가능성, 실전과의 괴리 등을 솔직히 서술하는 것이 오히려 전문성을 보여줍니다.

### 2-4. 모의투자 실적 정리법

모의투자 실적을 포트폴리오로 활용할 때는 다음 형식으로 정리하는 것이 효과적입니다.

```
■ 운용 개요
- 운용 기간: 2024.03 ~ 2024.09 (6개월)
- 초기 투자금: 1억 원 (모의)
- 벤치마크: KOSPI 200

■ 운용 성과
- 포트폴리오 수익률: +12.3%
- 벤치마크 수익률: +5.1%
- 초과수익률(알파): +7.2%
- 최대낙폭(MDD): -8.5%
- 샤프비율: 1.34

■ 투자 전략
- [구체적인 투자 전략 서술]

■ 주요 종목 및 투자 논리
- 종목 A: [투자 사유, 매수/매도 시점, 수익률]
- 종목 B: [투자 사유, 매수/매도 시점, 수익률]

■ 교훈 및 반성
- [잘한 점과 개선할 점을 솔직하게 서술]
```

**모의투자 실적에 대한 주의사항:**
- 모의투자는 실전 투자와 다릅니다 (심리적 요소, 유동성, 거래비용 등). 이 점을 인지하고 있음을 보여주는 것이 중요합니다.
- 수익률 자체보다 **투자 프로세스와 논리**가 더 중요하게 평가됩니다.
- 손실 경험과 그로부터 배운 점을 솔직하게 서술하는 것이 오히려 성숙한 투자자로서의 인상을 줍니다.

### 2-5. 엑셀 모델링 샘플

IB, 리서치, 운용 직무를 목표로 한다면 엑셀 모델링 능력은 핵심 역량입니다.

#### 주요 모델링 유형

| 모델 유형 | 난이도 | 관련 직무 | 설명 |
|----------|--------|----------|------|
| **3 Statement Model** | ★★★☆☆ | IB, 리서치, 운용 | 손익계산서, 재무상태표, 현금흐름표를 연결하는 기본 모델. 가장 기초적이면서 중요한 모델 |
| **DCF Model** | ★★★★☆ | IB, 리서치, 운용, PE | 현금흐름할인 밸류에이션 모델. 3 Statement Model 기반으로 구축 |
| **LBO Model** | ★★★★★ | PE, IB | 차입매수 모델. PE 취업 시 필수적 모델링 스킬 |
| **Merger Model (합병 모델)** | ★★★★★ | IB (M&A) | 합병 시너지, EPS 희석/증가 분석 등 |
| **Comparable Analysis** | ★★☆☆☆ | IB, 리서치 | 동종업계 비교 밸류에이션. 상대적으로 간단하나 분석적 시각이 필요 |

#### 모델링 학습 자원

- **온라인 강의**: Wall Street Prep, Breaking Into Wall Street(BIWS), CFI(Corporate Finance Institute) 등의 유료 강의가 체계적. 국내에서는 관련 학원이나 온라인 강의도 존재
- **실습**: 실제 기업의 사업보고서를 기반으로 직접 모델을 구축하는 것이 가장 효과적
- **Excel 단축키**: 금융 모델링에서 마우스 없이 키보드로 작업하는 것이 기본. 단축키 숙달이 필수

## 4. 스펙 현실 점검

### 과대평가되는 스펙

| 스펙 | 과대평가되는 이유 | 현실 |
|------|-----------------|------|
| **자격증 개수** | "자격증이 많을수록 유리하다"는 통념 | 관련성 없는 자격증을 나열하면 오히려 "방향성이 없다"는 인상을 줄 수 있음. 목표 직무에 맞는 2~3개면 충분 |
| **주식투자 대회 수상** | "투자 실력 증명"이라는 인식 | 단기 수익률은 운의 요소가 크며, 업계 종사자들도 이를 인지. 투자 논리와 프로세스가 더 중요 |
| **토익 고득점 (900+)** | "영어 점수가 높으면 유리"라는 인식 | 최소 기준을 충족하면 토익 점수 차이는 합격에 결정적이지 않음. 실제 영어 커뮤니케이션이 더 중요 |
| **해외 어학연수** | "글로벌 경험"이라는 인식 | 단순 어학연수는 큰 차별화가 되지 않음. 해외 인턴이나 교환학생(학점 이수)이 더 유의미 |
| **다수의 대외활동** | "활동이 많으면 적극적"이라는 인식 | 깊이 없이 이름만 올린 활동은 면접에서 오히려 약점이 될 수 있음 |
| **봉사활동 시간** | 공공기관에서 봉사시간을 평가한다는 인식 | NCS 기반 채용에서 봉사시간의 가중치는 크지 않음 (기관별 상이) |

### 과소평가되는 스펙

| 스펙 | 과소평가되는 이유 | 현실적 가치 |
|------|-----------------|------------|
| **심층 투자 분석 리포트** | 작성에 시간과 노력이 많이 들어 기피 | 리서치/운용/IB 직무에서 **가장 직접적으로 역량을 보여주는 포트폴리오**. 잘 작성된 리포트 3~5편은 자격증 여러 개보다 효과적일 수 있음 |
| **실무 경험 (인턴)** | 경쟁률이 높아 "어차피 안 된다"며 포기 | 채용연계형 인턴은 가장 확실한 취업 경로. 체험형이라도 경험의 가치가 큼 |
| **금융 관련 학회 활동** | 화려한 스펙이 아니라는 인식 | 1년 이상 체계적으로 활동하며 리포트를 작성하고 발표한 경험은 강력한 자산 |
| **프로그래밍/데이터 역량** | "금융은 엑셀만 하면 된다"는 인식 | Python, SQL 등의 역량은 점점 더 중요해지고 있으며, 전통 금융 직무에서도 차별화 요소 |
| **산업 전문 지식** | "범용적인 금융 지식이 더 중요하다"는 인식 | 특정 산업(반도체, 제약, 에너지 등)에 대한 깊은 이해는 리서치/IB에서 매우 높이 평가됨 |
| **면접 준비** | 서류 스펙에만 집중하고 면접 준비를 소홀히 함 | 서류를 통과한 후에는 면접이 결정적. 금융 시사, 투자 관점, 직무 이해도를 체계적으로 준비해야 함 |
| **네트워크** | "줄을 대는 것"이라는 부정적 인식 | 선배, 현직자와의 건전한 네트워크는 채용 정보, 멘토링, 추천(레퍼럴)에서 실질적 도움이 됨 |
