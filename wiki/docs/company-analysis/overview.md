# 기업분석 개요

## 이 섹션의 목적

Layer 1~2에서 재무제표를 읽고 분석하는 능력을 배웠다면, Layer 4에서는 **개별 기업을 종합적으로 분석하고 투자 판단을 내리는 방법**을 배웁니다.

기업분석은 두 축으로 구성됩니다:
- **정성적 분석 (Qualitative)**: 비즈니스 모델, 경쟁 우위, 경영진, 거버넌스
- **정량적 분석 (Quantitative)**: 재무 모델링, 시나리오 분석, 적정가치 산출

## 기업분석 프레임워크

### Top-Down vs Bottom-Up

| 접근법 | 순서 | 적합한 상황 |
|--------|------|------------|
| **Top-Down** | 거시경제 → 산업 → 기업 | 매크로 영향 큰 산업 (금융, 건설) |
| **Bottom-Up** | 기업 → 산업 → 거시경제 | 종목 선정 중심 투자 |
| **혼합** | 산업 스크리닝 + 기업 심층 분석 | 실무에서 가장 보편적 |

### 기업분석 5단계 프로세스

```
1단계: 사업 이해 (Business Understanding)
  → 무엇을 하는 기업인가?
  ↓
2단계: 경쟁 우위 분석 (Competitive Advantage)
  → 왜 이 기업이 돈을 벌 수 있는가?
  ↓
3단계: 재무 분석 (Financial Analysis)
  → 얼마나 잘 벌고 있는가? (Layer 2 활용)
  ↓
4단계: 밸류에이션 (Valuation)
  → 적정 가치는 얼마인가?
  ↓
5단계: 투자 결정 (Investment Decision)
  → 매수/매도/관망?
```

## 정성적 분석 영역

### 비즈니스 모델 분석

| 항목 | 핵심 질문 |
|------|-----------|
| 수익 모델 | 어떻게 돈을 버는가? |
| 가치 제안 | 고객에게 어떤 가치를 제공하는가? |
| 고객 세그먼트 | 누가 고객인가? |
| 핵심 자원 | 무엇이 경쟁력의 원천인가? |
| 비용 구조 | 주요 비용 항목은 무엇인가? |

### 경쟁 우위 (Economic Moat)

워런 버핏이 강조하는 **경제적 해자(Moat)**:

| 해자 유형 | 설명 | 예시 |
|-----------|------|------|
| 네트워크 효과 | 사용자 증가 → 가치 증가 | 카카오, 네이버 |
| 전환 비용 | 다른 제품으로 바꾸기 어려움 | SAP, Microsoft |
| 비용 우위 | 경쟁사보다 저렴하게 생산 | 삼성전자 반도체 |
| 무형 자산 | 브랜드, 특허, 라이선스 | 아모레퍼시픽, 셀트리온 |
| 규모의 경제 | 대량 생산/유통의 효율 | 현대자동차, CJ제일제당 |

### 경영진 평가

| 평가 항목 | 확인 방법 |
|-----------|----------|
| 실적 이행 능력 | 과거 가이던스 달성률 |
| 자본 배분 능력 | ROIC 추세, M&A 성과 |
| 주주 친화성 | 배당 정책, 자사주 매입, IR 활동 |
| 지배구조 | 이사회 독립성, 감사위원회 |
| 보상 구조 | 성과 연동 보상 비중 |

## 정량적 분석 영역

### 재무 모델링

Layer 2에서 배운 분석을 바탕으로 **미래 재무제표를 추정**합니다:

```
과거 5년 재무 분석 (Layer 2)
  ↓
핵심 가정 수립 (매출 성장률, 마진, CAPEX 등)
  ↓
3표 추정 (손익계산서 → 재무상태표 → 현금흐름표)
  ↓
FCF 산출 → DCF 밸류에이션
  ↓
시나리오 분석 (Bull / Base / Bear)
```

### 밸류에이션 방법론

| 방법 | 접근법 | 적합한 상황 |
|------|--------|------------|
| **DCF** | 절대가치 | 안정적 현금흐름 기업 |
| **상대가치** | PER, PBR 등 멀티플 | 비교 대상 풍부한 업종 |
| **NAV** | 순자산가치 | 지주회사, 부동산 |
| **SOTP** | 사업부별 합산 | 다각화 기업 |
| **잔여이익** | RI 모델 | 금융업 |

## 한국 기업 분석 특수 고려사항

### 재벌 구조 분석

| 요소 | 분석 포인트 |
|------|-----------|
| 지배구조 | 순환출자, 일감몰아주기, 사익편취 |
| 그룹 시너지 | 계열사 간 거래 비중, 상호 보증 |
| 지주회사 할인 | NAV 대비 할인율 (보통 30~50%) |
| 계열사 리스크 | 부실 계열사의 연쇄 영향 |

### DART 공시 활용 체크리스트

| 공시 문서 | 확인 사항 |
|-----------|----------|
| 사업보고서 | 사업 내용, 재무제표, 주석, 관계사 거래 |
| 감사보고서 | 감사 의견, 핵심감사사항(KAM) |
| 주요사항보고서 | 유상증자, CB/BW 발행, M&A |
| 지분변동공시 | 대주주 지분 변동, 임원 매매 |
| 공정공시 | 실적 전망, 사업 계획 |

### 한국 특유의 밸류에이션 이슈

1. **코리아 디스카운트 정량화**: 글로벌 Peer 대비 할인율 산출
2. **지배구조 프리미엄/할인**: ESG 지배구조 점수 반영
3. **연결/별도 차이**: 어떤 기준으로 밸류에이션할 것인가
4. **우선주 할인**: 보통주 대비 우선주 할인율 (보통 20~40%)
