# 재무 모델링 기초

> **Layer 4 - 정량적 분석** | 난이도: 🟡중급 → 🔴심화 | 선수 지식: [재무제표 분석](../../financial-analysis/overview.md), [3표 연결](../../foundation/three-statement-link/intro.md)

## 재무 모델링이란?

재무 모델링은 기업의 **미래 재무제표를 체계적으로 추정**하는 작업입니다. DCF 밸류에이션의 전단계로, 재무제표 분석(Layer 2)에서 파악한 패턴을 바탕으로 미래를 예측합니다.

## 3표 통합 모델 구축 프로세스

### Step 1: 과거 데이터 정리

5개년 재무 데이터를 표준화합니다:

| 항목 | 처리 방법 |
|------|----------|
| 손익계산서 | 공통형(매출=100%) 분석 |
| 재무상태표 | 총자산 대비 비중 분석 |
| 현금흐름표 | 주요 항목 추출 |
| 비율 분석 | Layer 2 지표 산출 |
| 일회성 항목 | 정상화(Normalization) |

### Step 2: 핵심 가정 수립

**손익계산서 가정**

| 항목 | 가정 방법 | 근거 |
|------|----------|------|
| 매출 성장률 | Top-down + Bottom-up | 시장 규모 × 점유율 × ASP |
| 매출원가율 | 과거 추세 + 원가 전망 | 원재료비, 인건비 변동 |
| 판관비율 | 과거 추세 + 전략 | R&D, 마케팅 계획 |
| 영업외 수익/비용 | 보수적 가정 | 이자비용은 부채 수준에 연동 |
| 법인세율 | 실효세율 적용 | 과거 실효세율 or 법정세율 |

**재무상태표 가정**

| 항목 | 가정 방법 | 근거 |
|------|----------|------|
| 매출채권 | 회전일수 유지 | 과거 평균 DSO |
| 재고자산 | 회전일수 유지 | 과거 평균 DIO |
| 매입채무 | 회전일수 유지 | 과거 평균 DPO |
| 유형자산 | CAPEX - 감가상각 | 투자 계획, 과거 패턴 |
| 차입금 | 만기 도래 + 신규 차입 | 자금 계획 |

**현금흐름표**: 손익계산서와 재무상태표에서 자동 도출

### Step 3: 3표 연결 구축

```
손익계산서 추정
  → 당기순이익 → 재무상태표 이익잉여금
  → 감가상각비 → 현금흐름표
  ↓
재무상태표 추정
  → 자산/부채 변동 → 현금흐름표 (운전자본, 투자)
  → 차입금 변동 → 이자비용 → 손익계산서
  ↓
현금흐름표 도출
  → 기말 현금 → 재무상태표 현금 (검증)
```

### Step 4: FCF 산출

```
매출액 (추정)
× 영업이익률 (가정)
= 영업이익
× (1 - 세율)
= NOPAT
+ 감가상각비
- CAPEX
- 순운전자본 변동
= FCFF (잉여현금흐름)
```

## 모델링 실전 가이드

### 매출 추정 방법

**방법 1: 물량 × 단가**
```
시장 규모 (TAM)
× 시장 점유율
= 판매 물량
× 평균 판매 단가 (ASP)
= 매출액
```

**방법 2: 세그먼트별 추정**
```
사업부 A 매출 (성장률 적용)
+ 사업부 B 매출 (성장률 적용)
+ 신규 사업 매출
= 총 매출
```

**방법 3: 드라이버 기반**
```
기존 고객 수 × 유지율 × ARPU
+ 신규 고객 수 × ARPU
= 매출액
```

### CAPEX 추정

| 구분 | 추정 방법 |
|------|----------|
| 유지 CAPEX | ≈ 감가상각비 (현 수준 유지) |
| 성장 CAPEX | 매출 성장에 필요한 추가 투자 |
| 총 CAPEX | 유지 + 성장 CAPEX |

과거 CAPEX/매출 비율, CAPEX/감가상각 비율을 참고

### 감가상각 추정

$$\text{감가상각비} = \frac{\text{유형자산 총액}}{\text{평균 내용연수}}$$

또는 과거 감가상각률(감가상각비/기초 유형자산)을 적용

## 📊 한국 기업 모델링 특수 고려

### 연결 vs 별도 모델

| 구분 | 용도 |
|------|------|
| **연결 모델** | 그룹 전체 가치 산출 (표준) |
| **별도 모델** | 해당 법인 고유 가치, 지주회사 NAV |
| **세그먼트 모델** | 사업부별 SOTP 밸류에이션 |

### 한국 기업 특수 항목

1. **환율 가정**: 수출 비중 높은 기업은 환율 시나리오 필수
2. **반도체 사이클**: ASP와 물량의 사이클 반영
3. **관계사 거래**: 내부 거래 제거 (연결) 또는 지속성 가정 (별도)
4. **K-IFRS 영향**: 수익 인식 기준, 리스 회계 반영

### DART 데이터 활용

1. 사업보고서: 5개년 요약 재무정보 (기초 데이터)
2. 분기보고서: 분기별 세그먼트 매출/영업이익
3. 주석: 감가상각 내역, 차입금 만기 구조
4. 사업 내용: 투자 계획, 시설 현황

## 💡 실무 포인트

### 모델링 체크리스트

- [ ] 과거 5년 재무 데이터를 정리했는가?
- [ ] 핵심 가정(매출 성장률, 마진, CAPEX)이 합리적인가?
- [ ] 3표가 올바르게 연결되어 있는가?
- [ ] 재무상태표가 밸런스하는가? (자산 = 부채 + 자본)
- [ ] 현금흐름표의 기말 현금 = 재무상태표 현금인가?
- [ ] 시나리오별 민감도 분석을 수행했는가?
- [ ] 가정의 출처와 근거를 문서화했는가?

### 모델링 → DCF 연결

```
재무 모델 (3표 추정)
  → FCFF 산출 (5~10년)
  → 터미널 밸류 산출
  → WACC으로 할인
  → 기업가치(EV)
  → 순차입금 차감
  → 주주가치(Equity Value)
  → 주당 가치
  → 현재 주가와 비교
  → 투자 판단
```
