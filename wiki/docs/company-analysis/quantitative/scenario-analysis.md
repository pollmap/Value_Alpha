# 시나리오 분석

> **Layer 4 - 정량적 분석** | 난이도: 🔴심화 | 선수 지식: [재무 모델링](./financial-modeling.md)

## 시나리오 분석이란?

단일 추정치가 아닌, 복수의 가능한 미래를 설정하여 기업 가치의 **범위(range)**를 산출하는 분석 기법입니다. 불확실성을 인정하고 관리하는 방법론.

## 시나리오 분석 프레임워크

### 3-시나리오 모델 (기본)

| 시나리오 | 확률 | 성격 | 핵심 가정 변동 |
|---------|------|------|--------------|
| **Bull (낙관)** | 25% | 최선의 상황 | 성장률 ↑, 마진 ↑ |
| **Base (기본)** | 50% | 가장 가능성 높은 상황 | 현재 추세 지속 |
| **Bear (비관)** | 25% | 최악의 상황 | 성장률 ↓, 마진 ↓ |

### 확률 가중 기업가치

$$\text{적정 가치} = P_{Bull} \times V_{Bull} + P_{Base} \times V_{Base} + P_{Bear} \times V_{Bear}$$

### 시나리오 설정 가이드

**Bull Case 가정 예시**:
- 시장 성장률: 상위 추정치
- 시장 점유율: 확대
- 마진: 개선 (원가 절감, 가격 인상 성공)
- 신규 사업: 성공적 진출
- 환율: 유리한 방향

**Bear Case 가정 예시**:
- 시장 성장률: 둔화 또는 역성장
- 시장 점유율: 축소 (경쟁 심화)
- 마진: 축소 (원가 상승, 가격 경쟁)
- 신규 사업: 실패 또는 지연
- 환율: 불리한 방향

## 민감도 분석 (Sensitivity Analysis)

### 2변수 민감도 테이블

DCF에서 가장 영향이 큰 두 변수를 선정하여 교차 분석:

**예시: WACC × 영구성장률**

| WACC \ g | 1.0% | 1.5% | 2.0% | 2.5% | 3.0% |
|----------|------|------|------|------|------|
| **8.0%** | 85,000 | 92,000 | 100,000 | 110,000 | 125,000 |
| **8.5%** | 78,000 | 84,000 | 90,000 | 98,000 | 108,000 |
| **9.0%** | 72,000 | 77,000 | 82,000 | 88,000 | 96,000 |
| **9.5%** | 67,000 | 71,000 | 75,000 | 80,000 | 87,000 |
| **10.0%** | 62,000 | 65,000 | 69,000 | 74,000 | 79,000 |

(단위: 원/주)

### 주요 민감도 변수

| 변수 | 영향도 | 불확실성 |
|------|--------|---------|
| 매출 성장률 | 높음 | 중간 |
| 영업이익률 | 높음 | 중간 |
| WACC | 매우 높음 | 높음 |
| 영구 성장률 | 매우 높음 | 높음 |
| CAPEX/매출 | 중간 | 낮음 |
| 운전자본/매출 | 낮음 | 낮음 |

## 몬테카를로 시뮬레이션

### 개념

각 입력 변수에 확률 분포를 부여하고, 수천~수만 번의 랜덤 시뮬레이션을 실행하여 기업 가치의 확률 분포를 산출합니다.

### 적용 예시

```
입력 변수 분포 설정:
- 매출 성장률: 정규분포 (μ=10%, σ=3%)
- 영업이익률: 정규분포 (μ=15%, σ=2%)
- WACC: 삼각분포 (min=8%, mode=9%, max=11%)
- 영구성장률: 균등분포 (1.5%~2.5%)

→ 10,000회 시뮬레이션 실행
→ 결과: 주당 가치 분포
  - 중앙값: 75,000원
  - 25th percentile: 62,000원
  - 75th percentile: 90,000원
  - 90% 신뢰구간: 50,000 ~ 105,000원
```

## 📊 한국 기업 시나리오 분석

### 한국 기업 특유의 시나리오 변수

| 변수 | Bull | Base | Bear |
|------|------|------|------|
| 원/달러 환율 | 약세 (1,200원) | 보합 (1,350원) | 강세 (1,500원) |
| 반도체 사이클 | 초호황 | 정상 수준 | 불황 |
| 중국 리스크 | 해소 | 현상 유지 | 갈등 심화 |
| 금리 환경 | 인하 | 동결 | 인상 |
| 지배구조 개선 | 밸류업 성공 | 점진적 개선 | 변화 없음 |

### 사이클 기업 시나리오

한국의 주요 수출 기업(반도체, 석유화학, 조선)은 사이클 위치에 따른 시나리오가 중요:

| 사이클 | 매출 가정 | 마진 가정 | 밸류에이션 접근 |
|--------|---------|----------|---------------|
| 상승기 | +20~30% | 마진 확대 | Forward 멀티플 |
| 피크 | +5~10% | 최고 마진 | 정상화 이익 기준 |
| 하락기 | -10~20% | 마진 축소 | Trough 밸류에이션 |
| 바닥 | -5~0% | 최저/적자 | 회복기 이익 기준 |

## 💡 실무 포인트

### 시나리오 분석 체크리스트

- [ ] Bull/Base/Bear 시나리오의 가정이 명확한가?
- [ ] 각 시나리오의 확률 배분이 합리적인가?
- [ ] 핵심 변수에 대한 민감도 분석을 수행했는가?
- [ ] 시나리오별 기업가치 범위가 의미있는가?
- [ ] 현재 주가가 어떤 시나리오를 반영하고 있는가?
- [ ] 가장 중요한 불확실성 요인은 무엇인가?

### 투자 판단에의 활용

| 상황 | 투자 판단 |
|------|----------|
| 현재 주가 < Bear Case 가치 | 강력 매수 (안전마진 충분) |
| Bear < 현재 주가 < Base | 매수 고려 (합리적 가격) |
| Base < 현재 주가 < Bull | 보유/관망 (적정~고평가) |
| 현재 주가 > Bull Case 가치 | 매도 고려 (과대평가) |

**벤저민 그레이엄의 안전마진(Margin of Safety)**: Base Case 적정가치 대비 30~40% 이상 할인된 가격에서 매수
