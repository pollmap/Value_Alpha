## Step 2: 핵심 가정 수립

### 매출 예측 (5~10년)

```
시나리오별 매출 성장률:
Year 1~3: 높은 성장기 (구체적 드라이버 기반)
Year 4~5: 성장 둔화기 (시장 성숙도 반영)
Year 6~10: 안정기 (GDP+α 수준으로 수렴)
```

### 마진 예측

```
영업이익률 가정:
- 현재 수준에서 출발
- 마진 개선/악화 요인 반영
- 장기적으로 업종 평균에 수렴하는지 확인
```

### CAPEX 및 운전자본

```
CAPEX:
- 유지 CAPEX ≈ 감가상각비
- 성장 CAPEX: 매출 성장에 비례
- CAPEX/매출 비율의 장기 수렴

운전자본:
- 과거 NWC/매출 비율 적용
- CCC 변동 가정 반영
```

## Step 4: WACC 산출

### WACC 구성 요소

$$\text{WACC} = \frac{E}{V} \times K_e + \frac{D}{V} \times K_d \times (1-t)$$

**자기자본비용 (Ke) - CAPM**:

$$K_e = R_f + \beta \times (R_m - R_f)$$

| 항목 | 한국 시장 기준 |
|------|--------------|
| 무위험수익률 (Rf) | 국고채 10년물 (3~4%) |
| 시장 리스크프리미엄 | 5~7% (한국 시장) |
| 베타 (β) | 개별 기업 (Bloomberg, FnGuide) |
| 추가 프리미엄 | 소형주 프리미엄, 유동성 프리미엄 |

**타인자본비용 (Kd)**:
- 회사채 수익률 또는 차입금 가중평균이자율
- 세후 기준 적용: Kd × (1-t)

**자본 구조 (D/E)**:
- 현재 자본 구조 또는 목표 자본 구조
- 시장가치 기준 (장부가치 X)

### 한국 기업 WACC 범위

| 유형 | WACC 범위 |
|------|----------|
| 대형 우량주 | 8~10% |
| 중형주 | 9~12% |
| 소형/성장주 | 11~15% |
| 바이오 (적자) | 13~18% |

## Step 5: 터미널 밸류

### 영구성장모델 (Gordon Growth)

$$\text{TV} = \frac{\text{FCFF}_{n+1}}{\text{WACC} - g}$$

**영구성장률(g) 설정**:
- 한국 장기 명목 GDP 성장률: 3~4%
- 일반적으로 2~3% 적용
- g < WACC (반드시)
- g ≤ 장기 GDP 성장률 (보수적)

### Exit Multiple 방법

$$\text{TV} = \text{EBITDA}_n \times \text{Exit EV/EBITDA}$$

Exit Multiple은 과거 평균 또는 업종 평균 적용

### 교차 검증

두 방법으로 산출한 TV를 비교하여 정합성 확인. TV가 전체 기업가치의 60~80%를 차지하는 것이 일반적.

## Step 6: 기업가치 산출

```
FCFF PV (명시적 예측기간) + TV PV (터미널밸류 현재가치)
= Enterprise Value (EV)
- 순차입금 (이자부부채 - 현금)
- 비지배지분 가치
+ 비영업용 자산 (투자자산 등)
= Equity Value (주주가치)
÷ 발행주식수 (자기주식 제외)
= 주당 적정가치
```

## Step 7: 검증 및 투자 판단

### 산출 결과 검증

| 검증 항목 | 확인 사항 |
|-----------|----------|
| TV 비중 | 60~80% 범위 내인가? |
| Implied 멀티플 | PER, PBR 등이 합리적인가? |
| Implied 성장률 | 현실적인 성장률인가? |
| 민감도 범위 | 합리적 범위 내 변동인가? |

### 투자 판단

```
적정가치: 80,000원 (Base Case)
현재 주가: 60,000원
할인율(Upside): 33%
안전마진: 충분 (>30%)
→ 투자 의견: 매수
```

## 한국 기업 DCF 특수 고려

### 한국 특유의 DCF 조정 사항

1. **지배구조 할인**: 적정가치에서 10~30% 할인
2. **비영업자산 가산**: 투자부동산, 지분법 투자 별도 평가
3. **우선주**: 우선주가 있는 경우 Equity Value에서 우선주 가치 차감
4. **비지배지분**: 연결 기준 EV에서 비지배지분 가치 차감
5. **교차 보유**: 그룹사 간 상호 보유 지분 처리

### SOTP(Sum-of-the-Parts) 필요 시

다각화 기업은 사업부별 DCF 또는 멀티플을 적용:

```
사업부 A 가치 (DCF or EV/EBITDA × EBITDA)
+ 사업부 B 가치
+ 비영업자산 가치
- 본사 비용 현재가치
= 총 기업가치
```
