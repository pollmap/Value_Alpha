# SOTP 밸류에이션 (Sum-of-the-Parts)

## 학습 목표

- SOTP 분석의 적용 조건과 방법론을 이해한다
- 각 사업부의 적정 밸류에이션 방법을 선택할 수 있다
- 지주회사 할인과 그 이유를 분석할 수 있다

---

## SOTP란?

SOTP는 다각화된 기업의 각 사업부(또는 자회사)를 **개별적으로 밸류에이션**한 후 합산하여 전체 기업 가치를 산출하는 방법입니다.

핵심 전제는 **기업의 가치가 개별 부분의 합과 다를 수 있다**는 것입니다. 시장이 복합기업 전체에 하나의 멀티플을 부여하면, 고성장 사업부의 가치가 저성장 사업부에 묻혀 저평가되는 경우가 빈번합니다. SOTP는 이러한 가치 왜곡을 해소하기 위한 분석 도구입니다.

### SOTP vs 통합 DCF: 언제 괴리가 발생하는가

SOTP 가치가 통합 DCF보다 **높게** 나오는 경우:

- 고성장 사업부가 저마진 사업부에 가려져 있을 때
- 비영업자산의 장부가치가 시장가치보다 현저히 낮을 때
- 사업부 간 시너지가 미미하여 분리 시 가치 훼손이 적을 때

SOTP 가치가 통합 DCF보다 **낮게** 나오는 경우:

- 사업부 간 강한 수직계열화 시너지가 존재할 때
- 본사 공유 서비스 비용을 사업부별로 배분하면 개별 수익성이 악화될 때
- 내부 거래 제거 시 각 사업부의 매출이 크게 감소할 때

### 해체 가치 개념

**해체 가치**란 기업을 각 사업부로 분리하여 개별 매각했을 때 실현 가능한 총 가치입니다. 해체 가치가 현재 시가총액보다 유의미하게 높으면, 이는 행동주의 투자자에게 강력한 투자 논거가 됩니다. 한국에서는 밸류업 프로그램 도입 이후 이러한 관점의 분석이 더욱 중요해졌습니다.

## SOTP가 필요한 경우

| 상황 | 이유 |
|------|------|
| 다각화 기업 | 사업부별 성격이 다름 |
| 지주회사 | 자회사 지분 가치의 합 |
| 비영업자산 보유 | 투자부동산, 금융자산 별도 평가 |
| 사업 재편 | 분할, 매각 시 부분 가치 산출 |
| 숨겨진 가치 발굴 | 시장이 일부 사업의 가치를 저평가 |

## 세분화 방법론 심화

### K-IFRS 주석에서 보고 세그먼트 식별

한국 상장기업은 K-IFRS 제1108호에 따라 **영업 세그먼트**를 공시합니다. DART에서 사업보고서를 열람하면 다음 정보를 확보할 수 있습니다:

1. **세그먼트별 매출액**: 외부 매출과 내부 거래 매출을 구분
2. **세그먼트별 영업이익**: 사업부 수익성의 핵심 지표
3. **세그먼트별 자산 및 부채**: 자본 집약도 파악에 필수
4. **지역별 세분화**: 내수 대 수출 비중 확인

실무적으로 기업이 공시하는 세그먼트 구분이 SOTP 분석에 최적이 아닌 경우가 많습니다. 예를 들어, 하나의 세그먼트에 성격이 전혀 다른 사업이 포함되어 있거나, 지나치게 세분화되어 있을 수 있습니다.

### 세그먼트 재분류 기준

| 판단 기준 | 세분화 | 통합 |
|----------|--------|------|
| 마진 차이 | 영업이익률 차이 5%p 이상 | 유사한 마진 구조 |
| 성장률 차이 | 매출 성장률 차이 10%p 이상 | 유사한 성장 궤적 |
| Peer 그룹 | 적용 가능한 Peer가 다름 | 동일 Peer 적용 가능 |
| 자본 집약도 | CAPEX/매출 비율이 크게 다름 | 유사한 투자 패턴 |
| 사업 사이클 | 경기 민감도가 상이 | 동일 사이클 노출 |

### 세그먼트 수익성 분석 프레임워크

세분화 후에는 각 세그먼트의 수익성을 다음 지표로 분석합니다:

- **영업이익률 추이**: 최소 3~5년 추세 확인
- **EBITDA 마진**: 감가상각 정책 차이를 제거한 수익성
- **투하자본이익률**: 각 사업부의 자본 효율성
- **한계이익률**: 매출 증가분 대비 이익 증가분으로 미래 수익성 방향 판단

## SOTP 프로세스

### Step 1: 사업부 세분화

```
기업 전체
├── 사업부 A (핵심 사업)
├── 사업부 B (성장 사업)
├── 사업부 C (비핵심 사업)
├── 비영업자산 (투자부동산, 금융자산)
└── 본사 비용 (미배분 비용)
```

### Step 2: 사업부별 밸류에이션

각 사업부에 적합한 밸류에이션 방법 적용:

| 사업부 | 방법 | 적용 멀티플/방법 |
|--------|------|----------------|
| 제조/핵심 사업 | EV/EBITDA | 동종업체 평균 적용 |
| 성장 사업 | DCF 또는 EV/Sales | 성장성 반영 |
| 금융 사업 | PBR | 은행/보험 Peer 적용 |
| 부동산 | NAV | 시가 평가 |
| 지분 투자 | 시장가치 | 상장: 시가, 비상장: 장부가 |

### Step 3: 합산 및 조정

```
사업부 A EV
+ 사업부 B EV
+ 사업부 C EV
+ 비영업자산 가치
= 총 기업가치 (Gross EV)
- 본사 비용 현재가치
- 순차입금
- 비지배지분
= 주주가치 (Equity Value)
÷ 주식수
= 주당 적정가치
```

### Step 4: 할인 적용

$$\text{적정 주가} = \text{SOTP 주당가치} \times (1 - \text{복합기업 할인율})$$

**복합기업 할인**:
- 다각화 할인: 10~20%
- 지배구조 할인: 10~30% (한국 재벌)
- 유동성 할인: 5~15% (비상장 자회사)

## 사업부별 밸류에이션 심화

### Peer 선정 방법론

SOTP의 정확도는 각 세그먼트에 적용하는 Peer 그룹의 품질에 좌우됩니다. Peer 선정 시 다음 기준을 순차적으로 적용합니다:

1. **동일 산업 분류**: GICS 코드 기준 동일 하위 산업
2. **사업 모델 유사성**: 매출 구조, 고객 구성, 가치사슬 위치
3. **규모 유사성**: 매출 기준 0.3~3배 범위
4. **지역적 비교 가능성**: 국내 Peer 우선, 부재 시 글로벌 Peer 활용
5. **수익성 구간**: 극단적 마진 차이가 있는 기업은 제외

실무에서는 국내 순수 사업자가 존재하지 않는 경우가 많으므로, 글로벌 Peer를 활용하되 **국가별 마진 차이**를 조정해야 합니다.

### 밸류에이션 방법 선택 기준

| 사업부 특성 | 최적 방법 | 이유 |
|------------|----------|------|
| 안정적 이익, 자본 집약적 | EV/EBITDA | 감가상각 차이 제거, 현금창출력 반영 |
| 적자 또는 초기 단계 | EV/Sales 또는 DCF | 현재 이익이 미래 가치를 대변하지 못함 |
| 금융업 | PBR | 자산-부채 스프레드가 핵심 수익원 |
| 자원/부동산 | NAV | 보유 자산의 시장가치가 핵심 |
| 고성장 기술 사업 | DCF | 장기 성장성을 현재가치로 반영 |
| 바이오/신약 | 파이프라인 DCF | 임상 단계별 성공 확률 가중 |
| 플랫폼 사업 | EV/매출 또는 EV/사용자 | 네트워크 효과로 수익화 지연 |

### EV/EBITDA 적용 시 핵심 점검 사항

- **정상화 EBITDA 사용**: 일회성 비용/수익 제거
- **리스 조정**: K-IFRS 제1116호 적용에 따른 EBITDA 왜곡 확인
- **Peer 멀티플의 중앙값 vs 평균**: 극단치 영향을 줄이기 위해 중앙값 권장
- **멀티플 시점**: 12개월 선행 실적 기준 적용이 표준

가상의 복합 지주회사 **K지주**를 가정합니다. K지주는 반도체, 에너지, 통신, 바이오 자회사를 보유하고 있습니다.

**보유 지분 현황 및 가치 산출**:

| 구분 | 보유지분 | 자회사 시총 | 지분가치 | 비고 |
|------|---------|-----------|---------|------|
| K반도체 | 25% | 80조 | 20.0조 | 상장, 시가 적용 |
| K에너지 | 35% | 30조 | 10.5조 | 상장, 시가 적용 |
| K텔레콤 | 30% | 15조 | 4.5조 | 상장, 시가 적용 |
| K바이오 | 45% | 8조 | 3.6조 | 상장, 시가 적용 |
| K머티리얼즈 | 100% | - | 2.5조 | 비상장, EV/EBITDA 적용 |
| 기타 비상장 | 다수 | - | 1.5조 | 장부가 기준 |
| **총 지분가치** | | | **42.6조** | |

**자체사업 및 조정 항목**:

| 구분 | 가치 | 산출 근거 |
|------|------|----------|
| 브랜드 로열티 수입 | 1.2조 | 연간 로열티 1,500억 × 8배 |

K지주의 NAV 할인율 44%는 일반 지주회사 평균(30~40%)보다 높습니다. 이는 복잡한 지배구조, 비상장 자회사 비중, 그리고 자회사 간 내부거래 규모가 원인일 수 있습니다.

가상의 제조기업 **M산업**을 가정합니다. M산업은 자동차부품, 방산, 로봇, 건설장비 사업부를 운영합니다.

| 건설장비 | 2조 | 2,400억 | 12% | 글로벌 경기 민감 |

**사업부별 밸류에이션**:

| 자동차부품 | EV/EBITDA | 국내 부품사 Peer 6.5배 | 5.2조 |

| 로봇 | EV/Sales | 글로벌 로봇 Peer 4배 | 2.0조 |
| 건설장비 | EV/EBITDA | 글로벌 장비 Peer 8배 | 1.9조 |
| **총 EV** | | | **13.4조** |
| 본사 비용 현재가치 | | 연 1,500억 × 6배 | -0.9조 |
| 순차입금 | | | -1.5조 |
| **주주가치** | | | **11.0조** |
| 발행주식수 | | | 1억주 |

M산업의 현재 주가가 75,000원이라면, SOTP 기준 **약 47%의 상승 여력**이 있습니다. 특히 방산 사업부의 가치가 전체의 32%를 차지하지만, 시장은 M산업을 단순 자동차부품 제조사로 인식하고 있어 방산 프리미엄이 반영되지 않은 상태입니다.

## 지주회사 NAV 분석 심화

### 한국 지주회사 구조의 특수성

한국의 지주회사 체제는 2000년대 이후 순환출자 해소 과정에서 형성되었습니다. 공정거래법상 지주회사 요건을 충족해야 하며, 상장 자회사는 20% 이상, 비상장 자회사는 40% 이상의 지분을 보유해야 합니다. 이 구조적 특성이 NAV 분석에 중요한 영향을 미칩니다.

### 자기주식 처리

지주회사가 보유한 자기주식의 가치 처리는 NAV 산출에서 논쟁이 많은 영역입니다:

- **전량 가치 인정**: 자기주식을 매각 또는 소각 가능한 자산으로 간주
- **가치 미인정**: 지배주주의 경영권 유지를 위해 소각 가능성이 낮다고 판단
- **부분 인정**: 현실적으로 50% 수준의 가치만 반영

실무에서는 자기주식 처분 이력, 경영진 발언, 지배구조 환경을 종합적으로 고려하여 판단합니다. 밸류업 정책 이후 자기주식 소각 가능성이 높아진 기업은 더 많은 가치를 인정하는 추세입니다.

### 브랜드 로열티 밸류에이션

한국 지주회사의 자체사업 중 상당 부분은 계열사로부터 수취하는 **브랜드 로열티**입니다:

| 평가 방법 | 산출 방식 | 적정 멀티플 |
|----------|----------|-----------|
| 수익 접근법 | 연간 로열티 수입 × 멀티플 | 7~10배 |
| 시장 접근법 | 글로벌 유사 로열티 거래 사례 참고 | 사례별 상이 |
| 원가 접근법 | 브랜드 구축 비용의 누적 합계 | 참고용으로만 사용 |

브랜드 로열티 수입의 **지속성과 성장성**이 핵심입니다. 계열사 매출에 연동되는 구조라면 계열사 성장률을 반영하여 DCF로 평가하는 것이 가장 정밀합니다.

### 한국 지주회사 NAV 할인율 범위

| 유형 | 할인율 |
|------|--------|
| 우수 지배구조 지주회사 | 20~30% |
| 일반 지주회사 | 30~40% |
| 지배구조 이슈 지주회사 | 40~50% |
| 해외 지주회사 (참고) | 10~20% |

## 복합기업 할인 심화

### 학술 연구 결과

복합기업 할인에 대한 주요 연구 결과는 다음과 같습니다:

- **Berger & Ofek (1995)**: 미국 복합기업의 평균 할인율은 13~15% 수준
- **Lins & Servaes (1999)**: 선진국 대비 신흥국의 복합기업 할인이 더 크며, 지배구조가 핵심 변수
- **국내 연구**: 한국 재벌 그룹의 복합기업 할인은 평균 20~40%로 글로벌 평균보다 높음

### 한국 특유의 할인 요인

| 요인 | 설명 | 할인 기여도 |
|------|------|-----------|
| 총수 일가 지배구조 | 소유-경영 괴리, 사익 편취 우려 | 10~20% |
| 불투명한 내부거래 | 계열사 간 부당 지원 가능성 | 5~10% |
| 비효율적 자본 배분 | 수익성 낮은 사업에 대한 지속 투자 | 5~15% |
| 비상장 자회사 정보 비대칭 | 재무 정보 제한으로 가치 평가 어려움 | 5~10% |
| 승계 불확실성 | 지배구조 변동 리스크 | 상황에 따라 상이 |

### 적정 할인율 추정 방법

1. **역사적 할인율 분석**: 해당 기업의 과거 3~5년 NAV 할인율 추이를 확인하고 평균과 표준편차 산출
2. **Peer 비교**: 유사 지주회사/복합기업 대비 상대적 할인 수준 비교
3. **이벤트 조정**: 지배구조 개선, 사업 재편, 밸류업 프로그램 참여 등의 변화를 반영하여 역사적 평균 대비 할인 축소/확대 판단
4. **정성적 판단**: 경영진의 주주환원 의지, 소통 수준, IR 품질 등 반영

## 한국 기업 SOTP 특수 고려

### 재벌 지주회사 SOTP 주의사항

1. **순환출자 제거**: 교차 보유 지분은 이중 계산 방지
2. **자기주식**: 지주회사 보유 자사주의 가치 처리
3. **브랜드 로열티**: 지주회사의 자체사업 중 로열티 수입 별도 평가
4. **배당 수입**: 자회사 배당에 의존하는 현금흐름 특성

### 사업 다각화 기업 SOTP

한국 대기업의 사업 세그먼트가 다양한 경우:

| 세그먼트 | 밸류에이션 방법 | Peer Group |
|---------|---------------|-----------|
| 전자 | EV/EBITDA | 글로벌 전자 기업 |
| 바이오 | DCF + Pipeline | 글로벌 바이오 |
| 금융 | PBR | 국내 금융사 |
| 유통 | EV/EBITDA | 국내 유통사 |

## 비상장 자회사 밸류에이션

한국 재벌 그룹의 SOTP 분석에서 비상장 자회사의 가치 평가는 가장 까다로운 영역입니다. 공개 정보가 제한적이므로 다양한 접근법을 교차 검증해야 합니다.

### 유사 거래 비교법

최근 동일 업종에서 발생한 M&A, 지분 투자, 상장 사례의 거래 멀티플을 참고합니다:

- DART의 **주요사항보고서**에서 계열사 지분 거래 내역 확인
- IR 자료에서 경영진이 언급하는 비상장 자회사의 매출, 이익 정보 수집
- 공정거래위원회 **대기업집단 공시**에서 비상장사 재무제표 확인

### 비상장 할인율

비상장 기업은 상장 기업 대비 다음의 추가 할인을 적용합니다:

| 할인 유형 | 적용 범위 | 적용 조건 |
|----------|----------|----------|
| 비유동성 할인 | 15~30% | 상장 Peer 멀티플에서 차감 |
| 규모 할인 | 5~15% | 소규모 기업일수록 높은 할인 |
| 정보 비대칭 할인 | 5~10% | 재무 공시가 제한적인 경우 |
| 소수지분 할인 | 10~20% | 경영권이 없는 소수 지분 보유 시 |

반대로, 비상장 자회사가 **상장 가능성이 높은 경우**에는 할인을 축소하거나 상장 프리미엄을 반영할 수 있습니다. 상장 예비심사 청구, 주관사 선정 등의 구체적 진행 상황을 확인해야 합니다.

### 실무적 가치 추정 절차

1. DART에서 비상장 자회사의 감사보고서 확보
2. 매출, 영업이익, 순자산 등 핵심 재무지표 추출
3. 상장 Peer의 멀티플 산출 후 비상장 할인 적용
4. 유사 거래 사례가 있으면 교차 검증
5. 최소 두 가지 방법의 평균값을 적정 가치로 제시

## SOTP 투자 전략

### 촉매 기반 SOTP 투자

SOTP 분석에서 가치 괴리를 발견한 것만으로는 투자 논거가 불충분합니다. 할인이 축소될 **촉매**가 존재하는지 확인해야 합니다:

| 촉매 유형 | 구체적 사례 | 할인 축소 강도 |
|----------|-----------|-------------|
| 자회사 상장 | 비상장 자회사 상장 예비심사 | 강 |
| 사업부 분할 | 인적/물적 분할 공시 | 강 |
| 자산 매각 | 비핵심 자산 매각 발표 | 중 |
| 밸류업 프로그램 | 기업가치 제고 계획 발표 | 중 |
| 자사주 매입/소각 | 주주환원 정책 강화 | 중 |
| 경영진 교체 | 주주 친화적 경영진 선임 | 상황에 따라 상이 |

### 행동주의 투자와 SOTP

한국에서 행동주의 투자가 활성화되면서 SOTP 분석의 가치가 높아지고 있습니다:

- 행동주의 펀드는 NAV 할인이 큰 기업을 주요 타겟으로 선정
- 주주제안을 통해 사업부 분할, 자사주 소각, 배당 확대 등을 요구
- 행동주의 펀드 진입이 알려지면 할인 축소가 **즉각적**으로 발생하는 경향

### 분할/분사 기회 포착

기업분할이 예상되는 상황에서의 SOTP 투자 전략:

1. **분할 전**: 분할 대상 사업부가 과소평가되어 있는지 SOTP로 확인
2. **분할 공시**: 분할 비율과 SOTP 가치를 비교하여 유불리 판단
3. **분할 후**: 분할된 각 회사의 독립적 밸류에이션 수행 및 재평가 가능성 분석

특히 물적분할의 경우 기존 주주의 가치 희석 여부를 SOTP 관점에서 면밀히 검토해야 합니다.

## 실무 포인트

### SOTP 분석 체크리스트

- [ ] 세그먼트 분류가 적절한가?
- [ ] 각 세그먼트에 적합한 밸류에이션 방법을 적용했는가?
- [ ] 비영업자산을 빠뜨리지 않았는가?
- [ ] 본사 비용(미배분 비용)을 반영했는가?
- [ ] 교차 보유 지분의 이중 계산을 방지했는가?
- [ ] 적절한 할인율을 적용했는가?
- [ ] 할인/할증 요인의 변화 가능성은?

### SOTP vs 통합 DCF 비교

| 방법 | 장점 | 단점 |
|------|------|------|
| SOTP | 사업별 특성 반영, 숨겨진 가치 발견 | 데이터 요구량 많음, 할인율 주관적 |
| 통합 DCF | 간결함, 시너지 반영 | 사업별 차이 무시 |

두 방법을 모두 사용하여 교차 검증하는 것이 바람직합니다.
