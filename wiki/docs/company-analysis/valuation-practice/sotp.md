# SOTP 밸류에이션 (Sum-of-the-Parts)

> **Layer 4 - 밸류에이션 실전** | 난이도: 🔴심화 | 선수 지식: [DCF 실전](./dcf-practice.md), [멀티플](../../financial-analysis/valuation-multiples/overview.md)

## SOTP란?

SOTP는 다각화된 기업의 각 사업부(또는 자회사)를 **개별적으로 밸류에이션**한 후 합산하여 전체 기업 가치를 산출하는 방법입니다.

## SOTP가 필요한 경우

| 상황 | 이유 |
|------|------|
| 다각화 기업 | 사업부별 성격이 다름 |
| 지주회사 | 자회사 지분 가치의 합 |
| 비영업자산 보유 | 투자부동산, 금융자산 별도 평가 |
| 사업 재편 | 분할, 매각 시 부분 가치 산출 |
| 숨겨진 가치 발굴 | 시장이 일부 사업의 가치를 저평가 |

## SOTP 프로세스

### Step 1: 사업부 세분화

```
기업 전체
├── 사업부 A (핵심 사업)
├── 사업부 B (성장 사업)
├── 사업부 C (비핵심 사업)
├── 비영업자산 (투자부동산, 금융자산)
└── 본사 비용 (미배분 비용)
```

### Step 2: 사업부별 밸류에이션

각 사업부에 적합한 밸류에이션 방법 적용:

| 사업부 | 방법 | 적용 멀티플/방법 |
|--------|------|----------------|
| 제조/핵심 사업 | EV/EBITDA | 동종업체 평균 적용 |
| 성장 사업 | DCF 또는 EV/Sales | 성장성 반영 |
| 금융 사업 | PBR | 은행/보험 Peer 적용 |
| 부동산 | NAV | 시가 평가 |
| 지분 투자 | 시장가치 | 상장: 시가, 비상장: 장부가 |

### Step 3: 합산 및 조정

```
사업부 A EV
+ 사업부 B EV
+ 사업부 C EV
+ 비영업자산 가치
= 총 기업가치 (Gross EV)
- 본사 비용 현재가치
- 순차입금
- 비지배지분
= 주주가치 (Equity Value)
÷ 주식수
= 주당 적정가치
```

### Step 4: 할인 적용

$$\text{적정 주가} = \text{SOTP 주당가치} \times (1 - \text{복합기업 할인율})$$

**복합기업 할인 (Conglomerate Discount)**:
- 다각화 할인: 10~20%
- 지배구조 할인: 10~30% (한국 재벌)
- 유동성 할인: 5~15% (비상장 자회사)

## SOTP 실전 예시

### 한국 지주회사 SOTP (예시)

| 구분 | 보유지분 | 자회사 시총 | 지분가치 |
|------|---------|-----------|---------|
| 핵심 자회사 A | 30% | 50조 | 15조 |
| 핵심 자회사 B | 45% | 20조 | 9조 |
| 상장 자회사 C | 20% | 5조 | 1조 |
| 비상장 자회사 D | 100% | 장부 2조 | 2조 |
| **총 지분가치** | | | **27조** |
| 자체사업가치 | | | 3조 |
| 순차입금 | | | -1조 |
| **NAV** | | | **29조** |
| 현재 시총 | | | 17조 |
| **NAV 할인율** | | | **41%** |

### 한국 지주회사 NAV 할인율 범위

| 유형 | 할인율 |
|------|--------|
| 우수 지배구조 지주회사 | 20~30% |
| 일반 지주회사 | 30~40% |
| 지배구조 이슈 지주회사 | 40~50% |
| 해외 지주회사 (참고) | 10~20% |

## 📊 한국 기업 SOTP 특수 고려

### 재벌 지주회사 SOTP 주의사항

1. **순환출자 제거**: 교차 보유 지분은 이중 계산 방지
2. **자기주식**: 지주회사 보유 자사주의 가치 처리
3. **브랜드 로열티**: 지주회사의 자체사업 중 로열티 수입 별도 평가
4. **배당 수입**: 자회사 배당에 의존하는 현금흐름 특성

### 사업 다각화 기업 SOTP

한국 대기업의 사업 세그먼트가 다양한 경우:

| 세그먼트 | 밸류에이션 방법 | Peer Group |
|---------|---------------|-----------|
| 전자 | EV/EBITDA | 글로벌 전자 기업 |
| 바이오 | DCF + Pipeline | 글로벌 바이오 |
| 금융 | PBR | 국내 금융사 |
| 유통 | EV/EBITDA | 국내 유통사 |

### SOTP 활인/할증 트리거

| 트리거 | 방향 | 효과 |
|--------|------|------|
| 사업부 매각/분할 | 할인 축소 | 숨겨진 가치 실현 |
| 행동주의 펀드 | 할인 축소 | 가치 언락 기대 |
| 추가 비효율 투자 | 할인 확대 | 자본 배분 우려 |
| 지배구조 개선 | 할인 축소 | 밸류업 효과 |

## 💡 실무 포인트

### SOTP 분석 체크리스트

- [ ] 세그먼트 분류가 적절한가?
- [ ] 각 세그먼트에 적합한 밸류에이션 방법을 적용했는가?
- [ ] 비영업자산을 빠뜨리지 않았는가?
- [ ] 본사 비용(미배분 비용)을 반영했는가?
- [ ] 교차 보유 지분의 이중 계산을 방지했는가?
- [ ] 적절한 할인율을 적용했는가?
- [ ] 할인/할증 요인의 변화 가능성은?

### SOTP vs 통합 DCF 비교

| 방법 | 장점 | 단점 |
|------|------|------|
| SOTP | 사업별 특성 반영, 숨겨진 가치 발견 | 데이터 요구량 많음, 할인율 주관적 |
| 통합 DCF | 간결함, 시너지 반영 | 사업별 차이 무시 |

두 방법을 모두 사용하여 교차 검증하는 것이 바람직합니다.
