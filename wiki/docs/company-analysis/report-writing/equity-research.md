# 투자 보고서 작성

> **Layer 4 - 보고서 작성** | 난이도: 🔴심화 | 선수 지식: [DCF 실전](../valuation-practice/dcf-practice.md), [시나리오 분석](../quantitative/scenario-analysis.md)

## 투자 보고서의 목적

분석 결과를 체계적으로 정리하여 **투자 판단의 근거**를 제시하는 문서입니다. 개인 투자 기록이든 프로 리서치 보고서든, 구조화된 보고서는 투자 의사결정의 질을 높입니다.

## 보고서 구조 (Standard Format)

### 1. Executive Summary (핵심 요약)

| 항목 | 내용 |
|------|------|
| 기업명 / 종목코드 | |
| 투자 의견 | 매수 / 보유 / 매도 |
| 목표 주가 | (산출 근거 포함) |
| 현재 주가 | |
| 상승 여력 (Upside) | |
| 핵심 투자 포인트 (3줄) | |

### 2. 투자 포인트 (Investment Thesis)

**핵심 3가지**를 명확하게 제시:

1. **포인트 1**: 구체적 성장 동력 또는 가치 촉매
2. **포인트 2**: 재무적 강점 또는 개선점
3. **포인트 3**: 밸류에이션 매력도

### 3. 사업 분석

- 비즈니스 모델 요약
- 매출 구성 (세그먼트별)
- 시장 규모 및 성장성
- 경쟁 환경 및 포지셔닝
- 경쟁 우위 (Moat) 분석

### 4. 재무 분석

- 수익성 추세 (마진, ROE, ROIC)
- 성장성 (매출/이익 성장률, CAGR)
- 안정성 (부채비율, 이자보상배율)
- 효율성 (자산회전율, CCC)
- 주요 재무 비율 요약표

### 5. 밸류에이션

**DCF 밸류에이션**:
- 핵심 가정 (성장률, 마진, WACC, g)
- FCFF 추정표
- 적정가치 산출
- 민감도 분석

**상대가치 밸류에이션**:
- Peer Group 멀티플 비교
- Target 멀티플 근거
- 적정 주가 레인지

### 6. 리스크 요인

| 리스크 | 영향 | 발생 확률 |
|--------|------|----------|
| 매출 성장 둔화 | 높음 | 중간 |
| 마진 압축 | 높음 | 낮음 |
| 환율 변동 | 중간 | 높음 |
| 규제 변화 | 중간 | 낮음 |

## 보고서 작성 가이드

### 좋은 보고서의 특징

1. **논리적 구조**: 주장 → 근거 → 데이터의 흐름
2. **반대 의견 인정**: 리스크와 반론을 솔직하게 다룸
3. **수치 기반**: 정성적 판단도 정량적 근거 동반
4. **행동 가능**: 구체적 투자 의견과 목표 주가 제시
5. **간결함**: 핵심에 집중, 불필요한 장황함 배제

### 피해야 할 실수

| 실수 | 설명 |
|------|------|
| 확증 편향 | 자신의 결론에 유리한 증거만 나열 |
| 과도한 정밀성 | 소수점 이하까지 정확한 것처럼 제시 |
| 과거 의존 | 과거가 미래를 보장한다고 가정 |
| 매크로 무시 | 산업/경제 환경 변화 간과 |
| 리스크 경시 | 낙관적 시나리오에만 집중 |

### 보고서 작성 프로세스

```
1. 데이터 수집 (DART, IR 자료, 산업 보고서)
   ↓
2. 재무 분석 (Layer 2 프레임워크 적용)
   ↓
3. 정성적 분석 (비즈니스, 경쟁우위, 경영진)
   ↓
4. 재무 모델링 (3표 추정, FCF 산출)
   ↓
5. 밸류에이션 (DCF + 상대가치)
   ↓
6. 투자 판단 (목표주가, 투자의견)
   ↓
7. 보고서 작성 (구조화, 리뷰)
   ↓
8. 모니터링 (실적 추적, 가정 검증)
```

## 💡 실무 포인트

### 보고서 작성 체크리스트

- [ ] Executive Summary가 1페이지 내에 핵심을 전달하는가?
- [ ] 투자 포인트가 3가지 이내로 명확한가?
- [ ] 재무 분석이 데이터에 기반하는가?
- [ ] DCF와 상대가치 모두 수행했는가?
- [ ] 리스크 요인을 솔직하게 다루었는가?
- [ ] 목표 주가의 산출 근거가 투명한가?
- [ ] 실적 추정표가 포함되어 있는가?
- [ ] 한국 특수 요인을 반영했는가?

### Layer 4 완결: 분석에서 투자까지

```
  ↓

  ↓

  ↓

         → 보고서로 문서화한다
```
