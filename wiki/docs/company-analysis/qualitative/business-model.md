# 비즈니스 모델 분석

## 비즈니스 모델이란?

비즈니스 모델은 기업이 **가치를 창출하고, 전달하고, 수익화하는 메커니즘**입니다. 투자 분석의 출발점은 "이 기업이 어떻게 돈을 버는가?"를 이해하는 것입니다.

## 비즈니스 모델 분석 프레임워크

### 1. 수익 모델 (Revenue Model)

| 수익 모델 유형 | 설명 | 예시 |
|---------------|------|------|
| 제품 판매 | 제품을 만들어 판매 | 삼성전자, 현대차 |
| 서비스 수수료 | 서비스 대가 수취 | 은행, 카드사 |
| 구독형 | 정기적 이용료 | 넷플릭스, SaaS |
| 플랫폼/중개 | 거래 수수료 | 카카오, 쿠팡 |
| 광고 | 트래픽 기반 광고 수익 | 네이버, 유튜브 |
| 라이선스/로열티 | IP 사용료 | 특허 기업, 프랜차이즈 |
| 하이브리드 | 복합 수익 구조 | 아마존 (판매+구독+클라우드) |

### 2. 가치 사슬 분석 (Value Chain)

기업이 가치 사슬의 어디에 위치하는지 파악합니다:

| 위치 | 특성 | 마진 특성 |
|------|------|-----------|
| 상류 (원재료) | 원자재, 부품 공급 | 사이클에 민감, 변동 큼 |
| 중류 (제조) | 가공, 조립 | 규모의 경제 중요 |
| 하류 (유통/판매) | 최종 소비자 접점 | 브랜드/채널 중요 |
| 플랫폼 | 수요-공급 연결 | 네트워크 효과 |

### 3. 핵심 성공 요인 (KSF) 파악

업종별로 다른 핵심 성공 요인:

| 업종 | 핵심 성공 요인 |
|------|--------------|
| 반도체 | 기술력, 미세공정, 규모의 경제 |
| 소비재 | 브랜드 파워, 유통망 |
| 플랫폼 | 사용자 수, 네트워크 효과 |
| 제약 | 파이프라인, 특허 포트폴리오 |
| 은행 | 예대마진, 자산 건전성 |
| 건설 | 수주 능력, 원가 관리 |

### 4. 비용 구조 분석

| 비용 유형 | 고정비 비중 높은 기업 | 변동비 비중 높은 기업 |
|-----------|---------------------|---------------------|
| 특성 | 영업 레버리지 큼 | 매출 비례 비용 |
| 호황기 | 이익 급증 | 이익 완만 증가 |
| 불황기 | 이익 급감/적자 | 이익 완만 감소 |
| 예시 | 반도체, 통신, 항공 | 유통, 도매, 서비스 |

## 비즈니스 모델 평가 기준

### 좋은 비즈니스 모델의 특징

| 특징 | 설명 | 재무적 증거 |
|------|------|------------|
| **반복 수익** | 재구매/구독 기반 | 안정적 매출, 낮은 변동성 |
| **높은 마진** | 가격 결정력 | 영업이익률 15%+ |
| **낮은 자본 집약도** | 적은 설비투자 | 높은 ROIC, 낮은 CAPEX/매출 |
| **확장성** | 한계비용 낮음 | 매출 증가 시 마진 확대 |
| **전환 비용** | 고객 이탈 어려움 | 높은 고객 유지율 |
| **네트워크 효과** | 사용자 증가 → 가치 증가 | 가속 성장, 승자독식 |

### 비즈니스 모델과 밸류에이션

| 비즈니스 모델 | 전형적 밸류에이션 | 이유 |
|-------------|-----------------|------|
| SaaS/구독형 | EV/매출 10~30x | 반복 수익, 높은 성장성 |
| 플랫폼 | PER 30~60x | 네트워크 효과, 독점적 지위 |
| 제조업 | EV/EBITDA 6~10x | 자본 집약적, 사이클 |
| 유통 | PER 10~20x | 저마진, 안정적 |
| 금융 | PBR 0.5~1.5x | 레버리지 사업, 자산 기반 |

## 한국 기업 비즈니스 모델 특성

### 한국 대표 비즈니스 모델 유형

1. **수출 제조업** (삼성전자, 현대차, LG화학)
   - B2B/B2C 제품 수출 중심
   - 환율, 글로벌 수요 사이클에 민감
   - 대규모 R&D 및 설비투자

2. **내수 플랫폼** (카카오, 네이버, 쿠팡)
   - 국내 시장 중심 플랫폼
   - 광고/커머스/핀테크 하이브리드 수익
   - 규제 리스크 존재

3. **재벌 지주구조** (SK, LG, 한화)
   - 지주회사 + 핵심 사업자회사 구조
   - 사업 다각화, 그룹 시너지
   - 지배구조 디스카운트 이슈

4. **바이오/제약** (셀트리온, 삼성바이오)
   - 바이오시밀러/신약 파이프라인
   - 높은 불확실성, 바이너리 이벤트
   - 전통적 밸류에이션 적용 어려움

### DART 사업보고서에서 비즈니스 모델 파악

1. **"회사의 개요"**: 설립 경위, 주요 사업 내용
2. **"사업의 내용"**: 상세 사업 설명, 매출 구성, 시장 환경
3. **"재무에 관한 사항"**: 세그먼트별 매출/이익
4. **주석 - 부문정보**: 사업부별 상세 재무

## 실무 포인트

### 비즈니스 모델 분석 체크리스트

- [ ] 기업이 어떻게 돈을 버는지 한 문장으로 설명할 수 있는가?
- [ ] 주요 매출원의 구성 비율은?
- [ ] 수익 모델의 반복성/예측 가능성은?
- [ ] 비용 구조의 고정비/변동비 비율은?
- [ ] 고객 집중도 리스크는? (매출의 몇 %가 주요 고객?)
- [ ] 가치 사슬에서의 위치와 교섭력은?

### 밸류에이션 연결

비즈니스 모델 이해가 밸류에이션의 근본입니다:

| 비즈니스 모델 특성 | DCF 입력값 영향 |
|------------------|----------------|
| 수익 반복성 | 매출 예측 신뢰도 ↑ |
| 마진 수준 | EBIT/NOPAT 예측 |
| 자본 집약도 | CAPEX, 운전자본 가정 |
| 성장 잠재력 | 성장률 가정 |
| 리스크 수준 | 할인율 (WACC) |
