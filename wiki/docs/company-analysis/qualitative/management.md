# 경영진 분석

## 경영진 분석의 중요성

피터 린치는 "평범한 경영진이 이끄는 좋은 사업에 투자하라"고 했지만, 워런 버핏은 "우수한 경영진을 선호한다"고 말합니다. 경영진의 질은 장기적으로 기업 가치에 큰 영향을 미칩니다.

## 경영진 평가 프레임워크

### 1. 실행 능력 (Execution)

| 평가 항목 | 확인 방법 |
|-----------|----------|
| 가이던스 달성률 | 과거 제시한 목표 vs 실제 실적 |
| 전략 실행 | 발표한 전략/투자 계획의 이행 여부 |
| 위기 대응 | COVID, 금융위기 등 위기 시 대처 |
| 시장 점유율 변화 | 경쟁사 대비 실적 추이 |

### 2. 자본 배분 능력 (Capital Allocation)

경영진의 가장 중요한 역할 중 하나는 자본을 효율적으로 배분하는 것입니다.

| 자본 배분 옵션 | 좋은 신호 | 나쁜 신호 |
|---------------|----------|----------|
| 유기적 재투자 | ROIC > WACC인 사업에 투자 | 저수익 사업 확장 |
| M&A | 합리적 가격, 시너지 달성 | 고가 인수, 시너지 미실현 |
| 배당 | 안정적/점진적 증가 | 무리한 배당 (차입해서 배당) |
| 자사주 매입 | 저평가 시 매입, 소각 | 고평가 시 매입, 보유만 |
| 부채 상환 | 적정 레버리지 유지 | 과도한 차입 지속 |

**핵심 지표**: 장기 ROIC 추세
- ROIC가 지속적으로 WACC 초과 → 자본 배분 우수
- ROIC가 WACC 하회 추세 → 자본 배분 의문

### 3. 주주 커뮤니케이션

| 평가 항목 | 좋은 신호 | 나쁜 신호 |
|-----------|----------|----------|
| IR 활동 | 정기적, 투명한 소통 | 소극적, 불투명 |
| 실적 발표 | 솔직한 설명, 리스크 인정 | 과대 홍보, 리스크 은폐 |
| 가이던스 | 보수적이고 달성 가능한 목표 | 과도하게 낙관적 |
| 주주총회 | 소수주주 의견 존중 | 형식적 운영 |

### 4. 보상 구조 분석

| 항목 | 확인 사항 |
|------|----------|
| 고정급 vs 성과급 비율 | 성과 연동 비중 높을수록 긍정적 |
| 성과 지표 연동 | ROE, EPS, TSR 등 어떤 지표와 연동? |
| 스톡옵션/RSU | 장기 인센티브 구조 |
| 임원 자사주 보유 | 경영진의 스킨 인 더 게임 |

## 한국 기업 경영진 분석 특수 고려

### 오너 경영 vs 전문 경영

| 구분 | 오너 경영 | 전문 경영 |
|------|----------|----------|
| 장점 | 장기적 관점, 빠른 의사결정 | 전문성, 견제와 균형 |
| 단점 | 사익 편취 리스크, 독단 | 대리인 비용, 단기 실적주의 |
| 한국 현실 | 재벌 대부분 오너 경영 | 일부 대기업에서 확대 중 |
| 분석 포인트 | 지배구조, 총수 리스크 | 보상 구조, 연임 기준 |

### DART에서 경영진 정보 확인

1. **사업보고서 → 임원 현황**: 경력, 보유 주식 수
2. **사업보고서 → 보수 현황**: 이사/감사 보수 총액
3. **사업보고서 → 최대주주**: 지분 변동 내역
4. **주요사항보고서**: 대표이사 변경, 임원 선임
5. **지분변동공시**: 임원 주식 매매 내역

### 임원 주식 매매 신호

| 패턴 | 해석 |
|------|------|
| CEO 대규모 자사주 매수 | 강한 긍정 신호 |
| 복수 임원 동시 매수 | 강한 긍정 신호 |
| CEO 대규모 매도 | 주의 (세금, 개인 사유 확인) |
| IPO 후 임원 연속 매도 | 부정적 신호 |

## 실무 포인트

- [ ] CEO/주요 임원의 경력과 업계 경험은?
- [ ] 과거 가이던스 달성률은 어떤가?
- [ ] 자본 배분 의사결정의 질은? (ROIC 추세)
- [ ] M&A 실적이 있다면 성과는?
- [ ] 배당/자사주 정책은 주주 친화적인가?
- [ ] 임원 보상 구조가 주주 이익과 연동되는가?
- [ ] 임원 자사주 보유/거래 패턴은?
- [ ] 오너 경영이라면 지배구조 리스크는?

### 밸류에이션 연결

| 경영진 요소 | 밸류에이션 반영 |
|------------|---------------|
| 뛰어난 자본 배분 | ROIC 스프레드 프리미엄 |
| 약한 거버넌스 | 기업가치 할인 (20~30%) |
| 투명한 소통 | 낮은 정보 비대칭 → 낮은 할인율 |
| 임원 이해관계 일치 | 대리인 비용 감소 |
