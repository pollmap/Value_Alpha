# 지배구조 분석

## 지배구조(Corporate Governance)란?

기업의 경영 방향과 의사결정이 이루어지는 **제도적 체계**입니다. 주주, 이사회, 경영진 간의 권한과 책임의 분배를 다룹니다.

한국에서는 코리아 디스카운트의 핵심 원인으로 지목되어 특히 중요합니다.

## 지배구조 분석 프레임워크

### 1. 이사회 구성

| 항목 | 좋은 지배구조 | 나쁜 지배구조 |
|------|-------------|-------------|
| 사외이사 비율 | 과반수 이상 | 소수, 형식적 |
| 사외이사 독립성 | 회사와 무관한 전문가 | 총수 지인, 전관 |
| 이사회 의장 | CEO와 분리 | CEO가 겸임 |
| 전문성 | 업종/재무/법률 전문가 | 경력 불일치 |
| 이사회 활동 | 적극적 참여, 빈번한 개최 | 형식적, 연 2~3회 |

### 2. 감사 기능

| 항목 | 확인 사항 |
|------|----------|
| 감사위원회 | 설치 여부, 사외이사 구성 |
| 외부 감사인 | 4대 회계법인 여부, 교체 이력 |
| 감사 의견 | 적정, 한정, 부적정, 의견거절 |
| 핵심감사사항(KAM) | 감사인이 중요하게 본 회계 이슈 |
| 내부회계관리 | 내부통제 평가 의견 |

### 3. 소수주주 보호

| 항목 | 좋은 관행 | 나쁜 관행 |
|------|----------|----------|
| 배당 정책 | 명확한 배당 정책 공시 | 불규칙, 불투명 |
| 자사주 처분 | 소각 또는 투명한 사용 | 경영권 방어, 우호세력 매각 |
| 관계사 거래 | 시장가격 기준, 이사회 승인 | 불리한 조건, 일감 몰아주기 |
| 유상증자 | 합리적 할인율, 기존주주 우선 | 과도한 할인, 제3자 배정 |
| 주주총회 | 전자투표, 소수주주 의결권 행사 | 접근 어려운 장소/시간 |

### 4. 경영 투명성

| 항목 | 확인 사항 |
|------|----------|
| 공시 적시성 | 법정 기한 준수, 자발적 공시 |
| 재무 투명성 | 주석 상세도, 세그먼트 공시 |
| ESG 공시 | 지속가능경영보고서 발행 |
| IR 활동 | 정기 NDR, 컨퍼런스 콜, IR 자료 |

## 한국 지배구조 특수 이슈

### 재벌 지배구조

| 이슈 | 설명 | 투자 영향 |
|------|------|----------|
| **순환출자** | A→B→C→A 출자 구조 | 의결권 왜곡, 지분 희석 |
| **지주회사 전환** | 순환출자 해소 과정 | 지배구조 개선, 단기 이벤트 |
| **일감 몰아주기** | 총수 일가 기업에 물량 배정 | 수익성 왜곡, 소수주주 피해 |
| **사익 편취** | 유리한 조건의 내부거래 | 기업가치 유출 |
| **경영권 승계** | 세대교체 과정 | 불확실성, 지배구조 변화 |

### 코리아 디스카운트와 지배구조

한국 시장이 받는 할인의 상당 부분이 지배구조에 기인:

| 요인 | 할인 수준 (추정) |
|------|----------------|
| 총수 리스크 | 5~15% |
| 낮은 배당성향 | 5~10% |
| 관계사 거래 | 5~10% |
| 소수주주 보호 미흡 | 5~10% |
| **합계** | **20~45%** |

### 지배구조 개선 트리거

| 트리거 | 예상 효과 |
|--------|----------|
| 행동주의 펀드 개입 | 배당 증가, 자사주 소각, 구조 개선 |
| 지주회사 전환 | 출자 구조 단순화, 투명성 개선 |
| 스튜어드십 코드 | 기관투자자 의결권 행사 확대 |
| 정부 규제 강화 | 일감 몰아주기 규제, 사외이사 확대 |
| 밸류업 프로그램 | PBR 1배 미만 기업 개선 압박 |

### DART에서 지배구조 정보 확인

1. **사업보고서 → 이사회 구성**: 이사 명단, 사외이사 비율
2. **사업보고서 → 대주주 현황**: 최대주주 및 특수관계인 지분
3. **사업보고서 → 관계사 거래**: 특수관계자 거래 내역
4. **감사보고서**: 감사 의견, 핵심감사사항
5. **지배구조보고서**: 지배구조 원칙 및 현황 (대형사)

## 실무 포인트

### 지배구조 분석 체크리스트

- [ ] 사외이사 비율과 독립성은 충분한가?
- [ ] 감사위원회가 실질적으로 기능하는가?
- [ ] 대주주와 소수주주의 이해가 일치하는가?
- [ ] 관계사 거래의 규모와 조건은 합리적인가?
- [ ] 배당 정책이 명확하고 주주 친화적인가?
- [ ] 경영 승계 계획이 투명한가?
- [ ] ESG 지배구조 등급은 어떤가?

### 밸류에이션 연결

| 지배구조 수준 | 밸류에이션 반영 |
|-------------|---------------|
| 우수 | 프리미엄 적용 (할인율 하향) |
| 보통 | 시장 평균 멀티플 |
| 부실 | 디스카운트 적용 (20~40%) |
| 개선 기대 | 리레이팅(Re-rating) 가능성 |

**NAV 밸류에이션에서**: 지주회사의 경우 NAV에 지배구조 할인율 적용

$$\text{적정 주가} = \text{NAV} \times (1 - \text{지배구조 할인율})$$
