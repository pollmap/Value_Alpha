# DART 공시 분석 가이드

## DART 시스템 개요

**DART(Data Analysis, Retrieval and Transfer System)**는 금융감독원이 운영하는 전자공시시스템으로, 상장기업 및 일정 규모 이상 비상장기업의 공시를 의무적으로 등록하도록 하는 국내 최대의 기업 정보 플랫폼입니다.

- 공식 URL: [https://dart.fss.or.kr](https://dart.fss.or.kr)
- 운영 주체: 금융감독원
- 대상: 상장법인, 사업보고서 제출대상 비상장법인, 금융회사 등

DART는 투자자가 기업의 재무 상태와 경영 활동을 객관적으로 파악할 수 있는 **1차 정보원(Primary Source)**입니다. 증권사 리포트나 뉴스 기사가 2차 정보라면, DART 공시는 기업이 법적 의무에 따라 직접 제출하는 원본 자료입니다.

:::tip 왜 DART를 직접 봐야 하나요?
증권사 리포트는 애널리스트의 해석이 포함된 2차 자료입니다. DART 공시 원문을 직접 읽는 습관을 들이면, 시장 참여자 대부분이 놓치는 정보를 선제적으로 포착할 수 있습니다.
:::

## 주요 공시 유형

### 정기공시

기업이 일정 주기로 의무적으로 제출하는 보고서입니다.

| 공시 유형 | 제출 주기 | 제출 기한 | 주요 내용 |
|:----------|:----------|:----------|:----------|
| **사업보고서** | 연 1회 | 사업연도 종료 후 90일 | 기업 전반에 대한 종합 보고서 (가장 상세) |
| **반기보고서** | 연 1회 | 반기 종료 후 45일 | 상반기 경영실적 및 재무상태 |
| **분기보고서** | 연 2회 | 분기 종료 후 45일 | 1분기, 3분기 경영실적 (2분기는 반기보고서) |

### 비정기공시

특정 사건 발생 시 즉시 또는 단기간 내에 제출하는 보고서입니다.

| 공시 유형 | 제출 시기 | 중요도 |
|:----------|:----------|:------:|
| **주요사항보고서** | 사유 발생 즉시 | 매우 높음 |
| **감사보고서** | 정기주총 1주 전 | 높음 |
| **지분변동공시** | 변동 발생 5일 이내 | 중간 |
| **공개매수공고** | 공개매수 결정 시 | 매우 높음 |
| **자기주식 취득/처분** | 결정 즉시 | 높음 |

:::caution 공시 시간대에 주의하세요
장중 공시는 주가에 즉각 반영됩니다. 장 마감 후(15:30 이후) 공시는 다음 날 시가에 반영되므로, **매일 장 마감 후 DART를 확인하는 루틴**을 만드세요.
:::

## 사업보고서 읽는 법

사업보고서는 기업 분석의 핵심 문서입니다. 보통 100~300페이지에 달하지만, 핵심 섹션을 중심으로 효율적으로 읽을 수 있습니다.

### 사업보고서 구성

| 섹션 | 내용 | 투자 분석 활용도 |
|:-----|:-----|:----------------:|
| I. 회사의 개요 | 설립일, 상장일, 주요 연혁 | ★★☆ |
| II. 사업의 내용 | 사업 모델, 시장 현황, 경쟁 구도 | ★★★ |
| III. 재무에 관한 사항 | 재무제표, 재무비율, 회계정책 | ★★★ |
| IV. 감사인의 감사의견 | 적정/한정/부적정/의견거절 | ★★★ |
| V. 이사의 경영진단 | 경영진의 실적 평가 및 전망 | ★★☆ |
| VI. 이사회 등 회사의 기관에 관한 사항 | 이사회 구성, 보수, 감사위원회 | ★★☆ |
| VII. 주주에 관한 사항 | 최대주주, 지분 변동, 배당 정책 | ★★★ |
| VIII. 임원 및 직원 등에 관한 사항 | 임원 현황, 직원 수, 평균 급여 | ★☆☆ |
| IX. 계열회사 등에 관한 사항 | 계열사 현황, 거래 관계 | ★★☆ |
| X. 이해관계자와의 거래내용 | 특수관계인 거래, 내부 거래 | ★★★ |

### 섹션별 핵심 체크포인트

**II. 사업의 내용 — 가장 먼저 읽어야 할 섹션**

- 매출 구성 비율: 사업부문별 매출 비중 변화 추적
- 시장 점유율 및 경쟁 구도: 경쟁사 대비 포지션 파악
- 신규 사업 및 투자 계획: 향후 성장 동력 확인
- 규제 환경: 업종별 규제 변화가 실적에 미칠 영향

**III. 재무에 관한 사항 — 숫자로 검증하는 섹션**

- 연결재무제표 vs 별도재무제표 구분 확인
- 최근 5개년 재무 추이 분석
- 회계정책 변경 여부 (수익인식 기준, 감가상각 방법 등)
- 우발채무 및 약정사항 확인

**VII. 주주에 관한 사항 — 지배구조 파악**

- 최대주주 지분율 변동: 경영권 안정성 판단
- 자기주식 보유/처분 현황
- 배당 정책 및 배당성향 추이

**X. 이해관계자와의 거래내용 — 숨겨진 리스크**

- 특수관계인과의 거래 규모 및 조건
- 대주주 관련 채무보증, 담보 제공
- 계열사 간 내부거래 비율

## 감사보고서 핵심 포인트

감사의견은 재무제표의 신뢰성을 판단하는 가장 중요한 지표입니다.

| 감사의견 | 의미 | 투자 시사점 |
|:---------|:-----|:-----------|
| **적정** | 재무제표가 회계기준에 따라 적정하게 작성됨 | 정상 — 추가 확인 불필요 |
| **한정** | 일부 항목에 대해 의견 제한 | 주의 — 한정 사유 반드시 확인 |
| **부적정** | 재무제표가 중대하게 왜곡됨 | 위험 — 투자 재고 필요 |
| **의견거절** | 감사 범위 제한으로 의견 표명 불가 | 매우 위험 — 상장폐지 사유 |

:::caution 감사의견 변경에 주의
적정 → 한정으로 변경되면 시장에서 매우 부정적으로 해석합니다. **계속기업 불확실성** 관련 강조사항이 추가된 경우에도 주의가 필요합니다.
:::

### 감사보고서에서 반드시 확인할 사항

1. **핵심감사사항(KAM)**: 감사인이 가장 중요하다고 판단한 사항
2. **계속기업 가정**: 기업의 존속 가능성에 대한 의문 여부
3. **강조사항**: 적정의견이지만 주의를 환기하는 사항
4. **내부회계관리제도 검토 의견**: 재무보고 내부통제의 적정성

## 공시에서 발견하는 적색 신호 (Red Flags)

다음 신호가 공시에서 발견되면 심층 분석이 필요합니다.

### 재무 관련 적색 신호

| 적색 신호 | 확인 위치 | 위험 수준 |
|:----------|:----------|:---------:|
| 감사의견 한정 또는 의견거절 | 감사보고서 | 매우 높음 |
| 계속기업 불확실성 강조 | 감사보고서 강조사항 | 매우 높음 |
| 매출채권 회전일수 급증 | 재무제표 주석 | 높음 |
| 재고자산 급증 (매출 대비) | 재무상태표 | 높음 |
| 회계정책 빈번한 변경 | 주석사항 | 중간 |
| 영업현금흐름 지속 적자 | 현금흐름표 | 높음 |
| 특수관계인 거래 급증 | 이해관계자 거래 | 중간 |

### 지배구조 관련 적색 신호

| 적색 신호 | 확인 위치 | 위험 수준 |
|:----------|:----------|:---------:|
| 대주주 지분 급락 | 주주에 관한 사항 | 높음 |
| 경영진 빈번한 교체 | 임원 현황 | 중간 |
| 사외이사 비율 급감 | 이사회 구성 | 중간 |
| 대주주 사익 편취 거래 | 이해관계자 거래 | 매우 높음 |
| 대규모 채무보증 제공 | 우발채무 | 높음 |

:::tip 적색 신호 활용법
적색 신호가 하나 발견되었다고 즉시 매도할 필요는 없습니다. 다만 **복수의 적색 신호가 동시에 나타나면** 해당 기업의 투자 비중을 줄이거나 면밀한 추가 분석이 필요합니다.
:::

## 주요사항보고서 해석

주요사항보고서는 기업의 중대한 경영 변동 사항을 시장에 알리는 공시입니다. 주가에 즉각적인 영향을 미치므로 빠른 해석 능력이 중요합니다.

### 주요 유형별 분석

**유상증자**

| 확인 항목 | 해석 포인트 |
|:----------|:-----------|
| 증자 목적 | 시설투자 vs 운영자금 vs 채무상환 — 성장 투자인지, 자금난인지 구분 |
| 증자 규모 | 기존 발행주식 대비 비율 — 희석 효과 산출 |
| 발행가 결정 방식 | 할인율이 클수록 기존 주주에게 불리 |
| 실권주 처리 방식 | 일반공모 전환 여부 확인 |

**전환사채(CB) / 신주인수권부사채(BW)**

| 확인 항목 | 해석 포인트 |
|:----------|:-----------|
| 전환가격 | 현재 주가 대비 프리미엄/디스카운트 |
| 리픽싱 조항 | 전환가 하향 조정 가능 여부 — 추가 희석 위험 |
| 발행 대상 | 특수관계인 대상 사모 발행 시 주의 |
| 만기 및 조기상환 조건 | Put/Call option 조건 확인 |

:::caution CB/BW 리픽싱 조항
리픽싱(Refixing) 조항이 있는 CB는 주가 하락 시 전환가격이 자동으로 낮아져 **기존 주주의 희석 효과가 가속화**됩니다. 최초 전환가 대비 70~80%까지 하향 조정되는 경우가 많으므로, 최악의 시나리오 기준 희석률을 반드시 계산하세요.
:::

**자기주식 취득/처분**

| 확인 항목 | 해석 포인트 |
|:----------|:-----------|
| 취득 목적 | 주가 안정 vs 소각 vs 임직원 보상 |
| 취득 규모 | 발행주식 대비 비율 |
| 소각 여부 | 소각해야 진정한 주주가치 제고 |
| 처분 사유 | 스톡옵션 행사 대응, 현금 확보 등 |

**합병 및 분할**

| 확인 항목 | 해석 포인트 |
|:----------|:-----------|
| 합병비율 산정 근거 | 공정가치 평가 방법론 확인 |
| 합병 시너지 추정치 | 경영진이 제시하는 시너지의 현실성 |
| 반대주주 매수청구권 | 행사 가격 및 기간 확인 |
| 분할 후 상장 계획 | 존속/신설법인 상장 여부 |

## DART Open API 활용

DART Open API를 활용하면 공시 정보를 프로그래밍 방식으로 자동 수집할 수 있습니다.

### API 키 발급

1. [DART Open API 포털](https://opendart.fss.or.kr) 접속
2. 회원가입 후 API 키 신청
3. 1일 요청 한도: 10,000건 (일반), 20,000건 (기관)

### 주요 API 엔드포인트

| API | 용도 | 엔드포인트 |
|:----|:-----|:----------|
| 공시검색 | 기업별/유형별 공시 목록 조회 | `/api/list.json` |
| 기업개황 | 기업 기본 정보 조회 | `/api/company.json` |
| 재무제표 | 단일/다중 기업 재무 데이터 | `/api/fnlttSinglAcnt.json` |
| 지분공시 | 대량보유, 임원 지분 변동 | `/api/majorstock.json` |
| 배당 정보 | 배당 현황 조회 | `/api/alotMatter.json` |

### 자동 모니터링 예시 (Python)

```python
import requests
import time

API_KEY = "YOUR_API_KEY"
BASE_URL = "https://opendart.fss.or.kr"

def get_recent_disclosures(corp_code, begin_date, end_date):
    """특정 기업의 최근 공시 목록을 조회합니다."""
    url = f"{BASE_URL}/api/list.json"
    params = {
        "crtfc_key": API_KEY,
        "corp_code": corp_code,
        "bgn_de": begin_date,
        "end_de": end_date,
        "page_count": 100
    }
    response = requests.get(url, params=params)
    return response.json()

def check_major_reports(data):
    """주요사항보고서 여부를 확인합니다."""
    major_keywords = ["주요사항", "유상증자", "전환사채", "합병", "분할"]
    alerts = []
    for item in data.get("list", []):
        if any(kw in item["report_nm"] for kw in major_keywords):
            alerts.append(item)
    return alerts
```

:::tip API 활용 팁
관심 종목 리스트를 만들어 매일 장 마감 후 자동으로 신규 공시를 확인하는 스크립트를 구축하면, 중요 공시를 빠르게 포착할 수 있습니다.
:::

## DART 기반 투자 스크리닝 워크플로우

DART 공시를 활용한 체계적인 종목 스크리닝 프로세스입니다.

### 1단계: 유니버스 설정

- DART 재무제표 API로 전체 상장기업 재무 데이터 수집
- 시가총액, 거래량 등 기본 필터 적용

### 2단계: 정량 스크리닝

- 영업이익률, ROE, 부채비율 등 재무비율 기준 필터링
- 3~5년 연속 실적 성장 기업 추출
- [ROIC 계산기](/calculators/roic)로 자본 효율성 검증

### 3단계: 공시 품질 확인

- 감사의견 적정 여부 확인
- 최근 1년 주요사항보고서 이력 확인
- 대주주 지분 변동 추적

### 4단계: 정성 분석

- 사업보고서 "사업의 내용" 섹션 정독
- 경쟁 우위 및 진입장벽 평가
- 경영진 역량 및 지배구조 확인

### 5단계: 밸류에이션

## 공시 유형별 중요도 및 대응 전략 요약

| 공시 유형 | 주가 영향 | 대응 속도 | 분석 난이도 | 핵심 대응 |
|:----------|:---------:|:---------:|:-----------:|:----------|
| 실적 공시 (잠정) | 높음 | 즉시 | 낮음 | 컨센서스 대비 서프라이즈 여부 확인 |
| 사업보고서 | 중간 | 1~2일 | 높음 | 전년 대비 변경 사항 중심 분석 |
| 유상증자 | 매우 높음 | 즉시 | 중간 | 희석률 및 자금 용도 확인 |
| CB/BW 발행 | 높음 | 즉시 | 높음 | 리픽싱 조항, 최대 희석률 계산 |
| 자기주식 취득 | 중간 | 당일 | 낮음 | 취득 목적 및 소각 계획 확인 |
| 합병/분할 | 매우 높음 | 즉시 | 매우 높음 | 합병비율 적정성, 시너지 분석 |
| 대주주 지분변동 | 중간 | 당일 | 낮음 | 매도 사유 확인 (세금, 경영권 등) |
| 감사의견 변경 | 매우 높음 | 즉시 | 중간 | 한정/부적정 사유 확인, 상폐 위험 점검 |

## 실전 팁 모음

:::tip DART 일일 루틴
1. 매일 장 마감 후 관심 종목 공시 확인 (15~20분)
2. 주 1회 업종별 주요사항보고서 스캔
3. 분기마다 보유 종목 사업보고서 정독
4. 연 1회 감사보고서 핵심감사사항 확인
:::

:::tip 비교 분석 활용법
동일 업종 내 3~5개 기업의 사업보고서를 나란히 비교하면, 개별 기업만 봤을 때 놓칠 수 있는 업종 공통 트렌드와 기업별 차별점을 동시에 파악할 수 있습니다. [Company Comparison 도구](/tools/company-comparison)를 함께 활용하세요.
:::

:::caution DART 데이터의 한계
DART 공시는 과거 및 현재 데이터만 제공합니다. 미래 전망은 경영진의 주관이 반영된 "이사의 경영진단" 섹션에서만 제한적으로 확인할 수 있으며, 이를 그대로 신뢰해서는 안 됩니다. 독자적인 미래 추정이 반드시 필요합니다.
:::
