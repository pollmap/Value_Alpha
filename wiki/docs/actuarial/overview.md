# 보험 계리 개요

보험 계리(Actuarial Science)는 보험회사의 재무적 건전성과 상품 설계의 수리적 기반을 제공하는 핵심 학문입니다. 보험이라는 산업 자체가 미래의 불확실한 사건에 대한 경제적 보장을 약속하는 비즈니스 모델이기 때문에, 그 약속의 이행 가능성을 수학적으로 검증하고 관리하는 계리 기능은 보험 경영의 근간이 됩니다.

---

## 1. 보험 계리란?

### 1.1 정의와 본질

보험 계리란 확률론, 통계학, 금융수학을 활용하여 보험과 관련된 리스크를 측정, 평가, 관리하는 전문 분야입니다. 구체적으로는 보험료를 산출하고, 보험금 지급에 필요한 준비금(리저브)을 적립하며, 보험회사의 재무 건전성을 유지하기 위한 일련의 수리적 업무를 수행합니다.

보험 계리의 핵심은 **"미래의 불확실성을 현재 시점에서 화폐 가치로 환산하는 것"**입니다. 이는 밸류에이션(가치평가)의 본질과 정확히 일치합니다. DCF(현금흐름할인법)에서 미래 현금흐름을 현재가치로 환산하듯, 보험 계리에서는 미래의 보험금 지급 의무를 현재 시점의 부채로 인식합니다.

### 1.2 보험회사 경영에서의 위치

보험회사의 경영 구조에서 계리 기능은 다음과 같은 위치를 차지합니다.

| 경영 영역 | 계리 기능의 역할 | 구체적 업무 |
|-----------|-----------------|------------|
| **상품 개발** | 보험료 산출, 상품 구조 설계 | 위험률 산출, 예정이율 설정, 해약환급금 설계 |
| **언더라이팅** | 위험 분류 기준 제공 | 경험통계 분석, 위험등급 체계 설계 |
| **준비금 관리** | 보험부채 평가 | 책임준비금 산출, IFRS 17 부채 평가 |
| **리스크 관리** | 리스크 계량화 | K-ICS 요구자본 산출, 스트레스 테스트 |
| **투자 전략** | ALM(자산부채관리) | 듀레이션 매칭, 투자수익률 목표 설정 |
| **경영 전략** | 수익성 분석, 사업 가치 평가 | Embedded Value 산출, 신계약 가치 분석 |

### 1.3 계리사(Actuary)의 역할

계리사는 보험회사에서 수리적 전문성을 바탕으로 핵심 의사결정을 지원하는 전문가입니다. 한국에서 계리사 자격은 금융감독원이 관리하며, 보험업법에 따라 모든 보험회사는 선임계리사(Appointed Actuary)를 의무적으로 선임해야 합니다.

선임계리사의 법적 책임은 다음과 같습니다.

- 보험료 및 책임준비금의 적정성 확인
- 배당금 산출의 공정성 검증
- 보험회사 재무 건전성에 대한 의견서 작성
- 금융감독원에 대한 계리 보고서 제출

## 2. 계리사의 업무 영역

### 2.1 상품 개발 (Product Development)

보험 상품 개발의 핵심은 **적정한 보험료 산출**입니다. 보험료는 크게 세 가지 요소로 구성됩니다.

```
영업보험료 = 순보험료 + 부가보험료

순보험료 = 위험보험료 + 저축보험료
부가보험료 = 사업비(신계약비 + 유지비 + 수금비)
```

**위험보험료**는 보험금 지급에 충당되는 부분으로, 위험률(사망률, 질병 발생률 등)에 기반하여 산출됩니다. **저축보험료**는 만기보험금이나 해약환급금 지급을 위해 적립되는 부분으로, 예정이율을 적용하여 계산합니다. **부가보험료**는 보험회사의 사업 운영 비용을 충당합니다.

### 2.2 준비금(리저브) 산출

보험회사가 미래의 보험금 지급 의무를 이행하기 위해 적립하는 부채를 **책임준비금**이라 합니다. 이는 보험회사 재무상태표에서 가장 큰 비중을 차지하는 항목입니다.

책임준비금의 종류는 다음과 같습니다.

| 준비금 유형 | 설명 | 해당 보험 |
|------------|------|----------|
| **보험료 적립금** | 장래 보험금 지급을 위해 보험료에서 적립 | 생명보험, 장기손해보험 |
| **미경과보험료 적립금** | 아직 보험기간이 경과하지 않은 보험료 | 일반손해보험 |
| **지급준비금** | 이미 발생했으나 아직 지급하지 않은 보험금 | 모든 보험 |
| **IBNR** | 발생했으나 아직 보고되지 않은 보험금(Incurred But Not Reported) | 주로 손해보험 |
| **계약자배당 준비금** | 계약자에게 배당하기 위한 준비금 | 유배당 보험 |

IFRS 17 도입 이후, 보험부채는 **BEL(최선추정부채) + RA(위험조정) + CSM(계약서비스마진)**으로 구성되며, 이에 대한 상세한 내용은 [IFRS 17 심층 분석](./ifrs17) 페이지에서 다룹니다.

### 2.3 리스크 관리 (Risk Management)

계리사는 보험회사가 직면하는 다양한 리스크를 계량화하고 관리하는 역할을 담당합니다. 주요 리스크 유형과 계리적 관리 방법은 다음과 같습니다.

- **보험위험(Insurance Risk)**: 실제 사망률, 유병률, 손해율이 예상과 다를 위험. 경험통계 분석과 위험률 모니터링으로 관리.
- **금리위험(Interest Rate Risk)**: 자산과 부채의 금리 민감도 차이에서 발생. 듀레이션 갭 분석과 ALM으로 관리.
- **해약위험(Lapse Risk)**: 해약률이 예상과 다를 위험. 해약률 시나리오 분석과 상품 설계(해약환급금 구조)로 관리.
- **사업비위험(Expense Risk)**: 실제 사업비가 예정 사업비를 초과할 위험.

이에 대한 상세한 분석은 [리스크와 자본 관리](./risk-capital) 페이지에서 다룹니다.

### 2.4 투자 전략과 ALM

보험회사는 보험료 수입으로 막대한 자산을 운용합니다. 한국 보험산업의 총 운용자산은 약 1,200조 원에 달하며, 이는 국내 금융시장에서 매우 중요한 기관투자자 역할을 합니다.

ALM(Asset-Liability Management, 자산부채관리)은 보험회사 투자 전략의 핵심 프레임워크입니다. 보험부채의 듀레이션(평균 잔존 만기)에 맞추어 자산의 듀레이션을 조정함으로써, 금리 변동에 따른 자산-부채 간 불일치 리스크를 관리합니다.

```
Duration Gap = 자산 Duration - 부채 Duration × (부채/자산)

목표: Duration Gap → 0 (면역화 전략)
```

한국 생명보험사의 경우, 과거 고금리 시대에 판매한 장기 확정금리 상품으로 인해 부채의 듀레이션이 자산의 듀레이션보다 현저히 긴 **"네거티브 듀레이션 갭"** 문제를 안고 있습니다. 이로 인해 저금리 환경에서 역마진(逆마진)이 발생하며, 이는 한국 보험산업의 구조적 과제입니다.

## 3. 보험의 기본 원리

### 3.1 대수의 법칙 (Law of Large Numbers)

보험의 수리적 기반이 되는 가장 근본적인 원리입니다. 개별적으로는 예측이 불가능한 사건(사망, 사고 등)도, 충분히 큰 집단에서는 그 발생 빈도가 일정한 통계적 규칙성을 보입니다.

예를 들어, 특정 연령대 남성의 1년 내 사망 여부를 개인 단위로 예측하는 것은 불가능하지만, 10만 명의 동일 연령대 남성 집단에서는 사망률이 통계적으로 안정적인 값을 보입니다. 이 원리를 통해 보험회사는 보험금 지급 총액을 상당한 정확도로 예측할 수 있으며, 이를 기반으로 적정 보험료를 산출합니다.

```
lim (n→∞) P(|X̄ₙ - μ| > ε) = 0

X̄ₙ: 표본평균 (실제 보험금 지급률)
μ: 모평균 (예상 보험금 지급률)
ε: 허용 오차
n: 표본 크기 (보험 계약 수)
```

### 3.2 수지상등의 원칙 (Equivalence Principle)

보험 계약 기간 동안 보험료의 수입 총액의 현재가치와 보험금 지출 총액의 현재가치가 같아야 한다는 원칙입니다.

```
PV(보험료 수입) = PV(보험금 지출 + 사업비 지출)

∑ Pₜ × vᵗ × ₜpₓ = ∑ Bₜ × vᵗ × ₜpₓ × qₓ₊ₜ + ∑ Eₜ × vᵗ × ₜpₓ

P: 보험료, B: 보험금, E: 사업비
v: 할인인자 (1/(1+i)), i: 예정이율
ₜpₓ: x세의 사람이 t년 후 생존할 확률
qₓ₊ₜ: x+t세의 사람이 1년 내 사망할 확률
```

이 원칙은 보험료 산출의 기본 등식이며, 보험회사가 장기적으로 손실 없이 보험 계약을 이행할 수 있는 균형점을 제시합니다.

### 3.3 급부반대급부의 원칙 (Principle of Benefit and Counter-Benefit)

각 보험 계약자가 납입하는 보험료는 그 계약자의 리스크(위험도)에 비례해야 한다는 원칙입니다. 즉, 위험이 높은 계약자는 더 높은 보험료를 납입하고, 위험이 낮은 계약자는 더 낮은 보험료를 납입해야 한다는 것입니다.

이 원칙은 **역선택(Adverse Selection)** 방지와 직결됩니다. 만약 모든 계약자에게 동일한 보험료를 부과한다면, 건강한 사람들은 보험 가입을 꺼리고 건강하지 못한 사람들만 가입하게 되어 보험 풀(pool)이 악화되는 현상이 발생합니다.

### 3.4 이익 3원 (Three Sources of Profit)

보험회사의 이익은 세 가지 원천에서 발생합니다.

| 이익 원천 | 정의 | 산출 방법 |
|-----------|------|----------|
| **사차익(死差益)** | 예정 사망률과 실제 사망률의 차이에서 발생 | 예정위험보험료 - 실제 보험금 지급액 |
| **이차익(利差益)** | 예정이율과 실제 투자수익률의 차이에서 발생 | 실제 투자수익 - 예정이율 적용 이자비용 |
| **비차익(費差益)** | 예정 사업비와 실제 사업비의 차이에서 발생 | 예정사업비 - 실제 사업비 |

```
보험이익 = 사차익 + 이차익 + 비차익

사차익 = (예정위험률 - 실제위험률) × 위험보험금
이차익 = (실제투자수익률 - 예정이율) × 책임준비금
비차익 = (예정사업비율 - 실제사업비율) × 영업보험료
```

한국 생명보험 산업에서는 오랫동안 **이차역마진(利差逆마진)** 문제가 지속되어 왔습니다. 과거 고금리 시대(1990년대)에 예정이율 7~8%로 판매한 장기 보험 계약이 현재까지 유효한데, 현재 투자수익률은 3~4%에 불과하여 이차손(利差損)이 발생하고 있습니다.

## 4. 생명보험 vs 손해보험

### 4.1 구조적 차이

생명보험과 손해보험은 보장하는 위험의 성격, 상품 구조, 준비금 체계 등에서 근본적인 차이가 있습니다.

| 구분 | 생명보험(Life Insurance) | 손해보험(Non-Life Insurance) |
|------|------------------------|---------------------------|
| **보장 대상** | 사람의 생사에 관한 위험 | 재산상의 손해 |
| **보험 기간** | 장기 (10~종신) | 단기 (1년) 또는 장기 |
| **보험금 지급** | 정액 (약정 금액) | 실손 (실제 손해액) |
| **언더라이팅** | 건강 상태, 직업 기반 | 물건, 환경, 이력 기반 |
| **준비금 핵심** | 보험료 적립금 | 지급준비금(IBNR 포함) |
| **주요 리스크** | 장수위험, 금리위험 | 자연재해, 사고빈도 |
| **규제 감독** | 금융위원회/금융감독원 | 금융위원회/금융감독원 |
| **자산 운용** | 장기 채권 중심 | 상대적으로 단기 운용 |

### 4.2 리스크 특성

**생명보험의 주요 리스크:**

- **사망위험(Mortality Risk)**: 실제 사망률이 예정 사망률을 초과할 위험
- **장수위험(Longevity Risk)**: 연금보험에서 계약자가 예상보다 오래 생존할 위험
- **질병위험(Morbidity Risk)**: 건강보험에서 질병 발생률이 예상을 초과할 위험
- **해약위험(Lapse Risk)**: 해약률 변동이 수익성에 미치는 영향

**손해보험의 주요 리스크:**

- **자연재해위험(Catastrophe Risk)**: 태풍, 지진, 홍수 등 대규모 자연재해
- **손해율 변동위험**: 자동차 사고율, 배상책임 판결 등의 변동
- **준비금 부족위험**: IBNR(발생했으나 미보고된 보험금) 추정 오류
- **장기 인플레이션위험**: 의료비 인상 등으로 인한 보험금 증가

### 4.3 주요 상품

**생명보험 주요 상품:**

| 상품 유형 | 설명 | IFRS 17 측정 모형 |
|-----------|------|------------------|
| 종신보험 | 사망 시 보험금 지급, 저축 기능 포함 | BBA |
| 정기보험 | 일정 기간 내 사망 시 보험금 지급 | BBA 또는 PAA |
| 연금보험 | 생존 시 연금 지급 | BBA |

## 6. IFRS 17 개요

### 6.1 왜 보험 회계가 바뀌었나

기존 IFRS 4는 보험계약에 대한 포괄적인 회계 기준이 아닌 **과도기적 기준**이었습니다. 각 국가별로 서로 다른 회계 처리를 허용했기 때문에, 국제적 비교가능성이 현저히 떨어졌습니다.

IFRS 17 도입의 핵심 배경은 다음과 같습니다.

1. **비교가능성 확보**: 글로벌 보험사 간 재무제표 비교 가능
2. **투명성 제고**: 보험부채를 시가(시장가치)로 평가하여 재무 건전성의 실질적 파악 가능
3. **수익 인식의 합리화**: 보험료 수취 시점이 아닌 보험서비스 제공 시점에 수익 인식
4. **미실현이익 이연**: 보험계약 체결 시점에 이익을 인식하지 않고 서비스 기간에 걸쳐 배분(CSM)

### 6.2 IFRS 4 vs IFRS 17 핵심 차이

| 구분 | IFRS 4 (구기준) | IFRS 17 (신기준) |
|------|----------------|-----------------|
| **부채 평가** | 원가(예정기초율) 기반 | 시가(현행추정치) 기반 |
| **할인율** | 계약 시점 예정이율 고정 | 현행 시장이율 반영 |
| **수익 인식** | 수입보험료 기준 | 보험서비스 제공 기준 |
| **미실현이익** | 계약 체결 시 인식 가능 | CSM으로 이연, 서비스 기간 배분 |
| **투자요소** | 매출에 포함 | 매출에서 제외 |
| **손실 계약** | 인식 불명확 | 즉시 인식 (손실부담계약) |
| **비교가능성** | 국가별 상이 | 글로벌 통일 기준 |

### 6.3 핵심 개념 소개

IFRS 17의 보험계약부채는 세 가지 구성요소로 이루어집니다.

```
보험계약부채 = BEL + RA + CSM

BEL (Best Estimate Liability, 최선추정부채)
  = 미래 보험금 지급 현금흐름의 현재가치
  = PV(보험금 유출 + 사업비) - PV(보험료 유입)

RA (Risk Adjustment, 위험조정)
  = 비금융위험에 대한 보상
  = 보험금 지급의 불확실성에 대한 추가 마진

CSM (Contractual Service Margin, 계약서비스마진)
  = 미실현이익
  = 보험서비스 제공 기간에 걸쳐 수익으로 인식
```

이 세 가지 구성요소에 대한 심층 분석은 [IFRS 17 심층 분석](./ifrs17) 페이지에서 다룹니다.

### 6.4 투자자를 위한 시사점

IFRS 17 도입 이후 보험주 투자에서 주목해야 할 핵심 포인트는 다음과 같습니다.

1. **CSM 규모**: 미래 이익의 파이프라인. CSM이 클수록 향후 안정적 이익 창출 가능
2. **신계약 CSM**: 신규 판매 상품의 수익성 지표. 보험사의 성장성 대리 변수
3. **CSM 상각률**: CSM이 보험수익으로 전환되는 속도
4. **K-ICS 비율**: 자본 건전성 지표. 배당 여력과 직결
5. **보험수익 구조 변화**: 저축성 보험 비중이 높은 생보사는 매출 규모가 대폭 감소
