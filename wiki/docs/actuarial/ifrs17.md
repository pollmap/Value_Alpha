# IFRS 17 보험계약 회계 심층 분석

IFRS 17(International Financial Reporting Standard 17)은 보험계약에 대한 국제회계기준으로, 2023년 1월 1일부터 전 세계적으로 시행되었습니다. 이 기준서는 보험부채를 시가 기반으로 평가하고, 보험수익을 서비스 제공 기준으로 인식하며, 미실현이익을 계약서비스마진(CSM)으로 이연하는 것을 핵심으로 합니다.

IFRS 17은 보험회사의 재무제표 체계를 근본적으로 바꾸었으며, 이는 보험주 밸류에이션에도 심대한 영향을 미칩니다. 본 페이지에서는 IFRS 17의 핵심 구조를 전문가 수준에서 분석합니다.

---

## 1. IFRS 17 보험계약부채 구조

### 1.1 보험계약부채의 3대 구성요소

IFRS 17에서 보험계약부채(Insurance Contract Liability)는 다음 세 가지 요소의 합으로 구성됩니다.

```
보험계약부채 = BEL + RA + CSM
```

각 구성요소를 상세히 살펴보겠습니다.

### 1.2 BEL (Best Estimate Liability, 최선추정부채)

BEL은 보험계약에서 발생하는 **미래현금흐름의 현재가치**입니다. 보험회사가 향후 지급해야 할 보험금, 사업비 등의 유출액에서 수취할 보험료 유입액을 차감한 순 현금흐름을 할인한 값입니다.

```
BEL = PV(미래 현금유출) - PV(미래 현금유입)

미래 현금유출:
  + 보험금 지급 (사망보험금, 만기보험금, 연금, 의료비 등)
  + 해약환급금 지급
  + 사업비 (유지비, 보험금 지급 관련 비용)
  + 계약자배당

미래 현금유입:
  - 보험료 수입 (향후 수취 예정인 보험료)
```

BEL 산출 시 핵심 가정은 다음과 같습니다.

- **사망률/유병률** — 설명: 경험통계 기반 위험률, 갱신 주기: 매 보고기간
- **해약률** — 설명: 계약 유지율/해약률, 갱신 주기: 매 보고기간
- **사업비율** — 설명: 유지비, 수금비 등, 갱신 주기: 매 보고기간
- **할인율** — 설명: 현행 시장이율 반영, 갱신 주기: 매 보고기간
- **인플레이션** — 설명: 사업비, 의료비 상승률, 갱신 주기: 매 보고기간

**핵심 포인트**: BEL은 매 보고기간마다 현행 가정(current estimates)으로 재평가됩니다. 이는 기존 IFRS 4에서 계약 시점의 가정을 고정(locked-in)했던 것과 근본적으로 다릅니다. BEL의 변동은 보험수익 또는 보험금융손익으로 반영됩니다.

### 1.3 RA (Risk Adjustment, 위험조정)

RA는 보험계약의 비금융위험(non-financial risk)에 대한 불확실성을 보상하기 위해 보험부채에 추가하는 마진입니다. 쉽게 말해, "예상대로 되지 않을 수 있는 위험"에 대한 완충 장치입니다.

```
RA의 경제적 의미:
  보험회사가 불확실한 현금흐름을 부담하는 대가로
  확실한 현금흐름과 교환하기 위해 기꺼이 지불할 금액

측정 방법:
  ① 신뢰수준법 (Confidence Level): 일반적으로 60\~90% 수준
  ② VaR (Value at Risk) 방식
  ③ CTE (Conditional Tail Expectation) 방식
```

RA의 주요 특성은 다음과 같습니다.

- **비금융위험만 반영**: 금리위험, 시장위험 등 금융위험은 할인율에 반영되므로 RA에서 제외
- **포트폴리오 수준에서 측정**: 분산효과(diversification benefit) 반영 가능
- **경과에 따라 해제**: 보험서비스를 제공하면서(위험이 소멸하면서) RA가 감소하고, 해제된 RA는 보험수익으로 인식
- **신뢰수준 공시 의무**: IFRS 17은 RA를 신뢰수준으로 환산하여 공시할 것을 요구

한국 주요 보험사의 RA 신뢰수준은 대체로 다음 범위에 있습니다.

- **삼성생명** — RA 신뢰수준: 약 75%, 비고: 보수적 접근
- **한화생명** — RA 신뢰수준: 약 70\~75%, 비고: 업계 평균 수준
- **교보생명** — RA 신뢰수준: 약 70\~75%, 비고: 비상장
- **삼성화재** — RA 신뢰수준: 약 75%, 비고: 손보 업계 선도
- **DB손해보험** — RA 신뢰수준: 약 70\~75%, 비고: 안정적 수준
- **현대해상** — RA 신뢰수준: 약 70\~75%, 비고: 업계 평균 수준

### 1.4 CSM (Contractual Service Margin, 계약서비스마진)

CSM은 IFRS 17에서 가장 핵심적이고 혁신적인 개념입니다. CSM은 보험계약에서 아직 제공하지 않은 보험서비스에 대한 **미실현이익(unearned profit)**을 의미합니다.

```
최초 인식 시 CSM 산출:

CSM = -(BEL + RA)  (단, CSM ≥ 0)

만약 BEL + RA > 0 (즉, 미래 유출 > 미래 유입):
  → 손실부담계약(Onerous Contract)
  → CSM = 0, 손실을 즉시 당기손실로 인식

만약 BEL + RA < 0 (즉, 미래 유입 > 미래 유출):
  → 수익성 계약
  → CSM = |BEL + RA|, 미실현이익으로 이연
```

**CSM의 핵심 의의:**

1. **이익의 이연(Deferral)**: 보험 계약 체결 시점에 이익을 인식하지 않음
2. **서비스 기간 배분**: CSM을 보험서비스 제공 기간에 걸쳐 체계적으로 상각하여 보험수익으로 인식
3. **미래 이익의 대리 변수**: CSM 잔액은 보험회사가 향후 인식할 수 있는 이익의 "파이프라인"
4. **밸류에이션 핵심 지표**: CSM 규모와 성장률은 보험주 밸류에이션의 핵심 요소

## 2. 측정 모형 (Measurement Models)

IFRS 17은 보험계약의 특성에 따라 세 가지 측정 모형을 제시합니다.

### 2.1 BBA (Building Block Approach, 일반 모형)

BBA는 IFRS 17의 기본(디폴트) 측정 모형입니다. 장기 보험계약에 적용되며, BEL + RA + CSM의 세 가지 빌딩 블록으로 보험부채를 측정합니다.

**적용 대상:**
- 종신보험, 연금보험, 건강보험 등 장기 생명보험
- 장기 손해보험
- 재보험 계약 (출재자 관점)

**특징:**
- 매 보고기간 현행 가정으로 BEL과 RA를 재측정
- 미래서비스 관련 가정변경은 CSM에서 조정
- 과거서비스 또는 당기서비스 관련 가정변경은 당기 보험수익/손실로 인식

### 2.2 VFA (Variable Fee Approach, 변동수수료 접근법)

VFA는 **직접참가 특성(direct participation features)**이 있는 보험계약에 적용되는 수정 모형입니다. 변액보험과 같이 기초항목(펀드 등)의 수익에 계약자가 직접 참여하는 상품에 적용됩니다.

**직접참가 특성의 세 가지 요건 (모두 충족해야 함):**

1. 계약 조건에 따라 계약자가 명확히 식별된 기초항목 풀(pool)에 대한 지분을 보유
2. 보험회사가 기초항목의 공정가치 수익 중 상당 부분을 계약자에게 지급
3. 보험회사가 계약자에게 지급할 현금흐름의 상당 부분이 기초항목의 공정가치에 따라 변동

**VFA의 핵심 차이점:**

```
BBA에서의 CSM 조정:
  비금융 가정변경 → CSM 조정
  금융 가정변경(할인율 등) → 보험금융손익 또는 OCI

VFA에서의 CSM 조정:
  비금융 가정변경 → CSM 조정
  금융 가정변경(할인율 등) → CSM 조정 (핵심 차이!)
```

VFA에서는 기초항목의 공정가치 변동이 CSM에 반영됩니다. 이는 변액보험에서 보험회사가 수취하는 "변동 수수료(variable fee)"의 성격을 반영한 것입니다.

**적용 대상:**
- 변액보험(Variable Life Insurance)
- 유니버설 보험(Universal Life)
- 실적배당형 연금

### 2.3 PAA (Premium Allocation Approach, 보험료배분 접근법)

PAA는 보장기간이 1년 이하이거나, BBA와 유사한 결과를 산출하는 경우 적용할 수 있는 **간편법**입니다.

**적용 요건 (택1):**
- 보장기간이 1년 이하
- BBA 적용 결과와 중요하게 다르지 않을 것으로 예상

**PAA의 핵심 차이점:**

```
PAA 잔여보장부채 = 미경과보험료 - 신계약비(상각)

→ CSM을 별도로 산출하지 않음
→ 보험부채 측정이 대폭 간소화
→ BEL + RA + CSM 구조가 아닌, 미경과보험료 기반
```

**적용 대상:**
- 자동차보험 (1년 갱신)
- 화재보험
- 배상책임보험
- 기타 단기 손해보험

### 2.4 측정 모형 비교 표

- **적용 대상** — BBA: 장기보험 (기본), VFA: 직접참가 특성 보험, PAA: 단기보험 (간편법)
- **CSM 산출** — BBA: O, VFA: O (변동수수료 반영), PAA: X (미경과보험료 기반)
- **금융변수 CSM 조정** — BBA: X (보험금융손익), VFA: O (CSM 조정), PAA: N/A
- **BEL 재측정** — BBA: 매 보고기간, VFA: 매 보고기간, PAA: 손실부담 판단 시
- **주요 적용 상품** — BBA: 종신, 연금, 건강, VFA: 변액보험, PAA: 자동차, 화재
- **한국 적용 비중** — BBA: 생보 대부분, VFA: 변액보험, PAA: 일반 손보 대부분

## 3. CSM 상각 (Roll-Forward)

CSM의 기간별 변동(Roll-Forward)은 IFRS 17 분석의 핵심입니다. CSM이 어떻게 증가·감소하는지를 이해하면, 보험회사의 미래 이익 흐름을 예측할 수 있습니다.

### 3.1 CSM Roll-Forward 구조

```
기초 CSM
  (+) 신계약 CSM                    ← 당기 신규 판매 계약의 CSM
  (+) 이자 부리 (locked-in rate)     ← CSM에 대한 이자 (시간가치)
  (-) CSM 상각 → 보험수익 인식       ← 보험서비스 제공에 따른 상각
  (±) 미래서비스 관련 가정변경        ← 비금융 가정(사망률, 해약률 등) 변경
  (±) 금융변수 변경 (VFA에 한함)      ← 기초항목 공정가치 변동
  (-) 손실부담 전환 시 차감           ← CSM이 0 이하로 떨어지면 손실 인식
기말 CSM
```

### 3.2 각 항목의 상세 설명

**신계약 CSM:**
- 당기에 새로 판매한 보험계약에서 발생하는 CSM
- 신계약 CSM = -(신계약 BEL + 신계약 RA) (단, ≥ 0)
- 보험사의 **성장성**과 **신계약 수익성**을 나타내는 핵심 지표
- 신계약 CSM이 클수록 미래 이익 파이프라인이 확대

**이자 부리:**
- CSM 잔액에 최초 인식 시점의 할인율(locked-in rate)을 적용하여 이자를 부리
- CSM의 화폐 시간가치를 반영
- 보험계약 체결 시점의 금리 수준에 따라 결정

**CSM 상각:**
- 보험서비스를 제공함에 따라 CSM의 일부를 보험수익으로 인식
- 상각 단위: **보장단위(Coverage Unit)** 기준
- 보장단위는 보험계약이 제공하는 보장의 양과 기대 보장기간을 반영
- 상각률 = 당기 보장단위 / (당기 보장단위 + 미래 보장단위)

```
CSM 상각 예시:

종신보험 (보장기간 30년):
  총 보장단위: 30단위 (매년 1단위)
  당기 상각률 = 1/30 = 3.3%

연금보험 (연금 지급기간 20년):
  보장단위는 연금 수령 기간에 연동
  연금 개시 전에는 상각 없음 (또는 적은 상각)
  연금 개시 후 20년간 상각
```

**미래서비스 관련 가정변경:**
- 미래 현금흐름에 영향을 미치는 비금융 가정의 변경
- 예: 사망률 개선, 해약률 변화, 사업비율 변동
- 유리한 변경(예: 사망률 하락) → CSM 증가
- 불리한 변경(예: 해약률 상승) → CSM 감소
- 단, CSM은 0 미만으로 떨어질 수 없으며, 초과 불리한 변경은 당기손실로 인식

### 3.3 실전 해석: CSM Roll-Forward 분석

투자자가 CSM Roll-Forward에서 주목해야 할 포인트는 다음과 같습니다.

- **기말 CSM** — 투자 시사점: 미래 이익 규모, 긍정 신호: 증가 추세, 부정 신호: 감소 추세
- **신계약 CSM** — 투자 시사점: 성장성, 신계약 수익성, 긍정 신호: 전년 대비 증가, 부정 신호: 전년 대비 감소
- **CSM 상각** — 투자 시사점: 당기 이익 기여, 긍정 신호: 안정적 상각, 부정 신호: 급격한 상각 증가
- **가정변경** — 투자 시사점: 경험 실적 반영, 긍정 신호: 유리한 변경, 부정 신호: 불리한 변경
- **신계약 CSM / 상각** — 투자 시사점: 지속가능성, 긍정 신호: 비율 > 1 (축적), 부정 신호: 비율 < 1 (소진)

**핵심 공식: CSM 지속가능성 비율**

```
CSM 지속가능성 비율 = 신계약 CSM / CSM 상각

> 1.0: CSM 축적 (미래 이익 파이프라인 확대)
= 1.0: CSM 유지 (안정적)
< 1.0: CSM 소진 (미래 이익 파이프라인 축소)
```

### 3.4 한국 보험사별 CSM 비교

한국 주요 보험사의 CSM 현황을 비교합니다 (IFRS 17 기준, 연간 실적 기준 추정치).

- **교보생명** — CSM 잔액 (조원): 약 6\~8, 신계약 CSM (조원): 약 1.0\~1.5, CSM 상각 (조원): 약 1.0\~1.3, 지속가능성 비율: 약 1.0\~1.2

## 4. 할인율 구조

### 4.1 할인율의 중요성

IFRS 17에서 할인율은 보험부채의 현재가치 산출에 사용되며, BEL의 크기와 CSM에 직접적인 영향을 미칩니다. 할인율 1%p 변동이 보험부채를 수조 원 단위로 변동시킬 수 있어, 할인율 결정 방법론은 IFRS 17 적용에서 가장 중요한 쟁점 중 하나입니다.

### 4.2 두 가지 접근법

IFRS 17은 할인율 결정에 두 가지 접근법을 허용합니다.

**상향식 접근법 (Bottom-up Approach):**

```
할인율 = 무위험이자율 + 유동성 프리미엄

무위험이자율: 국고채 수익률 (만기별 수익률 곡선)
유동성 프리미엄: 보험부채의 비유동적 특성을 반영하는 추가 스프레드
```

- 국고채 수익률 곡선에서 출발
- 보험부채는 시장에서 거래되지 않으므로 유동성이 낮음
- 이 비유동성을 반영하여 유동성 프리미엄을 가산
- 한국 보험사 대다수가 채택

**하향식 접근법 (Top-down Approach):**

```
할인율 = 자산 포트폴리오 기대수익률 - 신용위험 프리미엄

자산 기대수익률: 실제 운용 자산의 기대수익률
신용위험 프리미엄: 자산 포트폴리오의 예상 신용손실 차감
```

- 보험회사가 실제로 보유한 자산의 수익률에서 출발
- 신용위험(부도 가능성)을 차감
- 자산 운용 전략이 부채 할인율에 영향

### 4.3 할인율 구조의 세부 사항

**만기별 할인율 곡선:**

```
관측 가능 구간 (1\~30년):
  국고채 수익률 곡선 기반
  유동성 프리미엄 가산

관측 불가 구간 (30년 초과):
  장기선도이자율(LTFR)로 수렴
  한국: LTFR = 장기 균형금리 수준으로 결정
  보간법(Smith-Wilson 등)으로 수익률 곡선 연장
```

한국 보험사의 부채 듀레이션은 생명보험의 경우 15\~25년에 달하므로, 장기 할인율(특히 관측 불가 구간)의 결정이 매우 중요합니다.

**유동성 프리미엄 결정:**

유동성 프리미엄은 보험부채의 현금흐름 특성에 따라 차등 적용됩니다.

- **현금흐름 불확실성 낮음** — 유동성 프리미엄 수준: 높음 (전액 반영), 예시 상품: 확정급부 연금
- **현금흐름 불확실성 중간** — 유동성 프리미엄 수준: 중간, 예시 상품: 종신보험
- **현금흐름 불확실성 높음** — 유동성 프리미엄 수준: 낮음 (일부 반영), 예시 상품: 해약 환급금 비중 큰 상품

### 4.4 할인율이 CSM과 BEL에 미치는 영향

할인율 변동은 BEL과 CSM에 다음과 같이 영향을 미칩니다.

**금리 상승 시:**

```
BEL 감소 (미래현금흐름의 현재가치 하락)
  → 부채 감소 → 자본 증가

BBA 적용 계약:
  금리 변동 → BEL 변동 → 보험금융손익(P&L) 또는 OCI
  CSM에는 영향 없음 (locked-in rate 사용)

VFA 적용 계약:
  금리 변동 → 기초항목 가치 변동 → CSM 조정
```

**금리 하락 시:**

```
BEL 증가 (미래현금흐름의 현재가치 상승)
  → 부채 증가 → 자본 감소

장기 저금리 시:
  부채 듀레이션 > 자산 듀레이션인 경우
  자본 훼손 압력 증가
```

### 4.5 금리 민감도 분석

한국 보험사의 금리 민감도 예시 (금리 1%p 변동 시 추정):

- **대형 생보사** — BEL 변동: ±10\~15조, 자본 변동: ±3\~5조, CSM 변동 (BBA): 영향 없음, CSM 변동 (VFA): ±0.5\~1.0조
- **중형 생보사** — BEL 변동: ±3\~5조, 자본 변동: ±1\~2조, CSM 변동 (BBA): 영향 없음, CSM 변동 (VFA): ±0.1\~0.3조
- **대형 손보사** — BEL 변동: ±3\~5조, 자본 변동: ±1\~2조, CSM 변동 (BBA): 영향 없음, CSM 변동 (VFA): 미미

> **참고**: 보험사는 OCI 옵션 선택에 따라 금리 변동의 손익 인식 방식이 달라집니다. 현행이자율 기준으로 BEL을 재측정하되, BEL 변동 중 할인율 변경 효과를 OCI로 인식하는 방법을 대부분의 한국 보험사가 선택했습니다.

## 5. 보험수익 인식

### 5.1 IFRS 4 vs IFRS 17: 수익 인식의 패러다임 전환

IFRS 17 도입으로 보험수익 인식 방법이 근본적으로 변했습니다. 이 변화는 보험회사의 손익계산서 구조를 완전히 바꿨습니다.

**기존 IFRS 4 (수입보험료 기반):**

```
매출 = 수입보험료 (계약자가 납입한 보험료 전액)

포함 항목:
  ✓ 위험보험료 (보장 서비스 대가)
  ✓ 저축보험료 (투자요소, 만기환급금 재원)
  ✓ 부가보험료 (사업비)
```

**IFRS 17 (보험서비스 제공 기반):**

```
보험수익 = 예상보험금 + 예상사업비 + CSM 상각 + RA 해제

제외 항목:
  ✗ 투자요소(저축보험료) → 매출에서 제외
  ✗ 보험료 수취 시점과 무관

핵심: "보험서비스를 제공한 만큼만 수익으로 인식"
```

### 5.2 보험수익의 구성요소

IFRS 17에서 보험수익은 다음과 같이 구성됩니다.

```
보험수익 = ① + ② + ③ + ④

① 예상보험금 및 사업비 (당기 서비스 관련)
   = 당기에 발생할 것으로 예상되는 보험금 + 사업비
   → BEL에서 해제되는 금액

② CSM 상각
   = 당기 서비스에 배분된 CSM
   → 미실현이익의 실현

③ RA 해제
   = 당기 위험 소멸에 따른 RA 감소분
   → 비금융위험 부담 대가의 실현

④ 경험조정 (과거서비스 관련 변경 등)
   = 실제 현금흐름과 예상 현금흐름의 차이 중 당기분
```

### 5.3 보험서비스 비용의 구성

```
보험서비스비용 = 발생보험금 + 발생사업비 + 손실부담계약 조정

보험서비스 결과(이익) = 보험수익 - 보험서비스비용
```

### 5.4 투자요소의 제외

IFRS 17에서 가장 큰 변화 중 하나는 **투자요소(investment component)**가 보험수익에서 제외된다는 점입니다.

```
투자요소의 정의:
  "보험사건의 발생 여부와 관계없이 계약자에게 상환해야 하는 금액"

예시:
  - 종신보험의 해약환급금
  - 연금보험의 적립금
  - 저축성 보험의 만기환급금

투자요소 제외 효과:
  IFRS 4: 수입보험료 100 (위험 30 + 저축 50 + 사업비 20) → 매출 100
  IFRS 17: 보험수익 = 약 50 (위험보험료 상당 + CSM 상각 + RA 해제)
  → 매출 50% 감소
```

### 5.5 실전 영향: 매출 구조 변화

IFRS 17 도입으로 한국 보험사의 손익계산서는 다음과 같이 변화했습니다.

- **매출(보험수익)** — IFRS 4 시대: 수입보험료 중심, IFRS 17 시대: 서비스 제공 기반, 변화 방향: 생보 대폭 감소
- **비용 구조** — IFRS 4 시대: 지급보험금 + 사업비, IFRS 17 시대: 보험서비스비용, 변화 방향: 투자요소 상계
- **투자손익** — IFRS 4 시대: 투자영업이익으로 별도, IFRS 17 시대: 보험금융손익으로 통합, 변화 방향: 표시 방식 변경
- **이익 변동성** — IFRS 4 시대: 상대적으로 안정, IFRS 17 시대: CSM 상각 기반 안정, 금융변수 변동성, 변화 방향: 복합적

**저축성 보험 비중이 높은 생보사의 영향:**

```
예시: A 생명보험사

IFRS 4 기준:
  수입보험료: 20조 원
  (위험보험료 4조 + 저축보험료 12조 + 부가보험료 4조)

IFRS 17 기준:
  보험수익: 약 8조 원
  (예상보험금 3조 + 사업비 2조 + CSM 상각 2조 + RA 해제 1조)
  투자요소 12조 원 제외

→ 매출 규모 약 60% 감소
```

이러한 매출 감소는 회계적 표시 방식의 변화일 뿐, 보험회사의 실제 현금흐름이나 경제적 실질이 변한 것은 아닙니다. 그러나 매출 기반으로 보험사를 평가하던 기존 지표(매출 성장률, 사업비율 등)의 해석이 근본적으로 달라지게 됩니다.

## 6. 한국 보험사 IFRS 17 적용 현황

### 6.1 전환 개요

한국 보험산업은 2023년 1분기부터 IFRS 17을 적용했습니다. 전환 시 보험회사는 과거 보험계약에 대해 세 가지 방법 중 하나를 선택하여 재측정했습니다.

- **완전소급법** — 설명: 마치 최초 계약 시점부터 IFRS 17을 적용한 것처럼 재측정, 적용 조건: 과거 데이터가 충분한 경우
- **수정소급법** — 설명: 합리적이고 뒷받침 가능한 정보를 활용하여 완전소급법에 가장 가까운 결과 도출, 적용 조건: 완전소급법이 비현실적인 경우
- **공정가치법** — 설명: 전환일 시점의 보험계약 공정가치를 기준으로 CSM 산출, 적용 조건: 과거 데이터 확보가 어려운 경우

한국 보험사 대부분은 **수정소급법**을 주된 방법으로 적용하되, 일부 코호트(과거 계약 그룹)에 대해 공정가치법을 적용했습니다. 완전소급법은 데이터 요건이 매우 엄격하여 적용 범위가 제한적이었습니다.

### 6.2 전환 효과

IFRS 17 전환은 보험회사의 재무상태표와 손익 구조에 큰 변화를 가져왔습니다.

**자본 효과:**

- **부채 시가 평가 (저금리 → 부채 증가)** — 자본 영향: 자본 감소, 규모: 대형
- **CSM 인식 (미실현이익 부채 계상)** — 자본 영향: 자본 감소, 규모: 대형
- **RA 인식** — 자본 영향: 자본 감소, 규모: 중형
- **일부 자산 재분류 효과** — 자본 영향: 자본 증감 혼합, 규모: 중형
- **이연법인세 효과** — 자본 영향: 자본 증가, 규모: 중형

전환 시 대부분의 한국 보험사는 순자산(자본)이 감소했습니다. 다만, 이는 기존에 인식하지 않았던 미실현이익(CSM)을 부채로 계상한 것이므로, 경제적 실질이 악화된 것은 아닙니다.

### 6.3 CSM 규모 랭킹과 투자 의미

CSM은 IFRS 17 이후 보험주 투자에서 가장 중요한 지표 중 하나가 되었습니다. CSM 규모가 클수록 미래에 보험수익으로 인식할 수 있는 이익의 파이프라인이 풍부하다는 의미입니다.

**CSM 규모의 투자 해석:**

```
CSM이 큰 보험사:
  ✓ 미래 이익 가시성 높음
  ✓ 안정적인 보험수익 예상
  ✓ 배당 여력 풍부 (장기)
  ✓ 밸류에이션 프리미엄 정당화

CSM이 작은 보험사:
  ✗ 미래 이익 가시성 낮음
  ✗ 성장을 위한 신계약 확보 필요
  ✗ 신계약 수익성(마진) 점검 필요
```

**신계약 CSM의 중요성:**

신계약 CSM은 보험사의 성장성을 보여주는 핵심 지표입니다.

```
신계약 CSM = 신계약의 미래 이익 현재가치

신계약 CSM 마진 = 신계약 CSM / 신계약 보험료의 현재가치 × 100%

마진이 높을수록:
  → 수익성 좋은 상품을 판매하고 있음
  → 가격 경쟁력(pricing power) 보유
  → 미래 이익 품질 양호
```

### 6.4 IFRS 17 이후 투자자가 봐야 할 핵심 지표 10가지

IFRS 17 시대에 보험주 투자자가 반드시 모니터링해야 할 지표를 정리합니다.

- **1** — 지표: **CSM 잔액**, 정의: 기말 계약서비스마진, 투자 의미: 미래 이익 파이프라인
- **2** — 지표: **신계약 CSM**, 정의: 당기 신규 계약의 CSM, 투자 의미: 성장성, 신계약 수익성
- **3** — 지표: **CSM 상각률**, 정의: CSM 상각 / 기초 CSM, 투자 의미: 이익 인식 속도
- **4** — 지표: **보험수익**, 정의: IFRS 17 기준 매출, 투자 의미: 사업 규모 (투자요소 제외)
- **5** — 지표: **보험서비스 결과**, 정의: 보험수익 - 보험서비스비용, 투자 의미: 보험 본업의 수익성
- **6** — 지표: **보험금융손익**, 정의: 할인율 변동 등 금융 효과, 투자 의미: 금리 민감도, 변동성
- **7** — 지표: **K-ICS 비율**, 정의: 가용자본/요구자본, 투자 의미: 자본 건전성, 배당 여력
- **8** — 지표: **경험조정**, 정의: 실제 vs 예상 현금흐름 차이, 투자 의미: 가정의 적정성, 이익 품질
- **9** — 지표: **손실부담계약**, 정의: 손실이 예상되는 계약 규모, 투자 의미: 부실 계약 규모
- **10** — 지표: **해약률 추이**, 정의: 실제 해약률 변동, 투자 의미: CSM 영향, 유동성 리스크

## 7. IFRS 17과 밸류에이션

### 7.1 Embedded Value(EV)와의 관계

IFRS 17 도입 이전, 보험사 밸류에이션의 핵심 지표는 **Embedded Value(EV, 내재가치)**였습니다. EV는 보험회사의 경제적 가치를 나타내는 지표로, 다음과 같이 구성됩니다.

```
EV = ANW + VIF

ANW (Adjusted Net Worth, 수정순자산)
  = 순자산의 시가 평가 + 잠재적 자산가치

VIF (Value of In-Force, 보유계약 가치)
  = 기존 보유 보험계약에서 미래에 발생할 이익의 현재가치

추가:
신계약 가치(VNB, Value of New Business)
  = 당기 판매 신계약의 미래 이익 현재가치
```

**EV와 CSM의 비교:**

- **성격** — EV (내재가치): 경영자 추정 (비공인), IFRS 17 CSM: 회계 기준 (공인)
- **범위** — EV (내재가치): 순자산 + 보유계약 가치, IFRS 17 CSM: 미실현이익만 (부채 구성요소)
- **할인율** — EV (내재가치): 리스크할인율 (8\~12%), IFRS 17 CSM: 시장 할인율 (국고채+α)
- **리스크 반영** — EV (내재가치): 할인율에 포함, IFRS 17 CSM: RA로 별도 반영
- **공시 의무** — EV (내재가치): 자발적 공시, IFRS 17 CSM: 의무 공시
- **검증 가능성** — EV (내재가치): 제한적 (감사 대상 아님), IFRS 17 CSM: 높음 (외부감사 대상)
- **투자 활용** — EV (내재가치): P/EV 멀티플, IFRS 17 CSM: P/CSM 멀티플

**핵심**: CSM은 EV의 VIF(보유계약 가치)와 개념적으로 유사하지만, 할인율과 리스크 반영 방식이 다르기 때문에 수치가 동일하지는 않습니다. 일반적으로 CSM < VIF입니다 (CSM에는 RA가 별도 반영되고, 할인율이 EV보다 낮아 부채가 더 크게 인식되는 경향).

### 7.2 P/CSM 멀티플

IFRS 17 이후 등장한 새로운 밸류에이션 지표가 **P/CSM 멀티플**입니다.

```
P/CSM = 시가총액 / CSM 잔액

해석:
  P/CSM = 1.0x → 주가가 미래 미실현이익만큼 반영
  P/CSM < 1.0x → 주가가 미래 미실현이익 이하, 저평가 가능성
  P/CSM > 1.0x → 주가가 미래 미실현이익 초과, 성장 프리미엄 반영
```

그러나 P/CSM만으로는 보험사의 가치를 완전히 평가할 수 없습니다. CSM은 보유계약의 미실현이익만 반영하며, 보험회사의 순자산 가치, 미래 신계약 가치 등은 포함하지 않기 때문입니다.

### 7.3 보험사 밸류에이션 방법론

보험사 밸류에이션에 활용되는 주요 방법론을 종합적으로 정리합니다.

- **P/EV** — 산출식: 시가총액 / EV, 장점: 경제적 가치 반영, 단점: 비공인, 검증 한계
- **P/CSM** — 산출식: 시가총액 / CSM, 장점: 회계 기준, 검증 가능, 단점: 순자산 미반영
- **P/BV** — 산출식: 시가총액 / 순자산, 장점: 단순, 직관적, 단점: IFRS 17 순자산 왜곡 가능
- **DDM** — 산출식: 미래 배당의 현재가치 합, 장점: 현금 수취 기준, 단점: 배당정책 의존
- **SOTP** — 산출식: 사업부별 가치 합산, 장점: 정밀, 종합적, 단점: 복잡, 가정 많음

**IFRS 17 시대 권장 밸류에이션 프레임워크:**

```
보험사 적정가치 = 순자산(시가) + CSM + 미래 신계약 가치

순자산(시가):
  IFRS 17 자본 ± 미반영 잠재 가치

CSM:
  보유계약의 미실현이익

미래 신계약 가치:
  향후 판매할 신계약의 예상 CSM 현재가치
  = 신계약 CSM × 성장률 × 멀티플
```

### 7.4 한국 보험주 밸류에이션 현황과 투자 시사점

한국 보험주의 밸류에이션 현황을 주요 지표별로 정리합니다.

- **삼성생명** — P/BV (배): 0.3\~0.5, P/E (배): 8\~12, 배당수익률 (%): 3\~5%, 투자 포인트: 국내 최대, 높은 CSM
- **한화생명** — P/BV (배): 0.2\~0.4, P/E (배): 6\~10, 배당수익률 (%): 3\~5%, 투자 포인트: 구조조정 효과 기대
- **삼성화재** — P/BV (배): 0.7\~1.0, P/E (배): 8\~12, 배당수익률 (%): 3\~5%, 투자 포인트: 높은 수익성, 손보 1위
- **DB손해보험** — P/BV (배): 0.6\~0.9, P/E (배): 6\~10, 배당수익률 (%): 4\~6%, 투자 포인트: 안정적 실적, 높은 배당
- **현대해상** — P/BV (배): 0.4\~0.7, P/E (배): 6\~9, 배당수익률 (%): 4\~6%, 투자 포인트: 할인 매력, 자동차보험
- **메리츠화재** — P/BV (배): 0.8\~1.2, P/E (배): 7\~10, 배당수익률 (%): 3\~5%, 투자 포인트: 높은 성장성, 신계약 CSM

> **주의**: 위 수치는 시점에 따라 변동하며, 투자 의사결정 시 최신 공시 자료를 반드시 확인하시기 바랍니다.

**한국 보험주 저평가 원인 분석:**

한국 보험주는 글로벌 보험사 대비 할인(Korea Discount) 상태에 있습니다. 그 원인은 다음과 같습니다.

1. **이차역마진 우려**: 과거 고금리 확정 상품의 역마진 리스크 지속
2. **금리 변동성**: 금리에 민감한 재무구조로 인한 이익 변동성
3. **IFRS 17 불확실성**: 새로운 회계기준 적용 초기의 불확실성
4. **주주환원 부족**: 자본 적정성(K-ICS) 우선으로 배당 제한
5. **지배구조**: 일부 보험사의 지배구조 이슈
6. **성장성 우려**: 보험 침투율 포화, 인구 고령화에 따른 구조적 변화

**투자 시사점:**

```
IFRS 17 시대 보험주 투자 체크리스트:

□ CSM 잔액은 얼마인가? → 미래 이익 파이프라인
□ 신계약 CSM은 성장하고 있는가? → 성장성
□ CSM 지속가능성 비율 > 1인가? → CSM 축적 여부
□ K-ICS 비율은 충분한가? (150% 이상) → 배당 여력
□ 보험서비스 결과는 양호한가? → 본업 수익성
□ 금리 민감도는 어느 수준인가? → 리스크 관리
□ 주주환원 정책은 개선되고 있는가? → 밸류에이션 촉매
□ 손실부담계약 규모는 적정한가? → 부실 리스크
```

## 요약: IFRS 17 핵심 구조도

```
┌─────────────────────────────────────────────────────────────┐
│                     IFRS 17 보험계약부채                       │
│                                                             │
│  ┌──────────────┐  ┌──────────┐  ┌────────────────────┐     │
│  │   BEL        │  │   RA     │  │      CSM           │     │
│  │ (최선추정부채) │  │(위험조정) │  │ (계약서비스마진)    │     │
│  │              │  │          │  │                    │     │
│  │ 미래현금흐름  │  │ 비금융    │  │  미실현이익         │     │
│  │ 현재가치     │  │ 위험보상  │  │  → 서비스 기간     │     │
│  │              │  │          │  │    수익 배분       │     │
│  └──────────────┘  └──────────┘  └────────────────────┘     │
│                                                             │
│  측정 모형: BBA (일반) | VFA (변액) | PAA (단기)              │
│                                                             │
│  보험수익 = 예상보험금 + 사업비 + CSM 상각 + RA 해제           │
│  (투자요소 제외)                                              │
│                                                             │
│  밸류에이션: P/EV, P/CSM, P/BV, DDM                          │
└─────────────────────────────────────────────────────────────┘
```

:::tip 핵심 요약
IFRS 17의 핵심은 세 가지입니다. 첫째, 보험부채를 시가(현행추정치)로 평가합니다. 둘째, 미실현이익(CSM)을 서비스 제공 기간에 걸쳐 수익으로 인식합니다. 셋째, 보험수익에서 투자요소를 제외하여 보험 본연의 서비스 수익만을 보여줍니다. 투자자에게 CSM은 미래 이익의 파이프라인이며, 이를 중심으로 보험사의 가치를 평가하는 새로운 패러다임이 열렸습니다.
:::
