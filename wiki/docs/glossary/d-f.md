# 용어 사전 (D-F)

## D

### DCF (Discounted Cash Flow, 현금흐름할인법)
미래 현금흐름을 현재가치로 할인하여 기업가치를 산출하는 방법.

**관련:** [DCF 분석](/valuation/dcf/overview)

### Delta (델타)
기초자산 가격 변화에 대한 옵션 가격의 민감도. 옵션 그릭스 중 하나.

**관련:** [옵션 그릭스](/assets/derivatives/greeks)

### Dividend (배당)
기업이 이익의 일부를 주주에게 지급하는 현금.

### Duration (듀레이션)
채권 현금흐름의 가중평균 만기. 금리 민감도 측정에 사용.

**관련:** [듀레이션](/assets/bonds/duration)

## E

### EBITDA
영업이익에 감가상각비를 더한 값. Earnings Before Interest, Taxes, Depreciation, and Amortization.

**공식:**
```
EBITDA = 영업이익 + 감가상각비
```

**관련:** [EV/EBITDA](/valuation/relative/ev-ebitda)

### Enterprise Value (기업가치) {#enterprise-value}
시가총액에 순부채를 더한 값. 전체 자본 제공자에 대한 가치.

**공식:**
```
EV = 시가총액 + 순부채
```

### EPS (Earnings Per Share, 주당순이익)
당기순이익을 발행주식수로 나눈 값.

**관련:** [PER](/valuation/relative/per)

### Equity (자기자본)
총자산에서 총부채를 차감한 순자산.

### ERP (Equity Risk Premium, 시장위험프리미엄)
주식 투자에 대해 무위험이자율 대비 요구하는 추가 수익률. 보통 5-7%.

## F

### FCF (Free Cash Flow, 잉여현금흐름)
영업활동에서 창출된 현금 중 재투자를 제외하고 자본 제공자에게 배분 가능한 현금.

**공식:**
```
FCFF = EBIT(1-T) + 감가상각 - CapEx - ΔNWC
```

**관련:** [FCFF vs FCFE](/valuation/dcf/fcff-fcfe)

### Forward P/E (예상 PER)
현재 주가를 향후 12개월 예상 EPS로 나눈 값.

### Fundamental Analysis (기본적 분석)
재무제표, 산업, 경제 데이터를 분석하여 내재가치를 추정하는 방법.
