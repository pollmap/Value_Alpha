# 용어 사전 (A-C)

## A

### Alpha (알파)
벤치마크 대비 초과 수익률. 펀드 매니저의 실력을 측정하는 지표.

### Asset Allocation (자산배분)
포트폴리오를 주식, 채권, 대체자산 등 자산 클래스별로 배분하는 투자 전략.

## B

### Beta (베타) {#beta}
개별 주식의 시장 대비 변동성. β=1이면 시장과 동일, β>1이면 시장보다 변동성 높음.

**공식:**
```
β = Cov(Ri, Rm) / Var(Rm)
```

**관련:** [WACC](/valuation/dcf/wacc), [CAPM](#capm)

### Book Value (장부가치)
회계상 자산에서 부채를 차감한 순자산 가치. 주당순자산(BPS) 계산에 사용.

### Buyback (자사주 매입)
기업이 자사 주식을 시장에서 매입하는 행위. 주당 가치 상승 효과.

## C

### CapEx (Capital Expenditure, 자본적 지출)
유형자산 취득에 투자하는 금액. FCF 계산 시 차감.

### CAPM (Capital Asset Pricing Model) {#capm}
자본자산가격결정모형. 체계적 위험에 대한 보상으로 기대수익률을 설명.

**공식:**
```
E(R) = Rf + β × (Rm - Rf)
```

**관련:** [WACC](/valuation/dcf/wacc)

### Cash Flow (현금흐름)
기업의 현금 유입과 유출. 영업/투자/재무 활동으로 구분.

### Comparable (유사기업)
밸류에이션 비교를 위해 선정한 동종업계 기업들.

### Convexity (컨벡시티)
채권 가격과 금리의 비선형 관계를 측정. 듀레이션의 보정치.

**관련:** [듀레이션](/assets/bonds/duration)

### Cost of Capital (자본비용)
기업이 자본을 조달하는 데 드는 비용. WACC로 측정.

**관련:** [WACC](/valuation/dcf/wacc)
