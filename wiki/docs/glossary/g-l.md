# 용어 사전 (G-L)

## G

### Gordon Growth Model (고든 성장 모형)
배당이 일정 비율로 영구 성장한다고 가정하고 주식 가치를 계산하는 모형.

**공식:**
```
P = D1 / (r - g)
```

### Graham Number (그레이엄 넘버)
벤자민 그레이엄이 제시한 주식의 적정 가격 상한.

**공식:**
```
√(22.5 × EPS × BVPS)
```

**관련:** [그레이엄 넘버 계산기](/calculators/graham-number)

### Greeks (그릭스)
옵션의 위험 지표들: 델타, 감마, 세타, 베가, 로.

## I

### Intrinsic Value (내재가치)
자산의 본질적 가치. 시장가격과 비교하여 저평가/고평가 판단.

### IRR (Internal Rate of Return, 내부수익률)
투자의 NPV를 0으로 만드는 할인율.

## L

### Leverage (레버리지)
부채를 사용하여 자기자본 수익률을 증폭시키는 것.

### LBO (Leveraged Buyout)
차입금을 활용한 기업 인수.

**관련:** [LBO 모델링](/valuation/lbo)

### Liquidity (유동성)
자산을 현금으로 전환할 수 있는 용이성.
