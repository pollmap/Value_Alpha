# 조엘 그린블라트 (Joel Greenblatt)

> "좋은 기업을 싼 가격에 사라. 이것이 전부다."

## 인물 소개

**조엘 그린블라트 (1957~)** 는 고담 캐피탈의 창립자로, 1985년부터 2005년까지 연평균 **40%** 의 수익률을 기록한 전설적인 헤지펀드 매니저입니다.

| 항목 | 내용 |
|------|------|
| 운용사 | 고담 캐피탈 |
| 운용 기간 | 1985-2005 |
| 연평균 수익률 | 40% |
| 투자 스타일 | 특수상황, 마법공식 |
| 대표 저서 | 《주식시장을 이기는 작은 책》 |

## 핵심 투자 철학

### 1. 마법공식 (Magic Formula)

그린블라트의 가장 유명한 전략으로, 두 가지 지표만으로 종목을 선정합니다.

**공식:**
```
순위 = 수익률 순위 + 저평가 순위

수익률(이익수익률) = EBIT / Enterprise Value
저평가(자본수익률) = EBIT / (순운전자본 + 순고정자산)
```

**간소화 버전:**
```
수익률 = EBIT / EV (또는 영업이익/시가총액+순부채)
자본수익률 = ROIC (투하자본수익률)
```

### 마법공식 프로세스

```
1단계: 전체 상장주 중 시가총액 하위 제외
       (유틸리티, 금융주 제외)

2단계: 수익률(Earnings Yield) 기준 순위 부여
       높을수록 좋음 (저평가)

3단계: 자본수익률(ROIC) 기준 순위 부여
       높을수록 좋음 (우량)

4단계: 두 순위 합산하여 종합 순위 결정

5단계: 상위 20-30개 종목 매수

6단계: 1년 후 리밸런싱
```

### 마법공식 예시

| 기업 | EBIT/EV | ROIC | 수익률 순위 | ROIC 순위 | 종합 순위 |
|------|---------|------|------------|----------|----------|
| A사 | 15% | 40% | 3 | 2 | **5** |
| B사 | 20% | 25% | 1 | 5 | **6** |
| C사 | 12% | 50% | 5 | 1 | **6** |
| D사 | 18% | 30% | 2 | 3 | **5** |
| E사 | 10% | 28% | 6 | 4 | **10** |

→ A사, D사 매수 우선순위

## 2. 특수상황 투자 (Special Situations)

시장이 간과하는 특수한 상황에서 기회를 찾습니다.

### 스핀오프 (Spin-off)

기업 분할 시 발생하는 투자 기회:

```
모회사 A
    │
    ├── 핵심사업 (유지)
    │
    └── 비핵심사업 B (분할)
         │
         └── 기관투자자 강제 매도
              │
              └── 일시적 저평가 발생
                   │
                   └── 투자 기회!
```

**스핀오프 투자 포인트:**
1. 기관투자자의 강제 매도 (인덱스 이탈)
2. 분석가 커버리지 부재
3. 소형주로 전환 시 관심 감소
4. 경영진 인센티브 재조정

### 합병 차익거래 (Merger Arbitrage)

```
발표된 인수가: $50
현재 주가: $47
스프레드: $3 (6.4%)

리스크:
- 합병 무산 확률
- 완료까지 기간
- 조건 변경 가능성
```

### 구조조정 (Restructuring)

```
구조조정 기업 투자 체크리스트:
□ 핵심 사업의 가치는 건전한가?
□ 부채 구조 개선이 가능한가?
□ 신규 경영진의 역량은?
□ 구조조정 계획이 현실적인가?
□ 숨겨진 자산이 있는가?
```

## 마법공식 백테스트 결과

그린블라트의 책에서 제시한 역사적 성과:

| 기간 | 마법공식 | S&P 500 | 초과수익 |
|------|---------|---------|---------|
| 1988-2004 | 30.8% | 12.4% | +18.4% |

**주의사항:**
- 과거 성과가 미래를 보장하지 않음
- 실제 적용 시 거래비용, 슬리피지 발생
- 장기간(최소 3-5년) 유지 필요

## 마법공식 실전 적용

### 종목 선정 기준

```python
# 의사 코드
def magic_formula_screen(stocks):
    # 제외 조건
    exclude = [
        시가총액 < 50M,  # 소형주 제외
        sector in ['금융', '유틸리티'],  # 섹터 제외
        ADR,  # 외국주식 제외
    ]

    # 지표 계산
    for stock in eligible_stocks:
        earnings_yield = EBIT / EV
        roic = EBIT / (NWC + Net_Fixed_Assets)

    # 순위 부여
    rank_by(earnings_yield, descending=True)
    rank_by(roic, descending=True)

    # 종합 순위
    combined_rank = ey_rank + roic_rank

    return top(30, by=combined_rank)
```

### 포트폴리오 구성

| 항목 | 권장 |
|------|------|
| 종목 수 | 20-30개 |
| 매수 방식 | 분할 매수 (3-4개월에 걸쳐) |
| 보유 기간 | 1년 |
| 리밸런싱 | 연 1회 |
| 최소 투자 기간 | 3-5년 |

### 매도 규칙

```
매도 시점:
1. 보유 1년 경과 후 (세금 고려)
2. 단, 손실 종목은 1년 전 매도 가능 (손실 실현)
3. 새로운 마법공식 순위에서 탈락 시

매도하지 않을 때:
- 단기 주가 하락
- 일시적 부정적 뉴스
- 시장 전체 하락
```

## 그린블라트의 기타 인사이트

### 시장을 이기는 3가지 방법

1. **더 잘 알기** - 정보 우위 (어려움)
2. **더 길게 보기** - 시간 우위 (가능)
3. **더 깊게 보기** - 분석 우위 (가능)

### 가치투자의 작동 원리

> "시장은 단기적으로 인기투표, 장기적으로 저울"

```
단기: 감정, 뉴스, 모멘텀이 가격 지배
    │
    ▼
중기: 실적이 기대와 다르면 조정
    │
    ▼
장기: 내재가치로 수렴
```

### 인내의 중요성

마법공식의 연간 성과:

```
최고 연도: +50% 이상
최저 연도: -25% 이하
시장 하회 연도: 5회/17년

→ 3-5년 이상 유지해야 효과 발휘
```

## 체크리스트: 그린블라트 스타일

### 마법공식 체크리스트

```
□ 시가총액이 충분히 큰가? (최소 $50M)
□ 금융/유틸리티 섹터가 아닌가?
□ EBIT/EV(수익률)가 상위권인가?
□ ROIC(자본수익률)가 상위권인가?
□ 종합 순위가 상위 30위 이내인가?
□ 3년 이상 보유할 준비가 되었는가?
```

### 특수상황 체크리스트

```
□ 시장이 왜 이 기회를 놓치고 있는가?
□ 내재가치 대비 얼마나 할인되어 있는가?
□ 가치 실현의 촉매(catalyst)는 무엇인가?
□ 최악의 경우 손실은 얼마인가?
□ 시간이 내 편인가?
```

## 한국 시장 적용

### 마법공식 한국판 고려사항

```
1. 시가총액 기준: 1,000억원 이상
2. 제외 섹터: 금융, 유틸리티, 부동산
3. 지표:
   - 수익률: 영업이익/(시가총액+순부채)
   - ROIC: 영업이익/(자본+순부채-현금)
4. 데이터 소스: FnGuide, KRX
```

### 주의점

- 한국은 지배구조 할인 존재
- 재벌 그룹사 관계 확인 필요
- 회계 기준 차이 고려
