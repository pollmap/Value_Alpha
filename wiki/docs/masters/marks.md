# 하워드 막스 (Howard Marks)

> "투자에서 가장 중요한 것은 무엇이 좋은지 아는 게 아니라, 무엇이 가격에 반영되어 있는지 아는 것이다."

## 인물 소개

**하워드 막스 (1946~)** 는 오크트리 캐피탈의 공동 창립자로, **부실채권 투자**의 대가이며 투자 메모로 유명합니다. 워렌 버핏도 그의 메모가 오면 가장 먼저 읽는다고 합니다.

| 항목 | 내용 |
|------|------|
| 운용사 | 오크트리 캐피탈 매니지먼트 |
| 설립 | 1995년 |
| 운용 자산 | $170B+ |
| 전문 분야 | 부실채권, 하이일드 |
| 대표 저서 | 《투자에 대한 생각》, 《시장 사이클의 법칙》 |

## 핵심 투자 철학

### 1. 2차적 사고 (Second-Level Thinking)

1차적 사고를 넘어 **한 단계 더 깊이** 생각해야 합니다.

| 1차적 사고 | 2차적 사고 |
|-----------|-----------|
| "좋은 회사니까 매수" | "좋은 회사지만 기대가 이미 가격에 반영됨" |
| "전망이 어두우니 매도" | "모두가 비관적이라 가격이 충분히 낮음" |
| "금리가 오르니 채권 매도" | "금리 상승이 예상보다 적으면?" |
| "경기 침체가 오니 현금 보유" | "이미 주가가 침체를 반영했다면?" |

**2차적 사고를 위한 질문:**

2차적 사고를 실천하기 위해서는 다음 여섯 가지 질문을 던져야 합니다. 첫째, **컨센서스는 무엇인가?** 시장 참여자 대다수가 믿고 있는 것이 무엇인지 파악합니다. 둘째, **내 의견은 컨센서스와 어떻게 다른가?** 단순히 다르다는 것만으로는 부족하고, 왜 다른지 명확한 논리가 필요합니다. 셋째, **컨센서스가 틀릴 확률은 얼마인가?** 다수의 의견이 틀릴 가능성을 객관적으로 평가합니다. 넷째, **내가 틀릴 확률은 얼마인가?** 자신의 판단에 대한 겸손함이 필요합니다. 다섯째, **컨센서스가 틀렸을 때 얼마나 벌 수 있나?** 상방 잠재력을 계산합니다. 여섯째, **내가 틀렸을 때 얼마나 잃나?** 하방 위험을 반드시 고려해야 합니다.

### 2. 시장 사이클

> "사이클을 예측할 수는 없지만, 어디에 있는지는 알 수 있다."

**시장 심리 사이클의 단계:**

시장 심리는 극단적인 낙관과 극단적인 비관 사이를 진자처럼 오갑니다. 이 사이클은 감정의 연속적인 흐름으로 이해할 수 있습니다.

| 단계 | 심리 상태 | 특징 |
|------|----------|------|
| 1단계 | **비관/절망** | 사이클의 바닥, 모두가 포기, 최고의 매수 기회 |
| 2단계 | **희망** | 일부가 회복 가능성 인식, 저점 매수 시작 |
| 3단계 | **낙관** | 상승 추세 확인, 더 많은 투자자 참여 |
| 4단계 | **환희** | 상승장 지속, "쉽게 돈 버는" 분위기 |
| 5단계 | **탐욕** | 사이클의 정점, FOMO 만연, 위험 무시 |
| 6단계 | **걱정** | 일부가 경고 신호 감지, 스마트머니 이탈 시작 |
| 7단계 | **공포** | 하락 시작, 패닉 매도 증가 |
| 8단계 | **공황** | 급락, 투매, 다시 비관/절망으로 회귀 |

핵심 통찰은 **극단적인 탐욕일 때 매도하고, 극단적인 공포일 때 매수**하는 것입니다. 그러나 이것이 말처럼 쉽지 않은 이유는 극단에서는 모든 뉴스와 논리가 그 방향을 지지하기 때문입니다.

### 3. 위험의 정의

막스는 전통적 위험 정의에 이의를 제기합니다.

| 전통적 정의 | 막스의 정의 |
|------------|------------|
| 변동성 (σ) | **영구적 자본 손실 확률** |
| 측정 가능 | 측정 어려움 |
| 과거 기반 | 미래 지향 |
| 베타 | 펀더멘털 약화, 과대 평가 |

**위험의 역설:**

막스가 강조하는 핵심 개념 중 하나는 위험의 역설입니다. **인식된 위험이 높아지면 실제 위험은 낮아집니다.** 왜냐하면 사람들이 위험을 인식하면 가격이 하락하고, 하락한 가격은 손실 가능성을 줄이기 때문입니다. 반대로, **인식된 위험이 낮아지면 실제 위험은 높아집니다.** 사람들이 안심하면 가격이 상승하고, 높아진 가격은 하락 여지를 키웁니다.

> "위험은 가장 안전해 보일 때 가장 크다"

이 역설을 이해하면 왜 시장이 가장 좋아 보일 때 조심해야 하고, 가장 무서울 때 기회가 있는지 알 수 있습니다.

## 시장 사이클의 법칙

### 사이클의 3가지 특성

1. **피할 수 없다** - 인간 심리가 만듦
2. **극단으로 간다** - 평균에 머물지 않음
3. **회귀한다** - 결국 평균으로 돌아옴

### 사이클 위치 파악

**낙관적 시기 (위험 신호):**

다음 징후들이 나타나면 시장이 사이클의 고점에 가까워지고 있다는 경고 신호입니다.

| 징후 | 설명 |
|------|------|
| "이번엔 다르다" 논리 | 새로운 패러다임이라는 주장이 확산됨 |
| 신규 투자자 대거 유입 | 투자 경험이 없는 사람들이 시장에 진입 |
| 레버리지 증가 | 빚을 내서 투자하는 경우 급증 |
| IPO 급증 | 기업공개가 활발, 투기성 기업도 상장 |
| 긍정적 언론 보도 | 미디어가 시장 상승을 대대적으로 보도 |
| 가치평가 정당화 논리 | 높은 밸류에이션을 합리화하는 새 지표 등장 |
| FOMO 심리 | "기회를 놓치면 안 된다"는 조급함 만연 |

**비관적 시기 (기회 신호):**

반대로 다음 징후들이 나타나면 매수 기회가 가까워지고 있다는 신호입니다.

| 징후 | 설명 |
|------|------|
| "회복 불가능" 논리 | 상황이 영원히 나쁠 것이라는 주장 |
| 투매성 매도 | 가격에 관계없이 팔아야 하는 강제 매도 |
| 펀드 환매 급증 | 투자자들이 자금을 빼감 |
| 우량 자산도 폭락 | 좋은 기업의 주식도 동반 하락 |
| 부정적 언론 보도 일색 | 미디어가 비관론만 전함 |
| 절망적 심리 | "절대 회복 못한다"는 분위기 |
| 매수자 부재 | 아무도 사려고 하지 않음 |

### 막스의 포트폴리오 조절

| 사이클 위치 | 포지션 | 전략 |
|------------|--------|------|
| 사이클 저점 | 공격적 | 100% 투자, 레버리지 가능, 집중 투자 |
| 평균 | 중립적 | 정상 배분, 분산 투자 |
| 사이클 고점 | 방어적 | 현금 증가, 품질 상향, 위험 자산 축소 |

## 투자에서 가장 중요한 것

막스의 책 《투자에 대한 생각》의 핵심 20가지:

### 사고 (Thinking)

1. **2차적 사고** - 한 단계 더 깊이
2. **시장 효율성 이해** - 완전하지도, 무의미하지도 않음
3. **가치 이해** - 내재가치 추정 능력
4. **가격과 가치의 관계** - 좋은 매수 ≠ 좋은 기업
5. **위험 이해** - 변동성이 아닌 손실 가능성

### 실행 (Action)

6. **위험 인식** - 위험은 숨어 있다
7. **위험 통제** - 손실 회피가 수익보다 중요
8. **사이클 주의** - 역사는 반복된다
9. **진자 인식** - 극단에서 기회
10. **심리적 영향 대응** - 감정 통제

### 역발상 (Contrarian)

11. **역발상 투자** - 컨센서스에 맞서기
12. **저가 매수** - 떨어지는 칼 잡기
13. **인내** - 시간이 필요
14. **무지의 인식** - 모르는 것을 알기
15. **예측의 한계** - 미래는 불확실

### 방어 (Defense)

16. **합리적 기대** - 현실적 수익률
17. **방어적 투자** - 공격보다 방어
18. **함정 피하기** - 손실 회피
19. **부가가치 창출** - 알파 추구
20. **모든 것의 통합** - 종합적 접근

## 체크리스트: 막스 스타일 투자

### 매수 전 확인

막스 스타일로 투자하기 전에 다음 사항들을 반드시 확인해야 합니다.

| 확인 항목 | 질문 |
|----------|------|
| 차별화된 관점 | 컨센서스와 다른 나만의 관점이 있는가? |
| 하방 위험 | 내 관점이 틀릴 경우 손실은 얼마인가? |
| 사이클 위치 | 현재 시장은 사이클의 어디에 있는가? |
| 비관론 반영 | 가격이 비관론을 충분히 반영했는가? |
| 영구 손실 위험 | 영구적 자본 손실 위험은 낮은가? |
| 인내심 | 회복까지 기다릴 인내심이 있는가? |

### 매도 신호

다음 상황이 발생하면 매도를 고려해야 합니다.

| 매도 신호 | 설명 |
|----------|------|
| 낙관론 완전 반영 | 모든 좋은 뉴스가 이미 가격에 포함됨 |
| "이번엔 다르다" 논리 | 새 패러다임 주장이 등장함 |
| 더 나은 기회 | 더 좋은 위험/보상 비율의 투자처 발견 |
| 투자 논리 무효화 | 원래 매수 이유가 더 이상 유효하지 않음 |

## 부실채권 투자

막스의 전문 분야인 부실채권 투자:

### 기회 발생 시점

부실채권 투자 기회는 다음과 같은 과정을 거쳐 발생합니다.

**1단계: 과도한 신용 확장** - 경기 호황기에 대출 기준이 느슨해지고, 위험한 차입자에게도 자금이 공급됩니다.

**2단계: 버블 형성과 붕괴** - 신용 버블이 형성되고 결국 터지면서 패닉 매도가 발생합니다.

**3단계: 동반 하락** - 패닉 상황에서는 우량 채권도 투기등급 채권과 함께 폭락합니다.

**4단계: 저가 매수 기회** - 회복 가능성이 높은 채권을 액면가의 수십 퍼센트에 매수할 수 있습니다.

**5단계: 시장 정상화** - 시장이 안정되면서 채권 가격이 정상 수준으로 회복되어 큰 수익을 실현합니다.

### 부실채권 분석

| 확인 항목 | 내용 |
|----------|------|
| 기업 가치 | 청산가치 vs 계속기업가치 비교 |
| 자본구조 | 선순위 vs 후순위 채권의 위치 |
| 회생 가능성 | 구조조정 여력과 경영진 역량 |
| 회수율 | 원금 대비 회수 예상 비율 |
| 타이밍 | 정상화 예상 시점과 촉매 요인 |

## 불확실성에 대한 대응

### 알 수 있는 것 vs 알 수 없는 것

투자에서 겸손함은 무엇을 알 수 있고 무엇을 알 수 없는지 구분하는 데서 시작합니다.

**알 수 있는 것:**

현재 가치평가 수준을 분석하여 시장이 비싼지 싼지 판단할 수 있습니다. 현재 시장 심리가 탐욕인지 공포인지 파악할 수 있습니다. 과거의 역사적 패턴을 통해 유사한 상황에서 무슨 일이 일어났는지 알 수 있습니다. 그리고 기업의 현재 펀더멘털, 즉 재무 상태와 경쟁력을 분석할 수 있습니다.

**알 수 없는 것:**

정확한 미래는 아무도 모릅니다. 사이클의 정확한 전환점을 예측하는 것은 불가능합니다. 외부 충격인 블랙스완 이벤트는 예측할 수 없습니다. 그리고 완벽한 시장 타이밍도 불가능합니다.

### 불확실성 대응 원칙

1. **겸손** - 모른다는 것을 인정하고, 예측에 과신하지 않습니다.
2. **분산** - 한 곳에 집중하지 않고, 여러 자산과 전략에 분산합니다.
3. **안전마진** - 틀려도 괜찮은 가격에 매수하여 실수의 여지를 확보합니다.
4. **인내** - 시간의 편에 서서, 시장이 가치를 인식할 때까지 기다립니다.
