# 투자 대가별 실전 스크리닝 기준

투자 대가들의 철학을 실전에 적용하려면, 추상적인 원칙을 **구체적이고 정량적인 스크리닝 기준**으로 변환해야 합니다. 이 가이드는 각 대가의 핵심 기준을 수치화하고, 한국 주식시장에 적용하는 방법을 다룹니다.

---

## 벤저민 그레이엄 (Benjamin Graham)

그레이엄은 두 가지 명확한 스크리닝 전략을 제시했습니다: 극단적 저평가 종목을 찾는 **Net-Net 전략**과 안정성 중심의 **방어적 투자자 기준**입니다.

### Net-Net 전략 (초저평가 스크리닝)

기업의 순유동자산가치(NCAV)보다 시가총액이 낮은 종목을 찾는 전략입니다.

| 기준 | 조건 | 산출 방법 |
|:-----|:-----|:----------|
| NCAV | 시가총액 < NCAV x 2/3 | NCAV = 유동자산 - 총부채 |
| 유동비율 | > 150% | 유동자산 / 유동부채 |
| 부채비율 | < 100% | 총부채 / 자기자본 |
| 최근 실적 | 직전 연도 영업이익 > 0 | 영업손실 기업 제외 |

:::tip 한국 시장 Net-Net 현실
한국 시장에서 Net-Net 종목은 주로 소형주, 지주사, 부동산 보유 기업에서 발견됩니다. 다만 유동성이 낮고 지배구조 리스크가 있는 경우가 많으므로, **유동자산의 질**(현금성 자산 비중)을 반드시 확인하세요.
:::

### 방어적 투자자 기준 (Defensive Investor)

장기 보유를 전제로 안정적인 기업을 선별하는 기준입니다.

| 기준 | 조건 | 비고 |
|:-----|:-----|:-----|
| 시가총액 | 중형주 이상 | 한국 기준 시총 3,000억 이상 |
| PER | < 15배 | 최근 3년 평균 기준 |
| PBR | < 1.5배 | 또는 PER x PBR < 22.5 |
| 유동비율 | > 200% | 단기 지급 능력 확인 |
| 연속 배당 | 10년 이상 | 최소 5년으로 완화 가능 |
| 이익 성장 | 10년간 최소 33% | 연평균 약 3% 성장 |
| 부채비율 | < 100% | 장기부채 / 순유동자산 기준 |

**그레이엄 넘버 계산**: [Graham Number 계산기](/calculators/graham-number)에서 적정가격을 바로 산출할 수 있습니다.

```
Graham Number = √(22.5 × EPS × BPS)
```

## 워렌 버핏 (Warren Buffett)

버핏의 기준은 정량적 수치와 정성적 판단이 결합되어 있습니다. 정량 필터로 1차 스크리닝한 후, 정성 분석으로 최종 판단합니다.

### 정량 스크리닝 기준

| 기준 | 조건 | 확인 기간 |
|:-----|:-----|:----------|
| ROE | > 15% (연속) | 최근 10년 |
| 순이익률 | > 10% | 최근 5년 평균 |
| 부채비율 | < 80% | 최근 연도 |
| 이익 일관성 | 적자 연도 0회 | 최근 10년 |
| 잉여현금흐름 | 매년 양수 | 최근 5년 |
| 자본적 지출 | 순이익 대비 < 50% | 유지보수 CAPEX 중심 |
| 이익잉여금 | 꾸준히 증가 | 복리 성장의 증거 |

### 정성적 체크리스트 (경제적 해자)

| 해자 유형 | 확인 방법 | 한국 시장 예시 |
|:----------|:----------|:-------------|
| 브랜드 파워 | 가격 프리미엄 유지 가능 여부 | 소비재, 화장품 기업 |
| 전환비용 | 고객이 경쟁사로 이동하기 어려운 구조 | ERP, 반도체 장비 |
| 네트워크 효과 | 사용자 증가 시 가치 증가 | 플랫폼 기업 |
| 원가 우위 | 구조적 비용 우위 보유 | 대규모 제조업, 유통 |
| 규제 진입장벽 | 인허가, 라이선스 필요 | 금융, 통신, 제약 |

:::tip 버핏 스크리닝 핵심
버핏은 "훌륭한 기업을 적정 가격에 사라"고 합니다. 정량 기준을 통과한 기업이 **일시적 악재로 주가가 하락**했을 때가 매수 적기입니다. [DCF 계산기](/calculators/dcf)로 적정가치를 산출하고, 20~30% 안전마진을 확보하세요.
:::

## 피터 린치 (Peter Lynch)

린치는 성장주를 합리적인 가격에 매수하는 GARP(Growth at a Reasonable Price) 전략의 대가입니다.

### 정량 스크리닝 기준

| 기준 | 조건 | 산출 방법 |
|:-----|:-----|:----------|
| PEG | < 1.0 (이상적), < 1.5 (허용) | PER / 이익성장률 |
| 이익 성장률 | 15~25% (연간) | 최근 3~5년 평균 |
| 부채비율 | < 80% | 동종 업계 평균 이하 |
| 잉여현금흐름 | 양수 | 최근 3년 연속 |
| 내부자 매수 | 최근 6개월 내 순매수 | DART 임원 지분 변동 확인 |
| 재고자산 | 매출 대비 감소 추세 | 재고 회전일수 확인 |
| PER | 업종 평균 이하 | 동종업체 대비 저평가 |

### 린치의 6가지 종목 분류

| 분류 | 특징 | 적합한 기준 |
|:-----|:-----|:-----------|
| 저성장주 (Slow Growers) | 성장률 < 5%, 고배당 | 배당수익률 > 시장금리 |
| 대형 우량주 (Stalwarts) | 성장률 10~12% | PEG < 1.5, 꾸준한 실적 |
| 고성장주 (Fast Growers) | 성장률 > 20% | PEG < 1.0, 재무 안정성 |
| 경기순환주 (Cyclicals) | 실적이 경기에 연동 | 사이클 저점에서 매수 |
| 턴어라운드 (Turnarounds) | 위기 후 회복 기업 | 구조조정 진행, 부채 감소 |
| 자산주 (Asset Plays) | 숨겨진 자산 보유 | 순자산가치 대비 할인 |

**PEG 스크리닝**: [PEG Screener](/calculators/peg-screener)에서 PEG 기반 스크리닝을 바로 수행할 수 있습니다.

## 필립 피셔 (Philip Fisher)

피셔는 정량 지표보다 정성적 분석을 중시했지만, 일부 기준은 수치화할 수 있습니다.

### 정량 스크리닝 기준

| 기준 | 조건 | 확인 방법 |
|:-----|:-----|:----------|
| 매출 성장률 | > 업종 평균 | 최근 5년 연평균 |
| R&D 비율 | > 업종 상위 25% | 매출 대비 연구개발비 |
| 영업이익률 | 개선 추세 | 최근 5년 추이 |
| 매출 다각화 | 상위 고객 비중 < 30% | 고객 집중도 리스크 |

### 피셔의 15가지 질문 (정성 체크리스트)

핵심 질문을 정량화 가능한 형태로 변환하면 다음과 같습니다.

| 핵심 질문 | 정량적 지표 | 기준 |
|:----------|:-----------|:-----|
| 향후 수년간 매출 성장 가능한가? | 매출 CAGR | > 10% (3~5년 전망) |
| 경영진이 R&D에 충분히 투자하는가? | R&D/매출 | > 업종 중간값 |
| 영업 마진은 충분한가? | 영업이익률 | > 업종 상위 30% |
| 노사 관계는 양호한가? | 직원 이직률, 평균 근속연수 | 업종 평균 이상 |
| 내부자가 자사주를 보유하는가? | 경영진 지분율 | 유의미한 수준 |

:::tip 피셔 스타일 리서치
피셔의 "소문 조사법(Scuttlebutt)"을 현대적으로 적용하려면: (1) 해당 기업의 전/현직 직원 리뷰 사이트 확인, (2) 주요 거래처/경쟁사 관계자 의견 수집, (3) 업계 컨퍼런스 및 IR 자료 분석을 수행하세요.
:::

## 조엘 그린블라트 (Joel Greenblatt) — Magic Formula

그린블라트의 Magic Formula는 가장 체계적이고 기계적으로 적용 가능한 전략입니다.

### Magic Formula 스크리닝 기준

| 기준 | 산출 방법 | 순위 기준 |
|:-----|:----------|:----------|
| 이익수익률 (Earnings Yield) | EBIT / EV | 높을수록 좋음 (저평가) |
| 투하자본수익률 (ROIC) | EBIT / (순운전자본 + 순고정자산) | 높을수록 좋음 (고품질) |

### 적용 프로세스

1. **유니버스 설정**: 시가총액 하위 종목 및 금융/유틸리티 제외
2. **이익수익률 순위 산출**: 전 종목 대상 높은 순서로 순위 부여
3. **ROIC 순위 산출**: 전 종목 대상 높은 순서로 순위 부여
4. **합산 순위**: 이익수익률 순위 + ROIC 순위 = 종합 순위
5. **포트폴리오 구성**: 상위 20~30개 종목에 동일 비중 투자
6. **리밸런싱**: 연 1회 (보유 후 1년)

| 추가 필터 (한국 시장 적용) | 조건 |
|:---------------------------|:-----|
| 시가총액 | > 1,000억 원 |
| 업종 제외 | 금융, 유틸리티, 지주사 |
| 이익 안정성 | 최근 3년 중 적자 연도 없음 |
| 거래량 | 일평균 거래대금 > 5억 원 |

**ROIC 계산**: [ROIC 계산기](/calculators/roic)에서 개별 기업의 ROIC를 산출할 수 있습니다.

:::tip Magic Formula 실전 적용
그린블라트는 최소 3년 이상 기계적으로 운용해야 효과를 볼 수 있다고 강조합니다. 단기적으로 시장을 언더퍼폼하는 구간이 반드시 존재하므로, **규칙을 바꾸지 않는 인내심**이 핵심입니다.
:::

## 윌리엄 오닐 (William O'Neil) — CANSLIM

CANSLIM은 성장 모멘텀과 기술적 분석을 결합한 전략으로, 각 글자가 하나의 스크리닝 기준을 의미합니다.

| 글자 | 기준 | 정량적 조건 |
|:----:|:-----|:-----------|
| **C** | Current Quarterly Earnings (당분기 이익) | 전년 동기 대비 EPS 성장률 > 25% |
| **A** | Annual Earnings Growth (연간 이익 성장) | 최근 3~5년 EPS 연평균 성장률 > 25% |
| **N** | New Products/Management/Price High (새로운 변화) | 신제품 출시, 신규 경영진, 52주 신고가 근접 |
| **S** | Supply and Demand (수급) | 발행주식수 적정, 거래량 증가 동반 주가 상승 |
| **L** | Leader or Laggard (업종 내 위치) | 상대강도(RS) 순위 상위 20% |
| **I** | Institutional Sponsorship (기관 관심) | 기관 보유 비중 증가 추세, 우량 기관 편입 |
| **M** | Market Direction (시장 방향) | 주요 지수 상승 추세 확인 (약세장에서는 현금 보유) |

### CANSLIM 실전 적용 기준

| 단계 | 확인 사항 | 한국 시장 데이터 소스 |
|:-----|:----------|:---------------------|
| 1차 스크리닝 | C + A 기준 충족 | FnGuide, DART 분기보고서 |
| 2차 스크리닝 | L (상대강도) 확인 | 증권사 HTS, 네이버 금융 |
| 3차 확인 | I (기관 수급) 확인 | KRX 투자자별 매매동향 |
| 최종 확인 | M (시장 방향) 판단 | KOSPI/KOSDAQ 추세 분석 |
| 매수 타이밍 | 적절한 베이스 패턴 돌파 시 | 차트 패턴 분석 (Cup & Handle 등) |

:::caution CANSLIM 주의사항
CANSLIM은 **강세장에서 극대화**되는 전략입니다. 약세장이나 횡보장에서는 M 기준에 따라 현금 비중을 높이는 것이 원칙입니다. 시장 방향을 무시하고 매수하면 큰 손실을 입을 수 있습니다.
:::

### 시장 구조적 특성

| 특성 | 영향 | 대응 |
|:-----|:-----|:-----|
| 대주주 할인 | 지배구조 리스크로 PBR 할인 | 그레이엄 기준 PBR을 1.0으로 하향 조정 |

## 실전 스크리닝 워크플로우

아래 순서로 복수 대가의 기준을 조합하면 보다 견고한 종목 선별이 가능합니다.

### 추천 조합 전략

**가치+품질 조합 (그레이엄 + 버핏)**

1. 그레이엄 방어적 투자자 기준으로 1차 필터링
2. 버핏의 ROE > 15%, 이익 일관성 기준 추가 적용
3. [Graham Number 계산기](/calculators/graham-number)로 적정가격 확인
4. [DCF 계산기](/calculators/dcf)로 내재가치 교차 검증

**성장+가치 조합 (린치 + 그린블라트)**

1. PEG < 1.5 기준으로 성장 대비 저평가 종목 추출
2. Magic Formula 순위로 품질 + 저평가 동시 확인
3. [PEG Screener](/calculators/peg-screener)로 1차 스크리닝
4. [ROIC 계산기](/calculators/roic)로 자본 효율성 검증

:::tip 스크리닝은 시작일 뿐입니다
정량적 스크리닝은 분석 대상을 좁히는 첫 단계입니다. 스크리닝을 통과한 종목에 대해 반드시 **사업보고서 정독, 경쟁 우위 분석, 경영진 평가** 등 정성적 분석을 수행하세요. [DART 공시 분석 가이드](/company-analysis/dart-guide)를 참고하여 공시 원문을 직접 확인하는 것을 권장합니다.
:::
