# 벤자민 그레이엄 (1894-1976)

> **"가치투자의 아버지"**

<div className="master-quote">
"투자는 철저한 분석을 통해 원금의 안전과 적절한 수익을 약속하는 행위다. 이 기준을 충족하지 못하는 것은 투기다."

<span className="author">— 벤자민 그레이엄, "증권분석"</span>
</div>

## 인물 소개

벤자민 그레이엄은 **가치투자(Value Investing)**의 창시자로, 워렌 버핏의 스승입니다. 컬럼비아 비즈니스스쿨에서 투자론을 가르쳤으며, "증권분석(Security Analysis)"과 "현명한 투자자(The Intelligent Investor)"는 투자의 바이블로 불립니다.

| 항목 | 내용 |
|------|------|
| 생애 | 1894-1976 (82세) |
| 국적 | 미국 |
| 학력 | 컬럼비아 대학교 |
| 제자 | 워렌 버핏, 월터 슐로스 |
| 대표 저서 | 증권분석, 현명한 투자자 |

## 핵심 철학: 안전마진

<div className="formula-block">

**안전마진(Margin of Safety) = 내재가치 - 시장가격**

</div>

그레이엄의 핵심 원칙은 **안전마진**입니다. 내재가치보다 충분히 낮은 가격(20-40% 할인)에 매수해야 예측 오류나 불운에서도 손실을 최소화할 수 있습니다.

<div className="master-quote">
"투자의 비결을 세 단어로 요약하면: 안전마진(Margin of Safety)."

<span className="author">— 벤자민 그레이엄</span>
</div>

## 핵심 투자 기법

### 1. 그레이엄 넘버 (Graham Number)

주식의 최대 적정 가격을 계산하는 공식:

<div className="formula-block">

**Graham Number = √(22.5 × EPS × BVPS)**

</div>

| 요소 | 의미 | 기준 |
|------|------|------|
| 22.5 | PER 15 × PBR 1.5 | 그레이엄의 상한 기준 |
| EPS | 주당순이익 | 정상화된 수치 |
| BVPS | 주당순자산 | 장부가 기준 |

**예시:**
- EPS: 5,000원
- BVPS: 40,000원
- Graham Number = √(22.5 × 5,000 × 40,000) = **67,082원**
- 현재 주가가 50,000원이면 → **저평가** (안전마진 25%)

**[그레이엄 넘버 계산기 →](/calculators/graham-number)**

### 2. Net-Net 투자 (순유동자산 투자)

극단적 저평가 기업을 찾는 방법:

<div className="formula-block">

**NCAV (Net Current Asset Value) = 유동자산 - 총부채**

**Net-Net 기준: 시가총액 < NCAV × 2/3**

</div>

이 기준을 충족하면 기업을 청산해도 투자금의 1.5배를 회수할 수 있습니다.

| 항목 | 금액 |
|------|------|
| 유동자산 | 100억 |
| 총부채 | 40억 |
| **NCAV** | **60억** |
| 매수 기준 | < 40억 |

### 3. 방어적 투자자 기준 (7가지)

그레이엄이 "현명한 투자자"에서 제시한 보수적 종목 선정 기준:

| # | 기준 | 이유 |
|---|------|------|
| 1 | 시총 $100M 이상 | 충분한 규모와 유동성 |
| 2 | 유동비율 200% 이상 | 재무 안정성 |
| 3 | 10년 연속 흑자 | 수익의 안정성 |
| 4 | 20년 연속 배당 | 주주 환원 |
| 5 | 10년간 EPS 33% 이상 성장 | 최소한의 성장 |
| 6 | PER 15배 이하 | 가치 기준 |
| 7 | PBR 1.5배 이하 | 자산 기준 |

**복합 기준:** PER × PBR ≤ 22.5

### 4. 적극적 투자자 기준

더 적극적인 투자자를 위한 추가 기회:

- **특수 상황 (Special Situations)**: 구조조정, 스핀오프, 합병 차익거래
- **턴어라운드**: 일시적 어려움을 겪는 우량 기업
- **2류 기업**: 1류보다 저평가된 견실한 기업

## 미스터 마켓 (Mr. Market)

그레이엄의 유명한 비유:

> "시장을 당신의 파트너 '미스터 마켓'으로 생각하라. 그는 매일 당신에게 주식을 사거나 팔겠다고 제안한다. 그의 가격은 때때로 합리적이지만, 종종 그의 기분에 따라 터무니없이 높거나 낮다.
>
> 그의 감정에 휘둘리지 말고, 그가 불합리할 때만 거래하라."

**핵심 교훈:**
- 시장은 단기적으로 감정에 좌우됨
- 가격 변동은 기회, 위협이 아님
- 내재가치에 집중하라

## 투자 vs 투기

| 항목 | 투자 | 투기 |
|------|------|------|
| 분석 | 철저한 기본적 분석 | 시장 동향, 소문 |
| 목표 | 원금 안전 + 적절한 수익 | 빠른 수익 |
| 기간 | 장기 | 단기 |
| 가격 | 내재가치 대비 판단 | 가격 움직임 추종 |

## 현대적 적용

그레이엄의 원칙을 현대에 적용할 때 고려사항:

| 시대 차이 | 조정 방법 |
|----------|----------|
| 무형자산 중요성 증가 | 조정 BVPS (무형자산 가치 반영) |
| 배당 성향 감소 | 자사주 매입 포함 총주주환원 |
| 정보 효율성 증가 | 소형주, 해외 시장에서 기회 |
| 회계 기준 변화 | Adjusted 지표 사용 |

## 그레이엄 스타일 스크리너

현재 시장에서 그레이엄 기준 충족 종목을 찾는 조건:

```
조건 1: PER < 15
조건 2: PBR < 1.5
조건 3: PER × PBR < 22.5
조건 4: 부채비율 < 100%
조건 5: 5년 연속 흑자
조건 6: 시가총액 > 5,000억원
```
