# 보험 산업 완벽 가이드

보험 산업은 **미래의 불확실한 위험을 경제적으로 보장**하는 금융 산업의 핵심 축입니다. 이 가이드에서는 보험의 기본 원리부터 수익구조, 주요 직무, 규제체계, 최신 트렌드까지 보험 산업 전반을 체계적으로 다룹니다. 보험업 취업을 준비하는 학생, 계리사나 손해사정사를 목표로 하는 수험생, 보험사 주식에 투자하는 투자자 모두에게 실질적 도움이 되는 내용을 담았습니다.

---

## 1. 보험의 정의와 원리

### 1.1 보험이란 무엇인가

보험(Insurance)은 동일한 위험에 노출된 다수의 경제 주체가 위험의 발생 확률에 기초한 보험료를 공동으로 갹출하여, 실제로 위험이 발생한 구성원에게 약정된 보험금을 지급하는 경제적 제도입니다. 핵심은 **"위험의 이전과 분산"** 에 있습니다.

보험은 법적으로 「보험업법」에 따라 **생명보험업**, **손해보험업**, **제3보험업** 으로 구분되며, 보험사업을 영위하려면 금융위원회의 인가가 필요합니다.

### 1.2 보험의 기본 원리

보험이 경제적으로 성립하기 위해서는 다음의 수리적·법적 원리가 충족되어야 합니다.

| 원리 | 설명 | 구체적 의미 |
|------|------|------------|
| **대수의 법칙** | 관찰 대상이 많아질수록 실제 결과가 기대값에 수렴 | 가입자 수가 충분하면 사고 발생률을 정확히 예측 가능 |
| **수지상등의 원칙** | 보험료 수입 총액 = 보험금 지급 총액 | 보험 사업의 수리적 균형 조건 |
| **급부·반대급부 균등의 원칙** | 개인이 납입한 보험료 = 그 위험도에 비례 | 위험이 높은 사람은 더 높은 보험료를 납부 |
| **이익금지의 원칙** | 보험금은 실제 손해를 초과할 수 없음 | 손해보험에 적용, 도덕적 해이 방지 |

#### 대수의 법칙 상세

대수의 법칙(Law of Large Numbers)은 보험의 수학적 기반입니다. 예를 들어 자동차 사고율이 5%라고 할 때, 자동차 100대만 관찰하면 실제 사고율이 2~8%로 변동할 수 있지만, 100만대를 관찰하면 실제 사고율은 4.9~5.1%에 수렴합니다. 이 원리 덕분에 보험사는 보험료를 정확히 산출할 수 있습니다.

#### 수지상등의 원칙 상세

수지상등의 원칙을 수식으로 표현하면 다음과 같습니다.

```
P × n = S × n × q
(P: 순보험료, n: 가입자 수, S: 보험금, q: 사고발생확률)

따라서 순보험료 P = S × q
```

예를 들어 사망 확률이 0.1%이고 사망보험금이 1억원이면, 순보험료는 10만원(= 1억 × 0.001)이 됩니다.

### 1.3 보험료의 구조

보험료는 크게 **순보험료**와 **부가보험료**로 구성됩니다.

```
영업보험료(가입자가 실제 납부하는 보험료) = 순보험료 + 부가보험료
```

| 구분 | 하위 항목 | 설명 |
|------|----------|------|
| **순보험료** | 위험보험료 | 보험금 지급에 충당되는 부분 (사망률, 사고율 기반) |
| | 저축보험료 | 만기환급금·해약환급금에 충당되는 부분 (저축성보험에 해당) |
| **부가보험료** | 신계약비 | 모집인 수수료, 계약 체결 비용 |
| | 유지비 | 계약 유지·관리 비용 |
| | 수금비 | 보험료 수납 비용 |
| | 이익배당준비금 | 보험사 이익에 해당하는 부분 |

#### 예정기초율

순보험료 산출에는 세 가지 예정기초율이 사용됩니다.

| 예정기초율 | 설명 | 보험료에 미치는 영향 |
|-----------|------|------------------|
| **예정위험률** | 사고·사망 등의 예상 발생률 | 높을수록 보험료 상승 |
| **예정이율** | 보험사의 예상 투자수익률 | 높을수록 보험료 하락 |
| **예정사업비율** | 예상 운영비 비율 | 높을수록 보험료 상승 |

### 1.4 보험의 사회적 역할

- **경제적 안정 장치**: 예상치 못한 사고, 질병, 사망 등으로 인한 경제적 충격 완화
- **자본 축적 기능**: 장기 보험료 적립을 통한 대규모 자본 형성, 자본시장에 장기 투자자금 공급
- **사회보장 보완**: 국민건강보험, 국민연금 등 공적 보장의 부족분을 민간 보험이 보완
- **기업 리스크 관리**: 배상책임보험, 화재보험 등을 통한 기업 경영 안정
- **고용 창출**: 보험설계사, 손해사정사 등 대규모 고용 효과

## 2. 보험의 종류

### 2.1 생명보험 (Life Insurance)

생명보험은 사람의 생존과 사망에 관한 위험을 보장하는 보험입니다.

| 보험 종류 | 보장 내용 | 특징 | 대표 상품 |
|----------|----------|------|----------|
| **종신보험** | 사망 시 보험금 지급 (기간 제한 없음) | 평생 보장, 상속 설계에 활용 | 무배당 종신보험 |
| **정기보험** | 일정 기간 내 사망 시 보험금 지급 | 저렴한 보험료, 만기환급금 없음 | 10년/20년 정기보험 |
| **변액보험** | 보험료를 펀드에 투자, 실적에 따라 보험금 변동 | 투자 기능 결합, 원금 비보장 가능 | 변액유니버셜, 변액종신 |
| **연금보험** | 일정 연령부터 연금 형태로 보험금 수령 | 노후 대비, 세제 혜택 | 연금저축보험, 즉시연금 |
| **저축성보험** | 만기 시 환급금 지급 | 보장보다 저축 기능 중심 | 교육보험, 저축보험 |

#### 생명보험의 핵심 경험률

생명보험 상품 설계에는 다양한 경험통계가 활용됩니다.

| 경험률 | 설명 | 출처 |
|--------|------|------|
| **경험생명표** | 성별·연령별 사망률 | 보험개발원 |
| **경험질병발생률** | 암, 뇌졸중, 심근경색 등 발생률 | 보험개발원 |
| **해약률** | 계약 유지 기간별 해약 확률 | 개별 보험사 경험 데이터 |
| **경험사업비율** | 실제 운영비 비율 | 개별 보험사 실적 |

### 2.2 손해보험 (Non-Life Insurance / General Insurance)

손해보험은 우연한 사고로 인한 재산상의 손해를 보상하는 보험입니다.

| 보험 종류 | 보장 내용 | 의무 가입 여부 | 시장 규모 |
|----------|----------|--------------|----------|
| **자동차보험** | 교통사고로 인한 대인·대물·자차 손해 | 대인배상 I: 의무 | 손보 시장 최대 비중(약 30%) |
| **화재보험** | 화재로 인한 재산 손해 | 특수건물: 의무 | 기업 물건 중심 |
| **배상책임보험** | 타인에게 법률상 배상해야 할 손해 | 일부 업종 의무 | 성장세 |
| **해상보험** | 해상 운송 중 화물·선박 손해 | 임의 | 무역 의존도 높은 한국에서 중요 |
| **보증보험** | 채무불이행에 따른 손해 | 임의 | 서울보증보험 독점 |
| **기술보험** | 공사·기계 등 기술적 위험 | 임의 | 건설·제조업 관련 |

#### 자동차보험 구조 상세

자동차보험은 손해보험에서 가장 큰 비중을 차지하며, 그 구조를 이해하면 손해보험의 기본 원리를 파악할 수 있습니다.

| 담보 항목 | 성격 | 보장 내용 | 비고 |
|----------|------|----------|------|
| **대인배상 I** | 의무보험 | 타인 신체 상해 (법정 한도) | 미가입 시 과태료 |
| **대인배상 II** | 임의보험 | 대인배상 I 초과분 무한 보장 | 사실상 필수 |
| **대물배상** | 임의보험 | 타인 재산 손해 | 2천만원~10억원 선택 |
| **자기신체사고(자손)** | 임의보험 | 본인 신체 상해 | 과실 비율 무관 |
| **자기차량손해(자차)** | 임의보험 | 본인 차량 손해 | 자기부담금 존재 |
| **무보험차 상해** | 임의보험 | 무보험차에 의한 피해 | 뺑소니 포함 |

### 2.3 제3보험 (Third Sector Insurance)

제3보험은 생명보험(제1보험)과 손해보험(제2보험)의 중간 영역으로, 사람의 질병이나 상해를 보장하되 실제 발생한 손해를 보상하는 성격을 가집니다. 생명보험사와 손해보험사 모두 판매할 수 있습니다.

| 보험 종류 | 보장 내용 | 특징 | 시장 동향 |
|----------|----------|------|----------|
| **실손의료보험** | 실제 발생한 의료비 보상 | 국민건강보험 보완, 비급여 항목 보장 | 4세대 실손(2021~) |
| **암보험** | 암 진단 시 진단금 지급 | 정액 보장 | 가장 인기 있는 질병보험 |
| **상해보험** | 외부 사고로 인한 신체 상해 보장 | 골절, 화상, 교통사고 등 | 단체 상해보험 포함 |
| **건강보험** | 질병 입원·수술 시 보험금 | 일당, 수술비 등 정액 보장 | 고령화로 성장 |
| **간병보험** | 간병 필요 상태 시 보험금 | 치매, 장기간병 보장 | 신성장 분야 |

#### 실손의료보험의 세대별 변화

실손의료보험은 한국에서 "제2의 건강보험"으로 불릴 만큼 중요한 보험입니다.

| 세대 | 가입 시기 | 자기부담금 | 비급여 보장 | 주요 변화 |
|------|----------|-----------|-----------|----------|
| **1세대** | ~2009 | 10% | 전액 보장 | 도덕적 해이 문제 |
| **2세대** | 2009~2017 | 급여 10%, 비급여 20% | 보장 | 급여/비급여 분리 |
| **3세대** | 2017~2021 | 급여 10%, 비급여 20% | 보장(일부 제한) | 15년 재가입 |
| **4세대** | 2021~ | 급여 20%, 비급여 30% | 비급여 특약 분리 | 자기부담금 인상, 보험료 연동제 |

### 2.4 재보험 (Reinsurance)

재보험은 보험회사가 인수한 위험의 일부를 다른 보험회사(재보험사)에게 이전하는 것으로, **"보험회사의 보험"** 이라고 합니다.

#### 재보험의 구조

```
보험가입자 → [원수보험사] → 일부 위험 이전 → [재보험사] → 일부 위험 이전 → [재재보험사]
```

| 재보험 방식 | 설명 | 특징 |
|------------|------|------|
| **비례재보험** | 보험금과 보험료를 일정 비율로 분담 | Quota Share, Surplus |
| **비비례재보험** | 일정 금액 초과분만 재보험사 부담 | Excess of Loss, Stop Loss |
| **임의재보험** | 개별 건별로 재보험 체결 | 대형·특수 위험에 활용 |
| **특약재보험** | 일정 범주의 위험을 자동 출재 | 효율적, 포트폴리오 기반 |

#### 코리안리재보험 (Korean Re)

코리안리재보험은 한국 유일의 전업 재보험사이자 아시아 주요 재보험사 중 하나입니다.

- **설립**: 1963년(대한재보험공사), 2002년 민영화
- **역할**: 국내 원수보험사의 재보험 수요 처리, 해외 재보험 인수
- **시장 위치**: 국내 출재 재보험의 약 30~40% 인수
- **글로벌 순위**: 세계 재보험사 Top 20 이내
- **상장 여부**: KOSPI 상장

글로벌 주요 재보험사로는 **Munich Re**(독일), **Swiss Re**(스위스), **Hannover Re**(독일), **Berkshire Hathaway Re**(미국), **SCOR**(프랑스) 등이 있습니다.

## 3. 보험사의 수익구조

보험사의 손익은 크게 **보험영업손익**과 **투자영업손익**으로 구분됩니다. 보험사의 수익구조를 이해하는 것은 보험사 [재무 분석](/docs/financial-analysis/overview)과 [기업가치 평가](/docs/valuation)에 필수적입니다.

### 3.1 보험영업이익 (Underwriting Profit)

보험영업이익은 보험료 수입에서 보험금 지급과 사업비를 차감한 금액입니다.

```
보험영업이익 = 경과보험료 - 발생손해액 - 순사업비
```

| 구성 요소 | 설명 | 비고 |
|----------|------|------|
| **경과보험료** | 해당 기간에 수익으로 인식된 보험료 | 보험료 수입 ≠ 경과보험료 (시간 경과 기준) |
| **발생손해액** | 보험금 지급액 + 지급준비금 변동액 | 실제 지급 + 미래 지급 예상분 |
| **순사업비** | 보험모집, 유지, 관리에 소요된 비용 | 설계사 수수료, 광고비, 인건비 등 |

### 3.2 투자영업이익 (Investment Income)

보험사는 보험료를 수령한 후 보험금을 지급하기까지의 기간 동안 보유한 자금(보험 부채에 대응하는 자산)을 운용하여 투자 수익을 얻습니다.

| 투자자산 종류 | 비중(약) | 특징 |
|-------------|---------|------|
| **채권** | 50~60% | 안정적 이자수익, ALM 핵심 자산 |
| **대출** | 15~20% | 부동산 담보대출, 기업대출 |
| **주식** | 5~10% | 수익 변동성 높음 |
| **부동산** | 5~10% | 임대수익 + 자본이득 |
| **대체투자** | 10~15% | 인프라, PEF, 해외부동산 등 |

#### 생명보험사 vs 손해보험사 투자 특성

| 구분 | 생명보험사 | 손해보험사 |
|------|----------|-----------|
| **보유 자산 규모** | 매우 큼 (장기 적립) | 상대적으로 작음 |
| **투자 기간** | 장기 (20~30년) | 단~중기 (1~5년) |
| **ALM 중요도** | 매우 높음 | 보통 |
| **투자수익 의존도** | 높음 (수익의 핵심) | 보통 (언더라이팅 수익도 중요) |
| **금리 민감도** | 매우 높음 | 보통 |

### 3.3 합산비율 (Combined Ratio)

합산비율은 손해보험사의 보험영업 수익성을 판단하는 가장 핵심적인 지표입니다.

```
합산비율 = 손해율 + 사업비율
```

| 지표 | 산식 | 의미 | 기준 |
|------|------|------|------|
| **손해율** | 발생손해액 / 경과보험료 × 100 | 보험료 대비 보험금 지급 수준 | 낮을수록 양호 |
| **사업비율** | 순사업비 / 경과보험료 × 100 | 보험료 대비 운영비 수준 | 낮을수록 양호 |
| **합산비율** | 손해율 + 사업비율 | 보험영업의 종합 수익성 | 100% 미만이면 영업 흑자 |

#### 합산비율 해석

| 합산비율 | 의미 | 해석 |
|---------|------|------|
| **90% 미만** | 영업 대폭 흑자 | 매우 양호한 언더라이팅 실적 |
| **90~100%** | 영업 소폭 흑자 | 적정 수준 |
| **100~110%** | 영업 소폭 적자 | 투자수익으로 보완 필요 |
| **110% 초과** | 영업 대폭 적자 | 구조적 문제 가능성 |

:::tip 투자자 참고사항
합산비율이 100%를 초과해도 투자수익까지 고려하면 보험사 전체로는 흑자일 수 있습니다. 특히 생명보험사는 투자수익 의존도가 높아, 합산비율만으로 수익성을 판단하면 오류가 발생할 수 있습니다.
:::

## 4. 보험사 주요 직무

보험사에는 계리사 외에도 다양한 전문 직무가 존재합니다. 각 직무의 역할과 요구 역량을 상세히 살펴보겠습니다.

### 4.1 언더라이팅 (Underwriting / 인수심사)

언더라이팅은 보험 계약의 인수 여부를 결정하고, 적정한 보험료와 보장 조건을 설정하는 업무입니다. 보험사의 수익성을 좌우하는 핵심 직무입니다.

| 업무 영역 | 구체적 내용 |
|----------|------------|
| **위험 평가** | 청약서, 건강진단서, 직업 정보 등을 분석하여 피보험자의 위험도를 평가 |
| **인수 기준 설정** | 질병별·직업별·연령별 인수 가능 여부와 조건을 결정하는 기준표 작성 |
| **요율 산정** | 위험도에 따른 할인·할증 기준 설정, 보험료 조정 |
| **특별 조건 부과** | 고위험 피보험자에 대한 보험료 할증, 특정 부위·질병 부담보 조건 부과 |
| **재보험 협의** | 고액·고위험 건에 대한 재보험 출재 결정 |

#### 언더라이팅 의사결정 프로세스

```
청약 접수 → 기본 심사(자동심사) → 정밀 심사(언더라이터) →
  ├ 표준체 인수 (보통 조건)
  ├ 조건부 인수 (할증, 부담보)
  ├ 연기 (재검사, 추가 서류)
  └ 거절 (인수 불가)
```

### 4.2 보상 / 손해사정 (Claims Adjustment)

보상 업무는 보험사고가 발생했을 때 보험금 지급 여부와 금액을 결정하는 업무입니다. 손해사정사(Loss Adjuster)가 핵심 역할을 수행합니다.

| 업무 단계 | 구체적 내용 |
|----------|------------|
| **사고 접수** | 보험사고 발생 신고 접수, 초기 사고 내용 파악 |
| **사고 조사** | 사고 원인·경위 조사, 현장 방문, 관련 서류 수집 |
| **손해액 산정** | 약관에 따른 보험금 산출, 면책 사유 확인 |
| **과실 판정** | 교통사고 등에서 과실 비율 판정 |
| **보험금 지급** | 최종 보험금 결정 및 지급 |
| **구상권 행사** | 제3자에게 배상 청구할 수 있는 경우 구상 처리 |

### 4.3 상품개발 (Product Development)

상품개발은 시장 수요와 규제 환경을 반영하여 새로운 보험상품을 설계하고 인가를 받는 업무입니다.

| 단계 | 업무 내용 | 관련 부서 |
|------|----------|----------|
| **시장 조사** | 소비자 니즈 분석, 경쟁사 상품 분석 | 마케팅, 상품개발 |
| **상품 설계** | 보장 내용, 가입 조건, 특약 구조 설계 | 상품개발, 계리 |
| **요율 산출** | 보험료 산출, 예정기초율 설정 | 계리 |
| **약관 작성** | 보통약관, 특별약관 작성 | 법무, 상품개발 |
| **시스템 개발** | 청약·보상 시스템 반영 | IT |
| **인가 신청** | 금융감독원에 상품 인가(신고) 신청 | 상품개발, 준법감시 |
| **판매 교육** | 설계사·대리점 대상 신상품 교육 | 영업지원 |

### 4.4 자산운용 (Investment Management)

보험사의 자산운용은 보험 부채에 대응하는 자산을 투자하여 수익을 창출하는 업무입니다. 특히 **ALM(자산부채관리)** 이 핵심입니다.

#### ALM (Asset-Liability Management)

ALM은 자산과 부채의 만기, 금리, 통화 등의 특성을 매칭하여 리스크를 관리하는 체계입니다.

| ALM 요소 | 설명 | 관리 방법 |
|----------|------|----------|
| **듀레이션 매칭** | 자산과 부채의 만기(듀레이션) 일치 | 장기채 투자, 금리스왑 |
| **금리 리스크** | 금리 변동에 따른 자산·부채 가치 변동 | 금리 헤지, 매칭 전략 |
| **유동성 관리** | 보험금 지급에 필요한 유동성 확보 | 유동성 버퍼 유지 |
| **수익률 관리** | 예정이율 이상의 투자수익률 달성 | 포트폴리오 다변화 |

#### 역마진 문제

과거 고금리 시절(1990~2000년대 초) 판매된 **고금리 확정이율 보험**은 보험사가 약속한 이율(예: 연 7~8%)을 현재의 저금리 환경에서 운용수익으로 충족하지 못하는 **역마진** 문제를 발생시킵니다. 이는 특히 생명보험사의 수익성에 큰 부담을 주고 있으며, IFRS 17 도입과 함께 그 규모가 명확히 드러나게 되었습니다.

### 4.5 리스크관리 (Risk Management)

보험사의 리스크관리는 보험 고유의 다양한 위험을 식별, 측정, 통제하는 업무입니다.

| 리스크 유형 | 설명 | 관리 지표 |
|------------|------|----------|
| **보험리스크** | 보험금 지급이 예상보다 많아질 위험 | 손해율, 발생손해액 |
| **시장리스크** | 금리, 주가, 환율 변동 위험 | VaR, 금리민감도 |
| **신용리스크** | 투자 자산의 부도·신용등급 하락 위험 | 신용등급별 익스포저 |
| **운영리스크** | 시스템 오류, 사기, 법률 분쟁 등 | 운영손실 데이터 |
| **유동성리스크** | 보험금 지급을 위한 현금 부족 위험 | 유동성 비율 |

#### K-ICS (Korean Insurance Capital Standard)

K-ICS는 한국형 지급여력제도로, 2023년부터 시행된 보험사 건전성 규제입니다. 기존 RBC(Risk-Based Capital) 제도를 대체합니다.

```
K-ICS 비율 = 가용자본 / 요구자본 × 100
```

| 구분 | K-ICS(신) | RBC(구) |
|------|----------|---------|
| **자산·부채 평가** | 시가 평가 | 원가 기반 |
| **리스크 측정** | 충격시나리오 방식 | 위험계수 방식 |
| **최소 기준** | 100% (권고 150%) | 100% (권고 150%) |
| **IFRS 17 정합성** | 높음 | 낮음 |

### 4.6 계리 (Actuarial Science)

계리사(Actuary)는 보험료 산출, 책임준비금 평가, 수익성 분석 등 보험의 수리적 기반을 담당하는 전문가입니다. 상세한 내용은 [보험 계리 개요](/docs/actuarial/overview) 페이지를 참고하세요.

| 업무 영역 | 구체적 내용 |
|----------|------------|
| **보험료 산출** | 예정기초율을 적용하여 순보험료·영업보험료 산출 |
| **책임준비금 평가** | 보험부채의 적정성 평가, IFRS 17 부채 산출 |
| **수익성 분석** | 상품별·채널별 수익성 분석, 신계약가치(VNB) 산출 |
| **경험통계 분석** | 실제 사고율·해약률 등을 분석하여 예정기초율 검증 |
| **IFRS 17 대응** | CSM 산출, 보험수익 인식, 할인율 산출 등 |
| **선임계리사 업무** | 법정 책임준비금 적정성 의견서 작성 |

### 4.7 GA(법인대리점) 관리

GA(General Agency)는 보험사에 소속되지 않고 독립적으로 보험상품을 판매하는 법인대리점입니다. 여러 보험사의 상품을 비교·추천할 수 있어 최근 판매 채널에서 비중이 급증하고 있습니다.

| 항목 | 설명 |
|------|------|
| **GA의 역할** | 다수 보험사의 상품을 비교하여 소비자에게 추천·판매 |
| **시장 비중** | 생보 신계약의 약 50%+, 손보 신계약의 약 60%+ |
| **보험사의 GA 관리** | 수수료 체계 설계, 판매 품질 관리, 불완전판매 방지 |
| **주요 GA** | 리더스금융판매, GA코리아, 피플라이프 등 |

### 4.8 디지털 / 인슈어테크 (InsurTech)

디지털 기술을 보험업에 접목하여 상품 설계, 판매, 심사, 보상 프로세스를 혁신하는 분야입니다.

| 기술 영역 | 활용 분야 | 사례 |
|----------|----------|------|
| **AI/ML** | 언더라이팅 자동화, 사기 탐지 | AI 건강심사, 보험사기 탐지 |
| **빅데이터** | 맞춤형 요율 산정, 고객 분석 | 운전습관 연동 자동차보험(UBI) |
| **블록체인** | 보험금 자동 지급, 계약 관리 | 스마트 컨트랙트 기반 보험 |
| **IoT** | 실시간 위험 모니터링 | 웨어러블 기기 연동 건강보험 |
| **디지털 플랫폼** | 온라인 보험 판매·비교 | 보험다모아, 카카오페이 보험 |

## 5. 보험 규제체계

### 5.1 금융위원회 / 금융감독원

| 기관 | 역할 | 보험 관련 주요 업무 |
|------|------|-------------------|
| **금융위원회** | 금융정책 수립, 인허가 | 보험업 인가, 보험업법 제·개정, 정책 방향 설정 |
| **금융감독원** | 금융기관 검사·감독 | 경영실태평가, 보험상품 인가 심사, 불완전판매 점검 |
| **보험개발원** | 보험 통계·요율 산출 | 참조순보험요율 산출, 보험 통계 관리 |
| **보험연수원** | 보험 전문인력 양성 | 보험 관련 교육·자격 시험 |

### 5.2 보험업법의 주요 규제

| 규제 영역 | 주요 내용 |
|----------|----------|
| **진입 규제** | 보험업 인가 요건 (자본금: 생보 200억+, 손보 300억+) |
| **자산운용 규제** | 동일 차주 대출 한도, 위험자산 투자 비율 제한 |
| **건전성 규제** | K-ICS 지급여력비율 유지 의무 (최소 100%) |
| **모집 규제** | 설명의무, 적합성 원칙, 불완전판매 금지 |
| **공시 규제** | 경영공시, 상품공시, 비교공시 |
| **소비자 보호** | 보험금 지급 기한(3영업일), 분쟁조정 절차 |

### 5.3 K-ICS (한국형 지급여력제도)

K-ICS는 2023년부터 시행된 보험회사 건전성 규제로, 유럽의 Solvency II를 참고하여 설계되었습니다. 보험사의 [리스크 관리](/docs/risk-management/overview)와 직결되는 핵심 제도입니다.

#### K-ICS의 구조

```
가용자본 (Available Capital)
─────────────────────────── × 100 = K-ICS 비율
요구자본 (Required Capital)
```

| 구성 요소 | 세부 내용 |
|----------|----------|
| **가용자본** | 자산의 공정가치 - 부채의 공정가치 (IFRS 17 기준 부채) |
| **요구자본** | 생명·장기손해보험리스크 + 일반손해보험리스크 + 시장리스크 + 신용리스크 + 운영리스크 (분산효과 차감) |

#### K-ICS 등급 분류

| K-ICS 비율 | 등급 | 감독 조치 |
|-----------|------|----------|
| 150% 이상 | 적정 | 정상 감독 |
| 100~150% | 주의 | 개선 권고 |
| 50~100% | 부적정 | 경영개선 요구·명령 |
| 50% 미만 | 위험 | 경영개선 명령(최대: 인가 취소) |

### 5.4 IFRS 17 (보험계약 회계기준)

IFRS 17은 보험계약에 적용되는 국제회계기준으로, 한국에서는 2023년 1월 1일부터 시행되었습니다. 상세한 내용은 [IFRS 17 상세 가이드](/docs/actuarial/ifrs17) 페이지를 참고하세요.

#### IFRS 17 핵심 요약

| 항목 | IFRS 4 (구) | IFRS 17 (신) |
|------|------------|-------------|
| **부채 평가** | 원가 기반 | 시가(공정가치) 기반 |
| **수익 인식** | 보험료 수취 시점 | 보험서비스 제공 시점 (점진적 인식) |
| **할인율** | 고정 (장부이율) | 시장금리 반영 |
| **미래 이익** | 즉시 인식 가능 | CSM으로 이연 후 점진적 인식 |

#### IFRS 17의 3대 측정 요소

| 요소 | 영문 | 설명 |
|------|------|------|
| **최선추정부채** | BEL (Best Estimate Liability) | 미래 현금흐름의 현재가치 (기댓값) |
| **위험조정** | RA (Risk Adjustment) | 비금융 위험의 불확실성에 대한 보상 |
| **보험계약마진** | CSM (Contractual Service Margin) | 미래 이익의 현재가치 (점진적으로 수익 인식) |

```
보험부채 = BEL + RA + CSM
```

#### CSM의 중요성

CSM(보험계약마진)은 IFRS 17 하에서 보험사 가치평가의 핵심 지표로 부상했습니다.

- **CSM 잔액**: 보험사가 미래에 인식할 수 있는 이익의 총량 → 기업가치의 선행 지표
- **CSM 상각**: 매 기간 보험서비스 제공에 따라 이익으로 인식되는 금액 → 안정적 수익의 원천
- **신계약 CSM**: 신규 계약에서 발생하는 CSM → 성장성의 지표

:::info 투자자 참고사항
IFRS 17 도입 이후, 보험사 분석에서 CSM 관련 지표(CSM 잔액, CSM 상각률, 신계약 CSM)가 핵심 밸류에이션 요소가 되었습니다. 과거의 내재가치(EV) 기반 분석에서 CSM 기반 분석으로 패러다임이 전환되고 있습니다.
:::

## 6. 보험 관련 자격증

보험업 취업과 전문성 인정에 관련된 주요 자격증을 정리합니다. [자격증 가이드](/docs/certifications)도 함께 참고하세요.

### 6.1 보험계리사

| 항목 | 내용 |
|------|------|
| **시행 기관** | 금융감독원 |
| **시험 구성** | 1차(5과목: 보험수학, 경제학, 보험계약법, 회계학, 경영학) + 2차(4과목: 보험수리, 계리리스크관리, 보험회계, 연금수리) |
| **합격률** | 1차 약 15~20%, 2차 약 20~25% |
| **활용** | 보험사 선임계리사, 상품개발, 리스크관리, 컨설팅 |
| **전망** | IFRS 17, K-ICS 도입으로 수요 증가 |

### 6.2 손해사정사

| 종별 | 업무 범위 | 시험 과목(주요) |
|------|----------|---------------|
| **1종(신체)** | 사람의 신체에 관한 손해사정 | 의학이론, 근재보험 이론 |
| **2종(재물)** | 재물에 관한 손해사정 | 화재보험, 해상보험, 건설공사보험 |
| **3종(차량)** | 자동차 사고에 관한 손해사정 | 자동차보험 이론, 자동차구조 |
| **4종(종합)** | 1~3종 전 분야 | 종합적 손해사정 이론 |

:::note 참고
2023년부터 손해사정사 제도가 개편되어, 기존 4개 종별에서 신체·재물·차량·종합의 4개 분야로 재편되었습니다.
:::

### 6.3 보험중개사

| 항목 | 내용 |
|------|------|
| **역할** | 보험계약 체결을 중개하는 독립적 전문가 (보험사 소속이 아님) |
| **시험** | 보험중개사 시험 (생보/손보 구분) |
| **활용** | 보험중개법인 설립, 기업 보험 컨설팅 |
| **특징** | 보험설계사와 달리 보험사로부터 독립된 지위 |

### 6.4 국제 자격증

| 자격증 | 정식 명칭 | 주관 | 분야 |
|--------|----------|------|------|

## 7. 보험업 트렌드

### 7.1 IFRS 17 도입의 영향

2023년 IFRS 17 도입은 보험 산업 전반에 근본적인 변화를 가져왔습니다.

| 영향 영역 | 변화 내용 |
|----------|----------|
| **수익 인식** | 보험료 수취 시 즉시 인식 → 보험서비스 제공 기간에 걸쳐 점진적 인식(CSM 상각) |
| **부채 평가** | 원가 기반 → 시가 기반으로 전환, 금리 변동에 따라 부채 규모 변동 |
| **상품 전략** | CSM이 큰 보장성 보험 중심으로 판매 전략 전환 |
| **경영 지표** | 보험료 성장률 → CSM 성장률, 신계약가치 등으로 핵심 지표 변화 |
| **재무 변동성** | 시가 평가로 인한 자본 변동성 확대 → 자본 관리 중요성 증가 |

### 7.2 디지털 보험 (인슈어테크)

| 혁신 분야 | 현황과 전망 |
|----------|------------|
| **온라인 보험 판매** | 미니보험, 디지털 손해보험 중심으로 성장. 카카오페이, 토스 등 빅테크 플랫폼 진출 |
| **AI 언더라이팅** | 건강 데이터 기반 자동 심사, 청약 심사 시간 단축 |
| **UBI 보험** | 운전습관(Usage-Based Insurance) 데이터 기반 맞춤형 자동차보험료 산정 |
| **P2P 보험** | 소규모 그룹이 공동으로 위험을 분담하는 새로운 보험 모델 |
| **보험금 자동 지급** | 항공 지연, 날씨 등 데이터 기반 자동 보험금 지급(파라메트릭 보험) |

### 7.3 고령화와 연금시장

한국은 세계에서 가장 빠르게 고령화가 진행되는 국가 중 하나이며, 이는 보험 산업에 기회와 위기를 동시에 가져옵니다.

| 영향 | 기회 | 위기 |
|------|------|------|
| **연금보험** | 사적 연금 수요 급증 | 장수 리스크(예상보다 오래 살 위험) 확대 |
| **건강보험** | 의료비 보장 수요 증가 | 손해율 상승 압력 |
| **간병보험** | 치매·간병 보장 수요 신규 창출 | 요율 산정의 불확실성 |
| **자산운용** | 장기 투자자금 확대 | 초저금리 환경에서 수익률 달성 어려움 |

### 7.4 ESG와 기후리스크

| ESG 영역 | 보험업 관련 내용 |
|----------|----------------|
| **기후리스크** | 자연재해 빈도·심도 증가에 따른 보험손실 확대, 기후리스크 시나리오 분석 의무화 |
| **ESG 투자** | 보험사 자산운용에서 ESG 투자 원칙 도입, 석탄·화석연료 관련 투자 축소 |
| **녹색보험** | 재생에너지, 전기차 관련 신규 보험상품 개발 |
| **ESG 공시** | 기후 관련 재무정보 공시(TCFD) 의무화 추진 |
| **사회적 책임** | 보험 접근성 확대, 취약계층 보험 보장 강화 |

### 7.5 기타 주목할 트렌드

- **빅테크의 보험업 진출**: 카카오페이손해보험 출범, 네이버·토스 등 플랫폼 기업의 보험 시장 진입
- **임베디드 보험**: 상품·서비스 구매 시 자동으로 보험이 포함되는 내장형 보험 (예: 전자제품 구매 시 파손 보험)
- **마이크로 보험**: 소액·단기 보험으로 특정 상황에 맞춘 맞춤형 보장 (예: 여행 당일보험, 레저 보험)
- **보험사기 방지**: AI·빅데이터를 활용한 보험사기 탐지 고도화
- **규제 샌드박스**: 혁신적 보험상품·서비스의 시범 운영을 허용하는 규제 특례

## 8. 보험 핵심 용어

### 보험료·보험금 관련

| 용어 | 정의 |
|------|------|
| **보험료(Premium)** | 보험계약자가 보험사에 납입하는 금액. 순보험료 + 부가보험료로 구성 |
| **보험금(Claim)** | 보험사고 발생 시 보험사가 지급하는 금액 |
| **해약환급금** | 보험계약 해지 시 돌려받는 금액. 초기에는 납입보험료보다 적음 |
| **환급률** | 해약환급금 / 납입보험료. 계약 초기에 매우 낮음 |
| **보험가액** | 보험사고 발생 시 피보험자가 입을 수 있는 손해의 최고 한도액 |
| **보험가입금액** | 보험증권에 기재된 보험금의 최고 한도액 |

### IFRS 17 관련

| 용어 | 영문 | 정의 |
|------|------|------|
| **CSM** | Contractual Service Margin | 보험계약마진. 미래 서비스 제공에 대한 미실현이익으로, 보험서비스 기간에 걸쳐 점진적으로 수익 인식 |
| **BEL** | Best Estimate Liability | 최선추정부채. 미래 현금흐름(보험금 지급, 보험료 수취 등)의 현재가치 기댓값 |
| **RA** | Risk Adjustment | 위험조정. 비금융위험의 불확실성에 대한 보상금액. 보험부채에 가산 |
| **PAA** | Premium Allocation Approach | 보험료배분접근법. 단기보험에 적용하는 간편법 |
| **BBA/GMM** | Building Block Approach / General Measurement Model | 일반측정모형. 장기보험에 적용하는 원칙적 측정법 |
| **VFA** | Variable Fee Approach | 변동수수료접근법. 직접참여특성이 있는 보험(변액보험 등)에 적용 |
| **보험수익** | Insurance Revenue | 보험서비스 제공에 따라 인식하는 수익 (과거의 수입보험료와 다른 개념) |
| **보험서비스비용** | Insurance Service Expense | 발생보험금, 기타 보험서비스 비용 |

### 건전성·수익성 지표

| 용어 | 정의 | 기준 |
|------|------|------|
| **손해율** | 발생손해액 / 경과보험료 × 100 | 낮을수록 양호 |
| **사업비율** | 순사업비 / 경과보험료 × 100 | 낮을수록 양호 |
| **합산비율** | 손해율 + 사업비율 | 100% 미만이면 보험영업 흑자 |
| **K-ICS 비율** | 가용자본 / 요구자본 × 100 | 최소 100%, 권고 150% |
| **책임준비금** | 보험사가 미래 보험금 지급을 위해 적립하는 부채 | 법정 적립 의무 |
| **지급여력비율** | 가용자본 / 요구자본 (= K-ICS 비율) | 건전성의 핵심 지표 |

### 기타 주요 용어

| 용어 | 정의 |
|------|------|
| **언더라이팅** | 보험 인수심사. 청약 건의 인수 여부와 조건을 결정하는 과정 |
| **출재** | 원수보험사가 위험의 일부를 재보험사에 이전하는 것 |
| **수재** | 재보험사가 원수보험사로부터 위험을 인수하는 것 |
| **적립이율** | 보험사가 보험료 적립금에 적용하는 이율 |
| **예정이율** | 보험료 산출 시 가정하는 투자수익률 |
| **선임계리사** | 보험업법에 따라 보험사가 반드시 선임해야 하는 법정 전문가 |
| **완전판매** | 보험 가입 시 상품 내용을 충분히 설명하여 민원이 발생하지 않는 적정한 판매 |
| **불완전판매** | 중요 사항을 누락·오인하게 하여 가입시킨 판매, 민원·분쟁의 주요 원인 |
| **모집** | 보험계약의 체결을 중개하는 행위 (설계사, 대리점, 중개사를 통해 수행) |
| **역마진** | 보험사의 실제 투자수익률이 예정이율(고객에게 약속한 수익률)보다 낮은 상태 |
