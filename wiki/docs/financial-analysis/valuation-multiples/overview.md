# 밸류에이션 멀티플 개요

## 멀티플이란?

밸류에이션 멀티플(배수)은 기업의 **시장 가격을 재무 지표로 나눈 비율**입니다. 기업 간 상대적 가치를 비교하는 데 사용됩니다.

## 핵심 멀티플 분류

### 주가 기반 멀티플 (Equity Value)

| 멀티플 | 공식 | 핵심 의미 |
|--------|------|-----------|
| **PER** (주가수익비율) | 주가 / EPS | 이익 대비 주가 수준 |
| **PBR** (주가순자산비율) | 주가 / BPS | 순자산 대비 주가 수준 |
| **PSR** (주가매출비율) | 주가 / SPS | 매출 대비 주가 수준 |
| **PCR** (주가현금흐름비율) | 주가 / CFPS | 현금흐름 대비 주가 |
| **배당수익률** | DPS / 주가 | 주가 대비 배당 수준 |

### 기업가치 기반 멀티플 (Enterprise Value)

| 멀티플 | 공식 | 핵심 의미 |
|--------|------|-----------|
| **EV/EBITDA** | EV / EBITDA | 영업현금 창출력 대비 기업가치 |
| **EV/EBIT** | EV / EBIT | 영업이익 대비 기업가치 |
| **EV/Sales** | EV / 매출 | 매출 대비 기업가치 |
| **EV/IC** | EV / 투하자본 | 투하자본 대비 기업가치 |

### EV (Enterprise Value, 기업가치)

$$\text{EV} = \text{시가총액} + \text{순차입금}$$
$$\text{EV} = \text{시가총액} + \text{이자부부채} - \text{현금}$$

**EV를 사용하는 이유**: 자본 구조(부채/자본 비율)에 관계없이 기업의 영업 가치를 비교

## 주요 멀티플 심화

### PER (Price-to-Earnings Ratio)

**이론적 근거**: Gordon Growth Model에서

$$\text{PER} = \frac{1 - b}{K_e - g} = \frac{\text{배당성향}}{K_e - g}$$

여기서 b = 유보율, Ke = 자기자본비용, g = 성장률

**PER에 영향을 주는 요인**:
- 성장률(g) 높을수록 → PER 높음
- 자기자본비용(Ke) 높을수록 → PER 낮음
- 배당성향 높을수록 → PER 높음 (성장률 일정 시)

**한국 시장 PER 범위**:

| 구분 | 일반적 범위 |
|------|-----------|
| KOSPI 전체 | 10~15x |
| 대형 성장주 | 20~40x |
| 가치주/은행 | 5~8x |
| 바이오 (적자) | N/A (PSR 사용) |

### PBR (Price-to-Book Ratio)

**이론적 근거**:

$$\text{PBR} = \frac{\text{ROE} - g}{K_e - g}$$

- ROE > Ke → PBR > 1 (프리미엄)
- ROE = Ke → PBR = 1 (적정)
- ROE < Ke → PBR < 1 (디스카운트)

**한국 시장 "코리아 디스카운트"**: 한국 시장은 구조적으로 낮은 PBR
- KOSPI 평균 PBR: 0.9~1.1x (선진국 대비 저평가)
- 원인: 낮은 ROE, 지배구조 리스크, 지정학 리스크, 낮은 주주환원

### EV/EBITDA

**이론적 근거**:

$$\text{EV/EBITDA} = \frac{1 - \frac{D\&A}{EBITDA} - \frac{Tax}{EBITDA} - \frac{\Delta NWC}{EBITDA} - \frac{NetCapex}{EBITDA}}{WACC - g}$$

단순화하면 WACC과 성장률에 의해 결정됨.

**장점**:
- 감가상각 차이 제거 (회계 정책 차이 극복)
- 자본 구조 차이 제거 (EV 사용)
- 국가 간 세율 차이 제거 (세전 지표)

**한국 시장 EV/EBITDA 범위**:

| 업종 | 일반적 범위 |
|------|-----------|
| 반도체 | 5~12x (사이클) |
| IT/플랫폼 | 15~30x |
| 제조업 | 6~10x |
| 유통 | 5~8x |
| 통신 | 4~6x |

## 멀티플 활용 실전

### 멀티플 비교 분석 프로세스

1. **적절한 멀티플 선택**: 업종 특성에 맞는 멀티플
2. **비교 대상 선정**: 동종업체(Peer Group) 구성
3. **데이터 수집**: Trailing vs Forward 멀티플
4. **비교 분석**: 프리미엄/디스카운트 원인 분석
5. **적정가치 산출**: 타겟 멀티플 × 재무 지표

### 업종별 적합 멀티플

| 업종 | 1순위 | 2순위 | 비적합 |
|------|-------|-------|--------|
| 제조업 | EV/EBITDA | PER | PSR |
| IT/SW | PER | EV/Sales | PBR |
| 금융(은행) | PBR | PER | EV/EBITDA |
| 유통 | EV/EBITDA | PER | PBR |
| 바이오(적자) | EV/Sales | Pipeline 가치 | PER |
| 부동산 | PBR (NAV) | 배당수익률 | PER |

### Trailing vs Forward 멀티플

| 구분 | Trailing | Forward |
|------|---------|---------|
| 데이터 | 과거 12개월 실적 | 향후 12개월 추정 |
| 장점 | 확정된 데이터 | 미래 전망 반영 |
| 단점 | 과거 지향적 | 추정치 오류 가능 |
| 적합한 상황 | 안정적 기업 | 성장/변화 중인 기업 |

## 한국 기업 특성

### 코리아 디스카운트

한국 시장이 선진국 대비 낮은 멀티플을 받는 구조적 요인:

| 요인 | 설명 |
|------|------|
| 지배구조 | 재벌 총수 경영, 소수주주 이익 훼손 |
| 낮은 주주환원 | 배당성향 25~30%, 선진국 40~50% |
| 지정학 리스크 | 북한 리스크 |
| 복잡한 지주구조 | 순환출자, 상호출자 |
| 낮은 ROE | 자본 효율성 부족 |

### 한국 기업 멀티플 분석 시 주의

1. **연결 vs 별도**: 연결 기준 멀티플이 원칙
2. **비지배지분**: PER 산출 시 지배주주 순이익 사용
3. **우선주 시가총액**: EV 계산 시 우선주 포함 여부 확인
4. **자기주식**: BPS 계산 시 자기주식 차감

### Layer 2 → 밸류에이션 연결

재무제표 분석(Layer 2)의 결과가 멀티플 수준을 결정합니다:

| Layer 2 분석 | 높은 멀티플 정당화 | 낮은 멀티플 예상 |
|-------------|------------------|----------------|
| 수익성 | 높은 ROE/ROIC | 낮은 ROE, 자본비용 미달 |
| 성장성 | 높은 매출/이익 성장률 | 저성장, 역성장 |
| 안정성 | 건전한 재무구조 | 높은 부채, 유동성 부족 |
| 효율성 | 짧은 CCC, 높은 회전율 | 비효율적 자산 운용 |

**핵심 인사이트**: 멀티플은 결과값이지, 원인이 아닙니다. 왜 높은/낮은 멀티플인지를 수익성, 성장성, 안정성, 효율성으로 설명할 수 있어야 합니다.
