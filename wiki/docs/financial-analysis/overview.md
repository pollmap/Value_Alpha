# 재무제표 분석 개요

## 학습 목표

- 재무제표 분석의 4가지 핵심 차원(수익성, 성장성, 안정성, 효율성)을 이해한다
- 재무비율을 산출하고 해석하는 기본 역량을 습득한다
- 동종업계 비교와 시계열 추세 분석의 방법론을 학습한다
- 재무비율 분석의 한계와 보완 방법을 파악한다

---


## 이 섹션의 목적

Layer 1에서 재무제표의 구조와 항목을 배웠다면, Layer 2에서는 그 숫자들을 **해석하고 비교하는 방법**을 배웁니다.

재무제표 분석의 핵심 질문:
- 이 기업은 **돈을 잘 벌고 있는가?** → 수익성 분석
- 이 기업은 **성장하고 있는가?** → 성장성 분석
- 이 기업은 **재무적으로 안전한가?** → 안정성 분석
- 이 기업은 **자원을 효율적으로 쓰고 있는가?** → 효율성 분석
- 이 기업의 **주가는 적정한가?** → 밸류에이션 멀티플

## 분석 프레임워크

### 재무비율 분석의 4가지 차원

- **수익성** — 핵심 질문: 얼마나 벌고 있나?, 대표 지표: ROE, ROIC, 마진율, 밸류에이션 연결: DCF의 수익 예측
- **성장성** — 핵심 질문: 얼마나 빨리 성장하나?, 대표 지표: 매출성장률, EPS성장률, 밸류에이션 연결: DCF의 성장률 가정
- **안정성** — 핵심 질문: 재무적으로 안전한가?, 대표 지표: 부채비율, 이자보상배율, 밸류에이션 연결: WACC, 기업 리스크
- **효율성** — 핵심 질문: 자원을 잘 활용하나?, 대표 지표: 자산회전율, 재고회전율, 밸류에이션 연결: 운전자본, CAPEX 예측

### 분석 접근법

**1. 추세 분석 (Trend Analysis)**

같은 기업의 과거 데이터를 시계열로 비교합니다.

- 최소 3\~5년 데이터 필요
- 일관된 추세 vs 변동성 확인
- 추세의 방향과 속도 판단
- 한국 기업: DART에서 5개년 비교 재무제표 확인

**2. 동종업체 비교 (Peer Comparison)**

같은 산업 내 경쟁사와 비교합니다.

- 업종 특성 반영 (제조업 vs IT vs 금융)
- 규모 차이 고려 (대기업 vs 중소기업)
- 한국 기업: 같은 GICS 분류 또는 한국표준산업분류 기준
- KRX 업종 평균 대비 위치 확인

**3. 절대 기준 (Absolute Benchmarks)**

일반적으로 통용되는 기준값과 비교합니다.

- ROE > 자기자본비용 → 가치 창출
- 부채비율 < 100% → 보수적 재무구조 (한국 기준)
- 이자보상배율 > 3배 → 안정적 이자 상환
- 유동비율 > 150% → 단기 유동성 양호

## Layer 1 → Layer 2 연결

```
Layer 1: 재무제표의 '숫자'를 읽는다
    ↓
Layer 2: 숫자의 '의미'를 해석한다
    ↓
Layer 3: 산업 맥락에서 '비교'한다
    ↓
Layer 4: 개별 기업을 '분석'한다
```

### 각 재무제표에서 추출하는 분석 정보

- **재무상태표** — Layer 1에서 배운 것: 자산/부채/자본 구조, Layer 2에서 분석하는 것: 안정성, 효율성 비율
- **손익계산서** — Layer 1에서 배운 것: 매출\~순이익 흐름, Layer 2에서 분석하는 것: 수익성, 성장성 비율
- **현금흐름표** — Layer 1에서 배운 것: 영업/투자/재무 CF, Layer 2에서 분석하는 것: FCF 기반 밸류에이션
- **3표 연결** — Layer 1에서 배운 것: 항목 간 연결, Layer 2에서 분석하는 것: 종합 분석 (DuPont 등)

## 한국 시장 특수 고려사항

### 연결 vs 별도 재무제표
- **연결 재무제표**: 자회사 포함, 그룹 전체 실적 파악
- **별도 재무제표**: 해당 법인만의 실적
- 재벌 구조에서는 반드시 연결 기준으로 분석
- 지배구조 할인 고려 필요

### DART 공시 활용
- 사업보고서(연간), 분/반기보고서
- 감사보고서의 주석 사항 확인
- 주요사항보고서 (유상증자, M&A 등)
- 전자공시 XBRL 데이터 활용

### 한국 기업 분석 시 주의점
- **분기 실적의 계절성**: 4분기 일회성 비용 반영 빈번
- **관계사 거래**: 특수관계자 거래 비중 확인
- **환율 영향**: 수출 비중 높은 기업은 환율 민감도 분석
- **정부 규제**: 공정거래법, 대기업 규제 영향

## 밸류에이션 연결 포인트

재무비율 분석 결과가 밸류에이션에 직접 연결됩니다:

- **수익성 추세** — DCF 입력값: 미래 마진 예측, 상대가치 입력값: 적정 PER 수준
- **성장률 추세** — DCF 입력값: 매출/이익 성장률, 상대가치 입력값: PEG 비율
- **안정성 수준** — DCF 입력값: WACC (부채 비중), 상대가치 입력값: 적정 PBR 수준
- **효율성 수준** — DCF 입력값: 운전자본/CAPEX 가정, 상대가치 입력값: EV/IC 비율
