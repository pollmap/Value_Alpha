# 지속가능성장률 (SGR) 모델

## 지속가능성장률이란?

지속가능성장률(Sustainable Growth Rate, SGR)은 외부 자금 조달 없이, 현재의 수익성과 배당 정책을 유지하면서 달성할 수 있는 최대 매출 성장률입니다.

## SGR 공식

### 기본 공식

$$\text{SGR} = \text{ROE} \times b$$

여기서 b = 유보율 = 1 - 배당성향

### DuPont 확장 공식

$$\text{SGR} = \text{순이익률} \times \text{자산회전율} \times \text{재무레버리지} \times \text{유보율}$$

### 의미

| 구성 요소 | 높을수록 SGR | 의미 |
|-----------|-------------|------|
| 순이익률 | 상승 | 내부 자금 창출력 ↑ |
| 자산회전율 | 상승 | 자산 효율성 ↑ |
| 재무레버리지 | 상승 | 부채 활용도 ↑ (리스크도 ↑) |
| 유보율 | 상승 | 재투자 비중 ↑ (배당 ↓) |

## SGR 분석 실전

### 사례 분석: 기업 A

| 항목 | 값 |
|------|-----|
| ROE | 18% |
| 배당성향 | 25% |
| 유보율 (b) | 75% |
| **SGR** | **13.5%** |

**해석**: 기업 A는 외부 자금 없이 매년 13.5%까지 성장할 수 있음

### SGR vs 실제 성장률 비교

| 시나리오 | 상황 | 시사점 |
|----------|------|--------|
| 실제 성장률 < SGR | 잉여 자금 발생 | 배당 증가, 자사주 매입 여력 |
| 실제 성장률 = SGR | 균형 상태 | 안정적 성장 |
| 실제 성장률 > SGR | 자금 부족 | 유상증자 또는 차입 증가 필요 |

### SGR을 높이는 전략

| 전략 | 방법 | 부작용 |
|------|------|--------|
| 마진 개선 | 원가 절감, 가격 인상 | 경쟁력 저하 가능 |
| 효율성 향상 | 자산 활용도 개선 | 한계 있음 |
| 레버리지 확대 | 차입 증가 | 재무 리스크 증가 |
| 유보율 증가 | 배당 삭감 | 주주 반발 |

## SGR과 기업 생애주기

| 단계 | 성장률 vs SGR | 자금 조달 |
|------|-------------|----------|
| **성장기** | 실제 >> SGR | 유상증자, 벤처투자 빈번 |
| **성숙기** | 실제 ≈ SGR | 내부 자금으로 충분 |
| **안정기** | 실제 < SGR | 잉여 현금 → 배당, 자사주 |
| **쇠퇴기** | 실제 << SGR | 자산 매각, 구조조정 |

## 한국 기업 특성

### 한국 기업 SGR 특성

1. **배당성향이 낮은 편**
   - 한국 상장사 평균 배당성향: 25~30%
   - 선진국 대비 낮음 → 유보율 높음 → SGR 높게 산출
   - 그러나 실제 재투자 수익률(ROIC)이 낮으면 의미 없음

2. **재벌 그룹의 과잉 투자**
   - 실제 성장률 > SGR → 지속적 차입 또는 유상증자
   - 비핵심 사업 확장으로 ROIC 하락 가능
   - 사업 다각화가 가치를 창출하는지 확인 필요

3. **자사주 매입의 영향**
   - 자사주 소각 시 자기자본 감소 → ROE 상승 → SGR 상승
   - 한국은 자사주 소각보다 보유가 많아 효과 제한적

### SGR 계산 시 한국 기업 유의사항

- **연결 vs 별도**: 연결 기준 ROE로 계산이 원칙
- **비지배지분**: 연결 순이익에서 비지배지분 순이익 제외
- **이상치 처리**: ROE가 비정상적 연도는 정상화 ROE 사용

### 밸류에이션 연결

SGR은 DCF 모델의 **영구 성장률(g)** 설정에 참고됩니다:

- 장기적으로 기업의 성장률은 SGR을 넘기 어려움
- 터미널 성장률 ≤ SGR (보수적으로)
- 터미널 성장률 ≤ GDP 성장률 (더 보수적으로)

$$\text{Terminal Value} = \frac{\text{FCF}_{n+1}}{\text{WACC} - g}$$

여기서 g의 상한은 SGR과 장기 GDP 성장률 중 낮은 값
