# 마진 분석 (Margin Analysis)

## 마진이란?

마진(Margin)은 매출 대비 이익의 비율입니다. 기업이 매출 1원을 벌 때 실제로 얼마를 남기는지를 보여줍니다.

## 주요 마진 지표

### 1. 매출총이익률 (Gross Profit Margin)

$$\text{매출총이익률} = \frac{\text{매출총이익}}{\text{매출액}} \times 100$$

**의미**: 제품/서비스의 원가 경쟁력을 나타냅니다.

| 수준 | 해석 |
|------|------|
| 50% 이상 | 높은 부가가치 (SW, 브랜드 소비재) |
| 30~50% | 양호 (제조업 우량기업) |
| 15~30% | 보통 (일반 제조업) |
| 15% 미만 | 저마진 (유통, 건설) |

**분석 포인트**:
- 원재료 가격 변동 영향 (철강, 석유화학)
- 제품 믹스 변화 효과
- 생산 효율성 개선 여부
- 가격 인상/인하 효과

### 2. 영업이익률 (Operating Profit Margin)

$$\text{영업이익률} = \frac{\text{영업이익}}{\text{매출액}} \times 100$$

**의미**: 본업의 수익성을 나타냅니다. 판관비 관리 능력 포함.

| 수준 | 해석 |
|------|------|
| 20% 이상 | 우수 (독과점, 기술 우위) |
| 10~20% | 양호 |
| 5~10% | 보통 |
| 5% 미만 | 주의 필요 |

**분석 포인트**:
- 매출총이익률과의 차이 = 판관비 비중
- R&D 비용 비중 (기술 기업)
- 인건비 비중 변화
- 규모의 경제 효과

### 3. 순이익률 (Net Profit Margin)

$$\text{순이익률} = \frac{\text{당기순이익}}{\text{매출액}} \times 100$$

**의미**: 최종적으로 주주에게 귀속되는 이익의 비율입니다.

**분석 포인트**:
- 영업이익률과의 차이 분석 → 영업외 항목 영향
- 이자비용 부담 수준
- 법인세 실효세율
- 일회성 항목 제거 후 경상적 순이익률

### 4. EBITDA 마진

$$\text{EBITDA 마진} = \frac{\text{EBITDA}}{\text{매출액}} \times 100$$

**의미**: 감가상각 전 영업 현금 창출력을 나타냅니다. 설비투자가 큰 산업에서 특히 유용합니다.

## 마진 분석 실전 가이드

### 마진 추세 분석

5개년 마진 추세를 확인합니다:

| 연도 | 매출총이익률 | 영업이익률 | 순이익률 | EBITDA마진 |
|------|-------------|-----------|---------|-----------|
| 2019 | 35.2% | 12.1% | 8.5% | 18.3% |
| 2020 | 34.8% | 10.5% | 6.2% | 17.1% |
| 2021 | 36.1% | 13.2% | 9.8% | 19.5% |
| 2022 | 37.5% | 14.8% | 11.2% | 21.0% |
| 2023 | 38.2% | 15.5% | 12.0% | 22.3% |

**해석**: 매출총이익률이 꾸준히 상승 → 원가 경쟁력 강화 또는 제품 믹스 개선

### 마진 스프레드 분석

각 단계별 마진의 차이를 분석합니다:

- **매출총이익률 - 영업이익률** = 판관비 부담률
  - 이 차이가 줄어들면 → 비용 효율화 진행
  - 이 차이가 늘어나면 → R&D 확대, 마케팅 증가 등

- **영업이익률 - 순이익률** = 영업외/세금 부담률
  - 이 차이가 크면 → 높은 이자비용 또는 비경상 손실

### 업종별 벤치마크

| 업종 | 매출총이익률 | 영업이익률 | 특징 |
|------|-------------|-----------|------|
| 반도체 | 45~65% | 25~45% | 사이클에 따라 변동 큼 |
| 소프트웨어 | 60~80% | 15~30% | 높은 총마진, R&D 비용 큼 |
| 자동차 | 18~25% | 5~10% | 낮은 마진, 대량 판매 |
| 유통 | 25~35% | 3~8% | 저마진, 회전율 중요 |
| 제약/바이오 | 55~75% | 10~25% | R&D 비용 비중 높음 |
| 건설 | 12~18% | 3~7% | 수주 기반, 원가 변동 |
| 금융(은행) | N/A | N/A | NIM 사용 (마진율 부적합) |

## 한국 기업 특성

### 한국 기업 마진 분석 시 주의점

1. **K-IFRS 연결 기준**: 자회사 매출이 포함되므로, 별도 기준과 차이 발생
2. **재벌 그룹 내부거래**: 관계사 매출 비중이 높으면 마진이 왜곡될 수 있음
3. **환율 효과**: 수출 비중이 높은 기업은 원/달러 환율에 따라 마진 변동
4. **분기별 계절성**: 4분기 일회성 비용(성과급, 감손 등) 반영으로 마진 하락 빈번

### DART에서 마진 데이터 확인

1. DART → 기업명 검색 → 사업보고서
2. "손익계산서" 또는 "포괄손익계산서" 확인
3. 매출액, 매출원가, 영업이익, 당기순이익 추출
4. 5개년 추세 비교표 작성

## 실무 포인트

### 마진 분석 체크리스트

- [ ] 최소 5년간의 마진 추세가 안정적 또는 개선되고 있는가?
- [ ] 동종업체 대비 마진 수준은 어떠한가?
- [ ] 매출총이익률 변동의 원인은 무엇인가? (원가? 가격?)
- [ ] 판관비/매출 비율은 안정적인가?
- [ ] 일회성 항목을 제외한 경상적 마진은?
- [ ] 분기별 마진 변동의 계절성이 있는가?

### 마진과 밸류에이션 연결

| 마진 특성 | 밸류에이션 시사점 |
|-----------|------------------|
| 높고 안정적인 마진 | 프리미엄 밸류에이션 정당화 |
| 개선 추세 마진 | 이익 성장 → 높은 성장률 적용 |
| 하락 추세 마진 | 보수적 이익 예측 필요 |
| 변동성 높은 마진 | 높은 할인율(리스크 프리미엄) |

**DCF 연결**: 마진 분석 결과는 미래 손익계산서 추정의 핵심 입력값입니다. 현재 마진이 지속가능한지, 개선/악화 중인지 판단하여 예측 마진을 설정합니다.
