# 수익률 분석 (ROE / ROIC / ROA)

## 수익률 지표의 의미

마진율이 "매출 대비 이익"을 보여준다면, 수익률 지표는 **"투입한 자본 대비 이익"**을 보여줍니다. 투자자 관점에서 가장 중요한 지표입니다.

## 핵심 수익률 지표

### 1. ROE (Return on Equity, 자기자본이익률)

$$\text{ROE} = \frac{\text{당기순이익}}{\text{평균 자기자본}} \times 100$$

**의미**: 주주가 투자한 자본 대비 얼마의 이익을 창출했는가?

| 수준 | 해석 |
|------|------|
| 20% 이상 | 매우 우수 |
| 15~20% | 우수 |
| 10~15% | 양호 |
| 5~10% | 보통 |
| 5% 미만 | 자본비용 미달 가능 |

**핵심 해석 원칙**:
- ROE > 자기자본비용(Ke) → 가치 창출
- ROE < Ke → 가치 파괴
- 한국 시장 Ke: 일반적으로 8~12% 수준

**주의사항**:
- 부채를 많이 쓰면 ROE가 높아질 수 있음 → DuPont 분석 필요
- 자본이 적으면 ROE가 비정상적으로 높아질 수 있음
- 순이익에 일회성 항목이 포함될 수 있음
- 평균 자기자본 사용 (기초+기말)/2

### 2. ROA (Return on Assets, 총자산이익률)

$$\text{ROA} = \frac{\text{당기순이익}}{\text{평균 총자산}} \times 100$$

**의미**: 기업이 보유한 전체 자산으로 얼마의 이익을 만들었는가?

**ROE와의 관계**:
$$\text{ROE} = \text{ROA} \times \text{재무 레버리지}$$

ROE가 높은데 ROA가 낮다면 → 레버리지(부채)에 의한 ROE 상승

### 3. ROIC (Return on Invested Capital, 투하자본수익률)

$$\text{ROIC} = \frac{\text{NOPAT}}{\text{투하자본(IC)}} \times 100$$

여기서:
- **NOPAT** = 영업이익 × (1 - 실효세율) = 세후영업이익
- **투하자본(IC)** = 순운전자본 + 순고정자산 = 자기자본 + 이자부부채 - 현금

**의미**: 영업에 실제로 투입된 자본 대비 세후 영업이익. 자본 구조(부채/자본 비율)의 영향을 받지 않는 순수한 사업 수익률입니다.

| 비교 | 의미 |
|------|------|
| ROIC > WACC | 경제적 가치 창출 (EVA > 0) |
| ROIC = WACC | 자본비용만큼만 벌고 있음 |
| ROIC < WACC | 경제적 가치 파괴 |

**ROIC가 ROE보다 우월한 이유**:
1. 자본 구조 중립적 (부채 효과 제거)
2. 영업이익 기반 (영업외 항목 제거)
3. 경제적 가치 창출 여부를 정확히 판단
4. DCF와 직접적으로 연결됨

### 4. ROCE (Return on Capital Employed)

$$\text{ROCE} = \frac{\text{EBIT}}{\text{Capital Employed}} \times 100$$

여기서 Capital Employed = 총자산 - 유동부채

**의미**: 세전 기준의 자본 효율성. 세율이 다른 국가 간 비교에 유용합니다.

## 수익률 분석 실전 가이드

### ROE vs ROIC 비교 분석

| 기업 | ROE | ROIC | 해석 |
|------|-----|------|------|
| A사 | 25% | 18% | 건전한 고수익. 레버리지 적절히 활용 |
| B사 | 22% | 8% | ROE가 과도한 부채로 부풀려짐. 주의 |
| C사 | 10% | 15% | 보수적 재무구조. 실제 사업은 우수 |
| D사 | 5% | 4% | 실질적으로 자본비용 미달 |

### ROIC 스프레드 분석

**ROIC 스프레드** = ROIC - WACC

이 값이 양수이면 기업이 경제적 이익(Economic Profit)을 창출하고 있습니다.

- 스프레드가 클수록 → 강력한 경쟁우위 (moat)
- 스프레드가 지속적으로 양수 → 구조적 경쟁력
- 스프레드가 축소 추세 → 경쟁 심화 또는 산업 성숙

### 수익률 추세 분석 예시

| 연도 | ROE | ROA | ROIC | WACC | 스프레드 |
|------|-----|-----|------|------|---------|
| 2019 | 15.2% | 7.8% | 12.5% | 9.0% | +3.5%p |
| 2020 | 12.8% | 6.5% | 10.2% | 8.5% | +1.7%p |
| 2021 | 18.5% | 9.2% | 14.8% | 8.8% | +6.0%p |
| 2022 | 20.1% | 10.5% | 16.2% | 9.2% | +7.0%p |
| 2023 | 19.5% | 10.1% | 15.5% | 9.0% | +6.5%p |

**해석**: ROIC 스프레드가 안정적으로 양수 → 지속적 가치 창출 기업

## 한국 기업 특성

### 한국 시장 수익률 특성

1. **재벌 그룹사**: 연결 ROE와 별도 ROE 차이 큼
   - 지주회사: 별도 ROE는 낮지만 연결 ROE는 자회사 실적 반영
   - 사업자회사: 별도 ROE가 더 의미있을 수 있음

2. **자사주 효과**: 자기주식이 많은 기업은 ROE가 높게 나타남
   - 자기주식을 자본에서 차감하므로 자기자본 감소 → ROE 상승
   - 한국 대기업들은 자사주 비중이 높은 경우 다수

3. **K-IFRS 적용 효과**:
   - 공정가치 평가로 자산가치 변동 → ROA 변동성 증가
   - 투자부동산 보유 기업: 평가이익으로 순이익 왜곡 가능

### DART 데이터 활용

1. 사업보고서 → "재무비율" 항목에서 ROE 직접 확인 가능
2. 금감원 전자공시시스템에서 동종업체 비교 가능
3. 한국은행 기업경영분석 자료에서 업종 평균 ROE 확인

## 실무 포인트

### 수익률 분석 체크리스트

- [ ] ROE가 자기자본비용(Ke)을 상회하는가?
- [ ] ROIC가 WACC을 상회하는가?
- [ ] ROE와 ROA의 차이가 합리적인가? (과도한 레버리지 아닌지)
- [ ] 수익률 추세가 안정적 또는 개선되고 있는가?
- [ ] 동종업체 대비 수익률 수준은?
- [ ] 일회성 항목 제외 시 수익률 변화는?

### 밸류에이션 연결

| 수익률 지표 | DCF 연결 | 상대가치 연결 |
|------------|---------|--------------|
| ROE | 자기자본 성장률 | PBR = ROE / Ke |
| ROIC | 경제적 이익 모델 | EV/IC 비율 |
| ROIC 스프레드 | 초과수익 기간 추정 | 프리미엄/디스카운트 |
| ROE 추세 | 미래 이익 예측 | 적정 PER 산출 |

**핵심 공식**: Gordon Growth Model에서
$$\text{PBR} = \frac{\text{ROE} - g}{\text{Ke} - g}$$

ROE > Ke이면 PBR > 1이 정당화됩니다.
