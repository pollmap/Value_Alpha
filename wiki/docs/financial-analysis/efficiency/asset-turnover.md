# 자산회전율 분석

## 효율성 분석의 의의

효율성 분석은 기업이 보유한 **자원(자산)을 얼마나 효과적으로 활용**하여 매출을 창출하는지 평가합니다. DuPont 분석에서 ROE의 구성 요소 중 하나인 자산회전율을 심층 분석합니다.

## 주요 효율성 지표

### 1. 총자산회전율 (Total Asset Turnover)

$$\text{총자산회전율} = \frac{\text{매출액}}{\text{평균 총자산}}$$

| 수준 | 해석 |
|------|------|
| 2.0x 이상 | 높은 효율 (유통, 소매) |
| 1.0~2.0x | 보통 (제조업) |
| 0.5~1.0x | 자본집약적 (중화학, 반도체) |
| 0.5x 미만 | 매우 자본집약적 (금융, 부동산) |

### 2. 유형자산회전율 (Fixed Asset Turnover)

$$\text{유형자산회전율} = \frac{\text{매출액}}{\text{평균 순유형자산}}$$

**의미**: 설비투자의 효율성. 제조업에서 특히 중요.

- 높을수록 설비 활용도가 좋음
- 신규 설비투자 직후에는 일시적으로 하락 (정상)
- 감가상각이 진행되면 자연적으로 상승

### 3. 매출채권회전율 (Receivables Turnover)

$$\text{매출채권회전율} = \frac{\text{매출액}}{\text{평균 매출채권}}$$

$$\text{매출채권 회전일수} = \frac{365}{\text{매출채권회전율}}$$

**해석**:
- 회전율 높을수록 → 매출채권 회수가 빠름
- 회전일수 짧을수록 → 현금 회수 빠름
- 업종별 차이 큼 (현금 결제 vs 외상 결제)

| 회전일수 | 해석 (일반 제조업 기준) |
|---------|---------------------|
| 30일 미만 | 매우 빠른 회수 |
| 30~60일 | 양호 |
| 60~90일 | 보통 |
| 90일 초과 | 회수 지연 주의 |

### 4. 재고자산회전율 (Inventory Turnover)

$$\text{재고자산회전율} = \frac{\text{매출원가}}{\text{평균 재고자산}}$$

$$\text{재고자산 회전일수} = \frac{365}{\text{재고자산회전율}}$$

**해석**:
- 회전율 높을수록 → 재고 관리 효율적
- 너무 높으면 → 재고 부족으로 판매 기회 상실 가능
- 재고 증가와 매출 감소가 동시에 나타나면 → 강력한 경고 신호

### 5. 매입채무회전율 (Payables Turnover)

$$\text{매입채무회전율} = \frac{\text{매출원가}}{\text{평균 매입채무}}$$

$$\text{매입채무 회전일수} = \frac{365}{\text{매입채무회전율}}$$

**해석**:
- 회전일수 길수록 → 대금 지불 여유 (자금 운용 유리)
- 너무 길면 → 협력업체와의 관계 악화 또는 자금난 의심
- 교섭력이 강한 대기업일수록 회전일수가 긴 경향

## 종합: 현금전환주기 (CCC)

$$\text{CCC} = \text{매출채권 회전일수} + \text{재고자산 회전일수} - \text{매입채무 회전일수}$$

### 업종별 CCC 비교

| 업종 | 매출채권 | 재고 | 매입채무 | CCC |
|------|---------|------|---------|-----|
| 반도체 | 45일 | 60일 | 30일 | 75일 |
| 유통 | 10일 | 30일 | 45일 | -5일 |
| 건설 | 90일 | 120일 | 60일 | 150일 |
| SW/플랫폼 | 30일 | 0일 | 20일 | 10일 |
| 식음료 | 25일 | 40일 | 35일 | 30일 |

### CCC 개선 사례

| 전략 | 효과 | 예시 |
|------|------|------|
| 매출채권 회수 단축 | CCC 감소 | 조기결제 할인 제공 |
| 재고 감축 | CCC 감소 | JIT 생산, SCM 최적화 |
| 매입채무 지급 연장 | CCC 감소 | 협상력 활용 (대기업) |
| 선수금 모델 | CCC 음수 | 구독형 SaaS, 선결제 유통 |

## 한국 기업 특성

### 한국 기업 효율성 분석 포인트

1. **대기업-중소기업 거래 구조**
   - 대기업: 매입채무 회전일수가 긴 경향 (납품업체 대금 지급 지연)
   - 중소기업: 매출채권 회전일수가 긴 경향 (대금 회수 지연)
   - 상생협력 정책으로 개선 추세

2. **재고자산 분석 심화**
   - 반도체: 재고 급증 = 수요 둔화 신호 (주가 선행지표)
   - 건설: 미분양 주택 재고 주의
   - 원재료 비축: 원가 상승 기대 시 선제 구매

3. **K-IFRS 영향**
   - 수익 인식 기준(IFRS 15) 변경으로 매출채권 패턴 변화 가능
   - 건설업 진행기준 적용에 따른 미청구공사 확인

## 실무 포인트

### 효율성 분석 체크리스트

- [ ] 총자산회전율의 추세는? (개선/악화)
- [ ] 매출채권 회전일수가 증가하고 있지 않은가?
- [ ] 재고자산 회전일수가 업종 평균 대비 적절한가?
- [ ] CCC가 단축 추세인가?
- [ ] 유형자산회전율이 설비투자 후 회복되고 있는가?
- [ ] 동종업체 대비 효율성 수준은?

### 밸류에이션 연결

| 효율성 지표 | DCF 영향 |
|------------|---------|
| CCC 단축 | NWC 감소 → FCF 증가 → 기업가치 상승 |
| CCC 증가 | NWC 증가 → FCF 감소 → 기업가치 하락 |
| 유형자산회전율 개선 | 추가 CAPEX 불필요 → FCF 증가 |
| 총자산회전율 개선 | 자본 효율성 → ROE/ROIC 개선 |

**DCF 모델에서**: 매출 성장 시 운전자본 소요를 추정할 때 회전율 지표를 직접 사용합니다.

$$\Delta\text{NWC} = \Delta\text{매출채권} + \Delta\text{재고자산} - \Delta\text{매입채무}$$

과거 회전일수를 유지한다고 가정하면 매출 예측에서 운전자본 변동을 자동으로 추정할 수 있습니다.
