# 부채비율 분석

## 안정성 분석의 의의

안정성 분석은 기업의 **재무적 건전성**을 평가합니다. 아무리 성장성과 수익성이 좋아도, 재무구조가 취약하면 위기 시 생존이 어렵습니다.

밸류에이션 관점에서 안정성은 **할인율(WACC)**에 직접 영향을 미칩니다.

## 주요 부채 관련 지표

### 1. 부채비율 (Debt-to-Equity Ratio)

$$\text{부채비율} = \frac{\text{총부채}}{\text{자기자본}} \times 100$$

| 수준 | 해석 (일반 기업 기준) |
|------|---------------------|
| 50% 미만 | 매우 보수적 |
| 50~100% | 보수적, 양호 |
| 100~200% | 보통 |
| 200~300% | 적극적 |
| 300% 초과 | 고위험 |

**주의**: 업종별로 정상 범위가 크게 다름
- 금융업: 1,000% 이상도 정상 (예금이 부채)
- 건설업: 200~300% 보통
- IT/SW: 50% 미만도 흔함

### 2. 순차입금비율 (Net Debt-to-Equity)

$$\text{순차입금비율} = \frac{\text{이자부부채} - \text{현금 및 현금성자산}}{\text{자기자본}} \times 100$$

**부채비율보다 정확한 이유**:
- 매입채무 같은 영업부채를 제외
- 보유 현금을 차감하여 실질적 차입 부담 파악
- 순현금(Net Cash) 기업은 음수 → 실질적으로 무차입

### 3. 이자부부채비율

$$\text{이자부부채비율} = \frac{\text{단기차입금 + 유동성장기부채 + 사채 + 장기차입금}}{\text{총자산}} \times 100$$

**이자를 발생시키는 부채만** 따로 파악. 영업부채(매입채무 등)와 구분.

### 4. 총부채/EBITDA (Debt/EBITDA)

$$\text{Debt/EBITDA} = \frac{\text{순이자부부채}}{\text{EBITDA}}$$

**의미**: 현재 이익 수준으로 부채를 상환하는 데 몇 년이 걸리는가?

| 수준 | 해석 |
|------|------|
| 1x 미만 | 매우 건전 |
| 1~2x | 건전 |
| 2~3x | 보통 |
| 3~5x | 주의 |
| 5x 초과 | 고위험 |

**신용등급과의 관계**: 국제 신용평가사들이 가장 중시하는 지표 중 하나

### 5. 이자보상배율 (Interest Coverage Ratio)

$$\text{이자보상배율} = \frac{\text{영업이익}}{\text{이자비용}}$$

| 수준 | 해석 |
|------|------|
| 10x 이상 | 매우 안정 |
| 5~10x | 안정 |
| 3~5x | 보통 |
| 1~3x | 주의 |
| 1x 미만 | 이자도 못 내는 상태 (위험) |

**한국은행 기준**: 이자보상배율 1 미만 기업을 "한계기업"으로 분류

## 부채 구조 분석

### 만기 구조 (Maturity Profile)

단기 부채와 장기 부채의 비율을 확인합니다:

| 구분 | 리스크 |
|------|--------|
| 단기 차입 비중 높음 | 차환 리스크 (금리 상승, 신용경색 시 위험) |
| 장기 차입 비중 높음 | 안정적이나 금리 부담 클 수 있음 |
| 균형 잡힌 만기 구조 | 이상적 |

### 고정금리 vs 변동금리

- **고정금리 비중 높음**: 금리 상승 시 유리, 금리 하락 시 불리
- **변동금리 비중 높음**: 금리 변동에 노출, 금리 상승 시 이자 부담 급증

### 부채의 질적 분석

| 부채 유형 | 특성 |
|-----------|------|
| 회사채 (공모) | 시장에서 거래, 신용등급 필요 |
| 은행 차입금 | 은행 관계, 담보 조건 |
| 전환사채(CB) | 전환 시 지분 희석 |
| 교환사채(EB) | 보유 주식 교환 가능 |
| 리스부채 | K-IFRS 16 적용, 운용리스도 부채화 |

## 한국 기업 특성

### 한국 기업 부채 분석 포인트

1. **재벌 그룹 보증 구조**
   - 계열사 간 지급보증, 채무인수약정 확인
   - 우발부채로 인한 연쇄 위험 가능성
   - 주석의 "우발채무 및 약정사항" 반드시 확인

2. **한국 기업 부채비율 트렌드**
   - IMF 이후 부채비율 200% 미만 가이드라인
   - 최근 한국 대기업 평균 부채비율: 80~120%
   - 중소기업은 여전히 변동성 높음

3. **K-IFRS 16 리스부채 영향**
   - 2019년부터 운용리스도 재무상태표에 부채 인식
   - 임대차 비중 높은 기업(유통, 항공)은 부채비율 급등
   - K-IFRS 16 적용 전후 비교 시 조정 필요

4. **차입 구조의 특수성**
   - 한국 기업은 은행 차입 비중이 높은 편
   - 회사채 시장 활용은 대기업 중심
   - CP(기업어음) 의존도가 높은 기업 주의

### 한계기업 판단 기준

한국은행/금융감독원 기준:
- 이자보상배율 < 1 이 3년 연속 → 한계기업
- 한국 전체 상장사의 약 15% 내외가 한계기업에 해당

- [ ] 순차입금비율은 어떤 수준인가? (현금 보유 고려)
- [ ] 이자보상배율이 3배 이상인가?
- [ ] Debt/EBITDA가 3배 이내인가?
- [ ] 부채 만기 구조가 분산되어 있는가?
- [ ] 변동금리 부채 비중은 적절한가?
- [ ] 우발부채(지급보증 등)는 없는가?
- [ ] K-IFRS 16 리스부채의 영향은?

### 밸류에이션 연결

| 부채 지표 | WACC 영향 | 밸류에이션 영향 |
|-----------|---------|----------------|
| 높은 부채비율 | Kd 상승, 하지만 세금 절감 효과 | 최적 자본구조까지는 긍정적 |
| 낮은 이자보상배율 | 부도 위험 프리미엄 반영 | WACC 상승 → 기업가치 하락 |
| 높은 Debt/EBITDA | 재무적 곤란 비용 증가 | 할인율 프리미엄 부과 |
| 순현금 상태 | 낮은 WACC | EV = 시가총액 - 순현금 |

**DCF에서**: 부채 수준은 WACC 계산의 핵심 입력값입니다. D/(D+E) 비율이 변하면 WACC이 변하고, 기업가치가 달라집니다.
