# 유동성 분석

## 유동성이란?

유동성(Liquidity)은 기업이 **단기 채무를 적시에 상환할 수 있는 능력**입니다. 장기적으로 수익성이 좋아도 단기 유동성이 부족하면 흑자부도에 이를 수 있습니다.

## 주요 유동성 지표

### 1. 유동비율 (Current Ratio)

$$\text{유동비율} = \frac{\text{유동자산}}{\text{유동부채}} \times 100$$

| 수준 | 해석 |
|------|------|
| 200% 이상 | 매우 양호 |
| 150~200% | 양호 |
| 100~150% | 보통 |
| 100% 미만 | 단기 유동성 부족 주의 |

### 2. 당좌비율 (Quick Ratio)

$$\text{당좌비율} = \frac{\text{유동자산} - \text{재고자산}}{\text{유동부채}} \times 100$$

재고자산은 현금화에 시간이 걸리므로 제외한 더 보수적인 지표.

| 수준 | 해석 |
|------|------|
| 100% 이상 | 양호 |
| 70~100% | 보통 |
| 70% 미만 | 주의 |

### 3. 현금비율 (Cash Ratio)

$$\text{현금비율} = \frac{\text{현금 + 단기금융상품}}{\text{유동부채}} \times 100$$

가장 보수적인 유동성 지표. 즉시 현금화 가능한 자산만 고려.

### 4. 영업현금흐름비율 (Operating CF Ratio)

$$\text{OCF비율} = \frac{\text{영업활동 현금흐름}}{\text{유동부채}} \times 100$$

**장점**: 실제 현금 창출력 기반 유동성 평가

## 유동성 심화 분석

### 운전자본 분석 (Working Capital Analysis)

$$\text{순운전자본(NWC)} = \text{유동자산} - \text{유동부채}$$

$$\text{영업운전자본} = \text{매출채권} + \text{재고자산} - \text{매입채무}$$

**현금전환주기 (CCC: Cash Conversion Cycle)**

$$\text{CCC} = \text{매출채권 회전일수} + \text{재고자산 회전일수} - \text{매입채무 회전일수}$$

| CCC | 의미 |
|-----|------|
| 짧을수록 | 현금이 빨리 회수됨 (유동성 양호) |
| 길수록 | 현금이 오래 묶임 (유동성 부담) |
| 음수 | 선수금 모델 (매입채무 > 매출채권+재고) |

### CCC 구성 요소

| 지표 | 계산 | 의미 |
|------|------|------|
| 매출채권 회전일수 | (매출채권/매출) × 365 | 외상 회수 기간 |
| 재고자산 회전일수 | (재고/매출원가) × 365 | 재고 보유 기간 |
| 매입채무 회전일수 | (매입채무/매출원가) × 365 | 외상 지급 기간 |

### 유동성 위기 경고 신호

| 경고 신호 | 의미 |
|-----------|------|
| 유동비율의 지속적 하락 | 단기 채무 상환 능력 악화 |
| CCC의 급격한 증가 | 현금 회수 지연 |
| 단기차입금 급증 | 장기 부채의 단기화 또는 운전자본 부족 |
| 매출채권 급증 (매출 정체 시) | 매출 인식 공격성 또는 회수 지연 |
| 재고자산 급증 | 판매 부진 또는 과잉 생산 |
| 현금 보유 급감 | 자금 유출 가속 |

## 한국 기업 특성

### 한국 기업 유동성 분석 포인트

1. **어음 결제 관행**
   - 한국 기업은 여전히 어음 결제 비중 존재
   - 전자결제시스템 도입으로 감소 추세
   - 받을어음의 부도 위험 확인

2. **분기별 계절성**
   - 건설업: 준공 시기에 따른 유동성 변동
   - 소매업: 연말 성수기 전 재고 축적
   - 제조업: 원재료 비축 시기

3. **그룹사 자금 거래**
   - 계열사 대여금/차입금 확인
   - 지주회사의 자회사 대여금 규모
   - 특수관계자 거래의 유동성 영향

4. **K-IFRS 16 영향**
   - 리스부채의 유동성 분류
   - 1년 내 만기 리스부채 → 유동부채 분류
   - 유동비율 하락 효과

## 실무 포인트

### 유동성 분석 체크리스트

- [ ] 유동비율 150% 이상인가?
- [ ] 당좌비율 100% 이상인가?
- [ ] CCC가 업종 평균 대비 적절한가?
- [ ] 매출채권/재고자산의 급격한 변화는 없는가?
- [ ] 단기차입금 만기 구조는 적절한가?
- [ ] 영업현금흐름이 양(+)인가?

### 밸류에이션 연결

| 유동성 상태 | DCF 영향 | 기업가치 영향 |
|------------|---------|-------------|
| 양호한 유동성 | 안정적 현금흐름 예측 | 낮은 할인율 |
| 개선 추세 CCC | 운전자본 감소 → FCF 증가 | 기업가치 상승 |
| 유동성 부족 | 할인율 프리미엄 | 기업가치 할인 |
| 과도한 현금 보유 | 재투자 기회비용 | 현금 할인 적용 가능 |

**DCF에서**: CCC 변화는 운전자본 변동을 통해 FCF에 직접 영향을 미칩니다.

$$\text{FCF} = \text{NOPAT} + \text{감가상각} - \text{CAPEX} - \Delta\text{NWC}$$

운전자본(NWC) 증가 → FCF 감소, NWC 감소 → FCF 증가
