# 부동산 섹터 분석

## 섹터 개요

부동산 섹터는 리츠(REITs), 부동산 개발, 부동산 운영/관리를 포함합니다. 한국 리츠 시장은 성장 초기 단계로, 제도 개선과 함께 확대되고 있습니다.

## 하위 산업 분류

| 하위 산업 | 한국 대표 기업 | 특성 |
|-----------|-------------|------|
| **상장 리츠** | 신한알파리츠, 이지스밸류리츠 | 배당 중심, 안정적 |
| **부동산 개발** | 현대건설, DL이앤씨 | 프로젝트 기반 |
| **부동산 운영** | 이랜드리테일 | 임대/관리 수익 |

## 핵심 분석 프레임워크

### 리츠(REITs) 분석

**핵심 KPI**:
- **NAV (순자산가치)**: 보유 부동산 시가 - 부채
- **NOI (순영업수익)**: 임대 수익 - 운영 비용
- **Cap Rate**: NOI / 부동산 가치
- **공실률**: 임대 가능 면적 중 미임대 비율
- **WALE**: 가중평균 잔여 임대기간
- **배당수익률**: 주당 배당 / 주가
- **FFO**: 리츠 이익의 실질 지표 (순이익 + 감가상각 - 처분이익)

**밸류에이션**:
- P/NAV: NAV 대비 프리미엄/디스카운트
- 배당수익률: 국채 대비 스프레드
- FFO 멀티플: FFO 기준 PER

### NAV 산출

```
보유 부동산 감정평가액
+ 기타 자산
- 총부채
= NAV
÷ 주식수
= 주당 NAV
```

## 한국 부동산 섹터 특성

1. **리츠 시장 성장 초기**: 상장 리츠 수 증가 추세
2. **오피스/물류 중심**: 서울 오피스, 물류센터 리츠 다수
3. **금리 민감**: 금리 상승 시 리츠 수익률 매력 감소
4. **개발 리스크**: PF(프로젝트 파이낸싱) 관련 리스크

## 재무비율 벤치마크

| 지표 | 리츠 | 부동산 개발 |
|------|------|-----------|
| 배당수익률 | 5~8% | N/A |
| P/NAV | 0.8~1.2x | N/A |
| LTV (담보인정비율) | 40~60% | 60~80% |
| NOI 마진 | 60~80% | N/A |
