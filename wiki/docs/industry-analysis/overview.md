# 산업분석 개요

## 이 섹션의 목적

Layer 2에서 배운 재무 비율은 **산업 맥락 없이는 제대로 해석할 수 없습니다**. 부채비율 200%가 IT 기업에서는 위험하지만, 은행에서는 극히 보수적입니다. Layer 3에서는 GICS 11대 섹터별 분석 프레임워크를 배웁니다.

## GICS 11대 섹터 분류

GICS (Global Industry Classification Standard)는 MSCI와 S&P가 개발한 산업 분류 체계입니다.

| # | 섹터 | 영문 | 한국 대표 기업 | 핵심 지표 |
|---|------|------|--------------|----------|
| 1 | **정보기술** | Information Technology | 삼성전자, SK하이닉스 | EV/EBITDA, R&D 비율 |
| 2 | **헬스케어** | Healthcare | 삼성바이오, 셀트리온 | Pipeline 가치, PER |
| 3 | **금융** | Financials | KB금융, 삼성화재 | PBR, NIM, ROE |
| 4 | **경기소비재** | Consumer Discretionary | 현대차, 아모레퍼시픽 | PER, 브랜드 가치 |
| 5 | **필수소비재** | Consumer Staples | CJ제일제당, 오리온 | PER, 배당수익률 |
| 6 | **산업재** | Industrials | 현대중공업, 두산에너빌리티 | 수주잔고, EV/EBITDA |
| 7 | **소재** | Materials | POSCO, LG화학 | EV/EBITDA, P/B |
| 8 | **에너지** | Energy | S-Oil, SK이노베이션 | EV/EBITDA, 정제마진 |
| 9 | **유틸리티** | Utilities | 한국전력, 한국가스공사 | PBR, 배당수익률 |
| 10 | **부동산** | Real Estate | 신한알파리츠, 이지스밸류리츠 | NAV, 배당수익률 |
| 11 | **커뮤니케이션** | Communication Services | 카카오, 네이버 | PER, EV/EBITDA |

## 산업분석 프레임워크

### 1. 산업 구조 분석 (Porter's Five Forces)

| 요인 | 분석 질문 |
|------|-----------|
| 기존 경쟁 강도 | 경쟁사 수, 산업 집중도, 차별화 |
| 신규 진입 위협 | 진입 장벽, 규모의 경제, 규제 |
| 대체재 위협 | 대안 제품/기술 존재 |
| 공급자 교섭력 | 원재료 공급 구조 |
| 구매자 교섭력 | 고객 집중도, 전환 비용 |

### 2. 산업 수명주기

| 단계 | 성장률 | 이익률 | 경쟁 | 투자 관점 |
|------|--------|--------|------|----------|
| **도입기** | 매우 높음 | 적자/저이익 | 낮음 | 성장주 투자 |
| **성장기** | 높음 | 개선 | 증가 | EV/Sales, PER |
| **성숙기** | 안정 | 안정/높음 | 안정 | 배당, EV/EBITDA |
| **쇠퇴기** | 감소 | 하락 | 통합 | NAV, 배당 |

### 3. 산업별 핵심 분석 지표 (KPI)

각 산업마다 재무비율 외에 고유한 운영 지표(KPI)가 존재합니다:

| 산업 | 핵심 KPI |
|------|---------|
| 반도체 | ASP, 가동률, 미세공정 노드 |
| 은행 | NIM, NPL 비율, BIS 비율 |
| 통신 | ARPU, 가입자 수, 해지율 |
| 유통 | 동일점포매출(SSSG), GMV |
| 건설 | 수주잔고, 미분양률 |
| 바이오 | 임상 단계별 파이프라인 수 |
| 정유 | 정제마진, 가동률 |
| 항공 | 탑승률(Load Factor), RASK |

### 4. 산업별 밸류에이션 접근

| 산업 | 주요 밸류에이션 | 이유 |
|------|---------------|------|
| IT/반도체 | EV/EBITDA | 감가상각 차이 제거, 사이클 |
| 바이오 | DCF(Pipeline) | 적자 기업 다수, 미래가치 |
| 은행 | PBR | 레버리지 사업, 자산 기반 |
| 유틸리티 | PBR, 배당수익률 | 안정적 현금흐름, 규제 수익 |
| 부동산(리츠) | NAV, 배당수익률 | 자산가치 기반 |
| 건설 | PER(수주기반) | 수주잔고 → 미래 매출 |
| 소비재 | PER | 안정적 이익 |

## 한국 시장 산업 구조

### KOSPI 섹터별 시가총액 비중

한국 시장은 IT(반도체)와 금융 비중이 압도적:

| 섹터 | 비중 (대략) | 특성 |
|------|-----------|------|
| IT | ~35% | 삼성전자 단일 기업 영향 막대 |
| 금융 | ~15% | 은행, 보험, 증권 |
| 산업재 | ~10% | 조선, 건설, 기계 |
| 소재 | ~10% | 화학, 철강 |
| 경기소비재 | ~10% | 자동차, 의류 |
| 헬스케어 | ~5% | 바이오, 제약 |
| 기타 | ~15% | 에너지, 유틸리티, 통신 등 |

### 한국 산업분석 특수 고려사항

1. **삼성전자 의존**: KOSPI의 ~20% 비중, 반도체 사이클이 시장 전체 영향
2. **수출 의존**: 주요 기업이 수출 중심, 글로벌 경기/환율 영향 큼
3. **정부 정책**: 산업 육성/규제 정책이 큰 영향 (배터리, 반도체, 바이오)
4. **중국 리스크**: 중국 경기/정책이 한국 수출산업에 직접 영향
5. **재벌 구조**: 산업 내 재벌 그룹 간 경쟁 구도
