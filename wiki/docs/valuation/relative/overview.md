# 상대가치평가 (Relative Valuation)

<span className="difficulty-badge difficulty-intermediate">중급</span>

## 정의

**상대가치평가**는 유사한 기업들의 시장 가격을 기준으로 대상 기업의 가치를 산정하는 방법입니다. "시장이 유사 기업에 부여한 가격이라면, 우리 기업도 비슷해야 한다"는 논리에 기반합니다.

## DCF vs 상대가치평가

| 항목 | DCF | 상대가치평가 |
|------|-----|-------------|
| **기반** | 내재가치 (Intrinsic) | 시장가치 (Relative) |
| **장점** | 이론적 정확성 | 간편함, 시장 검증 |
| **단점** | 가정에 민감 | 시장이 잘못될 수 있음 |
| **사용** | 정밀 분석, M&A | 빠른 스크리닝, 벤치마킹 |

:::tip 실무 활용
대부분의 실무에서는 **DCF와 상대가치평가를 함께** 사용합니다. DCF로 내재가치를 산출하고, 상대가치평가로 검증하거나 Football Field 차트로 범위를 제시합니다.
:::

## 핵심 멀티플

### Equity Value 기반

주가를 분자로 사용

| 멀티플 | 공식 | 적합 대상 |
|--------|------|-----------|
| **[PER](/valuation/relative/per)** | 주가 / EPS | 수익성 있는 성장기업 |
| **[PBR](/valuation/relative/pbr)** | 주가 / BPS | 금융, 자산주, 적자기업 |
| **[PSR](/valuation/relative/psr)** | 주가 / 매출 | 적자 성장기업 |
| **PEG** | PER / EPS 성장률 | GARP 투자 |

### Enterprise Value 기반

EV(시총 + 순부채)를 분자로 사용

| 멀티플 | 공식 | 적합 대상 |
|--------|------|-----------|
| **[EV/EBITDA](/valuation/relative/ev-ebitda)** | EV / EBITDA | M&A, 자본구조 중립 비교 |
| **EV/EBIT** | EV / EBIT | 감가상각 큰 기업 |
| **EV/Sales** | EV / 매출 | 적자 기업, 초기 스타트업 |

## Comparable Company Analysis (CCA)

### 유사 기업 선정 기준

1. **산업**: 동일 또는 유사 산업 분류
2. **규모**: 비슷한 매출/시총 규모
3. **성장성**: 유사한 성장률 전망
4. **수익성**: 비슷한 마진 수준
5. **지역**: 동일 시장 또는 유사 시장

### 분석 프로세스

```
┌─────────────────────────────────────────────────────────────────┐
│  Step 1: 유사 기업 선정 (5-10개)                                │
├─────────────────────────────────────────────────────────────────┤
│  Step 2: 재무 데이터 수집                                        │
│  - 주가, 시가총액, 순부채                                        │
│  - EPS, BPS, EBITDA, 매출                                       │
├─────────────────────────────────────────────────────────────────┤
│  Step 3: 멀티플 계산                                             │
│  - PER, PBR, EV/EBITDA 등                                       │
├─────────────────────────────────────────────────────────────────┤
│  Step 4: 이상치 제거 및 통계                                     │
│  - 평균, 중앙값, 사분위 범위                                     │
├─────────────────────────────────────────────────────────────────┤
│  Step 5: 대상 기업 가치 산출                                     │
│  - 대상 기업 지표 × 유사 기업 멀티플                             │
└─────────────────────────────────────────────────────────────────┘
```

### 예시: 네이버 유사 기업 분석

| 기업 | 국가 | PER | EV/EBITDA | 매출성장률 |
|------|------|-----|-----------|-----------|
| Alphabet | 미국 | 25x | 14x | 12% |
| Meta | 미국 | 22x | 11x | 15% |
| 카카오 | 한국 | 45x | 20x | 8% |
| 라인야후 | 일본 | 30x | 12x | 5% |
| **평균** | - | **30.5x** | **14.3x** | **10%** |
| **중앙값** | - | **27.5x** | **13x** | **10%** |

**네이버 적용:**
- 네이버 EPS: 30,000원
- 적용 PER: 27.5x (중앙값)
- **산출 주가: 825,000원**

## 멀티플 해석 시 주의점

### 1. 회계 기준 차이

| 항목 | K-IFRS | US-GAAP | 영향 |
|------|--------|---------|------|
| R&D | 일부 자산화 | 대부분 비용 | EPS 차이 |
| 리스 | IFRS 16 | ASC 842 | EBITDA 차이 |

### 2. 일회성 항목

- 정상 수익력 기준으로 **조정 EPS/EBITDA** 사용
- 구조조정비용, 자산손상, 일회성 이익 제거

### 3. 시점 차이

| 종류 | 설명 | 적용 |
|------|------|------|
| TTM | 최근 12개월 | 현재 상태 반영 |
| Forward | 향후 12개월 예상 | 성장성 반영 |
| NTM+1 | 내년 예상 | 턴어라운드 기업 |

### 4. 프리미엄/할인 요인

| 요인 | 프리미엄 | 할인 |
|------|----------|------|
| 규모 | 대형주 | 소형주 |
| 유동성 | 거래량 많음 | 거래량 적음 |
| 지배구조 | 투명 | 불투명 |
| 성장성 | 고성장 | 저성장 |
| 시장 지위 | 1위 | 후발주자 |
