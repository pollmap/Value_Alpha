# 잔여이익모형 (Residual Income Model)

## 개요

**잔여이익모형 (RIM)** 은 기업이 자기자본비용을 초과하여 창출하는 이익을 기반으로 기업가치를 평가하는 방법입니다.

```
잔여이익 = 순이익 - (자기자본 × 자기자본비용)
        = 순이익 - 자기자본에 대한 기회비용
```

## 핵심 공식

### 잔여이익 (Residual Income)

```
RI = NI - (BV × Ke)

RI: 잔여이익
NI: 순이익
BV: 자기자본 장부가치
Ke: 자기자본비용
```

### 기업가치 산정

```
V₀ = BV₀ + Σ[RIₜ / (1 + Ke)ᵗ]

V₀: 현재 주식가치
BV₀: 현재 자기자본 장부가치
RIₜ: t기의 잔여이익
Ke: 자기자본비용
```

## 계산 예시

```
가정:
- 현재 자기자본 (BV₀): 10,000억원
- 자기자본비용 (Ke): 10%
- 예상 순이익: 1,500억원/년 (5년간)
- 영구성장률: 2%

1년차 잔여이익:
RI₁ = 1,500 - (10,000 × 10%) = 500억원

기업가치:
V = 10,000 + 500/(1.1) + 500/(1.1)² + ... + TV
```

## DCF와의 비교

| 구분 | DCF | RIM |
|------|-----|-----|
| 기초 | 현금흐름 | 회계이익 |
| 시작점 | 0 | 장부가치 |
| 적용 | 현금흐름 예측 용이 시 | 회계정보 신뢰성 높을 시 |
| 장점 | 직관적 | 회계자료 활용 |

## 활용 분야

1. **금융기관 가치평가** - 현금흐름 정의 어려움
2. **규제산업** - 자본수익률 중심 평가
3. **M&A 실사** - 장부가치 기반 분석

## EVA (경제적부가가치)

```
EVA = NOPAT - (투하자본 × WACC)

NOPAT: 세후영업이익
투하자본: 영업에 사용된 자본
WACC: 가중평균자본비용
```

**EVA와 RIM의 관계:**
- EVA: 기업 전체 관점 (WACC 사용)
- RIM: 주주 관점 (Ke 사용)
