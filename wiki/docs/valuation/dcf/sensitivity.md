# 민감도 분석

<span className="difficulty-badge difficulty-intermediate">중급</span>

## 정의

**민감도 분석(Sensitivity Analysis)**은 DCF 모델의 핵심 가정이 변할 때 기업가치가 어떻게 변하는지 분석하는 기법입니다. 불확실성을 정량화하고 투자 결정의 위험을 평가하는 데 필수적입니다.

## 왜 필요한가?

DCF 모델은 수많은 가정에 의존합니다:

- 매출 성장률
- 영업이익률
- WACC
- 영구성장률
- CapEx 비율

이 중 **어떤 가정이 가장 중요한지**, **가정이 얼마나 변해도 투자 논리가 유지되는지** 파악해야 합니다.

## 분석 유형

### 1. 단일 변수 민감도

한 변수만 변경하고 나머지는 고정

<div className="formula-block">

WACC 변화에 따른 기업가치 변동

| WACC | 기업가치 | 변동률 |
|------|----------|--------|
| 8% | 1,500억 | +25% |
| 9% | 1,300억 | +8% |
| **10%** | **1,200억** | **Base** |
| 11% | 1,100억 | -8% |
| 12% | 1,000억 | -17% |

</div>

### 2. 이변수 민감도 (Data Table)

두 변수를 동시에 변경하여 2차원 표로 표시

**WACC vs 영구성장률**

| WACC \ g | 2.0% | 2.5% | 3.0% | 3.5% | 4.0% |
|----------|------|------|------|------|------|
| **8.0%** | 1,350 | 1,450 | 1,575 | 1,725 | 1,920 |
| **9.0%** | 1,200 | 1,280 | 1,375 | 1,490 | 1,630 |
| **10.0%** | 1,080 | 1,145 | 1,220 | 1,310 | 1,420 |
| **11.0%** | 980 | 1,035 | 1,095 | 1,170 | 1,260 |
| **12.0%** | 900 | 945 | 1,000 | 1,060 | 1,135 |

:::tip 활용법
현재 주가와 비교하여 어떤 가정 조합에서 저평가/고평가인지 확인합니다.
:::

### 3. 토네이도 차트 (Tornado Chart)

각 변수의 영향력을 시각적으로 비교

```
                    기업가치 변동 범위 (억원)
                    900    1,000   1,100   1,200   1,300   1,400
                     │       │       │       │       │       │
영구성장률 (2-4%)   ├───────────────────────────────────────────┤
WACC (9-11%)       ├─────────────────────────────────┤
매출성장률 (5-15%) ├───────────────────────────┤
영업이익률 (8-12%) ├─────────────────────┤
CapEx/매출 (3-7%)  ├───────────┤
                     │       │       │       │       │       │
                    Base Case: 1,200억원
```

**해석**: 막대가 길수록 해당 변수의 영향력이 큽니다. 위 예시에서 **영구성장률**과 **WACC**가 가장 중요한 변수입니다.

### 4. 시나리오 분석

여러 변수를 동시에 조정하여 3가지 시나리오 구성

| 항목 | Bear Case | Base Case | Bull Case |
|------|-----------|-----------|-----------|
| 매출 성장률 | 5% | 10% | 15% |
| 영업이익률 | 8% | 10% | 12% |
| WACC | 11% | 10% | 9% |
| 영구성장률 | 2% | 3% | 4% |
| **기업가치** | **850억** | **1,200억** | **1,700억** |
| **확률 가중** | 25% | 50% | 25% |
| **기대가치** | | **1,225억** | |

<div className="formula-block">

**기대가치 = 0.25 × 850 + 0.50 × 1,200 + 0.25 × 1,700 = 1,225억원**

</div>

## 핵심 변수 선정

### 일반적으로 민감도가 높은 변수

| 순위 | 변수 | 이유 |
|------|------|------|
| 1 | **영구성장률** | 터미널 가치에 직접 영향 |
| 2 | **WACC** | 모든 현금흐름의 할인율 |
| 3 | **매출 성장률** | 전체 현금흐름 규모 결정 |
| 4 | **영업이익률** | 현금 창출 능력 |
| 5 | **CapEx 비율** | FCF 직접 감소 |

### 산업별 핵심 변수

| 산업 | 핵심 변수 |
|------|----------|
| 테크 | 매출 성장률, 고객획득비용 |
| 제조 | 영업이익률, 원자재 비용 |
| 유틸리티 | WACC, 요금 인상률 |
| 부동산 | Cap Rate, 공실률 |
| 바이오 | 성공 확률, 출시 시점 |

## 실무 베스트 프랙티스

### 1. 현실적인 범위 설정

| 변수 | 권장 범위 |
|------|----------|
| WACC | Base ± 1-2%p |
| 영구성장률 | Base ± 1%p |
| 매출 성장률 | Base ± 50% |
| 마진 | Base ± 2-3%p |

### 2. Break-even 분석

현재 주가를 정당화하는 가정 찾기:

> "현재 주가 기준 시총이 1,500억원이라면, 이를 정당화하려면 WACC 8% 또는 영구성장률 4%가 필요합니다. 이 가정이 합리적인가?"

### 3. 투자 결정 프레임워크

```
┌─────────────────────────────────────────────────────────┐
│                    민감도 분석 결과                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  현재 주가: 1,000억원                                   │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │ Bear Case (1-σ):  850억원  → 15% 하락 위험     │   │
│  │ Base Case:        1,200억원 → 20% 상승 여력    │   │
│  │ Bull Case (1+σ):  1,700억원 → 70% 상승 가능    │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  ✅ Risk/Reward 비율: 상승 여력 > 하락 위험            │
│  ✅ Bear Case에서도 큰 손실 없음                       │
│  → 투자 결정 긍정적                                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 몬테카를로 시뮬레이션

고급 분석: 각 변수에 확률분포를 부여하고 수천 번 시뮬레이션

```
변수 분포 설정:
- 매출 성장률: Normal(10%, 3%)
- 영업이익률: Normal(10%, 2%)
- WACC: Triangular(9%, 10%, 11%)
- 영구성장률: Uniform(2%, 4%)

→ 10,000회 시뮬레이션 결과:
- 평균 기업가치: 1,180억원
- 표준편차: 220억원
- 95% 신뢰구간: 780억 ~ 1,620억원
- 현재 주가(1,000억) 초과 확률: 68%
```
