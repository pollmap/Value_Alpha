# FCFF vs FCFE

<span className="difficulty-badge difficulty-intermediate">중급</span>

## 개요

DCF 분석에서 사용하는 현금흐름은 두 가지입니다:

| 구분 | FCFF | FCFE |
|------|------|------|
| 전체 명칭 | Free Cash Flow to Firm | Free Cash Flow to Equity |
| 귀속 대상 | 전체 자본 제공자 (주주 + 채권자) | 주주만 |
| 할인율 | WACC | 자기자본비용 (Re) |
| 결과 | 기업가치 (Enterprise Value) | 자기자본가치 (Equity Value) |

## FCFF (기업잉여현금흐름)

### 정의

FCFF는 영업활동에서 창출된 현금 중 **모든 자본 제공자**(주주와 채권자)에게 배분 가능한 현금흐름입니다.

### 공식

<div className="formula-block">

**FCFF = EBIT × (1-T) + D&A - CapEx - ΔNWC**

</div>

또는

<div className="formula-block">

**FCFF = EBITDA × (1-T) + D&A × T - CapEx - ΔNWC**

</div>

### 구성요소 상세

| 항목 | 의미 | 계산 방법 |
|------|------|----------|
| **EBIT** | 영업이익 | 손익계산서에서 직접 확인 |
| **T** | 법인세율 | 유효세율 또는 한계세율 |
| **D&A** | 감가상각비 | 현금흐름표 또는 주석 |
| **CapEx** | 자본적 지출 | 유형자산 취득 - 처분 |
| **ΔNWC** | 운전자본 변동 | (유동자산 - 현금) - (유동부채 - 단기차입금) 변동 |

### FCFF 계산 예시

| 항목 | 금액 (억원) |
|------|------------|
| 영업이익 (EBIT) | 1,000 |
| 법인세 (22%) | (220) |
| **세후 영업이익** | **780** |
| (+) 감가상각비 | 200 |
| (-) 자본적 지출 | (300) |
| (-) 운전자본 증가 | (50) |
| **FCFF** | **630** |

## FCFE (주주잉여현금흐름)

### 정의

FCFE는 채권자에 대한 의무를 이행한 후 **주주에게 배분 가능한** 현금흐름입니다.

### 공식

<div className="formula-block">

**FCFE = FCFF - 이자비용 × (1-T) + 순차입금**

</div>

또는

<div className="formula-block">

**FCFE = 당기순이익 + D&A - CapEx - ΔNWC + 순차입금**

</div>

### FCFE 계산 예시

| 항목 | 금액 (억원) |
|------|------------|
| FCFF | 630 |
| (-) 이자비용 × (1-T) | (78) |
| (+) 신규 차입 | 100 |
| (-) 부채 상환 | (50) |
| **FCFE** | **602** |

## FCFF vs FCFE 비교

```
┌─────────────────────────────────────────────────────────────────┐
│                         EBIT (영업이익)                          │
│                              │                                  │
│                    (-) 법인세 × EBIT                            │
│                              ↓                                  │
│                    EBIT × (1-T) = NOPAT                         │
│                              │                                  │
│                    (+) 감가상각                                  │
│                    (-) CapEx                                    │
│                    (-) ΔNWC                                     │
│                              ↓                                  │
│                  ┌──────────────────────┐                       │
│                  │        FCFF          │ ← WACC으로 할인       │
│                  │   (기업가치 산출)     │   → Enterprise Value │
│                  └──────────────────────┘                       │
│                              │                                  │
│                    (-) 이자비용 × (1-T)                         │
│                    (+) 순차입금                                  │
│                              ↓                                  │
│                  ┌──────────────────────┐                       │
│                  │        FCFE          │ ← Re로 할인           │
│                  │  (자기자본가치 산출)  │   → Equity Value     │
│                  └──────────────────────┘                       │
└─────────────────────────────────────────────────────────────────┘
```

## 언제 무엇을 사용하나?

### FCFF 사용 권장

| 상황 | 이유 |
|------|------|
| **자본구조 변화 예상** | 부채 상환/증가 계획이 있는 경우 |
| **M&A 분석** | 인수 후 자본구조 변경 가능 |
| **레버리지가 높거나 음(-)인 FCFE** | FCFE가 불안정할 때 |
| **일반적인 기업 밸류에이션** | 실무에서 가장 많이 사용 |

### FCFE 사용 권장

| 상황 | 이유 |
|------|------|
| **안정적 자본구조** | 부채비율이 일정하게 유지 |
| **금융회사** | 부채가 영업의 일부 |
| **배당 정책 분석** | 주주 관점의 가치 산정 |
| **LBO 분석** | 주주 수익률 계산 |

## 검증: 두 방법의 결과는 같아야 함

:::tip 일관성 검증
FCFF를 WACC로 할인한 기업가치에서 순부채를 차감한 자기자본가치는, FCFE를 Re로 할인한 자기자본가치와 **이론적으로 동일**해야 합니다.

**EV (from FCFF) - Net Debt = Equity Value (from FCFE)**
:::

결과가 다르다면:
1. 할인율 계산 오류
2. 자본구조 가정 불일치
3. 현금흐름 계산 오류

## 운전자본(NWC) 계산 시 주의점

### 포함 항목

| 유동자산 | 유동부채 |
|----------|----------|
| 매출채권 | 매입채무 |
| 재고자산 | 미지급비용 |
| 선급금 | 선수금 |

### 제외 항목

| 유동자산 | 유동부채 |
|----------|----------|
| **현금 및 현금성자산** | **단기차입금** |
| 단기금융상품 | 유동성장기부채 |

:::caution 주의
현금과 이자부부채는 운전자본 계산에서 제외합니다. 이들은 별도로 기업가치에서 조정됩니다.
:::
