# DCF 분석 (현금흐름할인법)

<span className="difficulty-badge difficulty-intermediate">중급</span>

## 정의

**DCF(Discounted Cash Flow)**는 기업이 미래에 창출할 것으로 예상되는 현금흐름을 적절한 할인율로 현재가치로 환산하여 기업의 내재가치를 산출하는 방법입니다.

<div className="master-quote">
"가격은 당신이 지불하는 것이고, 가치는 당신이 얻는 것이다."

<span className="author">— 워렌 버핏</span>
</div>

## 핵심 공식

<div className="formula-block">

**기업가치 = Σ [FCFt / (1+WACC)^t] + TV / (1+WACC)^n**

</div>

### 구성요소

| 요소 | 설명 | 산출 방법 |
|------|------|----------|
| **FCF** | 잉여현금흐름 | EBIT(1-T) + D&A - CapEx - ΔNWC |
| **WACC** | 가중평균자본비용 | 자기자본비용과 부채비용의 가중평균 |
| **TV** | 터미널 가치 | 예측 기간 이후의 가치 |
| **n** | 예측 기간 | 보통 5-10년 |

## 왜 DCF인가?

### 장점

1. **내재가치 기반**: 시장 심리가 아닌 기업의 본질적 가치에 집중
2. **유연성**: 다양한 시나리오와 가정을 반영 가능
3. **논리적 일관성**: 현금흐름의 시간가치를 체계적으로 반영
4. **비교 가능**: 산업, 규모에 관계없이 적용 가능

### 한계점

1. **가정 민감도**: 성장률, WACC 가정에 따라 결과가 크게 달라짐
2. **터미널 가치 의존**: 전체 가치의 60-80%가 TV에서 결정
3. **예측의 불확실성**: 미래 현금흐름 예측의 어려움
4. **적자 기업 적용 곤란**: 양(+)의 현금흐름이 없으면 적용 어려움

## DCF 분석 5단계

```
┌─────────────────────────────────────────────────────────────────┐
│  Step 1: 과거 재무 분석                                         │
│  ├─ 매출 성장률 추이                                           │
│  ├─ 마진 변화 (GPM, OPM, NPM)                                  │
│  └─ 자본 지출 패턴                                              │
├─────────────────────────────────────────────────────────────────┤
│  Step 2: 재무 추정 (5-10년)                                     │
│  ├─ 매출 예측 (Top-down / Bottom-up)                           │
│  ├─ 비용 구조 예측                                              │
│  └─ 운전자본 및 CapEx 추정                                     │
├─────────────────────────────────────────────────────────────────┤
│  Step 3: FCFF/FCFE 산출                                         │
│  ├─ FCFF = EBIT(1-T) + D&A - CapEx - ΔNWC                     │
│  └─ FCFE = FCFF - 이자비용(1-T) + 순차입금                     │
├─────────────────────────────────────────────────────────────────┤
│  Step 4: 할인율(WACC) 산정                                      │
│  ├─ 자기자본비용 (CAPM)                                        │
│  ├─ 부채비용                                                    │
│  └─ 목표 자본구조                                               │
├─────────────────────────────────────────────────────────────────┤
│  Step 5: 터미널 가치 및 기업가치                                │
│  ├─ Gordon Growth Model 또는 Exit Multiple                     │
│  ├─ 기업가치 = PV(FCF) + PV(TV)                                │
│  └─ 자기자본가치 = 기업가치 - 순부채                           │
└─────────────────────────────────────────────────────────────────┘
```

## 실무 적용 예시

### 삼성전자 간이 DCF (가상)

| 연도 | 매출 (조원) | EBIT | FCFF | 현재가치 |
|------|------------|------|------|----------|
| 2024 | 280 | 42 | 30 | 27.3 |
| 2025 | 308 | 49 | 35 | 28.9 |
| 2026 | 339 | 54 | 40 | 30.0 |
| 2027 | 373 | 60 | 45 | 30.7 |
| 2028 | 410 | 66 | 50 | 31.0 |
| **TV** | - | - | - | **350.0** |
| **합계** | - | - | - | **497.9조** |

*가정: WACC 10%, 영구성장률 3%*

:::caution 주의사항
위 예시는 교육 목적의 가상 데이터입니다. 실제 투자 결정에는 보다 정밀한 분석이 필요합니다.
:::
