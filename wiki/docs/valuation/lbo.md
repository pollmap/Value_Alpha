# LBO 분석 (Leveraged Buyout)

## LBO란?

**차입매수 (LBO)** 는 인수자금의 대부분을 차입금으로 조달하여 기업을 인수하는 방식입니다.

```
LBO 자금조달 구조:
┌─────────────────────────────┐
│     인수가격 100억원         │
├─────────────────────────────┤
│  선순위 대출    50억 (50%)   │
│  메자닌 금융    20억 (20%)   │
│  자기자본       30억 (30%)   │
└─────────────────────────────┘
```

## 핵심 개념

### 1. 레버리지 효과

```
투자수익률 극대화:

시나리오: 기업가치 100억 → 150억 (50% 상승)

무차입 인수:
- 투자: 100억 (전액 자기자본)
- 수익: 50억
- ROI: 50%

LBO (70% 차입):
- 투자: 30억 (자기자본)
- 기업가치: 150억 - 70억(부채상환) = 80억
- 수익: 50억
- ROI: 167%
```

### 2. 이상적인 LBO 대상

```
□ 안정적인 현금흐름 (부채 상환 가능)
□ 낮은 자본지출 요구
□ 비핵심 자산 매각 가능
□ 비용 절감 여지
□ 경기 방어적 산업
□ 강한 시장 지위
```

## LBO 수익률 분석

### IRR (내부수익률)

```
IRR 구성요소:

1. 부채 상환 (Debt Paydown)
   - 영업현금흐름으로 부채 감소
   - 자기자본 비중 증가

2. EBITDA 성장
   - 매출 성장
   - 마진 개선

3. 멀티플 확대
   - Exit 시 더 높은 배수 적용
   - 구조개선, 성장성 증명

IRR = f(Entry Multiple, Exit Multiple, EBITDA Growth, Leverage)
```

### IRR 계산 예시

```
투자:
- 인수가격: 1,000억원 (EV/EBITDA 8x)
- 자기자본: 400억원
- 부채: 600억원

5년 후 Exit:
- EBITDA: 200억원 (연 5% 성장)
- 부채 상환: 300억원
- Exit 멀티플: 9x

Exit 가치:
- EV = 200억 × 9 = 1,800억원
- 잔여부채: 300억원
- 자기자본가치: 1,500억원

IRR = (1,500/400)^(1/5) - 1 ≈ 30%
```

## PE 펀드의 가치창출

### 가치창출 원천

```
1. 운영 개선 (Operational Improvement)
   - 비용 절감
   - 매출 성장
   - 운전자본 효율화

2. 재무 엔지니어링 (Financial Engineering)
   - 최적 자본구조
   - 부채 리파이낸싱
   - 배당 리캡

3. 거버넌스 개선 (Governance)
   - 경영진 인센티브 정렬
   - 이사회 강화
   - 전략적 방향 설정

4. 멀티플 차익 (Multiple Arbitrage)
   - 저평가 매입
   - 고평가 매각
```

## LBO 위험요소

| 위험 | 설명 | 완화 방안 |
|------|------|----------|
| 금리 위험 | 변동금리 부채 | 금리 헷지 |
| 경기 위험 | 현금흐름 감소 | 방어적 산업 선택 |
| 운영 위험 | 실적 미달 | 실사 강화 |
| Exit 위험 | 매각 지연/저가 | 다양한 Exit 경로 |
