# 암호화폐·디파이 산업 완벽 가이드

## 학습 목표

- 암호화폐 거래소(CeFi)와 탈중앙화 금융(DeFi)의 비즈니스 모델을 이해한다
- 거래대금, TVL, 활성 사용자 수 등 핵심 지표를 해석할 수 있다
- 한국 암호화폐 규제 환경과 특수성을 파악한다
- 암호화폐 관련 기업의 밸류에이션 접근법을 습득한다

## 1. 섹터 개요

### 암호화폐 산업의 구조

암호화폐 산업은 크게 **중앙화 금융(CeFi)**과 **탈중앙화 금융(DeFi)**으로 구분됩니다. 전통 금융과 달리 24시간 365일 운영되며, 국경을 초월한 글로벌 시장을 형성합니다.

| 구분 | CeFi (중앙화 금융) | DeFi (탈중앙화 금융) |
|:-----|:------------------|:-------------------|
| **운영 주체** | 기업 (거래소, 수탁사) | 스마트 컨트랙트 (코드) |
| **수익 모델** | 거래 수수료, 상장 수수료, 이자 | 프로토콜 수수료, 토큰 발행 |
| **규제** | 각국 금융 규제 적용 | 대부분 규제 사각지대 |
| **사용자 경험** | 전통 금융과 유사 | 지갑 연결, 복잡한 UX |
| **리스크** | 해킹, 거래소 파산, 규제 | 스마트 컨트랙트 취약점, 러그풀 |
| **대표 서비스** | 업비트, 바이낸스, 코인베이스 | Uniswap, Aave, Lido |

### 시장 규모 (2024년 기준)

| 지표 | 글로벌 | 한국 |
|:-----|:------|:-----|
| **암호화폐 시가총액** | 약 2.5조 달러 | - |
| **일일 거래대금** | 약 500\~1,000억 달러 | 약 50\~100억 달러 |
| **DeFi TVL** | 약 500억 달러 | 약 10억 달러 |
| **암호화폐 투자자 수** | 약 5억 명 | 약 600만 명 |
| **거래소 수** | 500개 이상 | 20개 이상 (등록 기준) |

한국은 글로벌 암호화폐 거래량의 약 5\~10%를 차지하는 주요 시장입니다. 특히 원화 거래 비중이 높고, "김치 프리미엄"이라 불리는 국내외 가격 괴리 현상이 자주 발생합니다.

### 중앙화 거래소 (CEX)

암호화폐 거래소는 사용자 간 암호화폐 매매를 중개하고 수수료를 수취하는 플랫폼입니다.

**비즈니스 모델**:
- **거래 수수료**: 매매 시 0.05\~0.5% 수수료 (핵심 수익원)
- **상장 수수료**: 신규 코인 상장 시 프로젝트로부터 수취
- **스테이킹 서비스**: 사용자 자산 스테이킹 후 수수료 분배
- **렌딩/차입**: 암호화폐 대출 서비스 이자 수익
- **NFT 마켓플레이스**: NFT 거래 중개 수수료

**핵심 지표**:

| 지표 | 정의 | 해석 |
|:-----|:-----|:-----|
| **거래대금** | 일/월별 총 거래금액 | 수수료 수익의 기반, 변동성 클수록 증가 |
| **MAU/DAU** | 월간/일간 활성 사용자 | 플랫폼 점유율, 성장성 지표 |
| **시장점유율** | 국내 거래대금 비중 | 경쟁력, 프랜차이즈 가치 |
| **예치자산(AUC)** | 고객 예치 암호화폐 총액 | 플랫폼 신뢰도, 스테이킹 수익 기반 |
| **상장 코인 수** | 거래 가능 코인 종류 | 서비스 다양성, 규제 리스크 |

**주요 기업**:
- **두나무 (업비트)**: 국내 점유율 80% 이상, 카카오 투자 유치
- **빗썸코리아 (빗썸)**: 국내 2위, 다수 대주주 변경 이력
- **코인원**: 국내 3위, KB금융 지분 투자
- **코빗**: SK그룹 계열, B2B 수탁 서비스 강점
- **고팍스**: 핀테크 스타트업 출신

### 탈중앙화 금융 (DeFi)

DeFi는 블록체인 스마트 컨트랙트를 통해 중개자 없이 금융 서비스를 제공하는 생태계입니다.

**DeFi 카테고리**:

| 카테고리 | 설명 | 대표 프로토콜 |
|:---------|:-----|:-------------|
| **DEX (탈중앙화 거래소)** | AMM 기반 토큰 스왑 | Uniswap, PancakeSwap |
| **렌딩 프로토콜** | 암호화폐 대출/차입 | Aave, Compound |
| **스테이킹/리퀴드 스테이킹** | PoS 자산 스테이킹 대행 | Lido, Rocket Pool |
| **스테이블코인** | 달러 페깅 암호화폐 | DAI (MakerDAO), USDC |
| **파생상품** | 선물, 옵션, 합성자산 | dYdX, GMX |
| **수익 최적화** | 이자율 최적화 | Yearn Finance |

**핵심 지표**:

| 지표 | 정의 | 해석 |
|:-----|:-----|:-----|
| **TVL (Total Value Locked)** | 프로토콜에 예치된 총 자산 | 프로토콜 규모, 신뢰도 |
| **프로토콜 수수료** | 거래/이용 시 발생 수수료 | 실질 수익 창출 능력 |
| **일일 활성 주소** | 프로토콜 이용 지갑 수 | 실제 사용량 |
| **토큰 시가총액** | 거버넌스 토큰 가치 | 프로토콜 밸류에이션 |
| **FDV (완전희석가치)** | 전체 토큰 발행 기준 시총 | 장기 희석 리스크 |

## 3. 한국 암호화폐 규제 환경

### 규제 체계

한국은 2021년 **특정금융정보법(특금법)** 개정을 통해 암호화폐 거래소를 금융당국 감독 하에 편입했습니다.

| 규제 항목 | 내용 | 영향 |
|:---------|:-----|:-----|
| **VASP 등록** | 가상자산사업자 금융정보분석원 신고 | 미신고 시 영업 불가 |
| **실명계좌 연동** | 은행 실명계좌 필수 연동 | 익명 거래 불가, 4대 거래소만 보유 |
| **ISMS 인증** | 정보보호관리체계 인증 의무 | 보안 기준 충족 필요 |
| **자금세탁방지** | AML/KYC 의무화 | 사용자 신원 확인 필수 |
| **과세** | 2025년부터 22% 양도소득세 (250만 원 공제) | 투자 수익에 과세 |

### 실명계좌 연동의 의미

한국에서 원화 입출금이 가능한 "원화마켓"을 운영하려면 시중은행의 실명계좌 연동이 필수입니다. 현재 이를 보유한 거래소는 **업비트(케이뱅크), 빗썸(NH농협), 코인원(카카오뱅크), 코빗(신한은행)** 4곳뿐입니다. 이 진입장벽이 사실상 과점 구조를 형성합니다.

```
┌─────────────────────────────────────────────────────────────┐
│              한국 암호화폐 거래소 시장 구조                     │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌──────────────────┐                                        │
│  │  원화마켓 4대 거래소  │ ← 실명계좌 연동 보유                 │
│  │  업비트 (80%+)      │                                      │
│  │  빗썸 (10%~)        │                                      │
│  │  코인원 (5%~)       │                                      │
│  │  코빗 (2%~)         │                                      │
│  └──────────────────┘                                        │
│           ↑                                                  │
│     진입장벽: 실명계좌                                         │
│           ↓                                                  │
│  ┌──────────────────┐                                        │
│  │  기타 거래소         │ ← 코인마켓만 운영 (원화 거래 불가)    │
│  │  고팍스, 플라이빗 등  │                                      │
│  └──────────────────┘                                        │
│                                                              │
```

## 4. 암호화폐 거래소 분석 심화

### 4.1 수익 구조

암호화폐 거래소의 수익은 **거래대금**에 직접 연동됩니다. 암호화폐 시장의 변동성이 클수록 거래가 활발해지고 수수료 수익이 증가합니다.

| 수익원 | 비중 | 설명 |
|:-------|:-----|:-----|
| **거래 수수료** | 70\~90% | 매수/매도 시 0.05\~0.5% 수수료 |
| **상장 수수료** | 5\~15% | 신규 코인 상장 시 일회성 수익 |
| **스테이킹 수수료** | 5\~10% | 사용자 스테이킹 보상의 일부 |
| **기타** | 5% 이하 | 렌딩, NFT, API 서비스 등 |

**거래대금과 수익의 관계**:

$$\text{거래 수수료 수익} = \text{거래대금} \times \text{수수료율}$$

예: 일 거래대금 10조 원 × 수수료율 0.1% = 일 수익 100억 원

### 4.2 리스크 요인

| 리스크 | 설명 | 사례 |
|:-------|:-----|:-----|
| **규제 리스크** | 정부 정책 변화, 규제 강화 | 중국 거래소 전면 금지 (2021) |
| **해킹 리스크** | 거래소 보안 침해, 자산 유출 | FTX 해킹 (2022) |
| **시장 리스크** | 암호화폐 가격 급락, 거래량 감소 | 2022년 크립토 윈터 |
| **거래상대방 리스크** | 거래소 파산, 자산 동결 | FTX 파산 (2022), 테라 사태 |
| **법적 리스크** | 증권성 판단, 상장폐지 | SEC vs 리플 소송 |

## 5. 밸류에이션 접근

### 거래소 밸류에이션

암호화폐 거래소는 전통 금융 거래소와 유사하게 **거래대금 기반 수익 멀티플**로 평가합니다.

| 방법론 | 적용 | 특징 |
|:-------|:-----|:-----|
| **P/E** | 수익 기반 | 수익 변동성 높아 적용 어려움 |
| **P/S** | 매출 기반 | 거래대금 변동에 민감 |
| **EV/거래대금** | 거래량 기반 | 거래소 간 비교에 유용 |
| **사용자당 가치** | MAU × 사용자당 가치 | 플랫폼 기업 접근법 |

**벤치마크**: 코인베이스 (나스닥 상장)
- 2024년 기준 P/S 약 5\~10배
- 거래대금 대비 시가총액으로 국내 거래소 가치 추정 가능

### DeFi 프로토콜 밸류에이션

DeFi 프로토콜은 **토큰 시가총액**을 기준으로 평가합니다.

| 지표 | 산식 | 해석 |
|:-----|:-----|:-----|
| **TVL 대비 시총** | 시가총액 / TVL | 1 이하면 저평가 가능성 |
| **P/F (Price to Fees)** | 시가총액 / 연간 프로토콜 수수료 | 낮을수록 저평가 |
| **P/E (토큰)** | 시가총액 / 프로토콜 순이익 | 전통 P/E와 유사 |

## 6. 투자 시 고려사항

### 암호화폐 거래소 투자 체크리스트

- [ ] 규제 환경 변화 모니터링 (특금법, 과세 정책)
- [ ] 거래대금 추이 확인 (암호화폐 시장 사이클 연동)
- [ ] 시장점유율 변화 (경쟁 구도)
- [ ] 보안 사고 이력 및 대응 체계
- [ ] 주요 주주 및 지배구조 안정성
- [ ] 글로벌 확장 계획 (해외 거래소 진출)

### 산업 사이클

암호화폐 산업은 **4년 주기 반감기 사이클**과 연동되는 경향이 있습니다.

```
┌─────────────────────────────────────────────────────────────┐
│                    암호화폐 시장 사이클                        │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  상승장 (Bull Market)              하락장 (Bear Market)       │
│  ┌─────────────┐                  ┌─────────────┐            │
│  │ 거래량 폭증    │                  │ 거래량 급감    │            │
│  │ 신규 사용자 유입│                  │ 사용자 이탈    │            │
│  │ 거래소 실적 호조│                  │ 거래소 실적 악화│            │
│  │ 상장 수수료 증가│                  │ 구조조정       │            │
│  └─────────────┘                  └─────────────┘            │
│         ↑                                  ↓                 │
│         └───────────── 반감기 ─────────────┘                 │
│              (약 4년 주기, 비트코인 공급 감소)                  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
