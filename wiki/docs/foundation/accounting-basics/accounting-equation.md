# 회계등식의 의미

## 학습 목표

이 문서를 읽고 나면:
1. 회계등식(자산 = 부채 + 자본)의 의미를 직관적으로 이해할 수 있다
2. 모든 거래가 회계등식을 유지하는 원리를 설명할 수 있다
3. 회계등식을 통해 재무상태표를 읽을 수 있다

## 핵심 개념

### 가장 기본적인 등식

회계의 전체 체계는 하나의 간단한 등식에서 출발합니다.

**자산 = 부채 + 자본**

이 등식이 의미하는 바를 이해하면 재무상태표의 절반은 이해한 것입니다.

**직관적 이해**: 기업이 가진 것(자산)은 어딘가에서 왔습니다. 남에게 빌린 것(부채)이거나, 주주가 낸 것(자본)입니다. 기업이 보유한 자산의 총액은 그 자산을 마련한 원천(부채 + 자본)과 항상 같습니다.

**비유로 이해하기**: 집을 사는 상황을 생각해보십시오.

- 집(자산) 가격: 5억원
- 주택담보대출(부채): 3억원
- 본인 돈(자본): 2억원

5억(자산) = 3억(부채) + 2억(자본)

이 등식은 항상 성립합니다. 집값이 6억으로 오르면 자산이 6억이 되고, 부채는 그대로 3억이므로 자본(순자산)이 3억으로 늘어납니다.

### 회계등식의 확장

회계등식을 재배열하면 다른 통찰을 얻을 수 있습니다.

**자본 = 자산 - 부채**

자본은 자산에서 부채를 뺀 **순자산(Net Assets)**입니다. 기업을 청산할 때 모든 자산을 매각하고 모든 부채를 갚으면 남는 것이 주주의 몫, 즉 자본입니다.

**부채 = 자산 - 자본**

자산 중 자본으로 마련하지 않은 부분이 부채입니다. 자기 돈으로 마련하지 않은 자산은 결국 남에게 갚아야 합니다.

### 거래와 회계등식

모든 회계 거래는 회계등식의 균형을 유지합니다. 한쪽이 변하면 다른 쪽도 같이 변합니다.

**거래 유형별 회계등식 변화**:

| 거래 | 자산 | 부채 | 자본 | 등식 유지 |
|------|------|------|------|-----------|
| 현금 출자 | +100 | - | +100 | ✓ |
| 은행 차입 | +50 | +50 | - | ✓ |
| 현금으로 장비 구매 | ±0 (현금↓, 장비↑) | - | - | ✓ |
| 외상으로 재료 구매 | +30 | +30 | - | ✓ |
| 매출 발생 (현금) | +80 | - | +80 | ✓ |
| 급여 지급 | -40 | - | -40 | ✓ |
| 차입금 상환 | -50 | -50 | - | ✓ |
| 배당금 지급 | -20 | - | -20 | ✓ |

**주의**: 자본의 변동은 두 가지 경로가 있습니다.
1. **자본거래**: 유상증자, 자사주 매입, 배당 등 주주와의 직접 거래
2. **손익거래**: 수익과 비용으로 인한 이익잉여금 변동

매출이 발생하면 자산(현금 또는 매출채권)이 늘고, 자본 중 이익잉여금이 늘어납니다. 비용이 발생하면 자산이 줄고 이익잉여금이 줄어듭니다.

### 이익잉여금과 자본의 연결

손익계산서의 순이익은 재무상태표의 이익잉여금으로 연결됩니다.

**기말 이익잉여금 = 기초 이익잉여금 + 당기순이익 - 배당금**

이 관계를 통해 손익계산서와 재무상태표가 연결됩니다. 1년간 벌어들인 이익(순이익)이 쌓여서 자본(이익잉여금)이 됩니다.

### 회계등식으로 재무상태표 검증

회계등식을 이용하면 재무상태표의 논리적 일관성을 검증할 수 있습니다.

**검증 1: 기본 등식 확인**

재무상태표의 자산 총계와 (부채 + 자본) 총계가 일치하는지 확인합니다. 일치하지 않으면 오류가 있는 것입니다.

**검증 2: 이익잉여금 변동 확인**

(전기 이익잉여금) + (당기순이익) - (배당금) = (당기 이익잉여금)

이 관계가 성립하는지 확인합니다. 차이가 있다면 회계정책 변경, 전기오류수정, 기타포괄손익 재분류 등을 확인합니다.

**검증 3: 현금 변동 확인**

(전기말 현금) + (현금흐름표 순현금흐름) = (당기말 현금)

현금흐름표의 현금 변동이 재무상태표의 현금 변동과 일치하는지 확인합니다.

## 실무 포인트

> **회계등식으로 빠르게 상황 파악하기**
>
> 재무상태표를 처음 볼 때, 회계등식의 구성비를 먼저 확인하십시오.
>
> - 자산 중 유동자산 vs 비유동자산 비율은?
> - 자본조달 중 부채 vs 자본 비율은? (부채비율)
> - 자본 중 자본금 vs 이익잉여금 비율은?
>
> 이 비율만으로도 기업의 대략적인 특성을 파악할 수 있습니다.
>
> - 이익잉여금이 크다 → 오래 벌어온 기업, 배당 여력
> - 부채 비중이 높다 → 레버리지 활용, 재무 위험
> - 유동자산 비중이 높다 → 유동성 양호, 또는 비효율적 자산 운용

## 한국 기업 특성

한국 기업의 재무상태표에서 주의 깊게 볼 항목들:

**자본잉여금과 이익잉여금**: 한국 기업은 유상증자, 합병, 주식발행초과금 등으로 자본잉여금이 큰 경우가 많습니다. 자본잉여금과 이익잉여금의 구성을 확인하십시오.

**기타포괄손익누계액**: 해외 종속기업이 있는 기업은 해외사업환산손익이, 금융자산을 많이 보유한 기업은 금융자산평가손익이 기타포괄손익누계액에 큽니다.

**자기주식**: 한국 기업 중 자사주 매입을 활발히 하는 기업은 자기주식이 자본에서 차감됩니다. 대규모 자사주는 주주환원 의지를 보여주지만, 소각 여부도 확인해야 합니다.

**비지배지분**: 연결재무제표에서 100% 소유하지 않은 종속기업이 있으면 비지배지분이 자본에 포함됩니다. 지배기업 주주 귀속 자본과 비지배지분을 구분해서 봐야 합니다.

## 계산 예시

**삼성전자 2023년 재무상태표 요약** (가상의 단순화된 예시):

| 구분 | 금액 (조원) |
|------|-------------|
| **자산** | |
| 유동자산 | 200 |
| 비유동자산 | 200 |
| **자산 총계** | **400** |
| **부채** | |
| 유동부채 | 80 |
| 비유동부채 | 40 |
| **부채 총계** | **120** |
| **자본** | |
| 자본금 | 1 |
| 자본잉여금 | 4 |
| 이익잉여금 | 270 |
| 기타 | 5 |
| **자본 총계** | **280** |

**회계등식 확인**: 자산 400 = 부채 120 + 자본 280 ✓

**해석**:
- 부채비율: 120/280 = 43% (안정적)
- 이익잉여금이 자본의 대부분: 오랜 기간 이익 축적
- 유동자산이 풍부: 유동비율 200/80 = 250% (우수)
