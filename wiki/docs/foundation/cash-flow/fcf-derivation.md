# FCF 도출 (밸류에이션 연결점)

## 🎯 학습 목표

이 문서를 읽고 나면:
1. FCFF와 FCFE의 차이를 이해하고 계산할 수 있다
2. 재무제표에서 FCF를 도출하는 방법을 알 수 있다
3. FCF가 DCF 밸류에이션에 어떻게 연결되는지 파악할 수 있다

## 📚 핵심 개념

### 🟢 입문: 잉여현금흐름이란

잉여현금흐름(Free Cash Flow, FCF)은 기업이 영업활동에서 창출한 현금에서 **사업 유지와 성장에 필요한 투자**를 차감한 후 남는 현금입니다. 채권자와 주주에게 **분배 가능한 현금**입니다.

**FCF의 중요성**:
- 기업가치의 본질적 원천
- DCF 밸류에이션의 핵심 Input
- 배당, 부채상환, 자사주 매입의 재원
- 경영진의 자본배분 능력 평가

### FCFF vs FCFE

**FCFF (Free Cash Flow to Firm, 기업잉여현금흐름)**:

모든 자본제공자(채권자 + 주주)에게 분배 가능한 현금입니다. 이자 지급 전의 현금흐름입니다.

**FCFE (Free Cash Flow to Equity, 주주잉여현금흐름)**:

주주에게만 분배 가능한 현금입니다. 이자 지급과 부채 변동 후의 현금흐름입니다.

**관계**:

FCFE = FCFF - 이자비용 × (1 - 세율) + 순차입금 증가

### 🟡 중급: FCFF 계산 방법

FCFF를 계산하는 여러 가지 방법이 있습니다.

**방법 1: EBIT에서 출발**

| 단계 | 항목 |
|------|------|
| | EBIT (영업이익) |
| (-) | 세금 (EBIT × 세율) |
| = | NOPAT (세후영업이익) |
| (+) | 감가상각비 |
| (-) | CAPEX (설비투자) |
| (-) | 순운전자본 증가 |
| = | **FCFF** |

**방법 2: 순이익에서 출발**

| 단계 | 항목 |
|------|------|
| | 순이익 |
| (+) | 이자비용 × (1 - 세율) |
| (+) | 감가상각비 |
| (-) | CAPEX |
| (-) | 순운전자본 증가 |
| = | **FCFF** |

**방법 3: 영업CF에서 출발**

| 단계 | 항목 |
|------|------|
| | 영업활동현금흐름 |
| (+) | 이자비용 × (1 - 세율) |
| (-) | CAPEX |
| = | **FCFF** |

### FCFE 계산 방법

**방법 1: FCFF에서 출발**

| 단계 | 항목 |
|------|------|
| | FCFF |
| (-) | 이자비용 × (1 - 세율) |
| (+) | 순차입금 증가 (차입 - 상환) |
| = | **FCFE** |

**방법 2: 순이익에서 출발**

| 단계 | 항목 |
|------|------|
| | 순이익 |
| (+) | 감가상각비 |
| (-) | CAPEX |
| (-) | 순운전자본 증가 |
| (+) | 순차입금 증가 |
| = | **FCFE** |

### 순운전자본 (Net Working Capital) 계산

순운전자본 = 영업자산 - 영업부채

**영업 운전자본**:

| 영업자산 | 영업부채 |
|----------|----------|
| 매출채권 | 매입채무 |
| 재고자산 | 미지급비용(영업) |
| 선급금 | 선수금 |

**순운전자본 증가 = 당기 NWC - 전기 NWC**

순운전자본이 증가하면 현금이 묶이므로 FCF에서 차감합니다.

### 🔴 심화: DCF 밸류에이션 연결

**FCFF와 기업가치**

기업가치(Enterprise Value) = 미래 FCFF의 현재가치

EV = Σ(FCFFt / (1 + WACC)^t) + TV / (1 + WACC)^n

TV(Terminal Value) = FCFFn+1 / (WACC - g)

**주주가치**: 주주가치 = 기업가치 - 순부채

**FCFE와 주주가치**

주주가치(Equity Value) = 미래 FCFE의 현재가치

Equity Value = Σ(FCFEt / (1 + Ke)^t) + TV / (1 + Ke)^n

Ke = 자기자본비용

**언제 어떤 방법을 쓰는가?**

| 방법 | 할인율 | 적합한 상황 |
|------|--------|-------------|
| FCFF-WACC | WACC | 자본구조 변동 시 |
| FCFE-Ke | 자기자본비용 | 자본구조 안정 시 |

일반적으로 FCFF-WACC 방법이 더 많이 사용됩니다.

### 재무제표에서 Input 찾기

**FCFF 계산에 필요한 Input과 출처**:

| Input | 출처 | 확인 위치 |
|-------|------|-----------|
| EBIT | 손익계산서 | 영업이익 |
| 세율 | 손익계산서 주석 | 유효세율 계산 |
| 감가상각비 | 현금흐름표 | 영업CF 조정항목 |
| CAPEX | 현금흐름표 | 투자CF - 유형자산 취득 |
| NWC 변동 | 현금흐름표 | 영업CF 운전자본 변동 |

**실제 계산 예시**:

손익계산서:
- 영업이익(EBIT): 1,000억원
- 법인세비용: 200억원 (유효세율 20%)

현금흐름표:
- 감가상각비: 300억원
- 유형자산 취득: 500억원
- 운전자본 변동: -100억원 (증가)

**FCFF 계산**:
| 항목 | 금액 |
|------|------|
| EBIT | 1,000 |
| (-) 세금 (EBIT × 20%) | (200) |
| = NOPAT | 800 |
| (+) 감가상각비 | 300 |
| (-) CAPEX | (500) |
| (-) NWC 증가 | (100) |
| = **FCFF** | **500** |

## 💡 실무 포인트

> **FCF 계산 체크리스트**
>
> 1. **EBIT 확인**: 영업이익, 일회성 제외 여부
> 2. **세율**: 유효세율 vs 법정세율
> 3. **감가상각비**: 유형+무형+사용권자산 포함
> 4. **CAPEX**: 유지 CAPEX vs 성장 CAPEX 구분
> 5. **NWC**: 영업 관련 항목만 포함
> 6. **일관성**: 예측 기간 동안 가정의 일관성

## 📊 한국 기업 특성

**감가상각비 확인**: 한국 기업의 현금흐름표에서 감가상각비는 영업CF 조정항목에 표시됩니다. 유형자산 + 무형자산 + 사용권자산 감가상각비를 모두 확인합니다.

**CAPEX 파악**: 현금흐름표의 투자CF에서 유형자산 취득, 무형자산 취득을 확인합니다. 유지보수 vs 성장 투자 구분이 중요하며, 주석의 경영진 설명을 참고합니다.

**운전자본 특성**: B2B 거래가 많은 한국 기업은 매출채권과 매입채무 변동이 큽니다. 어음 결제 관행으로 운전자본 부담이 클 수 있습니다.

## 🧮 분석 예시

**P기업 FCFF 도출** (단위: 억원):

**손익계산서 요약**:
| 항목 | 2024 |
|------|------|
| 매출액 | 10,000 |
| 영업이익 | 1,500 |
| 이자비용 | 200 |
| 법인세비용 | 280 |
| 당기순이익 | 1,020 |

**현금흐름표 요약**:
| 항목 | 2024 |
|------|------|
| 영업활동현금흐름 | 1,800 |
| - 감가상각비(가산) | 500 |
| - 운전자본 변동 | (200) |
| 유형자산 취득 | (800) |

**유효세율 계산**:
세전이익 = 1,500 - 200 = 1,300
유효세율 = 280 / 1,300 = 21.5%

**FCFF 계산 (방법 1)**:
| 항목 | 금액 |
|------|------|
| EBIT | 1,500 |
| (-) EBIT × 21.5% | (323) |
| = NOPAT | 1,177 |
| (+) 감가상각비 | 500 |
| (-) CAPEX | (800) |
| (-) NWC 증가 | (200) |
| = **FCFF** | **677** |

**검증 (방법 3)**:
| 항목 | 금액 |
|------|------|
| 영업CF | 1,800 |
| (+) 이자비용 × (1-21.5%) | 157 |
| (-) CAPEX | (800) |
| = FCFF | 1,157 |

차이 발생 시 조정 항목 확인 필요

**FCFE 계산**:
| 항목 | 금액 |
|------|------|
| FCFF | 677 |
| (-) 이자비용 × (1-세율) | (157) |
| (+) 순차입금 증가 | 300 |
| = **FCFE** | **820** |
