# 현금흐름표의 중요성

## 학습 목표

이 문서를 읽고 나면:
1. 현금흐름표의 세 가지 활동 구분을 이해할 수 있다
2. 손익계산서의 이익과 현금흐름의 차이를 파악할 수 있다
3. 현금흐름 분석의 투자적 의미를 알 수 있다

## 핵심 개념

### 현금흐름표란

현금흐름표(Statement of Cash Flows)는 **일정 기간** 동안 현금이 어디서 들어오고 어디로 나갔는지를 보여주는 재무제표입니다.

**왜 현금흐름표가 중요한가?**

손익계산서의 이익은 **발생주의**로 작성되어 실제 현금 흐름과 다릅니다. 이익을 내도 현금이 없으면 망할 수 있습니다 (흑자 도산). 현금흐름표는 기업의 **실제 현금 창출 능력**을 보여줍니다.

"**Cash is King**" - 기업가치의 본질은 현금 창출 능력입니다. DCF 밸류에이션도 미래 현금흐름을 기반으로 합니다.

### 현금흐름표의 구조

현금흐름표는 세 가지 활동으로 구분됩니다.

| 활동 구분 | 내용 | 예시 |
|-----------|------|------|
| 영업활동 | 주된 수익창출 활동 | 제품 판매, 원재료 구입, 급여 지급 |
| 투자활동 | 장기자산 취득/처분 | 설비 투자, 자회사 인수, 투자자산 매각 |
| 재무활동 | 자금조달/상환 활동 | 차입금 조달/상환, 유상증자, 배당금 지급 |

**현금흐름표 기본 등식**:

**기말 현금 = 기초 현금 + 영업CF + 투자CF + 재무CF**

### 영업활동 현금흐름 (Operating Cash Flow, OCF)

영업활동 현금흐름은 기업의 **핵심 영업에서 창출된 현금**입니다. 가장 중요한 현금흐름입니다.

**간접법 계산** (대부분의 기업이 사용):

| 항목 | 조정 |
|------|------|
| 당기순이익 | 출발점 |
| (+) 감가상각비 | 비현금 비용 가산 |
| (+) 무형자산상각비 | 비현금 비용 가산 |
| (+/-) 운전자본 변동 | 현금 영향 조정 |
| (+/-) 기타 비현금 항목 | 조정 |
| = **영업활동현금흐름** | |

**운전자본 변동의 영향**:

| 항목 | 증가 | 감소 |
|------|------|------|
| 매출채권 | 현금 유출 (-) | 현금 유입 (+) |
| 재고자산 | 현금 유출 (-) | 현금 유입 (+) |
| 매입채무 | 현금 유입 (+) | 현금 유출 (-) |

### 투자활동 현금흐름 (Investing Cash Flow)

투자활동 현금흐름은 **장기자산의 취득과 처분**에 관련된 현금 흐름입니다.

**주요 항목**:
- 유형자산 취득 (CAPEX) → 현금 유출
- 유형자산 처분 → 현금 유입
- 무형자산 취득/처분
- 자회사/관계기업 투자 (M&A)
- 금융자산 취득/처분

**해석**: 투자CF가 음수(-)인 것이 일반적입니다. 성장을 위해 설비투자를 하기 때문입니다. 투자CF가 양수(+)라면 자산을 매각하고 있다는 의미입니다.

### 재무활동 현금흐름 (Financing Cash Flow)

재무활동 현금흐름은 **자금 조달과 상환**에 관련된 현금 흐름입니다.

**주요 항목**:

| 현금 유입 | 현금 유출 |
|-----------|-----------|
| 차입금 조달 | 차입금 상환 |
| 사채 발행 | 사채 상환 |
| 유상증자 | 자사주 매입 |
| | 배당금 지급 |

### 잉여현금흐름 (FCF)

**FCF (Free Cash Flow, 잉여현금흐름)**는 영업활동에서 창출한 현금에서 필수 투자(CAPEX)를 차감한 금액입니다. 채권자와 주주에게 분배 가능한 현금입니다.

**FCF = 영업활동현금흐름 - CAPEX**

또는 더 정확하게:

**FCFF (Free Cash Flow to Firm)** = EBIT × (1-세율) + 감가상각비 - CAPEX - 운전자본 증가

**FCFE (Free Cash Flow to Equity)** = FCFF - 이자비용 × (1-세율) + 순차입금 증가

**FCF의 중요성**:
- DCF 밸류에이션의 핵심 Input
- 실제 분배 가능한 현금
- 배당, 부채 상환, 자사주 매입의 재원

### 현금흐름 패턴 분석

세 가지 현금흐름의 조합으로 기업의 상황을 파악할 수 있습니다.

| 영업CF | 투자CF | 재무CF | 상황 |
|--------|--------|--------|------|
| + | - | - | 성숙기업: 영업으로 투자+상환 |
| + | - | + | 성장기업: 영업+차입으로 투자 확대 |
| + | + | - | 구조조정: 자산 매각으로 부채 상환 |
| - | - | + | 위험: 차입으로 적자+투자 충당 |
| - | + | + | 위기: 자산매각+차입으로 연명 |
| - | + | - | 쇠퇴: 투자 철수, 배당 지속 |

### 영업CF vs 순이익

영업CF와 순이익의 괴리를 분석하면 **이익의 질**을 평가할 수 있습니다.

**영업CF > 순이익 (건전)**:
- 비현금 비용(감가상각) 큼
- 운전자본 효율적 관리
- 이익의 질 양호

**영업CF < 순이익 (주의)**:
- 매출채권 급증 (회수 지연)
- 재고자산 증가 (판매 부진)
- 발생액 이익, 현금 없음
- 이익의 질 의문

**지속적인 괴리는 위험 신호**:
- 매년 영업CF < 순이익이면 이익 조작 가능성
- 발생액 비율 분석 필요

## 한국 기업 특성

**간접법 사용**: 한국 기업 대부분은 간접법으로 영업CF를 작성합니다. 순이익에서 시작하여 비현금 항목과 운전자본 변동을 조정합니다.

**운전자본 변동**: 한국 기업은 B2B 거래가 많아 매출채권/매입채무 변동이 영업CF에 큰 영향을 줍니다. 어음 결제 관행으로 운전자본 부담이 클 수 있습니다.

**대규모 CAPEX**: 반도체, 자동차, 조선 등 한국 주력 산업은 설비투자가 큽니다. 투자CF가 크게 음수인 것이 일반적입니다.

**리스부채 상환**: K-IFRS 16 적용 후 리스료 지급이 재무CF로 분류됩니다. 리스 비중이 높은 기업은 재무CF가 크게 음수일 수 있습니다.

## 분석 예시

**O기업 현금흐름 분석** (단위: 억원):

| 항목 | 2022 | 2023 | 2024 |
|------|------|------|------|
| **당기순이익** | 1,000 | 1,200 | 1,500 |
| 감가상각비 | 500 | 600 | 700 |
| 운전자본 변동 | (200) | (400) | (600) |
| 기타 | 100 | 100 | 100 |
| **영업CF** | **1,400** | **1,500** | **1,700** |
| CAPEX | (800) | (1,200) | (1,500) |
| 기타투자 | (100) | (200) | (300) |
| **투자CF** | **(900)** | **(1,400)** | **(1,800)** |
| 차입금 증감 | 200 | 500 | 800 |
| 배당금 | (300) | (350) | (400) |
| **재무CF** | **(100)** | **150** | **400** |
| **현금 증감** | **400** | **250** | **300** |

**FCF 계산**:
| 연도 | 영업CF | CAPEX | FCF |
|------|--------|-------|-----|
| 2022 | 1,400 | 800 | 600 |
| 2023 | 1,500 | 1,200 | 300 |
| 2024 | 1,700 | 1,500 | 200 |

**분석**:
- 순이익 50% 성장 (1,000→1,500)
- 영업CF도 성장하지만 운전자본 부담 증가
- CAPEX 급증: 성장 투자 확대
- FCF 감소 추세: 투자 확대로 잉여현금 감소
- 차입금 증가로 투자 충당
- 성장기 패턴 (영업+, 투자-, 재무+)
