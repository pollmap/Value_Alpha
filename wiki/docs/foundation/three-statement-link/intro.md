# 3표 연결의 원리

## 학습 목표

이 문서를 읽고 나면:
1. 재무제표 3표가 어떻게 연결되는지 이해할 수 있다
2. 한 표의 변화가 다른 표에 미치는 영향을 추적할 수 있다
3. 재무모델링의 기초 원리를 파악할 수 있다

## 핵심 개념

### 3표 연결이란

재무제표 3표(재무상태표, 손익계산서, 현금흐름표)는 서로 독립된 것이 아니라 **긴밀하게 연결**되어 있습니다. 한 표의 숫자가 변하면 다른 표의 숫자도 변합니다.

**세 가지 주요 연결**:

| 연결 | 관계 |
|------|------|
| 손익계산서 → 재무상태표 | 순이익이 이익잉여금으로 |
| 손익계산서 → 현금흐름표 | 순이익이 영업CF의 출발점 |
| 현금흐름표 → 재무상태표 | 현금 증감이 현금자산 변동 |

### 손익계산서 → 재무상태표 연결

손익계산서의 **당기순이익**은 재무상태표의 **이익잉여금**으로 들어갑니다.

**기말 이익잉여금 = 기초 이익잉여금 + 당기순이익 - 배당금**

1년간 벌어들인 이익이 쌓여서 자본이 됩니다. 적자가 나면 이익잉여금이 감소합니다.

**예시**:
| 항목 | 금액 |
|------|------|
| 기초 이익잉여금 | 1,000 |
| 당기순이익 | 300 |
| 배당금 | (100) |
| 기말 이익잉여금 | 1,200 |

### 손익계산서 → 현금흐름표 연결

손익계산서의 **당기순이익**은 현금흐름표 **영업활동현금흐름**의 출발점입니다 (간접법).

| 항목 | 조정 |
|------|------|
| 당기순이익 | 출발 |
| (+) 감가상각비 | 비현금 비용 |
| (+/-) 운전자본 변동 | 현금 영향 |
| (+/-) 기타 | 기타 조정 |
| = 영업활동현금흐름 | |

순이익과 영업CF의 차이가 **발생액(Accruals)**입니다.

### 현금흐름표 → 재무상태표 연결

현금흐름표의 **순현금흐름**은 재무상태표의 **현금** 변동과 일치합니다.

**기말 현금 = 기초 현금 + 영업CF + 투자CF + 재무CF**

**예시**:
| 항목 | 금액 |
|------|------|
| 기초 현금 | 500 |
| 영업CF | 300 |
| 투자CF | (200) |
| 재무CF | (50) |
| 기말 현금 | 550 |

### 상세 연결 관계

**자산 항목별 연결**:

| 자산 | 손익계산서 영향 | 현금흐름표 영향 |
|------|-----------------|-----------------|
| 현금 | - | 최종 결과 |
| 매출채권 | 매출 인식 | 영업CF 조정 |
| 재고자산 | 매출원가 | 영업CF 조정 |
| 유형자산 | 감가상각비 | 투자CF (CAPEX) |
| 무형자산 | 상각비 | 투자CF |

**부채 항목별 연결**:

| 부채 | 손익계산서 영향 | 현금흐름표 영향 |
|------|-----------------|-----------------|
| 매입채무 | 매입 (원가) | 영업CF 조정 |
| 차입금 | 이자비용 | 재무CF |
| 사채 | 이자비용 | 재무CF |
| 미지급비용 | 비용 발생 | 영업CF 조정 |

**자본 항목별 연결**:

| 자본 | 손익계산서 영향 | 현금흐름표 영향 |
|------|-----------------|-----------------|
| 자본금 | - | 재무CF (증자) |
| 이익잉여금 | 순이익 | - |
| 자기주식 | - | 재무CF (매입/처분) |

### 거래별 3표 영향

**1. 외상 매출 발생 (100원)**

| 재무제표 | 영향 |
|----------|------|
| 손익계산서 | 매출 +100 |
| 재무상태표 | 매출채권 +100, 이익잉여금 +100(세전) |
| 현금흐름표 | 순이익에서 시작, 매출채권 증가로 차감 |

**2. 감가상각비 인식 (50원)**

| 재무제표 | 영향 |
|----------|------|
| 손익계산서 | 감가상각비 +50, 이익 -50 |
| 재무상태표 | 유형자산 -50, 이익잉여금 -50 |
| 현금흐름표 | 순이익 감소, 감가상각비 가산 → 영업CF 불변 |

**3. 설비투자 (CAPEX 200원, 현금)**

| 재무제표 | 영향 |
|----------|------|
| 손익계산서 | 즉시 영향 없음 (향후 감가상각) |
| 재무상태표 | 현금 -200, 유형자산 +200 |
| 현금흐름표 | 투자CF -200 |

**4. 배당금 지급 (30원)**

| 재무제표 | 영향 |
|----------|------|
| 손익계산서 | 영향 없음 |
| 재무상태표 | 현금 -30, 이익잉여금 -30 |
| 현금흐름표 | 재무CF -30 |

### 재무모델링 기초

**3표 연결 모델**

재무모델링에서는 3표를 연결하여 일관된 예측을 수행합니다.

**모델링 순서**:
1. 손익계산서 예측 (매출 → 비용 → 이익)
2. 재무상태표 예측 (운전자본, 고정자산)
3. 현금흐름표 도출 (영업CF, 투자CF)
4. 재무활동 결정 (차입/상환, 배당)
5. 재무상태표 완성 (현금, 부채)

**순환참조 (Circular Reference)**

이자비용은 부채 잔액에 의존하고, 부채 잔액은 현금 부족분(현금흐름)에 의존합니다. 현금흐름은 이자비용에 영향을 받습니다.

이자비용 → 순이익 → 현금흐름 → 현금 잔액 → 차입 필요액 → 부채 잔액 → 이자비용

이 순환을 처리하는 것이 재무모델링의 핵심 기술입니다.

## 실무 포인트

> **3표 연결 검증 체크리스트**
>
> 1. **이익잉여금 검증**: 기초 + 순이익 - 배당 = 기말
> 2. **현금 검증**: 기초 + CF합계 = 기말
> 3. **회계등식**: 자산 = 부채 + 자본 성립
> 4. **영업CF 시작점**: 순이익에서 출발
> 5. **일관성**: 각 표의 숫자가 상호 일치

## 한국 기업 특성

**연결재무제표 연결**: 한국 상장기업의 주재무제표는 연결재무제표입니다. 종속기업까지 포함한 3표가 연결됩니다.

**별도재무제표와의 차이**: 지주회사나 계열사 분석 시 별도재무제표도 확인합니다. 연결과 별도의 이익잉여금 차이는 종속기업 실적의 영향입니다.

**자본변동표**: K-IFRS에서 자본변동표는 필수 재무제표입니다. 이익잉여금 외에 자본의 모든 변동을 보여줍니다.

## 종합 예시

**Q기업 3표 연결 검증** (단위: 억원):

**손익계산서**:
| 항목 | 금액 |
|------|------|
| 매출 | 10,000 |
| 영업이익 | 1,500 |
| 당기순이익 | 1,000 |

**현금흐름표**:
| 항목 | 금액 |
|------|------|
| 영업CF | 1,400 |
| 투자CF | (800) |
| 재무CF | (400) |
| 현금 증감 | 200 |

**재무상태표 (기초 vs 기말)**:
| 항목 | 기초 | 기말 | 변동 |
|------|------|------|------|
| 현금 | 500 | 700 | +200 |
| 자산 합계 | 8,000 | 8,800 | +800 |
| 부채 합계 | 3,000 | 3,000 | 0 |
| 자본금 | 1,000 | 1,000 | 0 |
| 이익잉여금 | 4,000 | 4,800 | +800 |
| 자본 합계 | 5,000 | 5,800 | +800 |

**검증**:
1. 현금 변동: 500 + 200 = 700 ✓
2. 이익잉여금: 4,000 + 1,000 - 200(배당) = 4,800 ✓
3. 회계등식: 8,800 = 3,000 + 5,800 ✓
4. 재무CF -400 = 배당 -200 + 기타 -200
