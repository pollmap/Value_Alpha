# 자기주식 (한국 기업의 자사주 매입 특성)

## 학습 목표

이 문서를 읽고 나면:
1. 자기주식의 취득과 처분, 소각의 회계처리를 이해할 수 있다
2. 자사주 매입의 목적과 효과를 파악할 수 있다
3. 한국 기업의 자사주 매입 특성을 분석할 수 있다

## 핵심 개념

### 자기주식이란

자기주식(Treasury Stock)은 기업이 자사 주식을 매입하여 보유하는 것입니다. 재무상태표에서 **자본의 차감 항목**으로 표시됩니다.

**자기주식의 특성**:
- 의결권 없음
- 배당권 없음
- 발행주식수에는 포함, 유통주식수에서는 제외
- 자본의 마이너스 항목

**재무상태표 표시**:

| 항목 | 금액 |
|------|------|
| 자본금 | 1,000 |
| 자본잉여금 | 2,000 |
| 이익잉여금 | 5,000 |
| 기타포괄손익누계액 | 200 |
| **자기주식** | **(500)** |
| **자본 합계** | **7,700** |

### 자기주식 취득 목적

**주주환원**: 배당과 함께 주주에게 가치를 돌려주는 방법입니다. 매입 후 소각하면 주당가치가 상승합니다.

**주가 안정**: 주가가 저평가되었다고 판단할 때 매입합니다. 매수 수요 창출로 주가 하락을 방어합니다.

**임직원 보상**: 스톡옵션, 성과주 지급 재원으로 활용합니다. 신주 발행 대신 자사주로 지급합니다.

**M&A 대비**: 합병, 인수 시 대가로 자사주를 활용합니다.

**경영권 방어**: 적대적 M&A 방어 목적으로 매입합니다. 우호 세력에 처분하여 지분 확보합니다.

### 자기주식 회계처리

**취득 시**:
- 자기주식(자본 차감) 증가
- 현금 감소
- 자본 총액 감소

**처분 시 (취득가 < 처분가)**:
- 자기주식 감소
- 현금 증가
- 차익은 자기주식처분이익(자본잉여금)

**처분 시 (취득가 > 처분가)**:
- 자기주식 감소
- 현금 증가
- 차손은 자기주식처분손실(자본잉여금) 또는 이익잉여금 차감

**소각 시**:
- 자기주식 감소
- 자본금 감소 (액면금액) 또는
- 이익잉여금 감소 (취득가 - 액면가)

### 자사주 매입의 효과

**EPS 증가 효과**

자사주를 매입하면 유통주식수가 감소하여 EPS가 증가합니다.

**예시**:
- 당기순이익: 1,000억원
- 발행주식수: 1억주
- 자기주식: 1,000만주
- 유통주식수: 9,000만주

EPS 계산:
- 발행주식 기준: 1,000억 ÷ 1억 = 1,000원
- 유통주식 기준: 1,000억 ÷ 9,000만 = 1,111원 (+11%)

**자사주 매입 vs 배당 비교**:

| 항목 | 자사주 매입 | 현금배당 |
|------|-------------|----------|
| 주당가치 | 증가 (소각 시) | 중립 |
| 주가 영향 | 상승 신호 | 중립~긍정 |
| 세금 | 양도세 (처분 시) | 배당소득세 |
| 유연성 | 시기 선택 가능 | 정기 지급 |
| 신호 효과 | 저평가 인식 | 이익 지속 |

**소각의 중요성**

자사주 매입만으로는 완전한 주주환원이 아닙니다. 소각해야 발행주식수가 영구 감소합니다.

| 상황 | 발행주식수 | 효과 |
|------|-----------|------|
| 매입만 | 불변 | 일시적 |
| 매입 후 소각 | 감소 | 영구적 |
| 매입 후 임직원 지급 | 불변 | 희석 방지 |
| 매입 후 재매도 | 불변 | 효과 없음 |

### 자사주 매입의 함정

**자사주 매입이 항상 좋은 것은 아닙니다**

| 상황 | 문제점 |
|------|--------|
| 고가 매입 | 저평가 아닌 상황에서 매입 → 가치 파괴 |
| 소각 없이 보유만 | 언제든 재매도 가능 → 희석 위험 |
| 부채로 매입 | 재무위험 증가 |
| 대주주 지분율 유지 목적 | 소수주주 이익과 불일치 |
| 투자 기회 포기 | 성장 투자보다 자사주 → 장기 성장 저하 |

**자사주 매입 적정성 평가**

| 평가 항목 | 체크 포인트 |
|-----------|-------------|
| 주가 수준 | PBR, PER 등 밸류에이션 저평가 여부 |
| 현금 여력 | 투자, 배당 후 잉여 현금으로 매입 |
| 소각 계획 | 매입 후 소각 여부 공시 |
| ROE 영향 | 자본 감소로 ROE 개선 vs 실질 개선 |
| 대주주 의도 | 지배력 유지 목적인지 확인 |

## 실무 포인트

> **자사주 매입 분석 체크리스트**
>
> 1. **매입 규모**: 총 매입액, 발행주식 대비 비율
> 2. **매입 가격**: 평균 매입 단가 vs 현재 주가
> 3. **소각 여부**: 소각 계획 또는 이력 확인
> 4. **자금 원천**: 잉여 현금 vs 차입
> 5. **시기와 패턴**: 주가 하락 시 매입, 상승 시 중단?
> 6. **대주주 거래**: 대주주 지분 매입, 처분과 연계?

## 한국 기업 특성

**재벌 그룹의 자사주 활용**

한국 재벌 그룹은 자사주를 경영권 유지에 활용해왔습니다.

- 자사주 보유 후 우호세력 처분
- 순환출자 규제 회피 수단
- 지주회사 전환 시 활용

2020년 상법 개정으로 자사주 의결권 제한이 강화되었지만, 여전히 경영권 방어 목적의 자사주 매입이 있습니다.

**자사주 소각 확대**

최근 주주행동주의(행동주의 펀드) 압력으로 자사주 소각이 증가하고 있습니다. 삼성전자, 현대차, SK하이닉스 등이 대규모 소각을 발표했습니다.

**공시 규정**

자사주 매입/처분/소각은 공시 의무가 있습니다.
- 취득 결정: 즉시 공시
- 취득 결과: 매월 공시
- 처분/소각: 즉시 공시

DART에서 "자기주식" 검색으로 확인 가능합니다.

## 분석 예시

**L기업 자사주 분석**:

**현황**:
| 항목 | 내용 |
|------|------|
| 발행주식수 | 1억주 |
| 자기주식 | 1,500만주 (15%) |
| 유통주식수 | 8,500만주 |
| 자기주식 취득 원가 | 7,500억원 |
| 주당 평균 취득 단가 | 50,000원 |
| 현재 주가 | 60,000원 |

**최근 3년 자사주 활동**:
| 연도 | 취득 | 소각 | 임직원 지급 | 기말 보유 |
|------|------|------|-------------|----------|
| 2022 | 500만주 | 0 | 100만주 | 1,200만주 |
| 2023 | 400만주 | 0 | 100만주 | 1,500만주 |
| 2024 | 300만주 | 500만주 | 100만주 | 1,200만주 |

**분석**:
- 2024년 500만주 소각 발표: 긍정적 주주환원
- 소각 후 발행주식수: 9,500만주로 감소
- EPS 향상 효과: 약 5%
- 취득 단가 5만원 < 현재가 6만원: 적절한 타이밍
- 지속적 매입 + 소각 선순환 기대

**소각 효과 시뮬레이션**:
| 항목 | 소각 전 | 500만주 소각 후 |
|------|--------|----------------|
| 발행주식수 | 1억주 | 9,500만주 |
| 당기순이익 | 5,000억 | 5,000억 |
| EPS | 5,000원 | 5,263원 |
| BPS | 40,000원 | 42,105원 |
