# 자본의 구성요소

## 학습 목표

이 문서를 읽고 나면:
1. 자본의 각 구성요소를 이해할 수 있다
2. 자본금과 이익잉여금의 차이를 파악할 수 있다
3. 기타포괄손익누계액의 의미를 알 수 있다

## 핵심 개념

### 자본이란

자본(Equity)은 자산에서 부채를 차감한 **잔여지분**입니다. 주주에게 귀속되는 순자산이며, 순자산(Net Assets)이라고도 합니다.

**회계등식**: 자본 = 자산 - 부채

자본은 기업 청산 시 모든 부채를 상환한 후 주주에게 돌아가는 몫입니다.

**자본의 구성**:

| 항목 | 내용 | 원천 |
|------|------|------|
| 자본금 | 주식 발행 시 액면가 총액 | 주주 출자 |
| 자본잉여금 | 주식발행초과금, 기타자본잉여금 | 주주 출자 |
| 이익잉여금 | 누적 이익 - 배당 | 영업활동 |
| 기타포괄손익누계액 | 당기손익 외 자본 변동 | 공정가치 변동 등 |
| 기타자본항목 | 자기주식 등 | 자본거래 |

### 자본금 (Share Capital)

자본금은 발행 주식의 **액면가 총액**입니다.

**계산**: 자본금 = 발행주식수 × 주당 액면가

**예시**:
- 발행주식수: 1,000만주
- 액면가: 500원
- 자본금: 50억원

**특징**:
- 회사 설립이나 유상증자 시 증가
- 감자(자본금 감소) 시 감소
- 액면가는 보통 500원, 1,000원, 5,000원 등
- 실제 발행가격(시장가)과 액면가는 다름

### 자본잉여금 (Capital Surplus)

자본잉여금은 주주와의 거래에서 발생한 잉여금입니다. 영업활동에서 발생한 것이 아니라 자본거래에서 발생합니다.

**주요 항목**:

| 항목 | 발생 원인 |
|------|-----------|
| 주식발행초과금 | 주식을 액면 초과 금액으로 발행 |
| 감자차익 | 감자 시 주주 환급액 < 감소 자본금 |
| 자기주식처분이익 | 자사주 매입가 < 처분가 |
| 기타자본잉여금 | 합병차익 등 |

**주식발행초과금 예시**:
- 액면가 500원 주식을 10,000원에 발행 (1,000만주)
- 자본금: 500 × 1,000만 = 50억원
- 주식발행초과금: (10,000 - 500) × 1,000만 = 950억원
- 총 조달액: 1,000억원

### 이익잉여금 (Retained Earnings)

이익잉여금은 **영업활동으로 벌어들인 이익의 누적**에서 배당 등으로 사외 유출된 금액을 차감한 것입니다.

**계산**: 기말 이익잉여금 = 기초 이익잉여금 + 당기순이익 - 배당금 - 기타 변동

**이익잉여금의 의미**:
- 과거 수익성의 누적 결과
- 배당 여력의 척도
- 재투자 자금의 원천
- 장기적 성장의 밑천

**이익잉여금 구성**:

| 항목 | 내용 |
|------|------|
| 법정적립금 | 상법상 의무 적립 (자본금의 1/2까지) |
| 임의적립금 | 회사 재량 적립 (사업확장, 배당평준화 등) |
| 미처분이익잉여금 | 처분 미확정 이익 |

### 기타포괄손익누계액

기타포괄손익누계액(Accumulated Other Comprehensive Income, AOCI)은 당기순이익에 포함되지 않는 자본의 변동입니다.

**기타포괄손익 항목**:

| 항목 | 내용 |
|------|------|
| 기타포괄손익-공정가치 금융자산 평가손익 | 지분상품 공정가치 변동 |
| 해외사업환산손익 | 해외종속기업 환산 차이 |
| 현금흐름위험회피 파생상품 평가손익 | 위험회피 파생상품 평가 |
| 재평가잉여금 | 유형자산 재평가 증가분 |
| 확정급여제도 재측정요소 | 보험수리적 손익 |

**특징**:
- 손익계산서의 당기순이익에 포함되지 않음
- 자본에 직접 반영
- 일부 항목은 향후 당기손익으로 재분류(Reclassification)
- 총포괄손익 = 당기순이익 + 기타포괄손익

**재분류 여부**:

| 항목 | 재분류 |
|------|--------|
| 해외사업환산손익 | ○ (처분 시) |
| 현금흐름위험회피 | ○ (위험회피 대상 손익 인식 시) |
| 기타포괄손익-공정가치 금융자산 | × |
| 확정급여제도 재측정요소 | × |

### 자기주식 (Treasury Stock)

자기주식은 기업이 자사 주식을 매입하여 보유하는 것입니다. 자본의 차감 항목으로 표시됩니다.

**자기주식 취득 목적**:
- 주주환원 (주가 지지)
- 소각을 통한 주당가치 제고
- 임직원 보상 (스톡옵션)
- 경영권 방어

**회계처리**:
- 매입 시: 자기주식(자본 차감) 증가, 현금 감소
- 소각 시: 자기주식 감소, 자본금 또는 이익잉여금 감소
- 처분 시: 자기주식 감소, 현금 증가, 차익은 자본잉여금

### 자본변동표 분석

자본변동표(Statement of Changes in Equity)는 자본의 각 항목이 어떻게 변동했는지 보여줍니다.

**자본변동 원인**:

| 변동 원인 | 영향 항목 |
|-----------|-----------|
| 당기순이익 | 이익잉여금 ↑ |
| 배당금 지급 | 이익잉여금 ↓ |
| 유상증자 | 자본금 ↑, 자본잉여금 ↑ |
| 자사주 매입 | 자기주식 ↓ (차감 증가) |
| 자사주 소각 | 자기주식 ↑, 자본금/이익잉여금 ↓ |
| 기타포괄손익 | 기타포괄손익누계액 변동 |
| 전환사채 전환 | 자본금 ↑, 자본잉여금 ↑ |

**자본변동표 예시** (요약):

| 항목 | 자본금 | 자본잉여금 | 이익잉여금 | AOCI | 자기주식 | 합계 |
|------|--------|------------|------------|------|----------|------|
| 기초 | 500 | 1,000 | 3,000 | 200 | (100) | 4,600 |
| 당기순이익 | - | - | 400 | - | - | 400 |
| 배당 | - | - | (100) | - | - | (100) |
| 자사주 매입 | - | - | - | - | (50) | (50) |
| 기타포괄손익 | - | - | - | 50 | - | 50 |
| **기말** | **500** | **1,000** | **3,300** | **250** | **(150)** | **4,900** |

## 실무 포인트

> **자본 분석 체크리스트**
>
> 1. **이익잉여금 비중**: 자본 중 이익잉여금 비율 (높을수록 자력 성장)
> 2. **배당 여력**: 이익잉여금 > 배당 예정액 확인
> 3. **자기주식 현황**: 보유 규모, 취득 목적, 소각 여부
> 4. **기타포괄손익**: 큰 변동이 있으면 원인 파악
> 5. **유상증자 이력**: 잦은 증자는 희석 위험

## 한국 기업 특성

**자본잉여금**: 한국 기업은 과거 고가 유상증자로 주식발행초과금이 큰 경우가 많습니다. 합병, 분할 과정에서 발생한 자본잉여금도 있습니다.

**이익잉여금**: 재벌 그룹 계열사는 오랜 기간 누적된 이익잉여금이 상당합니다. 그러나 배당 성향이 낮아 이익잉여금만 쌓이는 경우도 있습니다.

**자기주식**: 최근 주주환원 강화로 자사주 매입이 활발합니다. 삼성전자, 현대차 등 대기업의 자사주 소각 사례가 증가하고 있습니다.

**비지배지분**: 연결재무제표에서 100% 소유하지 않은 종속기업의 자본 중 비지배주주 몫이 있습니다. 지배기업 소유주지분과 비지배지분을 구분합니다.

## 분석 예시

**J기업 자본 구조 분석** (단위: 억원):

| 항목 | 금액 | 비중 |
|------|------|------|
| 자본금 | 1,000 | 10% |
| 주식발행초과금 | 2,000 | 20% |
| 이익잉여금 | 7,500 | 75% |
| 기타포괄손익누계액 | 300 | 3% |
| 자기주식 | (800) | -8% |
| **자본 합계** | **10,000** | **100%** |

**분석**:
- 이익잉여금 비중 75%: 오랜 기간 이익 축적, 자력 성장
- 자기주식 800억: 주주환원 의지, 향후 소각 여부 확인
- 기타포괄손익누계액 300억: 양수면 금융자산 평가이익 등
- 배당 여력: 이익잉여금 충분
