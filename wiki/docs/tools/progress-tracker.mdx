## 추천 학습 순서

### 투자 분석 입문자

```
Layer 1 (기초 회계) → Layer 2 (재무제표 분석) → 밸류에이션 → Layer 3 (산업분석) → Layer 4 (기업분석)
```

### 금융권 취업 준비자

```
금융 성향 테스트 → 직무 종류 → 합격 요건 → 자격증 가이드 → 기업 총람
```

### 매매 전략 학습자

```
매매 전략 개요 → 관심 전략 심화 → 리스크 관리 → Market Survivor 실습
```

## 학습 팁

### 1. 매일 조금씩

- 하루 30분~1시간씩 꾸준히
- 한 번에 많이보다 반복 학습이 효과적

### 2. 능동적 학습

- 읽기만 하지 말고 계산기/템플릿 직접 사용
- 관심 기업에 적용해 보기

### 3. 복습의 힘

- 일주일 후 다시 읽기
- 퀴즈로 자가 점검
