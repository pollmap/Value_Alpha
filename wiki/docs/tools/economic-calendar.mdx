## 주요 경제 이벤트 해설

### 통화정책 이벤트

| 이벤트 | 주관 | 빈도 | 투자 영향 |
|:-------|:-----|:-----|:---------|
| **FOMC** | 미국 연준 | 연 8회 | 금리 결정, 점도표, 파월 기자회견 |
| **한은 금통위** | 한국은행 | 연 8회 | 기준금리 결정, 통화정책 방향 |
| **ECB 통화정책** | 유럽중앙은행 | 연 8회 | 유로존 금리 결정 |

### 고용 지표

| 지표 | 발표 | 의미 |
|:-----|:-----|:-----|
| **비농업고용 (NFP)** | 매월 첫 금요일 | 미국 고용 시장 건강도. 예상치 상회 시 금리 인상 기대 |
| **실업률** | 매월 첫 금요일 | 경기 후행 지표. 급등 시 경기 침체 신호 |
| **임금 상승률** | 매월 첫 금요일 | 인플레이션 선행 지표 |

### 물가 지표

| 지표 | 발표 | 의미 |
|:-----|:-----|:-----|
| **CPI** | 매월 중순 | 소비자물가. 인플레이션 핵심 지표 |
| **Core CPI** | 매월 중순 | 식품·에너지 제외. 기저 인플레이션 |
| **PCE** | 매월 말 | 연준 선호 물가 지표 |
| **PPI** | 매월 중순 | 생산자물가. CPI 선행 |

## 경제 지표 발표 시 대응 전략

### 예상치 vs 실제치

```
┌─────────────────────────────────────────────────┐
│                지표 발표 해석                     │
├─────────────────────────────────────────────────┤
│                                                  │
│  실제치 > 예상치     →    호재 (경기 지표)        │
│                      →    악재 (인플레이션 지표)   │
│                                                  │
│  실제치 < 예상치     →    악재 (경기 지표)        │
│                      →    호재 (인플레이션 지표)   │
│                                                  │
│  * 시장은 "예상 대비" 서프라이즈에 반응           │
└─────────────────────────────────────────────────┘
```

### FOMC 대응 체크리스트

1. **금리 결정**: 인상/동결/인하 여부
2. **점도표**: 향후 금리 경로 예측
3. **파월 기자회견**: 향후 정책 방향 힌트
4. **성명서 문구 변화**: "higher for longer" 등 키워드
