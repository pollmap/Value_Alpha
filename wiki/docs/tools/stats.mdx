## 상세 분석 (관리자용)

:::info Google Analytics 설정 필요
상세한 방문자 분석(국가별 방문자, 페이지별 체류 시간, 유입 경로 등)을 보려면 Google Analytics를 설정해야 합니다.

1. [Google Analytics](https://analytics.google.com)에서 계정 생성
2. 속성 생성 후 측정 ID (G-XXXXXXXXXX) 발급
3. `docusaurus.config.js`에서 `trackingID` 값을 발급받은 ID로 교체
4. 배포 후 24-48시간 내 데이터 수집 시작
:::

### Google Analytics로 확인 가능한 정보

- **실시간 방문자 수**: 현재 접속 중인 사용자
- **일/주/월별 방문자 추이**: 시간대별 트래픽 분석
- **국가/지역별 방문자**: 어디서 접속하는지
- **디바이스 정보**: PC vs 모바일 비율
- **인기 페이지**: 어떤 콘텐츠가 가장 많이 조회되는지
- **유입 경로**: 검색 엔진, 직접 접속, 외부 링크 등
- **이탈률/체류 시간**: 콘텐츠 품질 지표
