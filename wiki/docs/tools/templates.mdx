# 📊 기업 분석 템플릿

실무에서 바로 사용할 수 있는 **재무 분석 템플릿**을 제공합니다. 모든 템플릿은 Excel 및 Google Sheets 호환됩니다.

---

## 💡 템플릿 사용 가이드

| 구분 | 난이도 | 소요 시간 | 추천 대상 |
|:-----|:------:|:---------:|:----------|
| 재무비율 분석 | ⭐ 초급 | 30분 | 투자 입문자, 기업 스크리닝 |
| 3-Statement Model | ⭐⭐ 중급 | 2\~4시간 | 애널리스트 지망생, 실전 학습자 |
| DCF 밸류에이션 | ⭐⭐⭐ 고급 | 4\~8시간 | IB 취업 준비생, 가치투자자 |
| LBO Model | ⭐⭐⭐ 고급 | 6\~12시간 | PE 취업 준비생, M&A 관심자 |

3. 'DCF' 시트에서 자동 계산된 적정가치 확인

## 🔧 Google Sheets 사용법

Excel 대신 Google Sheets를 사용하려면:

1. **다운로드한 파일 업로드**: Google Drive → 새로 만들기 → 파일 업로드
2. **Sheets로 열기**: 업로드된 파일 우클릭 → 연결 앱 → Google 스프레드시트
3. **사본 만들기**: 파일 → 사본 만들기 (편집을 위해 필수)

:::warning 호환성 주의
일부 Excel 전용 함수(STOCKHISTORY 등)는 Google Sheets에서 작동하지 않을 수 있습니다.
:::

네, 모든 템플릿은 완전 무료입니다. 상업적 용도가 아닌 개인 학습/분석 목적으로 자유롭게 사용하세요.

</details>

<details>
<summary><strong>템플릿 수정해서 사용해도 되나요?</strong></summary>

네, 자유롭게 수정 가능합니다. 단, 재배포 시 출처(Value Alpha)를 명시해 주세요.

</details>

<details>
<summary><strong>오류나 개선 제안은 어디로 하나요?</strong></summary>

[GitHub Issues](https://github.com/your-repo/issues)에 제보해 주세요. 피드백을 반영하여 지속적으로 개선합니다.

</details>

<details>
<summary><strong>한글 폰트가 깨지는 경우</strong></summary>

Excel에서 열 때 인코딩 문제가 발생할 수 있습니다:
1. Excel → 데이터 → 텍스트/CSV에서 → UTF-8 선택
2. 또는 메모장으로 열고 UTF-8로 다시 저장 후 Excel에서 열기

</details>
