# 현업 분석 방법론

교과서의 이론과 현업의 실무 사이에는 큰 간극이 있습니다. 이 페이지에서는 증권사 애널리스트, 자산운용사 펀드매니저, IB 뱅커가 실제로 어떻게 분석하고 투자 의사결정을 내리는지 생생한 실무 관점에서 설명합니다.

## 1. 증권사 리서치 애널리스트의 하루

### 1.1 일과 타임라인

#### 오전 (07:00 ~ 12:00)

- **07:00-07:30** — 활동: 출근, 시황 체크, 상세: 전일 미국 시장, 야간 선물, 아시아 장 전 동향 확인
- **07:30-08:00** — 활동: Morning Meeting 준비, 상세: 오늘 이슈 정리, 커버리지 종목 관련 뉴스 체크
- **08:00-08:30** — 활동: Morning Meeting, 상세: 팀 전체 브리핑: 시황, 섹터별 이슈, 금일 발간 보고서 소개
- **08:30-09:00** — 활동: 프리마켓 대응, 상세: 장전 이슈 대응, 세일즈/운용사 문의 답변
- **09:00-10:00** — 활동: 장중 모니터링, 상세: 커버리지 종목 주가 확인, 비정상 움직임 분석
- **10:00-12:00** — 활동: 기업 탐방 / 모델 작업, 상세: 기업 방문(주 1\~2회) 또는 엑셀 모델 업데이트

#### 오후 (12:00 ~ 21:00+)

- **12:00-13:00** — 활동: 점심 (종종 미팅), 상세: 기업 IR 담당자, 산업 전문가와 점심 미팅
- **13:00-15:30** — 활동: 장중 모니터링 / 분석, 상세: 모델 업데이트 계속, 산업 리서치, 데이터 분석
- **15:30-16:00** — 활동: 장 마감 리뷰, 상세: 당일 커버리지 종목 주가 움직임 리뷰
- **16:00-18:00** — 활동: 보고서 작성, 상세: 분석 보고서 초안 작성 (실적 프리뷰, 기업 분석 등)
- **18:00-19:00** — 활동: 컴플라이언스 검토, 상세: 보고서 내부 준법 심사 제출
- **19:00-21:00+** — 활동: 보고서 마감 / 스터디, 상세: 최종 수정, 발간. 산업 공부, 해외 리서치 읽기

### 1.2 보고서 종류와 작성 주기

- **실적 프리뷰** — 주기: 분기별 (연 4회/종목), 분량: 2\~5페이지, 내용: 실적 발표 전 추정치, 컨센서스 대비 전망
- **실적 리뷰** — 주기: 분기별 (연 4회/종목), 분량: 3\~8페이지, 내용: 실적 발표 후 분석, 추정치 변경, 투자의견
- **기업 심층 분석** — 주기: 연 1\~2회/종목, 분량: 20\~50페이지, 내용: 사업구조, 경쟁력, 밸류에이션 종합 분석
- **산업 보고서** — 주기: 수시, 분량: 10\~30페이지, 내용: 산업 동향, 규제 변화, 기술 트렌드
- **Daily 코멘트** — 주기: 매일, 분량: 1\~2페이지, 내용: 속보성 이슈 대응 (공시, 뉴스 등)

### 1.3 커버리지와 투자의견 체계

- **커버리지 종목 수**: 보통 10\~20개 (섹터별 차이)
- **투자의견 단계**: 매수(Buy) / 중립(Hold, Neutral) / 매도(Sell, Underperform)
  - 현실: 매도 의견은 극히 드묾 (IB 관계, 기업 관계 등 이유)
- **목표주가 산정**: 12개월 Forward 기준, DCF 또는 Target Multiple 방식
- **목표주가 변경**: 실적 추정 변경 시 또는 밸류에이션 멀티플 변경 시

## 2. 자산운용사 펀드매니저의 투자 프로세스

### 2.1 투자 프로세스 전체 흐름

```
[유니버스 선정] → [스크리닝] → [심층 분석] → [투자 결정] → [포트폴리오 구성] → [모니터링/리밸런싱]
```

### 2.2 유니버스 선정

**투자 유니버스(Investment Universe)**: 펀드의 투자 대상이 되는 종목군

- **대형주 펀드** — 유니버스: KOSPI200 구성 종목, 종목 수 (대략): ~200
- **중소형주 펀드** — 유니버스: KOSPI 시총 200위 이하 + KOSDAQ, 종목 수 (대략): ~1,500
- **가치주 펀드** — 유니버스: PBR 1배 이하, 배당수익률 3% 이상 등, 종목 수 (대략): ~300
- **성장주 펀드** — 유니버스: 매출 성장률 15% 이상, ROE 15% 이상 등, 종목 수 (대략): ~200
- **배당주 펀드** — 유니버스: 배당수익률 상위, 배당성장 이력, 종목 수 (대략): ~150
- **글로벌 펀드** — 유니버스: MSCI ACWI 또는 특정 지역, 종목 수 (대략): 다양

### 2.3 스크리닝 (Quantitative Screening)

대량의 종목 중에서 투자 후보를 줄이는 단계입니다.

#### 일반적 스크리닝 기준

```
1차 필터 (Hard Filter):
  □ 시가총액 ≥ 3,000억원 (유동성 확보)
  □ 일평균 거래대금 ≥ 10억원
  □ 관리종목/투자유의 제외
  □ 최근 감사의견 "적정" 확인

2차 필터 (Quantitative):
  □ ROE ≥ 10%
  □ 부채비율 ≤ 200%
  □ 매출 성장률 ≥ 5% (3년 CAGR)
  □ PER ≤ 업종 평균
  □ FCF Yield ≥ 5%

3차 필터 (Qualitative):
  □ 경영진 신뢰도
  □ 산업 성장성
  □ 경쟁 우위 지속 가능성
```

### 2.4 심층 분석 (Deep Dive)

스크리닝을 통과한 30\~50개 종목에 대해 심층 분석을 수행합니다.

#### 분석 프레임워크

**1) 사업 이해 (Business Understanding)**
- 수익 모델은 무엇인가? (B2B vs B2C, 구독 vs 일회성, 플랫폼 vs 제조)
- 핵심 경쟁력(Moat)은 무엇인가? (브랜드, 네트워크 효과, 규모의 경제, 전환비용)
- 산업 내 포지셔닝은? (리더 vs 도전자 vs 틈새)

**2) 재무 분석 (Financial Analysis)**
- 과거 5년 이상 재무제표 분석
- 수익성 추세 (마진 확장/축소)
- 현금흐름 질적 분석 (영업이익 대비 OCF 비율)
- 자본 배분 정책 (배당, 자사주, M&A, CAPEX)

**3) 밸류에이션 (Valuation)**
- 적절한 밸류에이션 방법론 선택
- 보수적/기본/낙관적 시나리오별 적정 가치 산출
- 현재 주가 대비 업사이드/다운사이드 평가

**4) 리스크 평가 (Risk Assessment)**
- 사업 리스크: 기술 변화, 경쟁 심화, 고객 집중도
- 재무 리스크: 부채 수준, 환율 노출, 유동성
- 규제 리스크: 정부 정책 변화, 환경 규제
- ESG 리스크: 지배구조, 환경 이슈, 사회적 책임

### 2.5 투자 결정과 포트폴리오 구성

#### 투자 의사결정 회의 (Investment Committee)

```
투자 제안서 구성:
1. 투자 논거 (Investment Thesis) - 3줄 요약
2. 사업 개요 및 경쟁력 분석
3. 재무 분석 핵심 요약
4. 밸류에이션 (시나리오별)
5. 리스크 요인 및 완화 방안
6. 제안 비중 및 매수 가격대
7. 모니터링 포인트 (투자 논거가 틀릴 시 매도 기준)
```

#### 포트폴리오 구성 원칙

- **종목 수** — 설명: 적정 분산, 일반적 기준: 20\~40종목
- **최대 비중** — 설명: 단일 종목 집중 리스크 방지, 일반적 기준: 5\~10%
- **섹터 분산** — 설명: 특정 산업 편중 방지, 일반적 기준: 섹터당 30% 이하
- **현금 비중** — 설명: 시장 상황에 따른 방어, 일반적 기준: 0\~20%
- **벤치마크 대비** — 설명: 트래킹 에러 관리, 일반적 기준: 펀드 스타일에 따라 상이

### 2.6 모니터링 및 리밸런싱

**일상적 모니터링:**
- 매일: 포트폴리오 수익률, NAV 변동, 주요 종목 뉴스
- 매주: 섹터별 성과 리뷰, 벤치마크 대비 성과
- 매월: 포트폴리오 리밸런싱 검토, 리스크 지표 점검

**매도 결정 기준:**
1. 투자 논거 훼손 (Thesis broken)
2. 목표가 도달 (Valuation ceiling)
3. 더 좋은 기회 발견 (Opportunity cost)
4. 리스크 한도 초과 (Risk limit)
5. 펀드 유출 대응 (Liquidity need)

## 3. IB 밸류에이션 실무

### 3.1 M&A 딜에서의 밸류에이션

IB에서의 밸류에이션은 "정확한 가치"를 찾는 것이 아니라, **협상 범위(Valuation Range)**를 설정하는 것입니다.

#### M&A 밸류에이션 프로세스

```
1. 인수 대상(Target) 분석
   └─ 사업 이해, 재무 분석, 시너지 분석

2. 독립 밸류에이션 (Standalone Valuation)
   ├─ DCF (Discounted Cash Flow)
   ├─ Trading Comps (상장사 멀티플)
   └─ Transaction Comps (과거 거래 멀티플)

3. 시너지 밸류에이션
   ├─ Revenue Synergies (매출 시너지)
   ├─ Cost Synergies (비용 시너지)
   └─ Financial Synergies (재무 시너지)

4. LBO 분석 (PE 관점)
   └─ 재무적 투자자가 지불 가능한 최대 가격

5. Football Field Chart
   └─ 모든 방법론의 결과를 하나의 차트로 종합
```

#### Football Field Chart 구성

```
방법론                    밸류에이션 범위 (원/주)
─────────────────────────────────────────────────────
52주 최고/최저     ████████████████████████  50,000 ~ 90,000
Trading Comps        ██████████████████    60,000 ~ 85,000
Transaction Comps      ████████████████    65,000 ~ 90,000
DCF (WACC 9\~11%)       ████████████████    62,000 ~ 88,000
DCF (Exit Multiple)      ██████████████    68,000 ~ 85,000
LBO (15\~20% IRR)           ████████████    70,000 ~ 82,000
─────────────────────────────────────────────────────
                    50,000      70,000      90,000
```

### 3.2 Fairness Opinion (공정가치 의견)

M&A 거래에서 **이사회의 의사결정이 주주에게 공정한지(fair)** 독립적으로 평가하는 의견서입니다.

**작성 시 주요 분석:**
1. DCF 분석 (다수의 시나리오)
2. 유사 상장사 비교 분석
3. 유사 거래 비교 분석
4. 프리미엄 분석 (과거 유사 거래의 프리미엄 수준)
5. 52주 주가 범위 분석
6. 애널리스트 목표주가 범위

**한국 특수성:**
- 자본시장법상 합병가액 산정 기준: 기준시가 (2개월/1개월/1주 가중평균 중 낮은 가격)
- 비상장사 합병: 자산가치 + 수익가치 가중평균 (한국은 법정 산식이 존재)
- 감정평가 의무: 일정 규모 이상 거래 시 외부 평가 의무

### 3.3 IPO 공모가 산정

#### 공모가 산정 프로세스

```
1단계: 유사회사 선정 (Peer Group)
  - 동일 산업, 유사 사업모델, 유사 규모
  - 최소 3\~5개사 선정

2단계: 적용 멀티플 결정
  - PER, EV/EBITDA, EV/Sales, PBR 등
  - 업종/성장 단계에 따라 적절한 멀티플 선택

3단계: 기업가치 산출
  - 유사회사 평균 멀티플 × 대상 기업 지표
  - 복수의 멀티플 적용하여 범위 산출

4단계: IPO 할인율 적용
  - 통상 20\~30% 할인 (비유동성 할인, IPO 할인)
  - 한국: 증권신고서 기재 시 할인율 공시

5단계: 공모 희망가 밴드 설정
  - 상단/하단 차이: 통상 20\~30%
  - 예: 30,000원 ~ 38,000원

6단계: 수요예측 (Book Building)
  - 기관투자자 수요조사 → 최종 공모가 확정
```

## 4. 블룸버그 터미널 활용

### 4.1 블룸버그 주요 기능

Bloomberg Terminal은 금융 데이터의 "스위스 군용 칼"로, 월 약 $2,000 (연 약 2,500만원) 비용이 드는 전문 터미널입니다.

#### 필수 기능 코드

- **BQ** — 명칭: Bloomberg Quote, 용도: 개별 종목 종합 정보 대시보드
- **FA** — 명칭: Financial Analysis, 용도: 재무제표, 재무비율 상세 분석
- **COMP** — 명칭: Comparable Analysis, 용도: 유사기업 비교 분석 (멀티플, 재무)
- **RV** — 명칭: Relative Value, 용도: 상대가치 분석, 밸류에이션 비교
- **GP** — 명칭: Graph, 용도: 주가 차트, 멀티플 차트
- **ANR** — 명칭: Analyst Recommendations, 용도: 애널리스트 투자의견, 목표주가
- **EE** — 명칭: Earnings Estimates, 용도: 실적 컨센서스 추정치
- **DDIS** — 명칭: Debt Distribution, 용도: 부채 구조 분석
- **MGMT** — 명칭: Management, 용도: 경영진 정보
- **RELS** — 명칭: Related Securities, 용도: 관련 종목 (자회사, 채권 등)
- **PORT** — 명칭: Portfolio Analytics, 용도: 포트폴리오 분석 (수익률, 리스크)
- **FLDS** — 명칭: Field Search, 용도: 데이터 필드 검색
- **EQS** — 명칭: Equity Screening, 용도: 주식 스크리닝 도구
- **MEMB** — 명칭: Index Members, 용도: 지수 구성 종목 확인
- **CACS** — 명칭: Corporate Actions, 용도: 기업 이벤트 (배당, 분할 등)

#### 블룸버그 엑셀 연동 (BDH/BDP)

블룸버그의 가장 강력한 기능 중 하나는 Excel Add-in입니다.

```excel
// BDP: 단일 시점 데이터
=BDP("005930 KS Equity", "PX_LAST")          // 삼성전자 현재가
=BDP("005930 KS Equity", "BEST_PE_RATIO")    // 컨센서스 PER
=BDP("005930 KS Equity", "CUR_MKT_CAP")     // 시가총액

// BDH: 시계열 데이터
=BDH("005930 KS Equity", "PX_LAST", "20240101", "20241231")  // 기간 주가
=BDH("005930 KS Equity", "SALES_REV_TURN", "20200101", "20240101", "Period", "A")  // 연간 매출

## 5. 컨센서스 활용법

### 5.1 컨센서스란?

**컨센서스(Consensus)**는 증권사 애널리스트들의 실적 추정치 평균(또는 중앙값)입니다. 시장이 기대하는 실적 수준을 나타냅니다.

### 5.2 FnGuide 컨센서스 활용

```
FnGuide 컨센서스 확인 경로:
  comp.fnguide.com → 종목 검색 → "Consensus" 탭

주요 확인 항목:
  - 매출, 영업이익, 당기순이익 (분기/연간)
  - 12개월 Forward PER, PBR
  - 목표주가 평균/최고/최저
  - 투자의견 분포 (Buy/Hold/Sell)
  - 추정치 변동 추이 (Revision Trend)
```

### 5.3 시장 기대 vs 본인 추정

핵심은 **컨센서스와 내 추정치의 차이**에서 투자 기회를 찾는 것입니다.

```
[시나리오 매트릭스]

                  시장 기대 (컨센서스)
                  긍정적        부정적
본인    긍정적    (A) 이미 반영   (B) 매수 기회 ★
추정    부정적    (C) 매도 기회   (D) 이미 반영
```

- **(B) 매수 기회**: 시장은 비관적이지만 내 분석으로는 실적이 좋을 것으로 예상 → 어닝 서프라이즈 가능
- **(C) 매도 기회**: 시장은 낙관적이지만 내 분석으로는 실적이 나쁠 것으로 예상 → 어닝 쇼크 가능

### 5.4 어닝 서프라이즈 트레이딩

**PEAD (Post-Earnings Announcement Drift)**: 실적 발표 후 주가가 서프라이즈 방향으로 수주\~수개월 추가 이동하는 현상

```
전략 프레임워크:
1. 실적 발표 1\~2주 전에 "내 추정" vs "컨센서스" 비교
2. 괴리가 큰 종목 리스트업
3. 서프라이즈가 예상되는 방향으로 포지션 설정
4. 실적 발표 후 PEAD 활용 (수익 실현 또는 손절)

주의사항:
- 컨센서스도 틀릴 수 있고, 내 추정도 틀릴 수 있음
- 실적이 좋아도 가이던스(향후 전망)가 나쁘면 주가 하락 가능
- 이미 시장에 반영된 기대(Whisper Number)가 있을 수 있음
```

### 5.5 컨센서스 추정치 변동 추적

**Earnings Revision (추정치 수정)**은 강력한 투자 시그널입니다.

```
강력한 매수 신호:
  - 3개월 연속 컨센서스 상향 (Upward Revision)
  - 다수 애널리스트가 동시에 상향 (Breadth)
  - 상향 폭이 점점 커지는 추세

강력한 매도 신호:
  - 3개월 연속 컨센서스 하향 (Downward Revision)
  - "어닝 서프라이즈 후에도" 추가 하향 (추가 악재)
  - 투자의견 하향 동반
```

## 6. 기업 탐방(NDR) 노하우

### 6.1 기업 탐방이란?

**NDR(Non-Deal Roadshow)** 또는 **기업 탐방**은 투자자가 기업을 직접 방문하여 경영진과 대화하는 것입니다. 한국 증권 시장에서 기업 탐방은 가장 중요한 정보 수집 활동 중 하나입니다.

### 6.2 탐방 전 준비

#### 사전 조사 체크리스트

```
□ 최근 2개년 사업보고서 정독
□ 최근 4분기 실적발표 자료 (IR 프레젠테이션)
□ 최근 증권사 리포트 5개 이상 읽기
□ 경쟁사 분석 (동종업종 2\~3개사)
□ 산업 트렌드 (규제 변화, 기술 변화)
□ 주요 경영진 이력 확인
□ 주가 차트 및 밸류에이션 트렌드 확인
□ DART 공시 히스토리 (최근 6개월)
```

#### 질문 리스트 작성법

효과적인 질문은 **공개 정보에서 답을 찾을 수 없는 것**이어야 합니다.

**좋은 질문 예시:**

```
[사업 전략]
- 향후 3년 간 가장 큰 성장 드라이버는 무엇이라고 보시는지?
- 신규 사업 진출 계획이 있으신지? 기존 사업과의 시너지는?
- 경쟁사 대비 차별화 포인트를 어떻게 유지하실 계획인지?

[재무/실적]
- 올해 CAPEX 계획과 투자 우선순위는?
- 원재료 가격 변동에 대한 헤지 전략은?
- 운전자본 효율화 계획이 있으신지?
- R&D 투자 대비 성과를 어떻게 평가하시는지?

[산업/경쟁]
- 업계 전반의 수급 상황을 어떻게 보시는지?
- 신규 진입자의 위협을 어느 정도로 인식하시는지?
- 규제 환경 변화가 사업에 미치는 영향은?

[경영/지배구조]
- 주주환원 정책의 방향성은? (배당, 자사주)
- 후계 구도에 대해 어떻게 생각하시는지?
- ESG 관련 주요 이니셔티브는?
```

**나쁜 질문 예시 (피해야 할 것):**
- "올해 영업이익 얼마 나올까요?" → 미공개 정보 요청 (공정공시 위반 가능)
- "주가가 왜 빠졌다고 생각하세요?" → 기업이 답할 수 없는 질문
- DART에 다 나와 있는 기본적인 질문 → 사전 조사 부족 노출

### 6.3 탐방 시 관찰 포인트

숫자 이외에도 **비언어적 신호(Non-verbal Signals)**에서 중요한 정보를 얻을 수 있습니다.

```
[긍정적 신호]
+ 경영진이 자신감 있고 구체적으로 답변
+ 어려운 질문에도 솔직하게 인정하고 대안 제시
+ 직원들의 분위기가 활기참
+ 사무실/공장이 잘 정돈되어 있음
+ IR 담당자가 적극적으로 추가 자료 제공

[부정적 신호]
- 답변이 모호하거나 질문을 회피
- "그건 말씀드리기 어렵다"를 반복
- 경영진이 과도하게 낙관적 (구체적 근거 없이)
- 직원 이직률이 높아 보이는 징후
- 약속한 자료를 보내주지 않음
- 이전 탐방에서 한 말과 현재 말이 다름
```

### 6.4 탐방 후 보고서 작성

```
[기업 탐방 보고서 템플릿]

기업명: ___________
탐방일: ___________
참석자: ___________
작성자: ___________

1. 핵심 요약 (Key Takeaways)
   - 3\~5개 핵심 포인트

2. 사업 현황 업데이트
   - 분기별 실적 동향
   - 신규 수주/계약 현황
   - 시장 환경 변화

3. 경영진 발언 요약
   - 주요 Q&A 내용
   - 경영진 톤(긍정적/중립/부정적)

4. 투자 판단에 미치는 영향
   - 기존 투자 논거 변경 여부
   - 추정치 변경 필요 여부
   - 투자의견/목표주가 변경 여부

5. 후속 조치
   - 추가 확인 필요 사항
   - 다음 탐방 일정
```

## 7. IR 자료 분석

### 7.1 사업보고서 (Annual Report) 핵심 체크포인트

DART에 공시되는 사업보고서는 가장 포괄적인 기업 정보 소스입니다.

#### 필독 섹션과 체크포인트

- **I. 회사의 개요**: 주요 사업 내용 변경, 자회사 현황, 직원 수 추이
- **II. 사업의 내용**: 산업 특성, 매출 구성, 신규 사업, 연구개발 현황
- **III. 재무에 관한 사항**: 재무제표, 감사의견, 특수관계자 거래
- **IV. 감사인의 감사의견**: "적정" 확인, 강조사항/핵심감사사항
- **V. 이사의 경영진단**: 경영진의 사업 전망, 리스크 인식
- **VI. 주주에 관한 사항**: 최대주주, 지분 변동, 자기주식 현황
- **VII. 임원 및 직원**: 경영진 보수, 스톡옵션, 직원 평균 급여

### 7.2 분기보고서 포인트

분기보고서는 사업보고서보다 간략하지만, 분기별 실적 추이를 파악하는 데 필수입니다.

```
분기보고서 속독법:
1. 재무제표 먼저 확인 (IS → 매출/영업이익 추이)
2. 전년 동기 대비 변화율 계산
3. 주석 중 '특수관계자 거래' 확인
4. 우발부채/약정사항 변화 확인
5. 영업활동현금흐름 vs 영업이익 괴리 확인
```

### 7.3 감사보고서 핵심 체크포인트

```
□ 감사의견: "적정" 이외의 의견 시 즉시 주의 (한정/부적정/의견거절)
□ 강조사항: 계속기업 불확실성, 중대한 소송 등
□ 핵심감사사항 (KAM): 감사인이 특별히 주의를 기울인 항목
  - 매출 인식의 적정성 (건설/SW 업종)
  - 영업권/무형자산 손상 평가 (M&A 기업)
  - 재고자산 평가 (제조업)
  - 금융상품 공정가치 (금융업)
□ 내부회계관리제도 의견: "효과적" 확인
□ 전기 대비 감사인 변경 여부: 잦은 변경은 적신호
```

### 7.4 주주총회 참고서류

```
주주총회 안건 중 주의해야 할 것:
1. 이사 선임/해임 → 경영진 변화 신호
2. 정관 변경 → 사업 목적 변경, 이사회 권한 변경
3. 합병/분할 승인 → 밸류에이션 영향
4. 감사(감사위원) 선임 → 독립성 확인
5. 이사 보수 한도 → 과도한 보수는 지배구조 리스크
6. 자기주식 취득 결정 → 주주환원 신호
```

## 8. 한국 시장 특화 분석

### 8.1 공매도 잔고 분석

```
공매도 데이터 확인 경로:
  - KRX 정보데이터시스템 (data.krx.co.kr) → 공매도 → 공매도 잔고
  - 금융위원회 공매도 종합 포털

분석 프레임워크:
  공매도 비율 = 공매도 잔고 / 발행주식수
  공매도 잔고 금액 = 공매도 잔고 × 현재가

[해석]
  공매도 비율 급증 + 주가 하락: 부정적 시그널 강화
  공매도 비율 급증 + 주가 횡보: 향후 하락 가능성 (선행)
  공매도 비율 감소 (Short Covering): 단기 반등 가능성
  공매도 비율 최고점 → 감소: 숏 스퀴즈 가능성 (급등)

주의사항:
  - 한국은 주기적으로 공매도 금지/제한이 시행됨
  - 공매도 자체가 악재가 아님 (시장 효율성 기여)
  - 공매도 잔고는 후행 데이터 (T+2 기준)
```

### 8.2 외국인 / 기관 수급 분석

한국 시장의 가장 독특한 특성 중 하나는 **투자자별 매매 동향**이 매일 공개된다는 점입니다.

```
투자자 유형별 특성:

[외국인]
  - KOSPI 시총의 약 30% 보유
  - 대형주 위주 투자
  - 매크로(환율, 글로벌 유동성)에 민감
  - 연속 매수/매도 추세가 강함

[기관 (금융투자)]
  - 증권사 자기매매 부문
  - 단기 트레이딩 비중 높음
  - 프로그램 매매 비중 큼

[기관 (보험)]
  - 장기 투자 (ALM 관점)
  - 배당주/채권형 선호
  - 분기말 윈도우 드레싱 주의

[기관 (투신/운용)]
  - 펀드 자금 흐름에 좌우
  - 벤치마크 대비 운용
  - 대형주 비중 높음

[개인]
  - 단기 매매 비중 높음
  - 역추세(Contrarian) 성향
  - 소형주/테마주 선호

[연기금]
  - 국민연금, 공무원연금 등
  - 장기 투자 (3\~5년 관점)
  - 매수 시 시장 안정 효과
  - "연기금이 사면 바닥" 인식 (항상 맞지는 않음)
```

#### 수급 분석 실전 전략

```
[강한 매수 신호]
  외국인 + 기관 동시 순매수 (10일 이상 연속)
  + 거래량 증가
  + 신고가 경신

[경계 신호]
  외국인 순매도 전환 (특히 대량)
  + 원/달러 환율 급등
  + 신흥국 전반 자금 유출

[주의사항]
  - 수급은 후행 지표 (이미 일어난 일)
  - 수급만으로 투자 결정하지 말 것
  - 펀더멘탈 분석과 병행해야 의미 있음
```

### 8.3 프로그램 매매 해석

```
