# 커뮤니티

금융 위키 커뮤니티에 오신 것을 환영합니다. 여기서 질문하고, 토론하고, 정보를 공유하세요.

## 참여 방법

:::info GitHub 계정 필요
토론에 참여하려면 **GitHub 계정**이 필요합니다. GitHub 계정이 없다면 [여기서 무료로 생성](https://github.com/signup)할 수 있습니다.

GitHub Issues를 사용하면:
- 스팸/어뷰징 방지
- 마크다운, 코드 블록, 이미지 첨부 가능
- 답변 알림 수신
:::

## 커뮤니티 가이드라인

1. **예의를 지켜주세요** - 서로 존중하며 건설적인 토론을 해주세요
2. **질문 전 검색** - 이미 답변된 질문인지 먼저 확인해주세요
3. **구체적으로 질문** - 막연한 질문보다 구체적인 질문이 좋은 답변을 받습니다
4. **출처 명시** - 정보를 공유할 때 출처를 밝혀주세요
5. **홍보 금지** - 광고나 자기 홍보 목적의 글은 삭제됩니다

---

## 토론 참여하기

카테고리별로 필터링하거나, 새 글을 작성하세요. GitHub에서 직접 댓글을 달 수 있습니다.
