# PER 밴드 분석 계산기

과거 PER 범위를 기반으로 **적정주가 밴드**를 산출하고, 현재 주가가 어느 구간에 위치하는지 판단합니다.

Forward EPS에 과거 PER 범위(최고/+1σ/평균/-1σ/최저)를 적용하여 목표주가 범위를 시각화합니다.

## 활용 가이드

### PER 밴드 분석이 유용한 경우
- **안정적 이익 성장 기업**: PER 범위가 일정한 소비재, 통신, 유틸리티
- **밸류에이션 밴드가 좁은 기업**: 역사적으로 PER 6\~12배 수준의 가치주

### PER 밴드 분석이 위험한 경우
- **경기순환주**: 이익 변동성이 커서 PER 자체가 의미 없음
- **성장 단계 전환 기업**: 과거 PER 밴드가 미래에 적용되지 않을 수 있음
- **적자 전환 기업**: EPS가 음수면 PER 계산 불가

:::danger 실전에서의 한계
PER 밴드는 "과거 시장이 이 기업에 부여한 밸류에이션이 미래에도 유지된다"는 전제입니다. 기업의 펀더멘털이 구조적으로 변했다면 (예: 삼성전자의 HBM 진입, 현대차의 EV 전환) 과거 PER 밴드는 무의미할 수 있습니다.
:::
