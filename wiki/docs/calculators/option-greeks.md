# 옵션 그릭스 계산기

**Black-Scholes 모형**으로 옵션 가격과 위험 지표(그릭스)를 계산합니다.

## Black-Scholes 공식

<div className="formula-block">

**Call = S × N(d1) - K × e^(-rT) × N(d2)**

**Put = K × e^(-rT) × N(-d2) - S × N(-d1)**

</div>

## 그릭스 (Greeks)

| 그릭 | 의미 | 콜옵션 ATM |
|------|------|-----------|
| **델타 (Δ)** | 기초자산 $1 변동 시 옵션가 변동 | ≈ 0.50 |
| **감마 (Γ)** | 델타의 변동률 | ATM에서 최대 |
| **세타 (Θ)** | 시간가치 소멸 (일당) | 음수 |
| **베가 (ν)** | IV 1%p 변동 시 옵션가 변동 | 장기 ATM에서 최대 |
| **로 (ρ)** | 금리 1%p 변동 시 옵션가 변동 | 콜 양수, 풋 음수 |

## 그릭스 해석

### 델타 (Delta)
- 콜: 0 ~ +1
- 풋: -1 ~ 0
- 헷지 비율로 사용

### 감마 (Gamma)
- 항상 양수
- ATM에서 최대, 만기 근접 시 급등

### 세타 (Theta)
- 매수자에게 불리 (시간가치 소멸)
- 만기 근접 시 가속

### 베가 (Vega)
- 변동성 상승 → 옵션 가치 상승
- 장기 옵션에서 더 큼

## 예시

| 입력 | 값 |
|------|-----|
| 기초자산 (S) | 100 |
| 행사가 (K) | 100 |
| 만기 (T) | 0.5년 |
| 무위험이자율 (r) | 3% |
| 변동성 (σ) | 20% |

| 결과 | 콜 | 풋 |
|------|-----|-----|
| 가격 | 5.88 | 4.40 |
| 델타 | 0.54 | -0.46 |
| 감마 | 0.028 | 0.028 |
| 세타 | -5.44 | -4.00 |
| 베가 | 28.0 | 28.0 |
