# DCF 계산기

**현금흐름할인법(DCF)**으로 기업의 내재가치를 계산하는 인터랙티브 도구입니다.

## 계산기

<DCFCalculator />

## 사용 방법

### 1. 잉여현금흐름(FCF) 입력

향후 5년간의 예상 잉여현금흐름을 입력합니다.

**FCF 계산 방법:**
```
FCFF = EBIT × (1-T) + 감가상각 - CapEx - 운전자본 증가
```

### 2. WACC (할인율) 입력

기업의 가중평균자본비용을 입력합니다. 일반적으로 8-12% 범위입니다.

:::tip
정확한 WACC 계산이 필요하면 [WACC 계산기](/calculators/wacc)를 사용하세요.
:::

### 3. 영구성장률 입력

예측 기간 이후 영구적으로 성장하는 비율입니다. 보통 2-4% (장기 GDP 성장률 수준).

**주의:** 영구성장률은 반드시 WACC보다 낮아야 합니다.

### 4. 발행주식수 및 순부채

- **발행주식수**: 백만 주 단위
- **순부채**: 이자부부채 - 현금 (억원 단위)

## 결과 해석

| 결과 | 의미 |
|------|------|
| **기업가치 (EV)** | 전체 자본 제공자에 대한 가치 |
| **자기자본가치** | EV - 순부채 = 주주 몫 |
| **주당 내재가치** | 자기자본가치 / 발행주식수 |
| **TV 비중** | 터미널 가치가 전체에서 차지하는 비율 |

:::warning TV 비중 주의
TV 비중이 80%를 초과하면 예측의 불확실성이 높습니다. 예측 기간을 늘리거나 가정을 재검토하세요.
:::

## 민감도 분석

**민감도 분석 보기** 버튼을 클릭하면 WACC와 영구성장률 변화에 따른 주당 가치 변동을 확인할 수 있습니다.
