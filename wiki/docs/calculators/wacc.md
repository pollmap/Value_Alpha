# WACC 계산기

**가중평균자본비용(WACC)**을 계산하는 인터랙티브 도구입니다. DCF 분석의 핵심 할인율입니다.

## 사용 방법

### 자기자본비용 (CAPM)

CAPM(자본자산가격결정모형)으로 자기자본비용을 계산합니다.

| 입력값 | 설명 | 참고 기준 |
|--------|------|----------|
| **무위험이자율** | 국고채 10년물 수익률 | 한국: 3-4% |
| **베타 (β)** | 시장 대비 변동성 | 1.0 = 시장 평균 |
| **시장위험프리미엄** | Rm - Rf | 5-7% |

**공식:**
```
Re = Rf + β × (Rm - Rf)
```

### 부채비용

| 입력값 | 설명 |
|--------|------|
| **세전 부채비용** | 차입 이자율 |
| **법인세율** | 한국 기준 22-25% |

이자비용의 세금 절감 효과를 반영:
```
세후 Rd = Rd × (1-T)
```

### 자본구조

| 입력값 | 설명 |
|--------|------|
| **시가총액** | 주가 × 발행주식수 |
| **이자부부채** | 차입금, 사채 등 |

:::tip
장부가가 아닌 **시장가치** 기준으로 입력하세요.
:::

## WACC 공식

```
WACC = (E/V) × Re + (D/V) × Rd × (1-T)
```

- E/V: 자기자본 비중
- D/V: 부채 비중
- Re: 자기자본비용
- Rd: 부채비용
- T: 법인세율

## 산업별 참고 WACC

| 산업 | WACC 범위 |
|------|----------|
| 유틸리티 | 5-7% |
| 금융 | 6-8% |
| 소비재 | 7-9% |
| 기술 | 9-12% |
| 바이오 | 12-15%+ |
