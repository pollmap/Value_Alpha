## 5. SOTP 밸류에이션

### 5.1 밸류에이션 프레임워크

현대자동차의 SOTP는 **5개 핵심 가치 구성요소**로 분해합니다.

```
현대자동차 SOTP =
  ① 완성차 본업 가치
  + ② 기아 지분가치
  + ③ 금융 자회사 가치 (현대캐피탈/카드)
  + ④ 인도법인(HMI) 가치
  + ⑤ 신사업 가치 (로보틱스/UAM)
  - ⑥ 코리아 디스카운트 (할인)
```

### 5.2 ① 완성차 본업 가치

```
완성차 영업이익 (인도/금융 제외):
  연결 영업이익: 15.1조
  - 금융 영업이익: -2.5조
  - 기아 기여분: -4.0조 (연결 비중)
  = 순수 현대차 완성차 이익: 약 8.6조

밸류에이션:
  적용 PER: 5~7x (글로벌 완성차 평균)
  완성차 가치 = 8.6조 × 6x = 51.6조 원

글로벌 Peer PER (2023):
  Toyota: 10~12x
  VW: 3~4x
  GM: 5~6x
  Ford: 6~7x
  Stellantis: 3~4x
  → 현대차 적정: 5~7x (Toyota보다 할인, VW보다 프리미엄)
```

## 7. 실전에서의 한계

:::danger 반드시 인식해야 할 리스크

### 7.1 EV 전환의 양면성

EV 전환은 기회이자 리스크입니다.

| 리스크 | 설명 |
|--------|------|
| EV 마진 하락 | ICE 대비 EV 마진 2~5%p 낮음 (초기) |
| 투자 부담 | EV 공장, 배터리, 소프트웨어에 막대한 투자 필요 |
| 경쟁 심화 | Tesla, BYD, 중국 업체 공격적 확장 |
| 기술 불확실성 | 배터리 기술, 자율주행 기술 변화 속도 |
| 수요 불확실성 | EV 수요 성장 둔화 가능성 (2024년 현실화) |

### 7.2 중국 리스크

```
현대차 중국 시장:
  2016년: 판매 180만 대 (시장 점유율 7%)
  2023년: 판매 26만 대 (시장 점유율 1%)
  → 사실상 중국 시장 철수 수준

원인:
  - 사드 보복 (2017~)
  - BYD 등 로컬 EV 업체 급성장
  - 가격 경쟁력 상실

교훈: 지정학 리스크는 예측 불가능하며,
      특정 시장 의존도가 높을수록 위험
```

### 7.3 SOTP 할인의 구조적 한계

SOTP 분석에서 "저평가"라는 결론이 **수년간 지속**될 수 있습니다.

```
"현대차 SOTP 적정가 50만원, 현재 20만원 → 60% 저평가!"

이 논리의 문제점:
1. 코리아 디스카운트가 해소될 촉매(Catalyst)가 불명확
2. 지분가치(기아, HMI)는 현금화되지 않으면 가치 실현 불가
3. 신사업(로보틱스, UAM)은 적자 지속, 가치 검증 어려움
4. "저평가"가 10년간 지속된다면, 실제로는 "적정가"일 수 있음
```

### 7.4 자동차 산업의 구조적 변화

| 위협 요인 | 가능성 | 영향도 |
|-----------|--------|--------|
| 테슬라/중국 EV 가격 경쟁 | 높음 | 높음 |
| 자율주행 기술 (Waymo, Cruise) | 중간 | 매우 높음 |
| 차량 공유 확대 (Uber, 로보택시) | 중간 | 높음 |
| 탄소 규제 강화 | 높음 | 중간 |
| 소비자 취향 변화 (SUV→소형) | 중간 | 중간 |

## 8. 자주 하는 실수

:::tip 초보 투자자가 빠지기 쉬운 함정

### 실수 1: "자동차 PER 5배는 정말 싼 건가?"

```
자동차 주식이 "만년 저PER"인 이유:

1. 경기 민감 업종
   - 경기 침체 시 자동차 판매 급감
   - 이익 변동성이 커서 낮은 PER 적용

2. 높은 자본 집약도
   - 공장, 금형, 부품 재고 등 대규모 자산 필요
   - ROE가 구조적으로 10~15%에 머무름

3. 기술 전환 비용
   - ICE → EV 전환 투자 (수십조 원)
   - 투자 회수 불확실

4. 경쟁 강도
   - 글로벌 20+ 완성차 OEM 경쟁
   - 중국 신규 진입자 (BYD, NIO, XPeng 등)

결론: 자동차 PER 5배는 "저평가"가 아니라
      업종 특성상 "적정 수준"일 수 있음
```

### 실수 2: 환율 효과 무시

```
현대차 이익의 환율 민감도:
  원/달러 10원 상승 → 영업이익 약 800~1,000억 원 증가

  2023년 영업이익 15.1조의 구성:
  - 순수 영업 개선: ~10조
  - 환율 효과 (원화 약세): ~3~4조
  - Mix 개선: ~1~2조

  → 원/달러 1,300원 → 1,100원으로 하락 시
     영업이익 2조+ 감소 가능
     → "실적 호조"의 상당 부분이 환율 효과였음을 인식해야
```

### 실수 3: 기아와 현대를 동일하게 취급

| 구분 | 현대차 | 기아 |
|------|--------|------|
| 투자 부담 | 높음 (그룹 전체 투자 담당) | 낮음 |
| 신사업 | 보스턴 다이내믹스, 슈퍼널 등 | 제한적 |
| 지배구조 | 복잡 (중간 지주사) | 상대적 단순 |
| 배당성향 | 낮음 | 높음 (개선 중) |
| PER | 5~6x | 4~5x |
| ROE | 13% | 17%+ |

→ 기아가 **"더 싸고 더 높은 ROE"**를 제공하는 경우가 많음

### 실수 4: 인도법인 가치의 이중 계산

```
주의: 연결 기준 이익에 이미 HMI 이익이 포함됨

잘못된 계산:
  연결 PER × 연결 이익 = 현대차 가치
  + HMI 시가총액 × 지분율 = HMI 가치
  → 이중 계산 발생!

올바른 계산:
  (연결 이익 - HMI 기여분) × PER = 완성차 본업 가치
  + HMI 시가총액 × 지분율 = HMI 가치
  = SOTP 합계
```

:::
