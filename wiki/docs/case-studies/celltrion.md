# 셀트리온: rNPV와 바이오시밀러 밸류에이션

## 학습 목표

이 케이스 스터디를 통해 다음을 학습합니다:

- 바이오시밀러 기업의 밸류에이션 특수성

## 4. rNPV 밸류에이션

### 4.1 rNPV 개념

rNPV(risk-adjusted Net Present Value)는 **파이프라인의 성공확률을 반영한 현재가치**입니다.

```
rNPV 공식:

rNPV = Σ [ (성공확률 × 예상 현금흐름) / (1 + 할인율)^t ]

구성 요소:
  ① 각 파이프라인별 Peak Sales 추정
  ② 매출 성장/감소 커브 (Ramp-up → Peak → Decline)
  ③ 영업이익률 추정
  ④ 개발 단계별 성공확률 (PoS: Probability of Success)
  ⑤ 할인율 (WACC + 바이오 리스크 프리미엄)
```

### 4.2 바이오시밀러 개발 성공확률

바이오시밀러는 오리지널과 달리 **표적 분자가 이미 검증**되어 있어 성공확률이 높습니다.

| 개발 단계 | 일반 바이오신약 PoS | 바이오시밀러 PoS | 셀트리온 적용 |
|-----------|-------------------|-----------------|-------------|
| Pre-clinical | 10~15% | 60~70% | 65% |
| Phase 1 | 15~20% | 75~85% | 80% |
| Phase 3 | 50~60% | 85~95% | 90% |
| 허가 신청 | 80~90% | 90~98% | 95% |
| 승인 완료 | 100% | 100% | 100% |

### 4.3 파이프라인별 rNPV 산출

#### 짐펜트라 (CT-P43, Stelara 바이오시밀러)

```
짐펜트라 rNPV 산출:

기본 가정:
  Peak Sales: $3.5B (글로벌)
  셀트리온 귀속 매출: $3.5B × 100% = $3.5B
  영업이익률: 35% (바이오시밀러 평균)
  성공확률: 100% (이미 FDA 승인)
  할인율: 10%

  2025: 0.6   2026: 1.8   2027: 3.0   2028: 3.5 (Peak)
  2029: 3.3   2030: 2.8   2031: 2.2   2032: 1.5

  2025: 0.21  2026: 0.63  2027: 1.05  2028: 1.23
  2029: 1.16  2030: 0.98  2031: 0.77  2032: 0.53

  2025: 0.16  2026: 0.49  2027: 0.82  2028: 0.96

  PV of FCF: $3.6B

## 5. 기존 제품 DCF + 파이프라인 rNPV = SOTP

### 5.1 기존 제품 DCF

```
기존 판매 제품 (램시마, 트룩시마, 유플라이마, 베그젤마):

2023 매출: 약 $2.0B (2.6조 원)
성장률: 연 5~10% (유플라이마 확대 효과)
정상화 영업이익률: 25~30%
Terminal Growth Rate: 0% (특허 만료 약품 특성)

DCF 가정:

## 6. 할인율 결정: 바이오 리스크 프리미엄

### 6.1 WACC 산출

```
셀트리온 WACC 산출:

무위험수익률 (Rf): 3.5% (한국 국고채 10년)
시장 프리미엄 (Rm-Rf): 6.0%
베타 (β): 1.2 (바이오 업종 평균)
자기자본비용 (Ke): 3.5% + 1.2 × 6.0% = 10.7%
타인자본비용 (Kd): 4.0% × (1-0.22) = 3.1%
자본구조: 자기자본 80%, 타인자본 20%

WACC = 80% × 10.7% + 20% × 3.1% = 9.2%
```

### 6.2 바이오 리스크 프리미엄

```
파이프라인 단계별 적용 할인율:

기존 판매 제품:     WACC (9~10%)     → 확정 매출, 리스크 낮음
승인 완료 (짐펜트라): WACC (9~10%)     → 시장 침투만 불확실
Phase 3:            WACC + 1~2%p     → 승인 리스크
Phase 1:            WACC + 2~4%p     → 개발 리스크
Pre-clinical:       WACC + 4~6%p     → 높은 불확실성

혹은: 모든 파이프라인에 동일 할인율(10%) 적용하되,
      성공확률(PoS)로 리스크 조정 → rNPV 방식 (위에서 사용)
```

## 8. 자주 하는 실수

:::tip 초보 투자자가 빠지기 쉬운 함정

### 실수 1: Peak Sales 과대추정

```
흔한 오류:
  "Stelara 매출이 $21B이니까,
   짐펜트라 Peak Sales도 $10B 가능!"

현실 체크:
  - 바이오시밀러 전체 시장 ≠ 오리지널 매출
  - 가격 할인 50% → 시장 규모 50% 축소
  - 경쟁사 5개 → 셀트리온 점유율 25~35%
  - 실제 Peak Sales: $2.5~4.0B (오리지널의 12~19%)

Peak Sales 추정 공식:
  오리지널 매출 × 바이오시밀러 침투율 × (1-가격할인율) × 점유율
  = $21B × 70% × 50% × 35% = $2.6B
```

### 실수 2: 경쟁 바이오시밀러 무시

```
파이프라인 밸류에이션 시 흔한 오류:

"셀트리온이 최초 승인이니까 시장을 독점할 것"

현실:
  - Stelara 바이오시밀러: 5개+ 경쟁
  - Humira 바이오시밀러: 10개+ 경쟁
  - "First-to-market"의 이점은 1~2년 정도
  - 이후 가격 경쟁으로 수렴

분석 시 반드시 확인:
  ① 오리지널 특허 만료 일정 (미국/EU)
  ② 경쟁 바이오시밀러 개발 현황
  ③ 각 경쟁사의 예상 출시 시기
  ④ 시장별 규제 환경 (미국 vs EU vs 신흥국)
```

### 실수 3: rNPV 합산의 과신

```
rNPV 합산 시 주의점:

"전체 파이프라인 rNPV = 21.4조원!"

이 숫자의 한계:
  ① 각 파이프라인의 성공확률은 "독립적"이라 가정
     → 실제로는 규제 환경, 기술력 등 공통 요인 존재
  ② Peak Sales는 "점 추정치"
     → 실제로는 넓은 범위의 확률분포
  ③ 할인율 가정에 매우 민감
     → 10% vs 12%에 따라 수조 원 차이
  ④ 실행 리스크 미반영
     → 생산, 마케팅, 유통 등 실행 과정의 리스크

적정 활용법:
  rNPV는 "대략적 방향성" 확인 도구
  주당가치의 정확한 숫자가 아닌 "범위"를 제시
```

### 실수 4: 합병 효과 과대평가

```
"셀트리온+헬스케어 합병으로 영업이익률 10%p 개선!"

현실:
  합병 전 셀트리온 OPM: ~35% (제조)
  합병 전 헬스케어 OPM: ~10% (유통)
  합병 후 연결 OPM: ~20~25% (가중평균)

  중간 마진 제거 효과: ~3~5%p (실제)
  추가 비용 절감: ~1~2%p

  → 합병 전 연결 OPM ~19%
  → 합병 후 연결 OPM ~23~25%
  → 개선폭: ~4~6%p (10%p가 아님)
```

### 실수 5: 시장 규모만 보고 투자

```
"글로벌 바이오시밀러 시장이 $100B가 된다!"

이 숫자의 의미:
  - 전체 시장이 커져도 개별 기업 매출은 별개
  - 경쟁자 수 증가 → 개별 점유율 하락
  - 가격 하락 → 매출 증가 < 볼륨 증가

투자 판단 시 중요한 것:
  전체 시장 크기 < 셀트리온의 경쟁 포지션
  매출 성장률 < 이익률 + 이익 성장률
```
