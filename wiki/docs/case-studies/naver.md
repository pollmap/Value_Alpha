:::

### 2.3 매출 성장률 분해

```
NAVER 전체 매출 성장률 분해 (2023, YoY +18%):

서치플랫폼:  +7%  ×  매출비중 48%  =  기여도 +3.4%p
커머스:      +30% ×  매출비중 27%  =  기여도 +8.1%p
핀테크:      +30% ×  매출비중 13%  =  기여도 +3.9%p
콘텐츠:      +11% ×  매출비중  8%  =  기여도 +0.9%p
클라우드:    +40% ×  매출비중  4%  =  기여도 +1.6%p
                                      ─────────────
                         합계 기여도:   +17.9%p ≒ +18%

→ 커머스가 성장의 약 45%를 견인
→ 서치플랫폼은 성숙기 진입 (한자릿수 성장)
```

## 4. SOTP 밸류에이션

### 4.1 밸류에이션 프레임워크

```
NAVER SOTP =
  ① 서치플랫폼 (캐시카우)
  + ② 커머스 (성장 엔진)
  + ③ 핀테크 (잠재 가치)
  + ④ 콘텐츠/웹툰 (글로벌 확장)
  + ⑤ 클라우드/AI (미래 성장)
  + ⑥ 라인야후 지분 (해외 자산)
  + ⑦ 순현금
  - ⑧ 지주회사 할인
```

### 4.2 ① 서치플랫폼

```
밸류에이션 방법: PER or EV/EBITDA

영업이익: 약 1.65조 (OPM ~35%)
EBITDA: 약 1.8조

방법 1 - PER:
  순이익 환산: 1.65조 × 0.78 = 1.29조
  적용 PER: 15~18x (성숙 플랫폼)
  가치: 1.29 × 16.5x = 21.3조

방법 2 - EV/EBITDA:
  EBITDA: 1.8조
  적용 배수: 12~15x
  EV: 1.8 × 13.5x = 24.3조

채택 가치: 22~24조
```

**Peer 비교:**

| 회사 | EV/EBITDA | PER | 성장률 |
|------|----------|-----|--------|
| Google (검색) | 14~16x | 22~25x | +10% |
| Baidu | 6~8x | 10~12x | +5% |
| Yahoo Japan (과거) | 10~12x | 15~18x | +5~8% |
| NAVER 서치 적용 | 12~15x | 15~18x | +7% |

### 4.3 ② 커머스 (스마트스토어)

```
밸류에이션 방법: EV/GMV 또는 EV/Sales

GMV: 50조 원 (2023)
커머스 매출: 2.6조 (Take Rate ~5.2%)

방법 1 - EV/GMV:
  적용 배수: 0.3~0.5x (쿠팡 IPO 시 0.8x)
  가치: 50조 × 0.4x = 20조

방법 2 - EV/Sales:
  적용 배수: 5~8x (고성장 커머스)
  가치: 2.6조 × 6.5x = 16.9조

채택 가치: 17~20조
```

## 7. 자주 하는 실수

:::tip 초보 투자자가 빠지기 쉬운 함정

### 실수 1: SOTP 합산의 과대평가 함정

```
SOTP 과대평가의 전형적 패턴:

"각 사업부를 분리 상장하면 72.5조!"
"현재 시총 40조 → 81% 업사이드!"

왜 실현되지 않는가:
1. 지주회사 할인은 구조적 (분리상장 안 하면 영구적)
2. 각 사업부 시너지를 개별 밸류에이션 시 이중 계산
3. 적자 사업부의 비용은 합산에서 과소평가
4. 라인야후 등 지분 가치는 현금화 가능성 불투명
5. 시장은 "합산 가치"가 아닌 "연결 이익"에 가격을 매김

교훈:
  SOTP는 "이론적 가치"이며,
  시장에서 실현되려면 "카탈리스트"가 필요
  (사업 분할, 자회사 상장, 자사주 소각 등)
```

### 실수 2: 성장률만 보고 밸류에이션 무시

```
"커머스 성장률 +30%! 클라우드 +40%! → 매수!"

문제점:
  커머스 OPM: 7% (이익 기여 미미)
  클라우드 OPM: -20% (적자)
  → 고성장이지만 이익에 기여하지 않는 사업부

  연결 기준으로 보면:
  매출 성장 +18% but 영업이익 성장 +10%
  → 성장의 "질"이 낮음 (매출은 늘지만 이익은 덜 늘어남)
```

### 실수 3: 라인야후 가치를 액면 그대로 반영

```
흔한 오류:
  라인야후 시총 2.5조 엔
  NAVER 지분 65% = 1.6조 엔 = 약 15조 원
  → "NAVER 시총 40조 중 37.5%가 라인야후!"

현실:
  - 경영권 불확실 (일본 정부 압박)
  - 지분 매각 시 할인 (블록딜 할인 20~30%)
  - 라인야후 자체 실적 부진 (OPM 5~8%)
  - 환율 리스크 (엔/원 변동)
  → 실현 가능 가치는 8~10조 수준
```

### 실수 4: AI 프리미엄의 과도한 부여

```
"HyperCLOVA X 출시! → AI 기업으로 재평가! → PER 40x!"

현실 체크:
  Q: HyperCLOVA X가 얼마나 매출을 만드는가?
  A: 2023~2024년 기준 연간 수천억 원 미만 (추정)

  Q: 글로벌 AI 기업과 경쟁할 수 있는가?
  A: 한국어 특화에서는 우위, 글로벌은 제한적

  Q: AI가 기존 검색 매출을 잠식하지 않는가?
  A: 검색 → AI 전환 과정에서 오히려 광고 매출 감소 가능

  결론: AI 프리미엄은 0.5~1조 수준이 적절
       (전체 SOTP의 1~2% 수준)
```

:::
