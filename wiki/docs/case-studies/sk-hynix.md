## 3. 메모리 사이클 Deep Dive

### 3.1 DRAM 수급 분석 프레임워크

DRAM 시장의 가격은 **수급 밸런스**에 의해 결정됩니다.

```
DRAM 수급 모델:

공급 측:                          수요 측:
┌─────────────────────┐          ┌─────────────────────┐
│ Wafer Input         │          │ Server (클라우드)     │
│ × 수율(Yield)       │          │ + Mobile             │
│ × Die 크기 축소     │          │ + PC                 │
│ × 가동률(Utilization)│          │ + Consumer           │
│ = Bit Supply        │          │ + Automotive          │
│                     │          │ + AI (HBM)           │
│ YoY Bit Growth:     │          │ = Bit Demand          │
│ 보통 15~20%         │          │                      │
│ 감산 시 0~5%        │          │ YoY Bit Demand Growth:│
│                     │          │ 보통 15~20%          │
└─────────────────────┘          └─────────────────────┘
         │                                │
         └──────── Gap 분석 ──────────────┘
                     │
          공급 > 수요: 가격 하락
          수요 > 공급: 가격 상승
```

### 3.2 가격 결정 메커니즘

DRAM 가격은 크게 **3가지 채널**에서 형성됩니다.

| 채널 | 설명 | 비중 |
|------|------|------|
| **Fixed Deal** | 분기 계약, 대형 고객과 협상 | ~70% |
| **Spot 시장** | DRAMeXchange 등에서 실시간 거래 | ~5% |
| **OEM 직거래** | 모바일/PC OEM과의 직접 계약 | ~25% |

### 3.3 사이클 국면별 투자 전략

| 국면 | 신호 | DRAM ASP 방향 | 적절한 행동 |
|------|------|--------------|-------------|
| **저점** | 재고일수 8주+, 감산 발표, 적자 전환 | 바닥 횡보 | 점진적 매수 개시 |
| **회복** | 가격 전월비 상승 전환, 재고 감소 | 상승 초기 | 적극 매수 |
| **확장** | 가동률 95%+, 공급 부족, 납기 지연 | 가파른 상승 | 보유 유지 |
| **정점** | CAPEX 대규모 증가 발표, 신규 Fab | 상승 둔화 | 점진적 매도 |
| **수축** | 수요 둔화, 재고 증가, 가격 하락 전환 | 하락 초기 | 매도 완료 |
| **침체** | 감산, 적자 전환, 인력 구조조정 | 가파른 하락 | 저점 탐색 (매수 준비) |

### 3.4 사이클 타이밍 지표

**선행 지표 (Leading Indicators):**
1. DRAM Spot 가격 전월비 변화율
2. 주요 고객사 재고일수 (서버 OEM, 모바일 OEM)
3. NVIDIA/AMD GPU 출하량 전망
4. 클라우드 3사 (AWS, Azure, GCP) CAPEX 가이던스

**동행 지표 (Coincident Indicators):**
1. DRAM 계약가격 QoQ 변화
2. SK하이닉스 가동률 수준
3. 분기 매출/영업이익 추이

**후행 지표 (Lagging Indicators):**
1. 영업이익률 (실적 발표 시 이미 반영)
2. PBR/PER (주가에 이미 반영된 경우 많음)

### 4.4 TSV 기술 상세

TSV(Through Silicon Via)는 DRAM 칩을 수직으로 쌓아 연결하는 핵심 기술입니다.

| 기술 요소 | SK하이닉스 (TC-NCF) | 삼성전자 (MR-MUF) |
|-----------|--------------------|--------------------|
| **본딩 방식** | Thermo-Compression, Non-Conductive Film | Mass Reflow, Molded Underfill |
| **수율** | 높음 (검증된 공법) | 상대적으로 낮음 |
| **생산 속도** | 상대적으로 느림 | 빠름 (대량생산 유리) |
| **열 관리** | 우수 | 개선 중 |
| **12Hi 적용** | 양산 확인 | 양산 진행 중 |

## 6. Forward PER vs Normalized PER 밸류에이션

### 6.1 두 가지 밸류에이션 접근법

사이클 기업에는 **두 가지 PER**을 함께 사용해야 합니다.

#### Forward PER (12개월 선행 이익 기준)

```
Forward PER = 현재 시가총액 / 향후 12개월 예상 순이익

장점: 가까운 미래 실적 반영
단점: 사이클 정점에서 과소평가, 저점에서 과대평가
적용: 단기 (6~12개월) 투자 판단
```

#### Normalized PER (정상화 이익 기준)

```
Normalized PER = 현재 시가총액 / 정상화(Mid-cycle) 순이익

장점: 사이클 중립적 평가
단점: "정상화 이익"의 주관성
적용: 중장기 (2~3년) 투자 판단
```

### 6.2 SK하이닉스 적용 예시

| 항목 | Forward PER | Normalized PER |
|------|------------|----------------|
| 시가총액 | 150조 원 | 150조 원 |
| 적용 이익 | 2024E 순이익 18조 | Mid-cycle 순이익 10조 |
| 산출 PER | 8.3x | 15.0x |
| 해석 | "저평가" | "적정~고평가" |

### 6.3 정상화 이익 산출 방법

```
SK하이닉스 정상화 이익 산출:

방법 1: 과거 평균법
  최근 10년 평균 영업이익 = (3.3+13.7+20.8+2.7+5.0+12.4+7.0-7.7) / 8 = 7.2조
  최근 10년 평균 OPM = 약 20%
  정상화 매출 35조 × OPM 20% = 7.0조

방법 2: Mid-cycle ASP 기준
  DRAM Mid-cycle ASP: $3.5 (과거 10년 중앙값)
  DRAM Bit 출하량: 2024년 기준
  정상화 DRAM 매출: 약 28조
  정상화 DRAM OPM: 30%
  정상화 DRAM 영업이익: 8.4조
  NAND 정상화 이익: 1~2조
  합계: 9.4~10.4조

방법 3: ROE 기준
  정상화 ROE: 15% (과거 10년 평균)
  자기자본: 55조
  정상화 순이익: 55 × 0.15 = 8.3조
```

## 7. 실전에서의 한계

:::danger 반드시 인식해야 할 리스크

### 7.1 HBM 프리미엄 지속성 불확실

HBM은 현재 **과점 시장**이지만, 영원히 지속되지 않을 수 있습니다.

| 리스크 요인 | 설명 |
|------------|------|
| 삼성/Micron 추격 | 2025~2026년 HBM4에서 격차 축소 가능 |
| NVIDIA 공급선 다변화 | 특정 업체 의존도 축소 전략 |
| HBM 대체 기술 | PIM, CXL Memory 등 대안 기술 부상 |
| 가격 경쟁 | 공급 확대 시 ASP 하락 압력 |
| AI 투자 사이클 | AI CAPEX 사이클 둔화 가능성 |

### 7.2 NAND 적자 지속

SK하이닉스의 NAND 사업은 구조적 열위에 있습니다.

```
NAND 사업 현황:
매출 비중: ~25% (8~9조)
시장 점유율: ~18% (5위)
Solidigm 인수: Intel NAND 사업 인수 (2021, $9B)
  → 통합 비용 지속, 적자 기여
```

## 8. 자주 하는 실수

:::tip 초보 투자자가 빠지기 쉬운 함정

### 실수 1: PBR로 저점 매수 vs PER로 저점 매수 혼동

사이클 기업에서 **PBR과 PER은 정반대로 움직입니다**.

```
사이클 저점 (예: 2023):
  PBR: 1.0~1.2x → "저평가처럼 보임"
  PER: 적자 또는 50x+ → "고평가처럼 보임"

  정답: PBR 기준 매수가 적절 (PER은 무의미)

사이클 정점 (예: 2018):
  PBR: 2.0~2.5x → "고평가처럼 보임"
  PER: 3~4x → "저평가처럼 보임"

  정답: PER 저점은 매도 신호 (이익이 정점이므로)
```

**핵심 원칙:**
- **사이클 저점**: PBR로 판단 (이익이 0이거나 적자이므로 PER 무의미)
- **사이클 정점**: PER로 판단 (PER이 너무 낮으면 오히려 위험 신호)
- **Mid-cycle**: Normalized PER + PBR 함께 사용

### 실수 2: HBM 매출만 보고 투자 결정

"HBM 매출이 급증하니까 무조건 매수" 라는 단순 논리의 함정:

| 빠지기 쉬운 오류 | 현실 |
|------------------|------|
| HBM 매출 3배 증가 | 이미 주가에 반영 (Forward PER에 포함) |
| HBM 영업이익률 50% | 경쟁 심화 시 하락 가능 |
| NVIDIA 독점 공급 | 공급선 다변화는 시간 문제 |
| HBM4가 게임체인저 | 삼성도 HBM4에서 경쟁 가능 |

### 실수 3: 사이클 저점에서 "이번엔 다르다"

2023년 DRAM Down Cycle에서 자주 나온 주장들:

```
"이번 Down Cycle은 이전과 다르다" (2023년 초)
→ "AI 수요 때문에 빠르게 회복할 것이다"
→ 결과: 회복은 2024년 하반기에야 시작 (예상보다 늦음)

"이번 Up Cycle은 이전과 다르다" (2024년 초)
→ "HBM 때문에 Super Cycle이 올 것이다"
→ 결과: 범용 DRAM은 여전히 사이클 영향 받음
```

### 실수 4: Solidigm 인수 효과 과대평가

```
Intel NAND 인수 ($9B, 약 12조 원) 평가:
  기대: NAND 점유율 확대, 기술 시너지, Enterprise SSD 강화
  현실: 통합 지연, 적자 지속, 기술 격차 해소 미흡
  교훈: M&A 시너지는 항상 과대평가되는 경향
```
