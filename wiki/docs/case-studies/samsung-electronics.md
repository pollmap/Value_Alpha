# 삼성전자: DuPont 분해와 반도체 사이클

## 학습 목표

이 케이스 스터디를 통해 다음을 학습합니다:

- 대형 복합기업(Conglomerate)에 대한 DuPont 분해 적용법
- 반도체 사이클이 재무제표에 미치는 영향 분석
- SOTP 밸류에이션 실전 적용
- HBM 시대 메모리 반도체 밸류에이션의 변화

## 1. 기업 개요

### 1.1 사업부 구조

삼성전자는 크게 **4개 사업부**로 구성된 글로벌 복합기업입니다.

| 사업부 | 주요 제품 | 글로벌 포지션 |
|--------|-----------|---------------|
| **DS (Device Solutions)** | DRAM, NAND, 파운드리, 시스템LSI | DRAM 1위, NAND 1위, 파운드리 2위 |
| **MX (Mobile eXperience)** | 스마트폰, 태블릿, 웨어러블 | 글로벌 스마트폰 1위 |
| **DX (Device eXperience)** | TV, 가전, 모니터 | 글로벌 TV 1위 (18년 연속) |
| **SDC (Samsung Display)** | OLED, LCD 패널 | 소형 OLED 1위 |

### 1.2 매출 및 이익 구성 (2023년 기준)

```
매출 구성:                          영업이익 구성:
┌──────────────────────────┐       ┌──────────────────────────┐
│ DS       약 32% (66.6조) │       │ DS      적자 (-14.9조)   │
│ MX/DX    약 49% (101조)  │       │ MX/DX   약 147% (21.7조) │
│ SDC      약 11% (22.8조) │       │ SDC     흑자  (2.1조)    │
│ Harman   약  5% (10.8조) │       │ Harman  흑자  (0.8조)    │
└──────────────────────────┘       └──────────────────────────┘
```

### 1.3 글로벌 포지션

- **시가총액**: 약 350~450조 원 (코스피 시가총액의 약 20%)
- **종업원 수**: 약 27만 명 (국내 12만, 해외 15만)
- **R&D 투자**: 연간 약 28조 원 (매출 대비 약 10~12%)
- **설비투자(CAPEX)**: 연간 약 40~55조 원

:::info 핵심 투자 포인트
삼성전자의 주가는 **DS 사업부의 실적에 의해 80% 이상 결정**됩니다. MX 사업부는 안정적이지만 성장성이 제한적이며, 실질적으로 반도체 사이클 기업으로 분석해야 합니다.
:::

## 3. DuPont 5요소 분해

### 3.1 DuPont 분해 공식

$$ROE = \frac{순이익}{매출} \times \frac{매출}{자산} \times \frac{자산}{자본}$$

5요소 분해 시:

$$ROE = 세후영업이익률 \times 자산회전율 \times 재무레버리지 \times 이자부담률 \times 세금부담률$$

### 3.2 호황 (2018) vs 불황 (2023) 비교

| DuPont 요소 | 2018 (호황) | 2023 (불황) | 변동폭 | 해석 |
|-------------|------------|------------|--------|------|
| **세전영업이익률** | 24.2% | 2.5% | -21.7%p | DS 사업부 실적 급변 |
| **자산회전율** | 0.64x | 0.52x | -0.12x | 매출 감소, 재고 증가 |
| **재무레버리지** | 1.37x | 1.41x | +0.04x | 거의 변동 없음 (무차입) |
| **이자부담률** | 1.02x | 0.98x | -0.04x | 이자 수익 > 이자 비용 |
| **세금부담률** | 0.76x | 0.58x | -0.18x | 적자로 실효세율 하락 |
| **산출 ROE** | **20.1%** | **4.7%** | **-15.4%p** | |

### 3.3 각 요소 상세 분석

#### (1) 세전영업이익률: 사이클의 핵심 드라이버

```
영업이익률 분해:
┌─────────────────────────────────────────────┐
│ 2018년 호황                                  │
│ DS 영업이익률: ~50% (DRAM 이익률 60%+)       │
│ MX 영업이익률: ~10%                          │
│ 연결 영업이익률: 24.2%                       │
├─────────────────────────────────────────────┤
│ 2023년 불황                                  │
│ DS 영업이익률: -22% (DRAM/NAND 모두 적자)    │
│ MX 영업이익률: ~13% (갤럭시 프리미엄화 성공) │
│ 연결 영업이익률: 2.5%                        │
└─────────────────────────────────────────────┘
```

**핵심**: DS 사업부의 영업이익률이 +50%에서 -22%까지 72%p 스윙합니다. 이것이 삼성전자 ROE 변동의 **약 90%**를 설명합니다.

#### (2) 자산회전율: 재고와 설비의 영향

| 항목 | 2018 | 2023 | 변화 |
|------|------|------|------|
| 매출 | 243.8조 | 258.9조 | +6.2% |
| 총자산 | 382.4조 | 497.7조 | +30.2% |
| 재고자산 | 28.9조 | 41.7조 | +44.3% |
| 유형자산 | 115.4조 | 162.8조 | +41.1% |
| 자산회전율 | 0.64x | 0.52x | -18.7% |

자산회전율 하락의 **주요 원인**:
- DRAM/NAND 감산에도 재고 급증 (비자발적 재고 축적)
- 파운드리 공장 신규 투자로 유형자산 증가
- 평택 P3/P4 라인 가동 전 자산 증가

#### (3) 재무레버리지: 순현금 기업의 특징

```
재무 구조 (2023):
총자산: 497.7조
총부채: 144.9조 (부채비율 41%)
순차입금: -약 65조 (순현금 상태)
```

삼성전자는 **무차입 순현금 기업**으로, 재무레버리지는 ROE 분석에서 거의 중립적입니다.

#### (4) 이자부담률 및 세금부담률

- **이자부담률**: 순현금 기업이므로 이자수익이 이자비용을 초과 → ROE에 양(+)의 기여
- **세금부담률**: 2023년 DS 적자로 실효세율 급락 (법인세 이연)

### 3.4 DuPont 분석의 투자 함의

| 질문 | 답변 |
|------|------|
| ROE 변동의 주된 요인은? | 세전영업이익률 (DS 사업부 사이클) |
| 구조적 변화는 있는가? | 자산회전율 하락 추세 (CAPEX 증가) |
| 재무 리스크는? | 매우 낮음 (순현금 65조) |
| 정상화 ROE는 얼마인가? | 12~15% (Mid-cycle 기준) |

## 5. HBM과 AI 시대 밸류에이션

### 5.1 HBM 시장 개요

HBM(High Bandwidth Memory)은 AI 가속기(GPU/NPU)에 필수적인 고대역폭 메모리입니다.

| 세대 | 대역폭 | 용량 | 주요 적용 |
|------|--------|------|-----------|
| HBM2E | 460GB/s | 16GB | A100 GPU |
| HBM3 | 819GB/s | 24GB | H100 GPU |
| HBM3E | 1,180GB/s | 36GB | H200/B100 GPU |
| HBM4 | 2,000GB/s+ | 48GB+ | 차세대 GPU (2025~) |

### 5.2 HBM 시장 규모 전망

| 연도 | HBM 시장 규모 | 성장률 | 삼성전자 점유율 |
|------|--------------|--------|----------------|
| 2022 | $2.3B | - | ~40% |
| 2023 | $4.0B | +74% | ~35% |
| 2024E | $16B | +300% | ~25% (하락) |
| 2025E | $30B+ | +88% | ~30% (회복 목표) |

### 5.3 삼성전자 vs SK하이닉스 HBM 경쟁

| 항목 | 삼성전자 | SK하이닉스 |
|------|---------|-----------|
| HBM3E 양산 시점 | 2024 H2 | 2024 H1 (선행) |
| NVIDIA 공급 | 인증 지연 | 독점 공급 중 |
| HBM3E 수율 | 50~60% (추정) | 70%+ (추정) |
| TSV 기술 | MR-MUF (후발) | TC-NCF (선발) |
| HBM4 로드맵 | 2025 하반기 | 2025 상반기 |

:::warning HBM 경쟁 현황
2024년 기준, SK하이닉스가 HBM3E에서 **약 6개월 이상 선행**하고 있으며, NVIDIA의 핵심 공급사로 자리잡았습니다. 삼성전자는 HBM3E 인증 지연으로 시장 점유율이 하락하는 상황입니다.
:::

### 5.4 HBM 포함 밸류에이션 프레임워크

기존 메모리 밸류에이션과 HBM 프리미엄을 분리해서 평가합니다:

```
삼성전자 DS 사업부 가치 =
  [Commodity DRAM 가치] + [NAND 가치] + [파운드리 가치] + [HBM 프리미엄]

HBM 프리미엄 산출:
  HBM 매출 × HBM 영업이익률 × 적정 PER
  = $8B × 40% × 15x = $48B (~65조원)
```

### 6.1 사업부별 가치 산출

#### (1) DS 사업부 (반도체)

```
정상화 영업이익 산출:
  DRAM 정상화 이익: 20~25조 (Mid-cycle ASP 기준)
  NAND 정상화 이익: 3~5조
  파운드리: -1~0조 (적자 지속 가정)

  정상화 영업이익 × (1-법인세율) × PER
```

```
밸류에이션:
  16.5조 × 0.78 × 8x = 103조 원
```

**적용 PER 근거**: Apple 25~30x 대비 할인 (성장성 차이, 브랜드 프리미엄 차이)

## 7. 실전에서의 한계

:::danger 반드시 인식해야 할 리스크

### 7.1 사이클 예측 불가능성

반도체 사이클의 정확한 타이밍을 예측하는 것은 **사실상 불가능**합니다.

- 2017년 Super Cycle 시작 시점을 사전에 예측한 애널리스트는 거의 없었음
- 2023년 Down Cycle의 깊이(DS 15조 적자)를 예측한 애널리스트 역시 극소수
- "이번 사이클은 다르다"는 말은 항상 나오지만, 사이클은 반복됨

**교훈**: 사이클 타이밍에 베팅하기보다 **구조적 경쟁력**에 집중해야 합니다.

### 7.2 정상화 이익의 함정

"정상화 이익"은 분석가의 주관이 크게 개입됩니다.

```
정상화 이익 산출의 불확실성:
┌──────────────────────────────────────────┐
│ "정상화 DRAM 이익은 20조? 25조? 30조?"   │
│                                          │
│ ASP 가정:  $3.0? $3.5? $4.0?            │
│ Bit Growth: 15%? 20%? 25%?              │
│ 원가 개선: 10%? 15%? 20%?               │
│                                          │
│ 작은 가정의 차이가 주당가치 2만원 차이    │
│ 를 만들어냅니다.                         │
└──────────────────────────────────────────┘
```

### 7.3 순현금 환상

순현금 65조가 있지만, 실제 주주에게 돌아오는 현금은 제한적입니다.

| 항목 | 금액 (연간) |
|------|-----------|
| 연간 배당금 | 약 9.8조 (2023) |
| 자사주 매입/소각 | 불규칙 |
| CAPEX 소요 | 40~55조 (매년 필수) |
| 실제 FCF | 사이클에 따라 -10조 ~ +30조 |

### 7.4 코리아 디스카운트

한국 시장 특유의 할인 요인:

- **지배구조**: 이재용 회장 일가의 경영권, 순환출자 구조
- **주주환원 부족**: 글로벌 Peer 대비 낮은 배당성향
- **지정학 리스크**: 남북 긴장, 미중 갈등에 따른 반도체 수출규제
- **대주주 할증**: 경영권 프리미엄에 따른 소수주주 할인

## 8. 자주 하는 실수

:::tip 초보 투자자가 빠지기 쉬운 함정

### 실수 1: 저점 PER 함정

사이클 저점에서 PER이 높게 나타납니다 (이익이 바닥이므로). 이를 "비싸다"고 판단하면 최적의 매수 시점을 놓칩니다.

```
사이클 기업의 PER 역설:
  사이클 정점 → 이익 최대 → PER 최저 (5~8x) → 매도 시점
  사이클 저점 → 이익 최소 → PER 최고 (30x+) → 매수 시점

  "PER이 낮을수록 좋다"는 공식은 사이클 기업에 적용 불가!
```

### 실수 2: 분기 실적에 대한 과잉 반응

분기 실적이 컨센서스를 소폭 하회/상회했다고 장기 투자 판단을 바꾸는 것은 위험합니다.

- DRAM 가격이 전분기 대비 5% 하락 → "Down Cycle 시작!"
- DRAM 가격이 전분기 대비 5% 상승 → "Super Cycle 시작!"
- 실제로는 **3~4분기 연속 추세**를 봐야 사이클 방향 판단 가능

### 실수 3: HBM 낙관론의 과잉

HBM이 구조적 성장동력임은 맞지만, 과도한 기대는 위험합니다.

- HBM은 전체 DRAM 매출의 약 20~30% (2024~2025 기준)
- 나머지 70~80%는 여전히 Commodity DRAM (서버/모바일/PC)
- HBM 가격도 경쟁 심화 시 하락 가능성 존재
- "삼성이 HBM4에서 역전한다"는 시나리오는 아직 **확인되지 않은 가정**

### 실수 4: 과거 PBR 밴드의 기계적 적용

```
"삼성전자 PBR 하단이 1.0x이니까 여기서 사면 된다"

문제점:
  - 과거 PBR 하단은 과거의 BPS 기준
  - DS 구조적 적자 시 BPS가 감소할 수 있음
  - 파운드리 대규모 투자로 자본 효율성 하락 중
  - PBR 하단이 0.9x, 0.8x로 내려갈 가능성도 존재
```

:::

- [ ] SOTP 상 각 사업부 가치 적정성
