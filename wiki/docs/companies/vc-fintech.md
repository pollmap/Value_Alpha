# VC/PEF & 핀테크 기업 총람

VC(벤처캐피탈)와 PEF(사모펀드)는 비상장 기업에 대한 투자·인수를 전문으로 하는 대체투자 영역의 핵심 플레이어입니다. 핀테크 기업은 기술을 기반으로 혁신적 금융 서비스를 제공하며, 전통 금융의 패러다임을 변화시키고 있습니다. 두 업종 모두 높은 성장성과 역동적인 업무 환경이 특징입니다.

---

### VC/PEF 업권 특성

- **PEF 대형화**: MBK·한앤컴퍼니 등 메가 PEF의 펀드 규모 확대. 10조 원 이상 펀드 등장
- **VC 다양화**: 초기(시드·프리시리즈A)부터 후기(프리IPO)까지 전 단계 투자 활성화
- **CVC 성장**: 카카오·네이버·삼성·LG 등 대기업의 CVC(기업벤처캐피탈) 활동 확대
- **세컨더리 시장**: PEF·VC 지분의 유통시장(세컨더리) 발전
- **ESG 투자**: 환경·사회·지배구조 관련 투자 기준 강화 추세

## 핀테크

핀테크(FinTech) 기업은 기술을 기반으로 결제, 송금, 투자, 대출, 보험 등 금융 서비스를 혁신하는 기업입니다. 한국 핀테크 시장은 토스·카카오페이 등 유니콘 기업을 중심으로 빠르게 성장하고 있습니다.

| 기업명 | 카테고리 | 주요사업 | 특징 |
|--------|---------|---------|------|

## 채용 특성

### VC/PEF

| 항목 | 내용 |
|------|------|
| **채용 방식** | 수시채용 중심. 경력직 선호. 소수 정예(연간 1~5명 채용) |
| **선호 경력** | IB(증권사 투자은행), 회계법인(Big4), 경영 컨설팅, 기업 전략 |
| **우대 자격** | CFA, CPA/AICPA, 변호사, MBA, 해외 대학원 |
| **처우 수준** | PEF 주니어 초봉 약 6,000~8,000만 원 + 성과급(carry). VC는 4,000~6,000만 원 |
| **경력 경로** | Analyst → Associate → VP → Principal → Partner(MD) |

### 핀테크

| 항목 | 내용 |
|------|------|
| **채용 방식** | 수시채용 중심. 개발 직군 상시 채용. 스타트업 문화 |
| **선호 경력** | IT 기업 개발 경력, 금융사 IT/기획, 데이터 분석 경력 |
| **우대 자격** | 개발: CS 학위, 오픈소스 기여 / 금융: CFA, AFPK / 데이터: 석사 이상 |
| **처우 수준** | 대형 핀테크(토스·카카오페이) 개발직 초봉 약 5,000~7,000만 원 + 스톡옵션 |
| **경력 경로** | 주니어 → 시니어 → 리드 → 헤드 / 또는 스타트업 창업·이직 |

- 바이아웃 대형화: 10조 원 이상 블라인드 펀드 등장

- 임베디드 금융: 비금융 플랫폼에 금융 기능 내장(BaaS)

:::
