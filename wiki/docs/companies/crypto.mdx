## 암호화폐 거래소

국내 암호화폐 거래소 중 원화마켓(실명계좌 연동)을 보유한 4대 거래소와 주요 거래소를 소개합니다.

<TabItem value="top4" label="원화마켓 4대 거래소" default>

### 두나무 (업비트)

| 항목 | 내용 |
|:-----|:-----|
| **거래소명** | 업비트 (Upbit) |
| **설립** | 2017년 |
| **시장점유율** | 약 80% 이상 (국내 1위) |
| **실명계좌** | 케이뱅크 |
| **특징** | 카카오 투자 유치, 압도적 시장 지배력, 글로벌 확장 (싱가포르, 인도네시아, 태국) |
| **주요 주주** | 송치형 대표, 카카오 |

**핵심 경쟁력**:
- 국내 최대 거래량과 유동성
- 카카오 생태계 연계 가능성
- 해외 거래소 네트워크 구축
- 높은 수익성 (2021년 영업이익 약 3조 원)

**리스크 요인**:
- 규제 변화에 따른 수익 변동
- 시장 사이클에 대한 높은 의존도
- 경쟁 심화 가능성

### 빗썸코리아 (빗썸)

| 항목 | 내용 |
|:-----|:-----|
| **거래소명** | 빗썸 (Bithumb) |
| **설립** | 2014년 |
| **시장점유율** | 약 10\~15% (국내 2위) |
| **실명계좌** | NH농협은행 |
| **특징** | 국내 최초 거래소, 다수 대주주 변경 이력, 해외 진출 시도 |
| **주요 주주** | 비덴트 (지분 분쟁 이력) |

**주요 이력**:
- 2014년: 국내 최초 암호화폐 거래소 설립
- 2017\~2018년: 거래량 세계 1위 기록
- 2019년: 해킹 사고 (약 350억 원 피해)
- 이후: 대주주 변경, 지분 분쟁 지속

## 저축은행과의 비교

암호화폐 거래소와 저축은행은 금융 인프라 관점에서 유사점이 있습니다.

| 구분 | 저축은행 | 암호화폐 거래소 |
|:-----|:--------|:--------------|
| **규제 기관** | 금융위/금감원 | 금융위/금정원 |
| **인허가** | 상호저축은행법 | 특금법 (VASP 등록) |
| **고객 보호** | 예금자보호 5,000만 원 | 보호 장치 없음 |
| **수익 모델** | 예대마진 | 거래 수수료 |
| **리스크** | 대출 부실 | 시장 변동성, 해킹 |
| **진입장벽** | BIS비율 | 실명계좌, ISMS |

## 직무 안내

### 암호화폐 거래소 주요 직무

| 직무 | 업무 | 필요 역량 |
|:-----|:-----|:---------|
| **마케팅** | 사용자 유치, 이벤트 기획 | 디지털 마케팅, 커뮤니티 관리 |
| **리스크관리** | AML/KYC, 이상거래 탐지 | 금융 규제, 데이터 분석 |
| **상장 심사** | 코인 상장 검토, 실사 | 토큰이코노믹스, 기술 이해 |
| **개발** | 거래 시스템, 보안 | 블록체인, 보안, 분산시스템 |
| **고객지원** | 사용자 문의 대응 | 암호화폐 지식, CS 역량 |
