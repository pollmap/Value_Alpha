# 저축은행 & 자산운용사 총람

저축은행과 자산운용사는 각각 서민·중소기업 금융과 집합투자 상품 운용이라는 고유한 역할을 수행하는 금융기관입니다.

---

## 저축은행 산업 개요

### 저축은행이란?

저축은행(Savings Bank)은 **상호저축은행법**에 따라 서민·중소기업을 대상으로 예금·대출 서비스를 제공하는 **제2금융권 금융기관**입니다. 은행 대출이 어려운 중·저신용 고객과 중소기업에 금융 서비스를 제공하는 '서민금융'의 핵심 역할을 수행합니다.

| 구분 | 시중은행 | 저축은행 |
|:-----|:--------|:--------|
| **근거법** | 은행법 | 상호저축은행법 |
| **감독기관** | 금융위원회·금융감독원 | 금융위원회·금융감독원 |
| **예금자보호** | 5,000만 원 한도 | 5,000만 원 한도 |
| **주요 고객** | 프라임 등급 | 중·저신용 등급 (서브프라임) |
| **대출 금리** | 낮음 (2\~7%) | 높음 (8\~20%) |
| **BIS비율 규제** | 8% 이상 | 7\~8% 이상 (자산 규모별 차등) |
| **영업 범위** | 전국 | 본점 소재지 중심 (제한적) |

### 산업 규모 및 현황 (2024년 기준)

| 지표 | 수치 | 비고 |
|:-----|:-----|:-----|
| **저축은행 수** | 79개 | 2011년 106개에서 구조조정 후 감소 |
| **총자산** | 약 130조 원 | 상위 10개사가 전체의 50% 이상 |
| **총 대출** | 약 110조 원 | 가계대출 60%, 기업대출 40% |
| **총 예금** | 약 95조 원 | 정기예금 비중 80% 이상 |
| **평균 NIM** | 약 4.5\~5.5% | 은행(1.5\~2%) 대비 2\~3배 높음 |
| **평균 연체율** | 약 5\~7% | 은행(0.3\~0.5%) 대비 10배 이상 |

### 저축은행 비즈니스 모델

```
┌─────────────────────────────────────────────────────────────┐
│                    저축은행 수익 구조                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  [수신]                    [여신]                  [수익]     │
│  예금 유치 ──────────────> 대출 실행 ──────────> 예대마진     │
│  (예금금리 3\~4%)           (대출금리 10\~20%)     (NIM 4\~5%)   │
│                                                              │
│  ┌──────────────┐         ┌──────────────┐                   │
│  │ 정기예금     │         │ 개인신용대출 │ ← 중금리 대출     │
│  │ 정기적금     │         │ 부동산담보   │ ← PF대출          │
│  │ 자유예금     │         │ 기업대출     │ ← 중소기업 대출   │
│  └──────────────┘         └──────────────┘                   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

<TabItem value="revenue" label="수익원천" default>

**1. 이자수익 (전체 수익의 85\~90%)**
- **예대마진(NIM)**: 대출금리 - 예금금리 = 순이자마진
- 은행 대비 높은 대출금리(10\~20%)로 높은 마진 확보
- 단, 연체·부실 리스크도 높아 충당금 부담 큼

**2. 비이자수익 (10\~15%)**
- 송금·환전 수수료
- 대출 취급 수수료
- 기타 금융 서비스 수수료

</TabItem>
<TabItem value="cost" label="비용구조">

**1. 이자비용 (40\~50%)**
- 예금 이자 지급
- 은행 대비 높은 예금금리 제시 (유동성 확보 목적)

**2. 대손충당금 (15\~25%)**
- 연체·부실 대출에 대한 충당금 적립
- 저축은행 수익성의 핵심 변수
- 건전성 악화 시 급격히 증가

**3. 판관비 (20\~30%)**
- 인건비, 임차료, IT비용 등
- 디지털 전환으로 비대면 채널 비용 증가

</TabItem>
<TabItem value="risk" label="핵심 리스크">

**1. 신용 리스크 (Credit Risk)**
- 중·저신용 고객 대상 → 연체율 높음
- 경기 침체 시 부실 급증 위험
- 부동산 PF 대출 부실 우려 (2023\~2024 이슈)

**2. 금리 리스크 (Interest Rate Risk)**
- 금리 상승기: 예금금리 상승 → 마진 축소
- 금리 하락기: 차주 상환능력 개선 but 대출수요 감소

**3. 유동성 리스크 (Liquidity Risk)**
- 예금 이탈 시 유동성 위기
- 대형 저축은행 부실 시 전이 효과

**4. 평판 리스크 (Reputation Risk)**
- 2011년 사태의 트라우마
- 고금리 대출에 대한 부정적 인식

</TabItem>
</Tabs>

### 저축은행 핵심 지표

저축은행 분석 시 반드시 확인해야 할 핵심 지표입니다.

| 지표 | 정의 | 적정 수준 | 해석 |
|:-----|:-----|:---------|:-----|
| **BIS자기자본비율** | 자기자본 ÷ 위험가중자산 | 7\~8% 이상 | 건전성의 핵심 지표. 규제 기준 미달 시 적기시정조치 |
| **고정이하여신비율** | 고정+회수의문+추정손실 ÷ 총여신 | 5% 이하 | 부실자산 비중. 높을수록 위험 |
| **연체율** | 연체대출 ÷ 총대출 | 3\~5% | 대출 건전성. 상승 시 충당금 증가 |
| **대손충당금적립률** | 대손충당금 ÷ 고정이하여신 | 100% 이상 | 부실대비 충분성. 높을수록 안정 |
| **NIM (순이자마진)** | (이자수익-이자비용) ÷ 이자수익자산 | 4\~6% | 핵심 수익성 지표. 은행의 2\~3배 |
| **ROA** | 당기순이익 ÷ 총자산 | 1% 이상 | 자산 대비 수익성 |
| **ROE** | 당기순이익 ÷ 자기자본 | 8\~12% | 자본 대비 수익성 |
| **예대율** | 총대출 ÷ 총예금 | 100% 이하 | 유동성 지표. 100% 초과 시 자금조달 부담 |

### 저축은행 역사: 2011년 저축은행 사태

#### 사태 개요

2011년 발생한 **저축은행 부실 사태**는 한국 금융사에서 중요한 교훈을 남긴 사건입니다.

| 구분 | 내용 |
|:-----|:-----|
| **사태 기간** | 2011년 1월 ~ 2012년 12월 |
| **부실 저축은행** | 31개 (전체 106개 중) |
| **영업정지** | 16개 저축은행 |
| **피해 규모** | 예금자 약 10만 명, 예금보험공사 약 27조 원 투입 |
| **핵심 원인** | 부동산 PF 대출 부실, 분식회계, 대주주 사금고화 |

#### 사태 이후 변화

| 영역 | 변화 내용 |
|:-----|:---------|
| **규제 강화** | BIS비율 규제 강화, PF 대출 한도 규제, 대주주 적격성 심사 강화 |
| **감독 강화** | 상시 감시체계 도입, 현장검사 강화, 적기시정조치 엄격 적용 |

| **경영 투명성** | 분식회계 처벌 강화, 외부감사 확대, 공시 의무 강화 |
| **예금자 보호** | 예금보험한도 유지 (5,000만 원), 예금자 보호 강화 |

:::tip 투자자·취업자 체크포인트
저축은행 분석 시 **부동산 PF 대출 비중**, **대주주 지분 구조**, **충당금 적립률**을 반드시 확인하세요. 2011년 사태의 핵심 원인이었던 요인들입니다.
:::

## 자산운용사

자산운용사는 투자자로부터 자금을 모아 펀드·ETF·연금 등 집합투자 상품으로 운용하는 전문 금융기관입니다. 국내 약 300개 이상의 자산운용사가 등록되어 있으며, 운용자산(AUM) 규모는 약 1,500조 원에 달합니다.

### 삼성자산운용
- **카테고리**: 자산운용
- **주요사업**: ETF·공모펀드·사모펀드·연금
- **특징**: 삼성그룹 금융계열. 국내 AUM 1위(약 300조 원). KODEX ETF 시장 점유율 1위. 주식·채권·대체투자 전 영역. 패시브·액티브 균형

### 미래에셋자산운용

- **주요사업**: 글로벌펀드·ETF·대체투자·연금
- **특징**: 미래에셋금융그룹 핵심. AUM 2위. TIGER ETF 시장 2위. 글로벌 투자 선도. 해외 거점 12개국. 글로벌 ETF·해외펀드 강점

## 채용 특성

### 저축은행

| 항목 | 내용 |
|:-----|:-----|
| **채용 방식** | 수시채용 중심. 대형사는 공채 시행 경우도 있음 |
| **우대 자격** | 신용분석사, AFPK, 여신심사역 |
| **처우 수준** | 대형사 초봉 약 3,500\~4,500만 원. 은행 대비 다소 낮으나 성장세 |
| **경력 경로** | 여신심사 → 여신관리 → 지점장 / 본부 전문부서 이동 |

### 자산운용사

| 항목 | 내용 |
|:-----|:-----|
| **채용 방식** | 수시채용 중심. 신입보다 경력직 선호. 인턴 → 전환 경로 중요 |
| **우대 자격** | CFA, 투자자산운용사, CAIA, FRM, CPA |
| **처우 수준** | 대형사 초봉 약 5,000\~6,000만 원. 운용역은 성과급 포함 높은 보상 가능 |
| **경력 경로** | 주니어 운용역/RA → 시니어 운용역 → CIO / 또는 셀사이드·바이사이드 간 이동 |
